# War3 RPG + 塔防玩法地图整理

整理时间：2026-04-01  
范围口径：本清单聚焦《Warcraft III》自定义地图中，偏 `ORPG / 地牢RPG / Hero Defense / Tower Defense / Tower Wars` 的代表作与常见流派。不是“全网绝对全集”，而是基于公开可检索来源整理出的高相关样本池。

## 摘要

如果你的目标是研究 `War3 里 RPG + 塔防` 这一大类玩法，可以先把地图分成 4 个大组：

1. `传统塔防`：强调塔组合、摆位、迷宫、元素/宝石合成。
2. `对抗式塔防`：玩家不仅守怪，还会“发兵”或和对手做经济博弈。
3. `英雄防守 / RPG化防守`：主控英雄成长、刷装、守家，RPG味更浓。
4. `ORPG / 地牢RPG`：更偏角色成长、存档、任务、副本探索，但对 War3 玩家来说常和“守图”用户群重叠。

如果只想先抓最有代表性的切片，优先看这 10 张：

- Element TD
- Gem TD
- Green Circle TD
- Wintermaul
- Legion TD Mega
- Wintermaul Wars
- Enfo's Team Survival
- X Hero Siege
- The Black Road
- Gaias Retaliation ORPG

- 以 `建塔决策` 为主，英雄操作权重低。
- 强调 `经济节奏 + 塔系搭配 + 怪物路径理解`。
- 非常适合拆解“局内循环”。

### 这组的共同特征

- 不是纯粹“守 AI 波次”，而是 `守线 + 压人 + 运营` 三件事同时发生。
- 相比传统 TD，更强调 `博弈性` 和 `回合外心理预判`。
- 若你在研究现在手游里的“自走防守 / 对推塔防 / PvP塔防”，这组最值得深挖。

- 如果你研究的是“War3 地图如何长线留存”，这组价值非常高。

## 五、按研究方向再做一次归类

### 1. 如果你想研究“塔防底层设计”

优先顺序：

- Element TD
- Gem TD
- Green Circle TD
- Wintermaul

原因：

- 覆盖了 `组合塔系`、`随机资源`、`迷宫封路`、`多人协作守线`、`PvP 发兵塔防` 这 5 个关键分支。

### 2. 如果你想研究“RPG味最浓的守图”

- Enfo's Team Survival
- X Hero Siege

原因：

- 这批地图不是纯粹摆塔，而是更强调 `英雄技能、刷装、守家、合成成长`。

- The Black Road

## 七、初步结论

- `War3 的“RPG+塔防”并不是一个单点品类，而是一整条连续谱。`
- 一端是 `Element TD / Gem TD` 这种纯建塔设计。
- 中间是 `Legion TD / Wintermaul Wars / Enfo / X Hero Siege` 这种带明显成长或对抗的守图。
- 另一端则是 `The Black Road / Gaias / TKoK` 这样的完整 ORPG。

对今天做产品研究最有价值的，不是把这些图都叫成同一类，而是看它们分别解决了哪一个核心问题：

- `纯策略构筑`：Element TD、Gem TD
- `高复玩多人协作`：Wintermaul、Green Circle TD
- `PvP化塔防博弈`：Legion TD Mega、Wintermaul Wars
- `英雄成长守家`：Enfo、X Hero Siege
- `长线角色留存`：The Black Road、Gaias、TKoK

## 八、说明与边界

- 本次整理优先使用公开可检索的地图库页面作为证据口径。
- 由于 War3 地图长期存在大量 `民间魔改版 / 重传版 / 年代不同的分支版本`，这里更关注“地图流派与代表作品”本身，而不是断言某个版本就是唯一正统版本。
- 像 `Castle Fight` 这类地图严格说更接近“自动出兵对抗”，不属于纯塔防，但因为和 War3 塔防/守图玩家群高度重叠，所以作为邻近样本保留。
