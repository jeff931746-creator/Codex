# 立项流程机制

这里收纳立项阶段的流程材料。

按主步骤看就两块：

1. [01-立项评审/README.md](/Users/mt/Documents/Codex/projects/00-共享能力/立项/立项流程机制/01-立项评审/README.md)
2. [02-纸面原型/README.md](/Users/mt/Documents/Codex/projects/00-共享能力/立项/立项流程机制/02-纸面原型/README.md)

结构说明：

- `01-立项评审/`：把 `方向卡 -> 母型卡 -> 参与方式卡 -> 核心机制卡 -> 玩法循环卡` 放在同一个步骤里
- `02-纸面原型/`：立项评审通过后的下一步，用于把玩法压成可推演、可迭代、可推翻的单局模型
- `03-题材选择/`：题材和风格相关的辅助 Prompt，不属于主流程步骤

执行原则：

- 先做 `01-立项评审`
- `玩法循环` 通过后，再进入 `02-纸面原型`
- `02-纸面原型` 通过后，才建议进入白模、灰盒或可玩原型
