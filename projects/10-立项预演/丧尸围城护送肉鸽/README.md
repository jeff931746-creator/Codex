# 丧尸围城护送肉鸽

这个项目当前处于立项预演阶段。

## 本地结构

1. `00-立项文档/`：放正式立项文档和玩法模型正文
2. `01-评审/`：放初审稿、评审规则和评审阶段记录
3. `02-纸面原型/`：固定拆成两个子目录
   `01-生成内容/`：放纸面原型、概念图 prompt、辅助脚本、规则参考、参考输出和探索性材料
   `02-评审结果/`：放纸面原型评审、打分结果、结论记录和回滚/退回判断

这条方向先按“围城压力 + 目标护送 + 局内肉鸽成长”收敛，不拆成多个并行主案。
