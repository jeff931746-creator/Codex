# 【桥头卡组守线RPG】纸面原型补证稿：递进部署链 v1

- 项目名称：桥头卡组守线RPG

- 当前版本：v1
- 补证目标：验证“桥头压力下的递进部署链”是否能成为核心玩法，而不是普通卡牌按钮。

---

## 一、项目基线

- 目标用户：喜欢短局守线、怪潮压境、自动战斗部署、局内构筑和轻RPG成长的中轻度策略用户。
- 核心买量承诺：`按顺序出牌，连成防线，守住最后一座桥。`
- 核心玩法一句话：玩家在桥头防线前按“阻挡 -> 输出 -> 控制 -> 终端”的层级打出卡牌，连成防线部署链，顶住怪潮并击败Boss。
- 核心爽点一句话：乱出卡只能勉强防守，接对链条会让整条防线连续启动，把压到门口的怪潮顶回去。
- 主要变现方式：内购为主，广告为辅。首付应放大“尝试新指挥官/新链条”的欲望，而不是单纯解除卡点。

## 二、本轮补证范围

### 1. 本轮要回答的闸门

- 闸门1：三秒买点。用户能否在静帧或15秒素材里看懂“桥头 + 怪潮 + 手卡 + 连锁防线”。
- 闸门2：一分钟承接。用户能否在首局前60秒学会两段链，并理解顺序影响结果。
- 闸门3：十分钟成型。用户能否在1-2局内形成至少一种明确打法，例如硬顶反击流、火炮终端流、冰控拖延流。
- 闸门4：二刷目标。用户首局结束后是否会想换指挥官、换终端卡、补断点或尝试另一条链。

### 2. 每个闸门的通过阈值

- 三秒买点：3秒内能复述“这是守桥游戏，底部卡牌按顺序连起来会爆发”。
- 一分钟承接：60秒内能完成一次“盾兵 -> 弓手”两段链，并看到比单出弓手更强的守线反馈。
- 十分钟成型：10分钟内能至少跑通一条三段以上链，并能说出“我这局靠什么守住”。
- 二刷目标：首局结束时能产生一个具体下一局目标，例如“我要试火炮终端”“我要多拿万能牌”“我要换冰控指挥官”。

### 3. 本轮不回答的问题

- 不验证完整商业化数值。
- 不验证长线内容供给节奏。
- 不验证全卡池平衡。
- 不验证美术风格和题材包装最终形态。

### 4. Kill 条件

- 玩家只觉得是在“按按钮放技能”，不能感知链条顺序带来的差异。
- 玩家能理解守桥，但理解不了为什么要按层级接牌。
- 万能牌和弃牌资源不能降低卡手挫败，反而让系统更复杂。
- 两名指挥官在同一关卡里的打法差异不明显。

## 三、纸面原型核心规则

### 1. 场地

纸面地图只保留一条桥面，分为5格：

```text
怪潮入口 -> A远端 -> B中桥 -> C桥头 -> D城门
```

- 怪物每回合向右推进1格。
- 怪物进入D城门时，对城门造成伤害。
- 城门初始生命：20。
- 每回合开始抽到4张手牌。
- 每回合可打出最多3张牌；若使用万能牌，本回合可额外打出1张。

### 2. 防线层级

| 层级 | 标签 | 职责 | 典型效果 |
| --- | --- | --- | --- |
| 1 | 阻挡 | 顶线、嘲讽、护甲 | 生成前排、拦截推进 |
| 2 | 输出 | 持续伤害 | 对被阻挡目标增伤 |
| 3 | 控制 | 减速、冻结、聚怪 | 推迟怪潮压线 |
| 4 | 终端 | 爆发、全局放大 | 清屏、炮击、全队强化 |

### 3. 递进部署链

- 同回合内，玩家按层级递增出牌时形成部署链。
- 合法链示例：`阻挡 -> 输出`、`阻挡 -> 输出 -> 控制`、`阻挡 -> 输出 -> 控制 -> 终端`。
- 跳层可以出牌，但不增加链条层数。
- 同层连续出牌只取第一张计入链条，后续按基础效果结算。
- 链条层数越高，获得越强的连锁奖励。

### 4. 连锁奖励

| 链条层数 | 奖励 |
| --- | --- |
| 2层链 | 本回合所有守军伤害+1 |
| 3层链 | 额外冻结最靠近城门的1个怪物 |
| 4层链 | 触发终端爆发，额外对全场怪物造成3点伤害 |

### 5. 万能牌

- 援军令：可视为任意层级接入链条，并召唤1个1攻1血民兵。
- 临时工事：可视为阻挡层，城门获得2点护甲。
- 战术转令：可视为任意层级，但只用于接到终端层；本回合终端效果-1伤害。

### 6. 弃牌换资源

每回合最多弃2张牌。

| 弃牌类型 | 获得资源 | 用途 |
| --- | --- | --- |
| 阻挡/输出 | 士气+1 | 消耗2士气，抽1张牌 |
| 控制 | 能量+1 | 消耗2能量，本回合下一张控制牌范围+1 |
| 终端 | 工事值+2 | 消耗3工事值，城门恢复2生命 |
| 万能 | 工事值+1 + 士气+1 | 作为保底资源，但损失补断点能力 |

弃牌设计目标：
- 让不合适的卡仍有价值。
- 让玩家在卡手时能主动修桥或找下一张牌。
- 让“不要强卡”也可能是正确选择。

## 五、指挥官

### 1. 铁壁队长

- 起始牌组：
  盾兵x2、路障x1、弓手x2、火枪手x1、冰霜术x1、桥头炮台x1、援军令x1、临时工事x1
- 被动：
  每次打出阻挡层后，城门获得1护甲。
- 推荐打法：
  以阻挡层起链，靠护甲和炮台撑过高压。
- 玩家应能说出：
  “我这局是先把桥头顶住，再等炮台清场。”

### 2. 火炮军师

- 起始牌组：
  盾兵x1、枪兵x1、弓手x1、弩车x2、火油x1、震荡陷阱x1、桥头炮台x2、援军令x1
- 被动：
  每完成一次3层链，下一张终端牌伤害+1。
- 推荐打法：
  尽量打出3层链，围绕终端爆发清场。
- 玩家应能说出：
  “我这局不是硬顶，是攒一轮炮台爆发。”

## 七、关卡脚本

### 1. 初始参数

- 城门生命：20
- 初始资源：士气0、能量0、工事值0
- 手牌：每回合抽4张
- 出牌限制：每回合最多3张，使用万能牌时最多4张
- 怪物行动：每回合结束时推进1格；若在C被阻挡则停下攻击阻挡物

### 2. 怪物单位

| 怪物 | 生命 | 攻击 | 特性 |
| --- | --- | --- | --- |
| 小怪 | 2 | 1 | 无 |
| 重甲怪 | 5 | 2 | 受到单次1点伤害时减免为0 |
| 跑者 | 1 | 1 | 每回合推进2格 |
| 投石怪 | 3 | 1 | 在B格可直接攻击城门1点 |
| Boss冲城巨怪 | 14 | 3 | 每2回合冲锋1次，向前多推进1格 |

### 3. 波次1：学会两段链

- 回合1：小怪x2，从入口进入。
- 回合2：小怪x2 + 跑者x1。
- 目标体验：
  玩家用“盾兵 -> 弓手”看到2层链伤害+1，理解顺序收益。
- 推荐教学牌序：
  回合1给盾兵、弓手、路障、临时工事。
- 通过标准：
  玩家至少完成1次2层链，城门剩余生命大于15。

### 4. 波次2：引入弃牌换资源

- 回合3：重甲怪x1 + 小怪x2。
- 回合4：跑者x2 + 投石怪x1。
- 目标体验：
  玩家抽到暂时打不出的终端牌时，可弃牌换工事值修桥，或者保留等待3层链后爆发。
- 推荐压力：
  回合3给一张桥头炮台，但不保证控制牌。
- 通过标准：
  玩家使用至少1次弃牌换资源，并能说出“弃牌不是失败，是换修桥/抽牌资源”。

### 5. 波次3：形成3层链

- 回合5：小怪x3 + 投石怪x1。
- 回合6：重甲怪x1 + 跑者x2。
- 目标体验：
  玩家完成“阻挡 -> 输出 -> 控制”，触发冻结最近怪物，感知3层链的控场价值。
- 推荐牌序：
  盾兵/枪兵、弓手/弩车、冰霜术/震荡陷阱。
- 通过标准：
  玩家至少完成1次3层链，并使用或明确想使用万能牌补断点。

### 6. Boss波：验证终端爆发

- Boss：冲城巨怪x1，入口出现。
- 附带：每回合小怪x1。
- Boss规则：
  每2回合冲锋，额外推进1格；若被冻结，跳过本次冲锋。
- 推荐胜利路径：
  铁壁队长用 `盾兵 -> 弓手 -> 冰霜术 -> 桥头炮台`，靠护甲和冻结拖住Boss，再用炮台终端收束。
  火炮军师用 `枪兵 -> 弩车 -> 火油 -> 桥头炮台`，先攒终端加伤，再通过4层链打出爆发。
- 通过标准：
  玩家能完成至少1次4层链，或明确知道自己缺哪一层导致失败。

## 八、60秒首局承接脚本

### 0-10秒：建立战场

- 画面信息：桥头、城门生命、怪潮入口、底部4张手牌。
- 玩家应看懂：怪物会从左到右压到城门，我要用手牌守住桥头。

### 10-25秒：第一次两段链

- 系统给牌：盾兵、弓手、路障、临时工事。
- 引导：先打盾兵，再打弓手。
- 反馈：弓手伤害从2变3，防线出现“2层链”提示。

### 25-45秒：第一次错误对比

- 系统给牌：弓手、盾兵、火枪手、临时工事。
- 若玩家先打弓手：只造成基础伤害，提示“没有前排接住，输出未进入连锁”。
- 若玩家先打盾兵：触发2层链，守军伤害+1。

### 45-60秒：第一次资源出口

- 系统给牌：桥头炮台、盾兵、弓手、冰霜术。
- 事件：工事值不足，炮台暂时接不上。
- 玩家选择：保留炮台等终端，或弃炮台换工事值修桥。
- 关键结论：玩家知道“不合适的牌可以变成守线资源”。

## 九、15秒素材脚本

### 版本A：极限守线

- 0-3秒：城门生命1%，怪潮压到桥头，底部手牌闪烁。
- 3-8秒：玩家连续打出盾兵、弓手、冰霜术，链条UI从1段亮到3段。
- 8-12秒：援军令补上终端断点，桥头炮台落下。
- 12-15秒：全桥炮火爆发，Boss倒下，文案出现：“按顺序出牌，守住最后一座桥。”

### 版本B：错误对比

- 0-5秒：乱出桥头炮台，伤害不足，Boss冲进城门。
- 5-10秒：回放同一局，先盾兵、再弓手、再冰霜、再炮台。
- 10-15秒：四段链爆发，怪潮被顶回，文案出现：“强卡不够，接对才赢。”

## 十、三轮纸面推演

### 推演1：新手按教学走

- 玩家行为假设：
  玩家按引导先打盾兵再打弓手，波次2尝试弃牌修桥，Boss波用援军令补断点。
- 是否达到闸门阈值：
  一分钟承接通过，十分钟成型部分通过。
- 暴露的问题：
  如果UI不清楚提示“下一层”，玩家可能不知道为什么弓手变强。

### 推演2：玩家乱出强卡

- 玩家行为假设：
  玩家看到桥头炮台就先打，导致未形成链条，Boss波伤害不足。
- 是否达到闸门阈值：
  三秒买点可能通过，一分钟承接失败。
- 暴露的问题：
  必须在前60秒给出错误对比，让玩家看到乱出强卡和接链的差异。

### 推演3：玩家卡手

- 玩家行为假设：
  玩家抽到阻挡、终端、终端、控制，缺输出层。
- 是否达到闸门阈值：
  若有援军令或弃牌换士气，可继续推进；否则体验失败。
- 暴露的问题：
  起始牌组必须控制层级分布；万能牌和弃牌资源是必要系统，不是附属系统。

- 一分钟承接：已有首局60秒脚本和两段链教学。
