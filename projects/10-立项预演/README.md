# 立项预演

这里用于放还处在方向探索、评审准备和立项论证阶段的项目。

## 项目结构

1. `00-立项文档/`：放正式立项文档、玩法模型、GDD 骨架等正文材料
2. `01-评审/`：放初审稿、评审规则、评审意见和决策记录
3. `02-纸面原型/`：固定拆成两个子目录
   `01-生成内容/`：放纸面原型、概念图 prompt、辅助脚本和探索性材料
   `02-评审结果/`：放纸面原型评审、打分结果和结论记录
4. `03-实现/`：放代码、配置、资源、模块拆分和实现说明
5. `04-测试/`：放测试记录、Bug 清单、验证结果和调参日志
6. `05-发布/`：放外链、部署说明、发布记录和版本摘要
