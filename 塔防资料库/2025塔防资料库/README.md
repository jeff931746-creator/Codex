# 2025塔防资料库

这是 `2025 年塔防资料库` 的第一版本地底稿。

当前边界：

- 先收录 `2025 年已公开发售`、`公开可检索`、`玩法上明确属于塔防或塔防混合` 的样本
- 第一版优先覆盖 `PC / Steam`
- 不是“2025 全球塔防绝对全集”，而是可持续扩充的研究样本库

当前包含：

- `tower_defense_2025_library.csv`：2025 年塔防样本主表
- `tower_defense_2025_mobile_minigame_watchlist.csv`：2025 年手游与微信小游戏观察表
- `2025塔防来源地图.md`：官方商店、国内媒体、抖音、微信等来源地图
- `资料说明.md`：本期资料库的口径、来源和扩充建议

后续可继续扩的方向：

1. `2025 手游塔防样本`
2. `2025 微信小游戏塔防样本`
3. `2025 抖音传播强的塔防样本`
4. `2025 国内媒体与职业评论摘要`
