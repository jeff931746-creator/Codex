## 三、四象限评分

每项 1-5 分。任一项 < 3 为硬伤，不得用总分掩盖。

#### 事实

**有效获量轴（横轴）**

| 维度 | 评分 | 说明 |
|---|---|---|
| 需求规模：当前市场是否有真实可触达需求 | /5 | |
| 第一眼刺激：首图是否有强识别、强情绪、强冲突 | /5 | |
| 素材新鲜度：是否有一眼可见的新表达 | /5 | |
| 竞争压力：同题材同人群同表达是否高度拥挤（反向） | /5 | |
| CPI 风险：预估买量成本是否在可承接范围内 | /5 | |

有效获量均分：/5

**ROI 承接轴（纵轴）**

| 维度 | 评分 | 说明 |
|---|---|---|
| LTV 上限：是否支撑成长 / 收集 / 养成 / 外观付费 | /5 | |
| 留存深度：是否能形成持续目标和重复行为 | /5 | |
| 付费自然度：付费是否服务体验而非硬塞 | /5 | |
| 内容效率：内容能否持续生产且成本可控 | /5 | |
| 回本压力：预期回本周期是否匹配团队能力 | /5 | |

ROI 承接均分：/5

#### 判断

- 四象限位置：`高获量+高ROI / 高获量+低ROI / 低获量+高ROI / 低获量+低ROI`
- 是否有硬伤项（< 3 分）：`是 / 否，项目：`
- 若低项需迁移，迁移方式：

#### 结论

- 四象限结论：`通过 / 补证 / 不通过`

| 5 秒短视频：是否能展示冲突和反馈 | |

- 是否出现以下失败信号：

#### 结论

- 卖相验证结论：`通过 / 补证 / 不通过`
