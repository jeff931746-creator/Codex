# 题材母题模板

按本模板新增题材母题条目。完整字段定义见 [02-题材母题标准.md](../02-题材母题标准.md)。

---

## 模板结构

```markdown
---
id: THM-XXX-XXX                   # 必填，分配后不可修改
name: 母题名称                     # 必填，≤8字
category: XXX                      # 必填，大类代码（SUR/POW/EXP/HEA/GRO/BRE）
core_emotion: 核心情绪              # 必填，一句话
core_conflict: 核心冲突             # 必填，一句话
audience_ids:                      # 必填，关联人群簇 ID
  - AUD-X-XXX
  - AUD-Y-XXX
playstyle_ids:                     # 必填，关联玩法承接 ID
  - PLY-XX-XXX
  - PLY-YY-XXX
theme_examples:                    # 必填，具体题材实例 ≥3
  - 具体题材1
  - 具体题材2
  - 具体题材3
representative_games:              # 必填，代表游戏 ≥3
  - 游戏A
  - 游戏B
  - 游戏C
evidence_level: A                  # 必填，证据等级 S/A/B/C/D

theme_elements:                    # 选填，题材三要素
  environment: 环境/时代
  culture_paradigm: 文化范式/战斗体系
  narrative: 叙事
acquisition_score: 4               # 选填，1-5
roi_score: 4                       # 选填，1-5
quadrant: "右上"                   # 选填
material_hooks:                    # 选填
  - 素材钩子1
  - 素材钩子2
risk_notes:                        # 选填
  - 风险1
  - 风险2
related_motifs:                    # 选填，相似母题 ID
  - THM-XX-XXX
opposing_motifs:                   # 选填，对立母题 ID
  - THM-YY-YYY
---

## 详细描述

### 母题定义

（描述这个母题的稳定情绪+冲突结构，让读者一眼能识别）

### 核心情绪体验

（玩家在这个母题下能获得的核心情绪体验是什么）

### 适用题材范围

（这个母题能落到哪些具体题材标签上）

### 代表作品分析

（说明为什么这几个代表游戏属于这个母题，它们的共同点是什么）

### 与其他母题的关系

（说明可融合的相邻母题、对立母题）

### 立项参考

（针对这个母题做立项时的注意事项、典型素材策略、风险点）
```

## 示例：THM-SUR-001 末世求生（参考用，不入正式库）

```markdown
---
id: THM-SUR-001
name: 末世求生
category: SUR
core_emotion: 压力释放 + 短暂爽感
core_conflict: 在崩溃的世界中保住自己和重要的人
audience_ids:
  - AUD-Y-001    # 压力释放型休闲玩家
  - AUD-M-001    # SLG重氪用户
playstyle_ids:
  - PLY-TD-001   # 波次防守 + 局外塔型升级
  - PLY-SLG-001  # 末世建城 + 联盟玩法
  - PLY-RG-001   # Roguelike 跑图
theme_examples:
  - 末世丧尸
  - 末世铁路王国
  - 核冬天废土
  - 海底文明崩溃
  - 病毒爆发城市
representative_games:
  - 向僵尸开炮
  - Last War
  - State of Survival
  - 末日杂货铺
evidence_level: S

theme_elements:
  environment: 现代或近未来，文明崩溃后的废土
  culture_paradigm: 枪械 + 近战 + 资源管理 / 建造防御工事
  narrative: 玩家是幸存者，需要在危机中保护自己和队伍
acquisition_score: 5
roi_score: 4
quadrant: "右上（高获量 + 高ROI）"
material_hooks:
  - 尸潮压境
  - 据点防守
  - 救援人质
  - 资源争夺
risk_notes:
  - 素材同质化严重，必须有差异切口
  - 大泡泡，竞争密度高
  - CPI压力增大
related_motifs:
  - THM-GRO-001    # 弱者逆袭（末世求生中常融入成长复仇线）
opposing_motifs:
  - THM-HEA-001    # 治愈放松（情绪基调对立）
---

## 详细描述

### 母题定义

末世求生是"在崩溃的世界中保住自己和重要的人"这一稳定情绪+冲突结构。
核心情绪是**压力**（资源不够、危险逼近、时间紧迫），间或穿插**爽感**（成功击退危险、救出队友）。

### 核心情绪体验

玩家在这个母题下获得：
- 紧迫感（资源稀缺、敌人压境）
- 解决感（每次成功防御都是小胜利）
- 责任感（保护幸存者、建设据点）

### 适用题材范围

这个母题可以落到的具体题材：
- 末世丧尸（最经典）
- 末世铁路王国（新颖切口）
- 核冬天废土（视觉差异化）
- 海底文明崩溃（蓝海尝试）
- 病毒爆发城市（现实感强）

### 代表作品分析

向僵尸开炮、Last War、State of Survival 这几款的共同点：
- 都有"危险压境 + 资源管理 + 建造防御"的核心循环
- 都有明确的"幸存者身份"
- 都有"短期生存目标 + 长期重建希望"的双层叙事

### 与其他母题的关系

- 可融合：与"弱者逆袭"（THM-GRO-001）融合可形成"末世翻身爽剧"，常见于短剧改编游戏
- 对立：与"治愈放松"（THM-HEA-001）情绪基调对立，难以兼容（但部分末世建造游戏会弱化压力强化建造，变成偏HEA的体验）

### 立项参考

- 素材策略：必须有"差异化切口"——新场景（如末世铁路）、新机制（如丧尸+解谜）、新人群（如女性向末世）
- CPI预期：20-50元（取决于切口创新度）
- 留存关键：前 3 天必须建立"据点感"，让玩家产生"我有家要保护"的代入感
- 商业化关键：月卡 + 资源加速 + 数值养成，深度商业化路径成熟
```

---

## 注意事项

- 母题名称必须包含情绪+冲突结构，不能用具体题材标签
- 至少 3 个具体题材实例和 3 个代表游戏
- audience_ids 和 playstyle_ids 必须填写（这是三位一体的核心关联）
- evidence_level 低于 B 不能进入正式库
