# 人群簇模板

按本模板新增人群簇条目。完整字段定义见 [01-人群簇标准.md](../01-人群簇标准.md)。

---

## 模板结构

```markdown
---
id: AUD-X-XXX                    # 必填，分配后不可修改
name: 人群簇名称                  # 必填，≤12字
category: X                       # 必填，大类代码（Y/M/S/F/C/H）
description: 一句话描述            # 必填，≤30字
emotion_need:                     # 必填，核心情绪需求
  - 情绪1
  - 情绪2
playstyle_preference:             # 必填，偏好玩法
  - 玩法1
  - 玩法2
payment_pattern:                  # 必填，付费习惯
  - 付费方式1
  - 付费方式2
acquisition_difficulty: 3         # 必填，获量难度 1-5
ltv_potential: 4                  # 必填，LTV潜力 1-5
evidence_level: A                 # 必填，证据等级 S/A/B/C/D

age_range: "25-35岁"              # 选填
gender_lean: male                 # 选填 male/female/neutral
geo:                              # 选填
  - 国内一二线
  - 东南亚
info_channels:                    # 选填
  - 抖音短视频
  - B站
representative_games:             # 选填，代表游戏
  - 游戏A
  - 游戏B
boundary: |                       # 必填，与相邻人群的差异
  与 AUD-X-XXX 的区别：______
  与 AUD-Y-XXX 的区别：______
risk_notes: |                     # 选填
  - 素材疲劳快
  - 流失高
---

## 详细描述

### 核心特征

（一段话描述人群的核心识别特征，包括其情绪需求、消费习惯、信息渠道）

### 典型行为画像

（描述这个人群的典型一天/一周如何接触和消费游戏内容）

### 代表游戏分析

（说明为什么这几个代表游戏的用户重合度高，他们共同的偏好是什么）

### 边界说明

（详细说明与相邻人群簇的边界，避免重复定义）

### 立项参考

（针对这个人群簇做立项时的注意事项、典型素材策略、CPI预期）
```

## 示例：AUD-Y-001 压力释放型休闲玩家（参考用，不入正式库）

```markdown
---
id: AUD-Y-001
name: 压力释放型休闲玩家
category: Y
description: 工作压力大，需要快速获得满足感的年轻玩家
emotion_need:
  - 压力释放
  - 即时满足
  - 轻量陪伴
playstyle_preference:
  - 放置
  - 单局短爽
  - 自动化
payment_pattern:
  - 月卡
  - 小额数值养成
  - 广告变现
acquisition_difficulty: 2
ltv_potential: 3
evidence_level: A

age_range: "22-32岁"
gender_lean: neutral
geo:
  - 国内一二线
  - 国内三四线
info_channels:
  - 抖音短视频
  - 微信朋友圈
representative_games:
  - 咸鱼之王
  - 不朽家族
  - 寻道大千
boundary: |
  与 AUD-Y-002 "二次元抽卡用户" 的区别：付费动机不同，本人群偏向数值养成和长线放置，非抽卡爆点
  与 AUD-M-001 "SLG重氪用户" 的区别：付费力较弱，但获量难度低；不需要长时间在线决策
risk_notes:
  - 素材疲劳快，3-7天需要换素材
  - 留存断崖式衰减，需要持续投放
---

## 详细描述

### 核心特征

这类玩家通常工作压力大、时间碎片化，将游戏作为压力释放出口。
他们不追求深度体验或社交炫耀，而是希望快速获得满足感。
对游戏的核心需求是"上班路上能玩一会儿"、"睡前刷一下能爽到"。

### 典型行为画像

- 早晨通勤路上打开 3-5 分钟，看离线收益
- 午休时玩 10-15 分钟，推进关卡
- 睡前躺床上玩 20-30 分钟，做日常任务
- 周末偶尔会有较长游戏时间，但不会沉迷某一款超过 2 小时

### 代表游戏分析

咸鱼之王、不朽家族、寻道大千这三款的用户重合度高，因为它们都满足：
- 上手门槛极低
- 自动化程度高
- 数值反馈快
- 单局/单次操作时间 ≤5 分钟

### 边界说明

与"二次元抽卡用户"的核心区别是付费动机：本人群是为"数值变强"付费，二次元用户是为"角色喜爱+抽卡爆点"付费。
与"SLG重氪用户"的核心区别是时间投入：本人群不愿意长时间在线决策，SLG用户能接受联盟战通宵。

### 立项参考

- 素材策略：突出"自动化"、"离线收益"、"快速变强"
- CPI预期：30-50元
- 留存关键：前 7 天的成长曲线必须明显，第二周内容需要有变化
- 商业化关键：月卡 + 小额数值养成，避免大额抽卡（人群付费力不够）
```

---

## 注意事项

- ID 由用户批准后分配，不能自行决定
- evidence_level 低于 B 的条目不能进入正式库
- boundary 字段必须明确，否则会被驳回
