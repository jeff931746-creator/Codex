# 题材母题标准

## 这是什么

题材母题是玩家在题材中能识别的稳定情绪+冲突结构。本文档定义题材母题的字段、与具体题材的关系、与历史题材库的归并规则。

参考：[00-框架定义.md](00-框架定义.md) 中的"维度 2：题材母题"。

---

## 母题 vs 具体题材

**核心区分**：

| 概念 | 数量 | 例子 |
|---|---|---|
| 题材母题 | 15-25 个 | "末世求生"、"权力博弈"、"成长复仇" |
| 具体题材 | 数百个 | "末世铁路王国"、"维多利亚蒸汽朋克"、"修真宗门" |

一个母题对应多个具体题材：

```text
"末世求生" 母题 → 末世丧尸、末世铁路王国、核冬天废土、海底文明崩溃
"权力博弈" 母题 → 三国、宫斗、黑帮、政治、修真宗门
```

## 大类分类（按主情绪）

题材母题按"主情绪"分大类，沿用 [题材分类参考库](../../../../archive/方法论/立项/题材分类参考库.md) 的 6 大主情绪分类：

| 大类代码 | 主情绪 | 母题数量预估 | 典型母题 |
|---|---|---|---|
| `SUR` | 求生压力 | 3-5 | 末世求生、绝境逃脱、生存建造 |
| `POW` | 爽感释放 | 3-5 | 无双战斗、力量碾压、技能爆发 |
| `EXP` | 敬畏探索 | 3-5 | 未知探索、神秘解谜、世界发现 |
| `HEA` | 治愈放松 | 3-5 | 田园经营、宠物模拟、装扮收集 |
| `GRO` | 成长满足 | 3-5 | 弱者逆袭、复仇崛起、修炼飞升 |
| `BRE` | 紧张破局 | 3-5 | 权力博弈、悬疑推理、棋局对决 |

**注意**：母题之间可以有跨大类的关联（如"复仇崛起"既有成长也有爽感），但归大类时按**核心情绪驱动力**归。

## 字段定义

每个题材母题条目必须包含以下字段：

### 必填字段

| 字段 | 类型 | 说明 |
|---|---|---|
| `id` | 字符串 | 格式 `THM-{大类}-{编号}`，永不修改 |
| `name` | 字符串 | 母题名称（≤8字） |
| `category` | 枚举 | 大类代码（SUR/POW/EXP/HEA/GRO/BRE） |
| `core_emotion` | 字符串 | 核心情绪（一句话） |
| `core_conflict` | 字符串 | 核心冲突（一句话） |
| `audience_ids` | 数组 | 关联的人群簇 ID 列表 |
| `playstyle_ids` | 数组 | 关联的玩法承接 ID 列表 |
| `theme_examples` | 数组 | 具体题材实例（≥3个） |
| `representative_games` | 数组 | 代表游戏（≥3个） |
| `evidence_level` | 枚举 | 证据等级 S/A/B/C/D |

### 选填字段

| 字段 | 类型 | 说明 |
|---|---|---|
| `theme_elements` | 对象 | 题材三要素（环境/文化范式/叙事），见 [题材选择方法论](../../../../archive/方法论/立项/题材选择方法论_2026-05-08.md) |
| `acquisition_score` | 整数 1-5 | 有效获量评分（母题整体倾向） |
| `roi_score` | 整数 1-5 | ROI 承接评分（母题整体倾向） |
| `quadrant` | 枚举 | 四象限定位（参考[题材选择方法论](../../../../archive/方法论/立项/题材选择方法论_2026-05-08.md) ） |
| `material_hooks` | 数组 | 典型素材钩子 |
| `risk_notes` | 数组 | 主要风险 |
| `related_motifs` | 数组 | 相似/可融合的其他母题 ID |
| `opposing_motifs` | 数组 | 对立/难融合的其他母题 ID |

## 母题三要素（推荐填写）

母题可以借用 [题材选择方法论](../../../../archive/方法论/立项/题材选择方法论_2026-05-08.md) 中的"题材三要素"框架，但母题层面要素更抽象：

```text
环境/时代：母题适用的世界观范围（如"末世求生"适用于现代废土、未来太空、奇幻末世等）
文化范式/战斗体系：母题导出的核心系统（如"末世求生"导出资源管理+生存系统）
叙事：母题的核心叙事张力（如"末世求生"是"在崩溃世界中保住自己和重要的人"）
```

## 母题命名原则

### 名称要求

- ≤ 8 字（比具体题材标签更短，更抽象）
- 包含**情绪 + 冲突结构**两部分（如"末世+求生"、"权力+博弈"、"成长+复仇"）
- 避免使用具体题材标签（如"末世丧尸"是具体题材，不是母题）

### 好的名称示例

✅ "末世求生"（情绪：压力；冲突：在崩溃世界活下去）
✅ "权力博弈"（情绪：紧张；冲突：人与人的智斗）
✅ "成长复仇"（情绪：爽感+成长；冲突：弱者翻身）

### 不好的名称示例

❌ "末世丧尸"（具体题材，不够抽象）
❌ "丧尸+塔防+建造"（玩法混入，不是纯母题）
❌ "新颖的奇幻冒险"（无情绪冲突结构）

## 与具体题材的归并规则

[题材总表](../../../../archive/资料/买量组合库/题材总表.md) 中每条具体题材必须能映射到 1 个母题。母题归并规则：

```text
1. 看题材的"主情绪" → 决定大类
2. 看题材的"核心冲突" → 决定具体母题
3. 题材表中加 mother_theme_id 字段填入对应母题 ID
```

**多母题归并的处理**：

- 一个题材主要归属一个母题（主母题，必填）
- 如果题材跨多个母题，可以在 `secondary_motif_id` 字段中记录次母题（最多 2 个，选填）
- 但题材的评分依据**主母题**，不是次母题的平均

例：

```yaml
题材：末世铁路王国
mother_theme_id: THM-SUR-001    # 主母题：末世求生
secondary_motif_id: [THM-GRO-002]    # 次母题：建造成长
```

## 历史题材库的归并

[历史题材库](../../../../archive/资料/历史题材库/) 中 13042 部作品需要按母题归类。归并步骤：

1. 完成历史题材库构建（当前任务）
2. 对每部作品，按其"题材三要素"判断主母题
3. 在历史题材库总览中按母题分组统计代表作品数量
4. 形成每个母题的"代表作品列表"（参考价值高的 5-10 部）

**当前阶段**：本框架建立时不执行归并，但要在母题标准中说明归并规则。

## 评分维度说明

### 有效获量（acquisition_score）

母题层面的获量难度，参考 [题材评估标准](../../../../archive/资料/买量组合库/题材评估标准.md)。

| 分数 | 含义 |
|---|---|
| 1 | 母题本身无法形成第一眼买量素材 |
| 2 | 母题需要长解释，素材难做差异化 |
| 3 | 母题有标准素材模板，但竞争激烈 |
| 4 | 母题有多种差异表达，市场仍有空间 |
| 5 | 母题素材力天然强，能立即吸引点击 |

### ROI 承接（roi_score）

母题层面的商业化承接能力。

| 分数 | 含义 |
|---|---|
| 1 | 母题导出的玩法商业化弱，仅适合广告变现 |
| 2 | 母题有小额付费空间，但难成大规模 ROI |
| 3 | 母题有标准商业化路径（月卡、外观、关卡推进） |
| 4 | 母题有较强付费驱动力（养成、收集、社交） |
| 5 | 母题天然导出深度商业化（数值养成、抽卡、PVP） |
