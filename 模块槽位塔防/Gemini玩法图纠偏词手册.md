# Gemini 玩法图纠偏词手册

## 用途

这份手册用于 Gemini 出图“方向大致对了，但玩法识别不够准”的情况。

最推荐的使用方法：

1. 不要一口气重写整段 prompt。
2. 先判断图偏到哪一种错误。
3. 复制对应的纠偏词，加在原 prompt 后面，或作为“请按以下要求修正当前图”的补充说明。

---

## 1. 右侧面板像技能按钮，不像模块槽位

### 症状

- 右侧更像技能栏
- 像 4 个可点击按钮
- 没有“硬件槽位”感

### 纠偏词

```text
Important correction:
the right-side panel must not look like a skill-button menu.
It must look like a permanent hardware module panel with exactly 4 embedded machine slots.
Each slot should feel like a physical socket for a cartridge-like module.
The slots are part of the defense system architecture, not active ability buttons.
```

---

## 2. 槽位和防线没有物理连接感

### 症状

- 模块只是放在 UI 上
- 看不出它们驱动整条防线
- 缺少能量线、电缆、核心联动

### 纠偏词

```text
Important correction:
make the installed module slots visibly connected into the defense system with glowing energy cables.
The cables should run from the module panel into the reactor core and main defense line.
The whole defense system should feel powered by the modules.
```

---

## 3. 画面像普通守线塔防，不像模块槽位塔防

### 症状

- 主炮太抢眼
- 还是“普通塔防战斗 + UI”
- 看不出“系统装配”感

### 纠偏词

```text
Important correction:
this image should not feel like a generic tower defense battle.
It must feel like a defense system being reconfigured by installed modules.
The modular system identity must be visible at a glance.
Do not make the image about a single cannon or generic combat spectacle.
```

---

## 4. 掉落模块不够显眼

### 症状

- 掉落物太小
- 像普通战利品
- 看不出“高价值模块”

### 纠偏词

```text
Important correction:
the dropped module must be the visual focus.
Make it larger, brighter, and clearly collectible.
It should look like a physical machine component or energy cartridge, not a coin, gem, or card.
Add a strong loot beam or holographic highlight.
```

---

## 5. 联动效果像两个独立技能，不像 synergy

### 症状

- 火是火，电是电
- 看不出它们互相作用
- 没有“Build 成型”的感觉

### 纠偏词

```text
Important correction:
show module synergy, not separate isolated effects.
The battlefield effects should look linked and system-driven, for example chain-burning or lightning-linked explosions.
The image should communicate that module combinations create a stronger build through interaction.
```

---

## 6. 4 种效果太乱，可读性差

### 症状

- 火、电、冰、召唤全堆在一起
- 画面炫但看不清
- 不像真实游戏截图

### 纠偏词

```text
Important correction:
improve gameplay readability.
Separate the module effects into clear battlefield zones or layers.
Keep the image dramatic, but make the combat hierarchy readable like a real mobile game screenshot.
Reduce random visual clutter.
```

---

## 7. 镜头、场景、UI 与前一张不一致

### 症状

- 系列图看起来不像同一局
- 机位变化太大
- UI 每张都在变

### 纠偏词

```text
Important correction:
keep the exact same battlefield, same camera angle, same vertical composition, same defense line layout, same module panel layout, and same UI hierarchy as the previous image.
This must feel like the same match in the same scene, only with one gameplay state changed.
```

---

## 8. Boss 图像像单独 Boss 关，不像玩法链条最后一环

### 症状

- 场景突然变了
- 变成单独 Boss arena
- 看不出前面 Build 的延续性

### 纠偏词

```text
Important correction:
this must remain the same battlefield and the same modular defense system from the previous images.
Do not turn it into a separate boss arena.
The boss wave should feel like the final stress test of the existing build, not a different mode.
```

---

## 9. 画面太像信息图，不像真机截图

### 症状

- 结构太平
- 像蓝图或展示图
- 不像战斗中截图

### 纠偏词

```text
Important correction:
this must look like a real mobile game battle screenshot, not an infographic, blueprint, or abstract concept board.
Keep the UI grounded, the battlefield active, and the gameplay state readable.
```

---

## 10. 推荐组合模板

### 如果“像技能按钮 + 没有物理连接感”

```text
Important correction:
the right-side panel must not look like a skill-button menu.
It must look like a permanent hardware module panel with exactly 4 embedded machine slots.
Each slot should feel like a physical socket for a cartridge-like module.
Make the installed slots visibly connected into the defense system with glowing energy cables running into the reactor core and main defense line.
```

### 如果“普通塔防感太强 + Build感太弱”

```text
Important correction:
this image should not feel like a generic tower defense battle.
It must feel like a defense system being reconfigured by installed modules.
Show module synergy, not separate isolated effects.
The modular system identity must be visible at a glance.
```

### 如果“炫但乱，不够像真机”

```text
Important correction:
improve gameplay readability.
Separate the module effects into clear battlefield zones or layers.
Reduce random visual clutter.
This must still look like a real mobile game battle screenshot, not an infographic or abstract effect montage.
```
