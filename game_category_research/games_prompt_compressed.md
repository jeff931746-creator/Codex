# 压缩版

```text
请你作为资深游戏竞品分析师，帮我收集不同品类的代表游戏，并进行简要拆解和分类，用于竞品研究和立项参考。

任务要求：
1. 覆盖主流游戏品类
2. 每个品类选择 3-5 款代表游戏
3. 对每款游戏做统一维度的简要拆解
4. 最终输出尽量表格化，便于横向对比

请优先覆盖这些品类：
- MOBA
- FPS / TPS
- 战术射击
- RPG
- ARPG
- MMORPG
- 开放世界
- 卡牌
- SLG
- 二次元养成
- Roguelike / Roguelite
- 模拟经营
- 生存建造
- 放置 / Idle
- 休闲益智
- 派对游戏
- 体育竞速
- 沙盒创造

请按以下格式输出：

第一部分：品类总览表
字段包括：
- 品类
- 核心特点
- 核心循环
- 常见商业化方式
- 长线运营方式
- 代表游戏

第二部分：代表游戏拆解表
每行一款游戏，字段包括：
- 游戏名称
- 主品类
- 次级标签
- 平台
- 核心玩法
- 核心循环
- 成长线
- 付费模式
- 社交属性
- PVP/PVE 属性
- 节奏类型
- 操作门槛
- 局内体验特征
- 局外系统特征
- 核心卖点
- 目标用户
- 参考价值

第三部分：每款游戏简析
每款游戏补充 3 点：
1. 最值得关注的设计亮点
2. 最关键的商业/运营特征
3. 最适合参考的方向

第四部分：最终分类汇总
请把所有游戏按以下维度再汇总分类：
- 按核心玩法分类
- 按商业模式分类
- 按社交强度分类
- 按操作门槛分类
- 按项目体量分类
- 按参考价值分类（玩法参考 / 商业化参考 / 运营参考 / UI参考 / 立项参考）

第五部分：总结建议
请最后输出：
1. 最适合小团队研究的品类
2. 最适合长线运营的品类
3. 最适合创新混搭的品类
4. 当前竞争最激烈的品类
5. 最值得重点研究的 10 款游戏

输出要求：
- 以分析和归纳为主，不要写宣传文案
- 优先使用 Markdown 表格
- 信息不确定时标注“推测”或“待确认”
- 如果内容过多，优先保证表格完整
- 保持表达简洁，但要有信息密度
```
