# 产品立项分析版

```text
你现在是一名资深游戏产品策划兼市场分析师，请帮我对各个品类游戏进行信息收集、基础拆解和立项视角分析，用于判断哪些方向值得进入。

任务目标：
1. 梳理主要游戏品类的市场与产品特征
2. 对各品类代表游戏进行简要拆解
3. 从“立项可行性”角度进行横向对比
4. 帮助判断哪些品类更适合新项目切入

请按以下结构输出：

一、整体分类框架
请先列出主要游戏品类，并给出每个品类的一句话定义。

二、每个品类的立项分析
对每个品类，请输出以下内容：
1. 品类名称
2. 品类定义
3. 核心玩法特征
4. 用户核心需求
5. 典型核心循环
6. 内容生产方式
7. 常见成长线
8. 常见商业化方式
9. 长线运营方式
10. 开发成本判断（低 / 中 / 高）
11. 美术成本判断（低 / 中 / 高）
12. 技术复杂度判断（低 / 中 / 高）
13. 运营复杂度判断（低 / 中 / 高）
14. 市场竞争强度判断（低 / 中 / 高）
15. 适合切入的团队类型
16. 立项风险
17. 立项机会点

三、代表游戏拆解
每个品类选择 3-5 款代表游戏，并按以下维度简要拆解：
1. 游戏名称
2. 主品类
3. 次品类标签
4. 平台
5. 核心卖点
6. 核心玩法
7. 核心循环
8. 成长驱动
9. 付费驱动
10. 社交属性
11. 内容更新方式
12. 为什么成功
13. 可借鉴点
14. 不适合直接照搬的点

四、立项视角分类
请把这些品类按以下维度再做一轮分类：
1. 适合小团队切入的品类
2. 适合中型团队切入的品类
3. 适合大团队长期投入的品类
4. 适合做中轻度产品的品类
5. 适合做长线运营的品类
6. 适合做买量驱动项目的品类
7. 适合做社区驱动项目的品类
8. 适合玩法创新切入的品类
9. 红海品类
10. 蓝海或可细分突破的品类

五、最后输出结论
请在最后总结：
1. 当前最值得关注的 5 个品类
2. 每个品类的进入门槛
3. 每个品类的主要机会
4. 每个品类的主要风险
5. 如果是小团队第一次做游戏，优先建议关注哪些品类，并说明原因

输出要求：
1. 先给整体摘要
2. 再展开各品类分析
3. 最后给出立项建议
4. 保持结构化表达
5. 少写空话，多给判断依据
6. 如果存在不确定信息，请明确说明是行业推断
```
