# 游戏品类调研 Prompt

这个文件夹用于集中存放“游戏品类信息收集、拆解、分类、立项分析、竞品研究”相关 Prompt。

当前包含：

- `games_prompt_base.md`
  通用版，适合先做各品类信息收集、简单拆解和分类。

- `games_prompt_product.md`
  产品立项分析版，适合从立项、机会、风险、团队切入角度做判断。

- `games_prompt_competitor.md`
  竞品调研表格版，适合做统一字段整理、横向对比和竞品资料沉淀。

- `games_prompt_deep.md`
  深度版，适合一次性做系统化研究、分类框架搭建和代表产品拆解。

- `games_prompt_light.md`
  轻量版，适合快速扫品类、快速建立行业认知。

- `games_prompt_compressed.md`
  压缩版，适合直接丢给 Claude / GPT 这类模型快速产出结构化结果。

建议使用方式：

1. 想快速开始：先用 `games_prompt_light.md`
2. 想系统研究：用 `games_prompt_deep.md`
3. 想做竞品表：用 `games_prompt_competitor.md`
4. 想做立项判断：用 `games_prompt_product.md`
