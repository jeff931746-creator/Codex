# 竞品调研表格版

```text
你现在是一名资深游戏竞品分析师，请帮我收集各个品类的代表游戏，并用统一表格维度进行整理，方便做竞品调研、横向对比和后续筛选。

任务目标：
1. 按品类收集代表游戏
2. 用统一字段整理每款游戏信息
3. 对每款游戏做简单拆解和分类
4. 输出结果尽量表格化，方便复制到文档或表格工具中

请按以下步骤输出：

一、先建立分类框架
请先列出主要游戏品类，并说明每个品类的核心特征。

二、按品类整理竞品
每个品类选择 3-5 款代表游戏，并使用统一表格输出。

三、每款游戏的表格字段
请尽量以表格形式输出，每行一款游戏，字段至少包括：
1. 游戏名称
2. 主品类
3. 次级标签
4. 平台
5. 地区/市场
6. 上线时间
7. 核心玩法
8. 核心循环
9. 局内体验
10. 局外系统
11. 成长线
12. 付费模式
13. 社交属性
14. PVP/PVE 属性
15. 节奏类型（短局 / 长局 / 碎片化）
16. 操作门槛（低 / 中 / 高）
17. 内容更新方式
18. 主要用户画像
19. 核心卖点
20. 参考价值
21. 风险提示

四、简单拆解要求
除了表格外，请对每款游戏补充 3 条简要分析：
1. 这款游戏最核心的设计亮点
2. 这款游戏的商业/运营关键点
3. 这款游戏适合作为哪类项目的参考样本

五、分类汇总
请在最后增加一个汇总表，对所有游戏做横向分类，至少包括：
1. 按核心玩法分类
2. 按商业模式分类
3. 按社交强度分类
4. 按操作门槛分类
5. 按项目体量分类
6. 按适合参考的项目阶段分类（立项参考 / 玩法参考 / 商业化参考 / 运营参考 / UI参考）

六、输出要求
1. 优先使用表格
2. 表格字段尽量统一
3. 信息尽量简洁但有效
4. 如果某些信息不确定，可以留“待确认”或标注“推断”
5. 如果内容过多，请先保证表格完整，再补充简析
6. 不要写成宣传介绍，要写成调研资料

七、最终补充
请最后额外输出：
1. 最适合做玩法参考的 10 款游戏
2. 最适合做商业化参考的 10 款游戏
3. 最适合做长线运营参考的 10 款游戏
4. 最适合小团队研究的 10 款游戏
```
