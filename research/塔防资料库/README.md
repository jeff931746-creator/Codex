# 塔防资料库

这个资料库是本地版塔防产品研究底稿，方便持续补充 `War3 地图`、`手游`、`Steam`、`小游戏` 等不同平台的塔防样本。

当前包含：

- `tower_defense_library.csv`：主资料库，可直接导入表格工具继续维护
- `字段说明.md`：字段口径与填写规范

当前首批样本来自现有整理文档：

- `/Users/mt/Documents/Codex/war3_rpg_td_maps.md`

建议维护方式：

1. 新增产品时，优先补 `产品名`、`平台`、`发售状态`、`主品类`、`上线时间`、`时间口径`、`数据来源`
2. 如果无法确认精确时间，`上线时间` 填 `待确认`，并在 `时间口径` 写清楚
3. 收入规模只做 `头部 / 腰部 / 长尾 / 待确认` 分级，不写伪精确流水
4. 玩法判断和商业判断分开写，避免把“好玩”和“能赚钱”混成一列
