## 1. 项目摘要

### 1.1 一句话定义

- 项目一句话定义：`一款以多武器自动战斗Build为核心、以大世界探索为前期承载、以轻MMO社交和跨服对抗为长线承接的ARPG产品。`

### 1.2 立项结论

- 立项建议：`可试做`
- 核心理由：
  - `事实`：原企划明确将项目定义为“多武器战斗的大世界探索ARPG”，并叠加轻度MMO承接。
  - `事实`：内测平均分 `3.47/4`，专家团平均分 `3.3/4`、评星 `19.65/25`，素材吸量测试通过，平均点击率约 `2.44%`。
  - `推断`：项目最有价值的不是“暗黑童话”题材，而是“多武器 Build + 自动战斗 + 大世界探索”这一前台体验是否足够新。
  - `建议`：适合进入验证型立项，但要优先验证多武器核心是否真的支撑主循环，而不是只支撑素材。
- 最大机会点：
  - `事实`：ARPG赛道头部集中度较低，存在凭玩法创新或题材差异实现突破的案例。
  - `推断`：多武器玩法如果做透，确实有机会在小游戏ARPG里形成较强辨识度。
- 最大风险点：
  - `事实`：项目同时叠加大世界探索、Build策略、轻MMO、多人玩法、跨服对抗，功能量较大。
  - `推断`：如果多武器差异、探索承接、多人体验三者没有形成闭环，项目会迅速退化成“功能很多但核心不够硬”的混合产品。

### 1.3 项目概览

- 目标赛道：`ARPG为主，轻MMO为辅`
- 目标平台：`小游戏优先`
- 目标用户：`MMO、卡牌/RPG用户为核心，次核心为社交驱动和怀旧用户，泛用户为轻度探索和外观用户`
- 核心玩法：`多武器自动战斗 + 元素Build + 区块式大世界探索`
- 核心题材：`暗黑童话`
- 核心破局点：`多武器 Build 的独特战斗体验`

## 2. 找准切入点

### 2.1 市场时机判断

- 目标品类当前阶段：`红海期中的结构性机会`
- 判断结论：
  - `事实`：原企划判断 2025 年小游戏整体流水约 `745亿`，仍处于高速增长期。
  - `事实`：RPG整体赛道 2025 年预估约 `84亿`，ARPG 约 `36.3亿`、同比增速 `16%`，MMORPG 约 `22亿`、同比增速 `47%`。
  - `推断`：赛道并不轻松，但 ARPG 和 MMO 都还有新品突破空间，尤其当产品有明确差异点时。
- 判断依据：
  - `B级证据`：原企划的赛道规模、增速、竞品与投放成本整理。
  - `C级证据`：团队对题材与玩法趋势的综合判断。
- 当前平台是否存在阶段性流量红利：
  - `推断`：存在局部红利，尤其在玩法差异化 ARPG 和新题材 MMO 方向。
- 是否存在市场空白：
  - `推断`：存在“多武器大世界探索ARPG”这一局部空白，但不属于无人区。

### 2.2 用户判断

- 目标用户是谁：`想要战斗爽感、Build深度和探索承接的RPG/MMO用户`
- 年龄段：`16-45岁`
- 性别倾向：`男性偏高，但次核心和泛用户存在双性别扩展空间`
- 游戏偏好：`MMO、ARPG、卡牌RPG、轻探索`
- 消费能力：`中高，尤其在技能、元素、皮肤、宠物、装备词条上有较深付费空间`
- 主要需求：`成就感 / 收集 / 对抗 / 探索 / 社交炫耀`
- 用户迁移动机：
  - `事实`：原企划认为 MMO 用户追求深度与操作，卡牌/RPG用户追求成长与收集，泛用户看重简单有趣。
  - `推断`：项目试图用“战斗爽感 + 探索承接 + 渐进式社交”把这三类人群接到同一产品里。

### 2.3 获客判断

- 预期获客成本区间：
  - `事实`：企划列举竞品注册/付费成本时，部分 RPG 品类成本较高，但素材端多武器方向点击率通过。
  - `推断`：如果多武器玩法卖相稳定，获客成本有机会控制在可验证区间；如果卖相不稳定，很容易落入普通 RPG 买量竞争。
- 是否适合走买量：`是`
- 题材是否具备低成本吸量潜力：
  - `事实`：暗黑童话被定义为传播度高、可做差异化的题材。
  - `推断`：题材有帮助，但不如“多武器”“巨大化武器”“元素反应”等玩法表达更关键。
- 画风是否适合当前投放环境：
  - `事实`：项目采用 Q版、轻暗黑、低饱和高明度、巨大化武器的 2D 手绘方向，并已用 DEMO 素材测出达标点击率。
  - `推断`：画风适配小游戏投放。

### 2.4 阶段结论

- 当前切入是否成立：`基本成立`
- 若成立，成立的前提条件是什么：
  - `建议`：多武器必须形成用户一眼能感知的差异，而不是 UI 层面的复杂配置。
  - `建议`：探索部分必须服务于战斗和成长，不能只是弱化版开放世界。
  - `建议`：轻MMO承接要渐进，不要过早抬高理解成本。
- 若不成立，主要问题是什么：
  - `推断`：如果多武器 Build 只停留在素材高光，而玩家实际战斗中感知不到明显差别，项目核心就会失效。

## 5. 吸引力验证

### 5.1 题材向素材验证

- 验证方式：`题材和画风方向素材测试`
- 素材方向：`暗黑童话、巨大化武器、Q版轻暗黑`
- 表现结果：
  - `事实`：当前美术方向接近《迷雾大陆》但更明亮，点击率整体达标。
  - `推断`：题材能承接点击，但不是决定性变量。
- 结论：`题材可用，需继续服务于玩法表现`

### 5.2 玩法向素材验证

- 验证方式：`多轮素材测试`
- 素材方向：`多武器（核心）、元素反应、武器巨大化、武器锻造`
- 表现结果：
  - `事实`：多武器核心概念平均点击率约 `2.44%-2.48%`，元素反应约 `2.68%`，均达到“及格~潜力”。
  - `推断`：多武器方向已经证明值得继续深挖，元素反应也有助攻价值。
- 结论：`玩法素材成立`

### 5.3 题材+玩法综合验证

- 验证方式：`DEMO节奏素材验证`
- 素材方向：`多武器战斗 + 暗黑童话 + 探索氛围`
- 表现结果：`通过，但波动较大`
- 结论：`值得继续测试，但需找出最稳定的吸量表达`

### 5.4 综合判断

- 哪类素材表现最好：`多武器核心概念与元素反应`
- 素材表现是否稳定：`中等，存在波动`
- 素材拓展空间是否足够大：`高`
- 是否值得继续投放验证：`是`
