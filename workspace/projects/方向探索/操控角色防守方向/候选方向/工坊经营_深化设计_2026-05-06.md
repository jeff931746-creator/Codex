# 工坊经营方向 — 深化设计

## 核心定位

**底层玩法**：获取材料 → 制作装备 → 战斗使用 → 获得新材料 → 制作更强装备

**独立玩法验证**：制作类游戏本身是独立品类（我的世界、泰拉瑞亚的制作系统）

**与操控防守结合**：我制作的装备在战斗中使用，装备效果直接体现

## 两个子方向对比

| 维度 | 方向A：局外制作 | 方向B：局外建造 |
|---|---|---|
| 核心体验 | "我打造的装备" | "我建造的设施" |
| 制作对象 | 武器、防具、消耗品 | 防御设施、生产设施 |
| 制作深度 | 配方、材料、属性 | 布局、升级、协同 |
| 战斗应用 | 装备直接使用 | 设施提供支援 |
| 成果体现 | 装备效果可见 | 设施效果可见 |
| 参考对标 | 泰拉瑞亚、怪物猎人 | Thronefall、RimWorld |
| 与Thronefall区别 | 制作装备不是建造防线 | 建造功能设施不是防御布局 |

## 方向A：局外制作（装备工坊模式）

### 核心循环

```
材料获取阶段
├─ 战斗掉落材料
├─ 探索采集材料
└─ 分解装备获得材料
    ↓
制作阶段（局外）
├─ 查看配方
├─ 选择制作目标
├─ 投入材料制作
└─ 选择属性倾向
    ↓
装备强化阶段
├─ 升级装备等级
├─ 附魔添加属性
├─ 改造调整效果
└─ 镶嵌宝石增强
    ↓
战斗阶段（局内）
├─ 装备武器防具
├─ 使用消耗品
├─ 装备效果直接体现
└─ 感受制作成果
    ↓
战斗结束
├─ 获得新材料
├─ 解锁新配方
├─ 获得强化材料
└─ 装备耐久消耗
    ↓
回到材料获取
```

### 制作系统（核心玩法）

**设计原则**：制作本身必须有策略深度，不是"材料够了→点击制作→获得装备"

**制作流程**：

1. **配方解锁**
   - 战斗掉落配方
   - 完成任务解锁
   - 达到等级自动解锁
   - 特殊事件获得

2. **材料准备**
   - 基础材料：常见掉落（铁矿、木材、皮革）
   - 稀有材料：特定敌人掉落（龙鳞、魔核）
   - 特殊材料：特定条件获得（满月之夜、特定地点）

3. **制作选择**
   - 选择配方
   - 投入材料
   - 选择属性倾向（力量/敏捷/智力）
   - 选择特殊效果（火焰/冰霜/雷电）

4. **制作结果**
   - 基础属性：根据配方固定
   - 随机属性：根据材料品质随机
   - 特殊效果：根据选择确定
   - 制作品质：影响属性数值

**制作深度来源**：

1. **材料组合**
   - 同一配方可以用不同材料
   - 不同材料影响装备属性倾向
   - 稀有材料提升装备品质

2. **属性倾向**
   - 制作时选择强化哪个属性
   - 不同倾向适合不同战斗风格
   - 倾向影响随机属性范围

3. **特殊效果**
   - 添加元素效果（火焰/冰霜/雷电/毒素）
   - 添加特殊能力（吸血/反伤/加速）
   - 效果需要特殊材料

### 装备类型

**武器类**：

1. **近战武器**
   - 剑：平衡型，攻速中等
   - 斧：高伤害，攻速慢
   - 匕首：低伤害，攻速快
   - 长枪：范围攻击，攻速慢

2. **远程武器**
   - 弓：高伤害，需要瞄准
   - 弩：中伤害，自动瞄准
   - 法杖：魔法伤害，范围攻击
   - 投掷：消耗品，高爆发

3. **特殊武器**
   - 双持：两把武器，攻速快
   - 双手：单把武器，伤害高
   - 盾牌：防御型，可以格挡

**防具类**：

1. **头盔**：提升防御和特殊抗性
2. **胸甲**：提升生命值和防御
3. **护腿**：提升移动速度和防御
4. **手套**：提升攻击速度和暴击
5. **靴子**：提升移动速度和闪避

**消耗品类**：

1. **药水**：恢复生命/魔法
2. **炸弹**：范围伤害
3. **陷阱**：放置后触发
4. **增益药剂**：临时提升属性

### 强化系统

**升级强化**：

1. **等级提升**
   - 投入材料提升装备等级
   - 等级提升基础属性
   - 等级上限受装备品质限制

2. **品质提升**
   - 投入稀有材料提升品质
   - 品质提升属性上限
   - 品质：普通→稀有→史诗→传说

**附魔系统**：

1. **属性附魔**
   - 添加额外属性（力量+10、暴击+5%）
   - 每件装备有附魔槽位限制
   - 附魔可以替换但会消耗材料

2. **效果附魔**
   - 添加特殊效果（火焰伤害、生命偷取）
   - 效果附魔需要特殊材料
   - 效果可以叠加但有上限

**改造系统**：

1. **属性调整**
   - 重新分配装备属性点
   - 调整属性倾向
   - 改造需要消耗材料

2. **外观定制**
   - 改变装备外观
   - 不影响属性
   - 满足个性化需求

**镶嵌系统**：

1. **宝石镶嵌**
   - 装备有宝石槽位
   - 镶嵌宝石提升属性
   - 宝石可以拆卸但会损坏

2. **宝石类型**
   - 红宝石：提升攻击
   - 蓝宝石：提升防御
   - 绿宝石：提升生命
   - 黄宝石：提升速度

### 材料系统

**材料分类**：

1. **基础材料**（常见）
   - 铁矿：制作武器防具
   - 木材：制作弓箭陷阱
   - 皮革：制作轻甲
   - 布料：制作法袍

2. **稀有材料**（较难获得）
   - 精金：制作高级武器
   - 秘银：制作高级防具
   - 龙鳞：制作传说装备
   - 魔核：制作魔法装备

3. **特殊材料**（特定条件）
   - 火焰精华：添加火焰效果
   - 冰霜结晶：添加冰霜效果
   - 雷电精华：添加雷电效果
   - 暗影碎片：添加暗影效果

**材料获取**：

1. **战斗掉落**：击败敌人掉落
2. **探索采集**：地图上采集资源点
3. **分解装备**：分解不需要的装备
4. **商店购买**：用金币购买基础材料
5. **任务奖励**：完成任务获得稀有材料

### 配方系统

**配方解锁**：

1. **等级解锁**：达到特定等级自动解锁
2. **掉落解锁**：击败特定敌人掉落配方
3. **任务解锁**：完成任务获得配方
4. **探索解锁**：发现隐藏配方

**配方类型**：

1. **基础配方**：制作基础装备
2. **进阶配方**：制作高级装备
3. **特殊配方**：制作独特装备
4. **传说配方**：制作传说装备

**配方深度**：

1. **材料替代**：同一配方可以用不同材料
2. **属性变化**：不同材料影响装备属性
3. **效果添加**：特殊材料添加特殊效果

### 战斗应用

**装备效果体现**：

1. **武器效果**
   - 伤害数值直接体现
   - 攻击速度明显感知
   - 特殊效果视觉反馈（火焰、冰霜）

2. **防具效果**
   - 受到伤害减少
   - 移动速度提升
   - 特殊抗性生效

3. **消耗品效果**
   - 药水即时恢复
   - 炸弹范围爆炸
   - 增益药剂临时提升

**装备耐久**：

1. **耐久消耗**：战斗中装备耐久降低
2. **修理系统**：消耗材料修理装备
3. **损坏惩罚**：耐久为0装备失效

### 长线驱动

1. **配方收集**：解锁所有配方
2. **完美装备**：制作满属性装备
3. **套装收集**：收集套装装备（套装效果）
4. **传说装备**：制作传说级装备

### 包装变体

同样的底层玩法，可以包装成：

1. **铁匠铺**：打造武器防具
2. **魔法工坊**：炼制魔法装备
3. **机械车间**：组装机械装备
4. **炼金实验室**：调配药水炸弹
5. **符文工坊**：铭刻符文装备

**包装差异**：
- 铁匠铺：锻造武器，淬火强化
- 魔法工坊：炼制法杖，附魔增强
- 机械车间：组装机甲，改装升级
- 炼金实验室：调配药水，实验配方
- 符文工坊：铭刻符文，激活效果

## 方向B：局外建造（设施建造模式）

### 核心循环

```
资源获取阶段
├─ 战斗获得资源
├─ 设施自动产出
└─ 任务奖励资源
    ↓
建造阶段（局外）
├─ 规划设施布局
├─ 建造功能设施
├─ 升级现有设施
└─ 设施协同效果
    ↓
设施产出阶段
├─ 生产设施产出资源
├─ 训练设施提升能力
├─ 研究设施解锁科技
└─ 防御设施提供支援
    ↓
战斗阶段（局内）
├─ 操控角色防守
├─ 设施提供支援
├─ 设施产出使用
└─ 保护关键设施
    ↓
战斗结束
├─ 获得建造资源
├─ 解锁新设施
├─ 设施可能受损
└─ 修复受损设施
    ↓
回到资源获取
```

### 建造系统（核心玩法）

**设计原则**：建造的是**功能设施**，不是**防御布局**（与Thronefall的核心区别）

**设施类型**：

1. **生产设施**
   - 资源矿场：产出基础资源
   - 材料工厂：产出制作材料
   - 食物农场：产出恢复道具
   - 弹药工厂：产出弹药补给

2. **训练设施**
   - 训练场：提升角色属性
   - 靶场：提升射击精度
   - 道场：解锁新技能
   - 图书馆：提升经验获取

3. **研究设施**
   - 研究所：解锁新科技
   - 实验室：研究新装备
   - 观测站：解锁新地图
   - 档案馆：解锁新配方

4. **防御设施**（辅助，不是主要防御）
   - 哨塔：提供视野和预警
   - 补给站：战斗中补给
   - 医疗站：战斗后恢复
   - 陷阱工坊：产出陷阱道具

**与Thronefall的核心区别**：

| 维度 | Thronefall | 方向B（设施建造） |
|---|---|---|
| 建造对象 | 防御布局（塔、墙、兵营） | 功能设施（生产、训练、研究） |
| 建筑作用 | 建筑本身就是防御力量 | 设施产出资源/能力 |
| 战斗中 | 建筑是主要防御 | 玩家操控角色是主要防御 |
| 玩家角色 | 操控角色是辅助 | 设施提供支援是辅助 |
| 体验承诺 | "我布置的防线挡住了敌人" | "我建造的基地支撑我守住了阵地" |

### 建造机制

**建造流程**：

1. **选择位置**
   - 基地有限的建造空间
   - 不同设施有不同空间需求
   - 设施之间需要保持距离

2. **投入资源**
   - 消耗资源建造设施
   - 建造时间（可以加速）
   - 建造完成后立即生效

3. **设施升级**
   - 投入资源升级设施等级
   - 升级提升产出效率
   - 升级解锁新功能

4. **设施协同**
   - 相邻设施产生协同效果
   - 特定组合解锁特殊效果
   - 布局规划影响效率

### 设施详细设计

**生产设施示例：资源矿场**

- **功能**：每小时产出铁矿
- **升级效果**：
  - Lv1：每小时10铁矿
  - Lv2：每小时20铁矿
  - Lv3：每小时30铁矿，解锁精金产出
- **协同效果**：相邻材料工厂时，产出+20%
- **战斗应用**：产出的铁矿用于制作装备

**训练设施示例：训练场**

- **功能**：提升角色攻击力
- **升级效果**：
  - Lv1：攻击力+5
  - Lv2：攻击力+10
  - Lv3：攻击力+15，解锁暴击训练
- **协同效果**：相邻靶场时，暴击率+5%
- **战斗应用**：攻击力提升直接体现在战斗中

**研究设施示例：研究所**

- **功能**：解锁新科技
- **升级效果**：
  - Lv1：解锁基础科技
  - Lv2：解锁进阶科技
  - Lv3：解锁传说科技
- **协同效果**：相邻实验室时，研究速度+30%
- **战斗应用**：科技提供永久增益

**防御设施示例：补给站**

- **功能**：战斗中提供补给
- **升级效果**：
  - Lv1：提供基础补给（弹药、药水）
  - Lv2：提供高级补给（增益药剂）
  - Lv3：自动补给（靠近自动恢复）
- **协同效果**：相邻医疗站时，恢复效果+50%
- **战斗应用**：战斗中靠近补给站恢复资源

### 布局规划

**空间限制**：

1. **建造格子**：基地有固定的建造格子
2. **设施大小**：不同设施占用不同格子数
3. **扩展系统**：完成任务解锁更多格子

**布局策略**：

1. **生产区**：集中建造生产设施，提升产出
2. **训练区**：集中建造训练设施，提升能力
3. **防御区**：在战斗路径上建造防御设施
4. **混合布局**：平衡各类设施

**协同效果**：

1. **相邻加成**：相同类型设施相邻提升效果
2. **组合加成**：特定组合解锁特殊效果
3. **距离惩罚**：设施距离太远效果降低

### 战斗应用

**设施在战斗中的作用**：

1. **产出使用**
   - 生产设施产出的资源用于制作装备
   - 训练设施提升的能力直接体现
   - 研究设施解锁的科技永久生效

2. **战斗支援**
   - 补给站提供补给
   - 医疗站提供恢复
   - 哨塔提供视野

3. **设施保护**
   - 敌人可能攻击设施
   - 设施受损影响产出
   - 战斗后需要修复

### 长线驱动

1. **解锁所有设施**：解锁所有设施类型
2. **完美布局**：优化设施布局效率
3. **满级设施**：所有设施升级到满级
4. **科技树完成**：解锁所有科技

### 包装变体

同样的底层玩法，可以包装成：

1. **避难所**：建造避难所设施，生存废土
2. **太空站**：建造太空站模块，探索宇宙
3. **地下基地**：建造地下设施，抵御入侵
4. **移动堡垒**：建造堡垒模块，征战四方
5. **魔法塔**：建造魔法设施，研究魔法

## 两个方向的选择建议

**选择方向A（装备工坊）如果**：
- 想要直接的成就感（我打造的装备）
- 愿意投入装备系统开发
- 目标受众喜欢刷装备
- 有能力做好制作和强化系统

**选择方向B（设施建造）如果**：
- 想要策略规划体验
- 愿意投入建造系统开发
- 目标受众喜欢经营建造
- 有能力做好布局和协同系统

**核心区别**：
- 方向A：制作装备，战斗中直接使用
- 方向B：建造设施，设施产出支撑战斗
