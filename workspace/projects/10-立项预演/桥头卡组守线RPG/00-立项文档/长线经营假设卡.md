# 长线经营假设卡

项目名：桥头卡组守线RPG  
目标市场：微信小游戏 / 轻中度移动端  
主要变现方式：内购为主，广告为辅

## 1. 长线目标定义

- 长线成功的定义：
  - 玩家能在单局成长和跨局积累中找到持续动力
  - 内容供给能支撑至少3-6个月的自然更新节奏
  - 买量素材能从核心玩法中持续提取新角度
  
- 关键观察周期：
  - 短期：D1-D7（单局爽点和初期成长是否成立）
  - 中期：D7-D30（跨局积累和章节推进是否有粘性）
  - 长期：D30+（内容供给和创意刷新是否可持续）
  
- 目标区间：
  - D7留存：30%-40%
  - D30留存：10%-15%
  - 长线LTV：中等偏上（具体数值待小测验证）

## 2. 玩家体验三层

### 第一层：积累

- 局内积累：
  - 费用积累和单位部署节奏
  - 局内三选一强化路径（轻Roguelite）
  - 阵容成型感（前排+后排+法术的配合）
  - 波次推进进度和Boss击杀
  
- 局外积累：
  - 单位卡解锁和升级
  - 法术卡解锁和升级
  - 章节推进和新关卡解锁
  - 强化件/神器收集
  
- 可支撑局数：
  - 单一卡组配置：约10-15局
  - 单一章节内容：约20-30局
  - 全卡组+全章节：约100-150局（需配合内容更新）

### 第二层：兑现

- 上线动因：
  - 尝试新解锁的单位卡或法术卡
  - 推进下一个章节或挑战Boss关卡
  - 验证新的强化路径组合
  - 完成每日/每周任务奖励
  
- 下线时惦记什么：
  - 差一点就能通过的Boss关卡
  - 即将解锁的新单位卡
  - 想尝试但还没验证的强化路径组合
  - 下一个章节的新防守面和新怪潮
  
- 付费在放大什么：
  - 加速解锁更多单位卡，尝试更多阵容组合
  - 获得稀有强化件，体验更强的单局成长曲线
  - 快速推进章节，看到更多样的防守面和Boss设计
  - 不是在解除卡点，而是在加速"尝试新组合"的乐趣

### 第三层：刷新

- 无更新情况下自然支撑时长：
  - 约1-2个月（基于初始卡组数量和章节内容）
  - 详细内容见第4节内容与运营引擎

## 3. 重复付费链

- 首付理由：
  - 加速解锁第一批核心单位卡（如强力前排或高输出后排）
  - 购买首个稀有强化件礼包
  - 解锁下一章节内容
  
- 重复付费理由：
  - 持续解锁新单位卡，尝试新的阵容组合
  - 购买限时强化件或神器
  - 解锁新章节和新挑战模式
  - 购买皮肤或视觉定制（如有）
  
- 高付费追求：
  - 收集全卡组，尝试所有可能的阵容组合
  - 获得所有稀有强化件，体验极限单局成长
  - 快速推进到最高难度章节
  - 挑战排行榜或限时活动（如有）
  
- 非付费玩家仍会留下的原因：
  - 核心玩法（卡牌部署守线）本身有足够乐趣
  - 免费也能逐步解锁单位卡和章节
  - 局内强化的随机性提供足够变化
  - 每日任务和免费奖励能维持基本进度

## 4. 内容与运营引擎

- 最小内容单元：
  - 单个单位卡（含技能和数值）
  - 单个法术卡
  - 单个强化件/神器
  - 单个关卡（防守面+波次配置+Boss）
  
- 每周供给：
  - 1-2个新单位卡或法术卡
  - 1个限时强化件或活动
  - 每日任务和奖励刷新
  
- 每月供给：
  - 1个新章节（包含5-8个关卡）
  - 3-5个新单位卡
  - 2-3个新法术卡
  - 1-2个新Boss设计
  - 1个限时挑战模式或活动
  
- 最高成本项：
  - 新章节的美术资源（防守面、怪物、Boss）
  - 新单位卡的技能设计和平衡调整
  - 新Boss的机制设计和测试
  
- 供给风险：
  - 如果美术资源成本过高，章节更新速度可能跟不上玩家消耗
  - 如果单位卡平衡调整不当，可能影响核心体验
  - 如果Boss设计过于复杂，可能提高制作门槛

## 5. 创意持续性

- 当前素材角度：
  - 桥头最后防线 + 怪潮压境
  - 卡牌部署守军 + 自动战斗
  - 波次推进 + Boss击杀
  - 阵容成型 + 单局成长
  
- 可反哺新素材的更新：
  - 新章节的新防守面（如雪山关卡、沙漠关卡）
  - 新单位卡的新兵种（如飞行单位、重装单位）
  - 新Boss的新机制（如分裂、召唤、狂暴）
  - 新强化件的新效果（如连锁反应、范围爆发）
  
- 创意疲劳风险：
  - 如果只强调"守桥头"，可能在3-6个月后出现视觉疲劳
  - 如果单位卡种类不够多，阵容组合会快速穷尽
  - 如果Boss机制过于相似，挑战感会下降
  
- 刷新方案：
  - 引入新的防守面类型（如多路径、移动防守面）
  - 引入新的单位卡机制（如合体、变形、召唤）
  - 引入新的玩法模式（如无尽模式、限时挑战）
  - 引入新的视觉主题（如节日皮肤、特殊章节）

## 6. 后续验证节点

- 原型阶段：
  - 验证卡牌部署是否真有前台新鲜感
  - 验证单位存在感是否足够强
  - 验证局内强化是否能支撑单局成长
  
- 小测阶段：
  - 验证D1-D7留存是否达到目标区间
  - 验证首付率和首付金额是否合理
  - 验证玩家是否理解核心玩法
  - 验证买量素材的转化效率
  
- 扩量阶段：
  - 验证D7-D30留存是否稳定
  - 验证重复付费是否发生
  - 验证内容消耗速度是否可控
  - 验证买量ROI是否可持续
  
- 上线后：
  - 验证D30+留存是否达到目标
  - 验证长线LTV是否符合预期
  - 验证内容供给是否能跟上玩家消耗
  - 验证创意刷新是否有效

## 7. 止损条件

- 条件1：D7留存低于20%
  - 说明核心玩法或单局成长不成立
  - 降级方案：简化为纯守线产品，去掉卡牌部署
  - 或直接终止
  
- 条件2：首付率低于5%且玩家反馈"不知道为什么要付费"
  - 说明付费点没有挂回核心玩法欲望
  - 降级方案：重新设计付费点，或转为纯广告变现
  
- 条件3：玩家在小测中普遍反馈"这就是轻量CR"
  - 说明前台差异不成立
  - 降级方案：强化守线和波次要素，弱化卡牌对战感
  - 或直接终止
  
- 条件4：内容消耗速度远超供给能力（玩家在1周内耗尽1个月内容）
  - 说明内容供给引擎不成立
  - 降级方案：增加数值成长层，拉长单局时长
  - 或调整内容复用策略
