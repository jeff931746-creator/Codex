# 前 10 分钟核心循环设计

## 一、CM 底层逻辑提炼

从 CM 文档中提取的**可复用设计原则**：

1. **纯粹的资源流转循环**：体力 → Slot → 金币 → 建岛 → 体力，形成闭环
2. **数值成长驱动乐趣**：通过数值膨胀让玩家感受进度
3. **核心循环必须能自洽运转**：不依赖外围系统也能玩下去
4. **用"反比控制"提供新鲜感**：机台反比让每次体验不同，达到复玩目的
5. **外围系统解决核心循环的运转问题**：如攻防解决"不愿建岛"，卡牌解决"Slot 疲劳"

## 二、本游戏的核心循环框架

### 玩法特性
- **多生物收集养成**：类似宝可梦/数码宝贝，生物是核心资产
- **操控角色防守**：玩家直接参与战斗，有操作乐趣

### 核心资源定义
- **主资源**：生物碎片/蛋（用于孵化/升级生物）
- **次资源**：能量/体力（用于触发战斗/孵化）

### 核心循环（3 分钟一轮）

```
能量 → 防守战斗（操作） → 生物碎片 → 孵化/升级生物 → 解锁新区域/提升战力 → 获得能量
```

**循环节点拆解**：

1. **消耗能量触发防守战**
   - 玩家操控角色防守 3 波敌人（30-60 秒）
   - 操作体验：移动、攻击、技能释放
   - 反馈：击败敌人掉落碎片，视觉爽快

2. **获得生物碎片**
   - 每场战斗掉落 1-3 种生物碎片
   - 碎片类型由"当前区域池"决定（反比机制的载体）

3. **孵化/升级生物**
   - 碎片积累到阈值 → 孵化新生物（首次获得有演出）
   - 重复碎片 → 升级已有生物（数值成长）

4. **解锁新区域/提升战力**
   - 生物达到一定数量/等级 → 解锁新区域
   - 新区域 = 新敌人 + 新碎片池 + 更高收益

5. **获得能量回流**
   - 升级生物时返还部分能量
   - 解锁新区域时奖励能量
   - 形成正循环

## 三、前 10 分钟体验设计

### 目标
- 让玩家完成 **3 轮完整循环**
- 体验到"战斗 → 收集 → 成长 → 解锁"的完整流程
- 建立"想要更多生物"的驱动力

### 时间线设计

**0-2 分钟：新手引导 + 第一轮循环**
- 赠送初始能量（10 点）
- 引导第一场防守战（简化敌人，必胜）
- 获得 3 种碎片各 5 个
- 引导孵化第一只生物（即时孵化，有演出）
- 生物自动加入队伍，战力提升
- 解锁第二区域，奖励 10 能量

**2-5 分钟：第二轮循环**
- 玩家主动选择区域 1 或区域 2 进行防守
- 区域 2 敌人更强，但碎片掉落更多
- 获得新碎片 + 重复碎片
- 孵化第二只生物 OR 升级第一只生物
- 战力提升，解锁区域 3

**5-10 分钟：第三轮循环 + 引入"反比"**
- 玩家发现：区域 1/2/3 的碎片池不同
- 想要特定生物 → 需要选择对应区域刷
- 但每个区域的碎片掉落有"反比"：
  - 你缺的碎片掉率低
  - 你多的碎片掉率高
  - 制造"差一点就能孵化"的紧张感
- 能量开始紧张，引导玩家关注能量获取途径

**10 分钟节点状态**
- 拥有 3-5 只生物
- 解锁了 3 个区域
- 能量耗尽，自然引导到外围系统（能量恢复/付费/社交）

## 四、"反比"机制的应用

### CM 的反比：机台反比控制 Slot 产出
### 本游戏的反比：区域碎片池 + 掉落反比

**实现方式**：
- 每个区域有 5-8 种生物碎片
- 玩家背包中某种碎片越多，该碎片掉率越低
- 玩家背包中某种碎片越少（尤其是"差 1-2 个就能孵化"），该碎片掉率提升
- 但不是 100% 控制，保留随机性

**目的**：
- 让玩家"总是差一点"，产生"再刷一次就能孵化"的驱动
- 避免"一次刷够所有碎片"导致的目标缺失
- 制造"选择困境"：刷区域 1 还是区域 2？

## 五、核心循环的自洽性检验

### 问题 1：玩家不想孵化生物怎么办？
- **CM 的解法**：不建岛就无法获得体力，循环断裂
- **本游戏的解法**：
  - 不孵化生物 → 战力不提升 → 无法解锁新区域 → 碎片池单一 → 体验重复
  - 孵化生物时返还能量 → 不孵化就亏能量
  - 新生物有独特技能 → 影响战斗体验

### 问题 2：战斗疲劳怎么办？
- **CM 的解法**：Slot 是经典玩法，配合水池满足心流
- **本游戏的解法**：
  - 不同生物有不同技能 → 战斗策略变化
  - 不同区域敌人有不同机制 → 战斗体验变化
  - 战斗时长控制在 30-60 秒 → 不过度消耗注意力

### 问题 3：能量不够怎么办？
- **CM 的解法**：付费 + 攻防系统（抢别人的体力）
- **本游戏的解法**：
  - 自然恢复（5 分钟 1 点）
  - 升级生物返还能量
  - 解锁新区域奖励能量
  - 外围系统：每日任务、社交赠送、付费购买

## 六、与 CM 的差异点

| 维度 | CM | 本游戏 |
|------|----|----|
| 核心玩法 | Slot（纯数值） | 防守战斗（操作 + 数值） |
| 成长载体 | 建岛等级 | 生物收集 + 升级 |
| 反比载体 | 机台反比 | 区域碎片池 + 掉落反比 |
| 循环时长 | 1-2 分钟 | 3 分钟 |
| 操作参与度 | 低（点击） | 高（实时操作） |

## 七、后续扩展方向

基于这个核心循环，可以扩展：

1. **攻击系统**：抢夺其他玩家的生物碎片（解决能量/碎片短缺）
2. **生物图鉴**：收集驱动，稀有生物
3. **生物组合技**：特定生物组合解锁特殊技能
4. **赛季/活动**：限时区域，限时生物
5. **社交**：生物交换，协作防守

**但这些都是为了"让核心循环更好地运转"，而不是核心循环本身。**
