# 牧场收集闭环演示实现说明

## 目标

这个 Godot 演示用于验证第一段可玩闭环：

`能量 -> 防守战斗 -> 碎片 -> 孵化/升级 -> 战力提升 -> 区域解锁 -> 能量回流`

当前版本采用强制阶段推进，目的是让玩家不看说明也能体验到“战斗、收集、孵化、变强、解锁、再战”的完整路径。

## 运行方式

1. 用 Godot 4.x 打开本目录。
2. 运行主场景 `res://scenes/Main.tscn`。
3. 开局点击 `防守草地牧门`。
4. 战斗中使用方向键移动，按空格或鼠标左键攻击。
5. 第一场胜利后回到牧场，查看叶犬碎片进度。
6. 点击 `孵化叶犬`，观察伙伴加入、战力提升和果园围栏解锁。
7. 进入第二场战斗，观察叶犬在玩家身边自动支援。

## 评审重点

- 开局是否能在 5 秒内知道下一步要做什么。
- 第一场胜利后是否能明确看到碎片从 `0/5` 到 `5/5`。
- 孵化后是否能看到新伙伴加入和 `战力 2 -> 5`。
- 果园围栏解锁是否形成下一目标。
- 第二场战斗中叶犬支援是否足够明显。

## 当前限制

- 使用占位图形和文字反馈，没有正式美术和音效。
- 没有存档、付费、社交、任务和长期进化树。
- 随机掉落和反比权重暂时退到自由循环阶段，第一轮闭环优先保证体验成立。
