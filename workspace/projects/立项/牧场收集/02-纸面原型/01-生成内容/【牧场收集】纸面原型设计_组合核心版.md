# 【牧场收集】纸面原型设计_组合核心版

## 验证目标

**核心问题：** 玩家是否会为了"发现和验证强力组合"而主动收集生物？

**核心玩法：** 实验不同生物组合 → 战斗验证效果 → 发现强力组合 → 收集更多生物凑新组合

**验证范围：** 6只生物，10种组合效果，3场战斗，15分钟验证

**成功标准：** 
- 玩家主动尝试不同组合
- 玩家发现强力组合后表现出兴奋
- 玩家为了凑齐想要的组合而继续收集

## 一、组合系统设计

### 组合的核心价值

**组合 ≠ 简单加成**

错误示例：兔子+小鹿 = 移速+20%（这只是数值加成，不是组合）

正确示例：兔子+小鹿 = 解锁"闪电突袭"技能（这是1+1>2的组合效果）

### 6只生物设计

| 生物 | 单体效果 | 孵化需求 |
|------|---------|---------|
| 🐰兔子 | 快速移动，攻击力低 | 3个碎片 |
| 🦌小鹿 | 冲撞攻击，单体伤害高 | 3个碎片 |
| 🐑羊 | 治疗，恢复HP | 3个碎片 |
| 🐻熊 | 高血量，攻击慢 | 3个碎片 |
| 🦅鹰 | 飞行，攻击远程敌人 | 3个碎片 |
| 🐺狼 | 群体攻击，伤害中等 | 3个碎片 |

**设计意图：**
- 每只生物单独使用都有明确定位
- 孵化阈值统一为3个，加快节奏

### 10种组合效果

| 组合 | 组合效果 | 效果说明 | 强度 |
|------|---------|---------|------|
| 兔子+小鹿 | 闪电突袭 | 小鹿冲撞后，兔子立即追击，造成双倍伤害 | ★★★ |
| 羊+熊 | 铁壁防线 | 熊血量+50%，羊治疗效果+50% | ★★★ |
| 鹰+狼 | 天地夹击 | 鹰从空中标记敌人，狼群体攻击被标记敌人伤害+100% | ★★★★ |
| 兔子+鹰 | 侦察小队 | 提前5秒预警敌人位置，兔子移速+30% | ★★ |
| 小鹿+熊 | 重装冲锋 | 小鹿冲撞带动熊一起冲锋，造成范围伤害 | ★★★ |
| 羊+狼 | 野性治疗 | 狼攻击时为羊充能，羊治疗量+30% | ★★ |
| 兔子+羊 | 快速救援 | 兔子受伤时，羊立即传送到兔子身边治疗 | ★★ |
| 小鹿+鹰 | 精准打击 | 鹰标记敌人弱点，小鹿冲撞必定暴击 | ★★★ |
| 熊+鹰 | 空投重击 | 鹰抓起熊空投到敌人群中，造成范围眩晕 | ★★★★ |
| 狼+兔子 | 狼群战术 | 兔子引诱敌人，狼从侧面包抄，伤害+50% | ★★★ |

**设计意图：**
- 组合效果不是简单数值加成，而是新的战术
- 强度分为2-4星，让玩家有"发现强力组合"的惊喜
- 每只生物都有多个组合可能，鼓励实验

### 三生物组合（隐藏彩蛋）

| 组合 | 组合效果 | 效果说明 | 强度 |
|------|---------|---------|------|
| 兔子+小鹿+鹰 | 三位一体突袭 | 鹰标记→小鹿冲撞→兔子收割，连续技能链 | ★★★★★ |
| 羊+熊+狼 | 野兽军团 | 羊持续治疗，熊吸引火力，狼输出，完美配合 | ★★★★★ |

**设计意图：**
- 三生物组合是隐藏彩蛋，玩家需要自己发现
- 效果远超二生物组合，给玩家"发现秘密"的成就感

## 二、战斗系统（组合验证场）

### 战斗不是核心，是组合的验证场

**战斗目的：**
- 让玩家看到组合效果的实际表现
- 不同敌人配置需要不同组合应对

### 3场战斗设计

**战斗1：单体强敌**
- 敌人：1只红色BOSS，血量500，伤害高
- 最优组合：小鹿+鹰（精准打击），或兔子+小鹿（闪电突袭）
- 其他组合：可以通关，但用时更长或HP损失更多

**战斗2：群体小怪**
- 敌人：10只绿色小怪，血量50，数量多
- 最优组合：鹰+狼（天地夹击），或狼+兔子（狼群战术）
- 其他组合：可以通关，但容易被围攻

**战斗3：混合战场**
- 敌人：2只蓝色中型敌人+5只绿色小怪
- 最优组合：羊+熊+狼（野兽军团），或兔子+小鹿+鹰（三位一体突袭）
- 其他组合：需要更高的操作技巧

**战斗结算（固定结果卡）：**

| 组合 | 战斗1结果 | 战斗2结果 | 战斗3结果 |
|------|----------|----------|----------|
| 无组合（单只生物） | 失败 | 失败 | 失败 |
| 弱组合（★★） | 胜利，HP剩余30% | 胜利，HP剩余40% | 失败 |
| 中等组合（★★★） | 胜利，HP剩余60% | 胜利，HP剩余70% | 胜利，HP剩余30% |
| 强力组合（★★★★） | 胜利，HP剩余90% | 胜利，HP剩余90% | 胜利，HP剩余60% |
| 最强组合（★★★★★） | 完美胜利 | 完美胜利 | 胜利，HP剩余90% |

**设计意图：**
- 战斗结果直接反映组合强度
- 玩家能明显感受到"找到正确组合"的差异
- 不需要实时操作，只需要选择组合

## 三、收集系统（为组合服务）

### 收集的目的：凑齐想要的组合

**掉落规则（简化版）：**
- 每场战斗掉落2个碎片
- 掉落池：当前未孵化的生物
- 如果所有生物都已孵化，随机掉落（用于升级，本轮不验证）

**孵化规则：**
- 每只生物需要3个碎片
- 孵化后立即可用

**示例路径：**

| 局数 | 掉落 | 已孵化生物 | 玩家可尝试的组合 |
|------|------|-----------|----------------|
| 1 | 兔2/鹿1 | 无 | 无法战斗 |
| 2 | 兔1/羊1 | 兔子 | 只能单只 |
| 3 | 鹿2 | 兔子、小鹿 | 兔子+小鹿（闪电突袭★★★） |
| 4 | 羊2 | 兔子、小鹿、羊 | 可尝试兔子+羊、小鹿+羊 |
| 5 | 熊2/狼1 | 兔子、小鹿、羊、熊 | 可尝试羊+熊（铁壁防线★★★） |
| 6 | 熊1/狼2 | 兔子、小鹿、羊、熊、狼 | 可尝试羊+熊+狼（野兽军团★★★★★） |

**设计意图：**
- 6局内孵化5只生物，节奏快
- 每孵化一只新生物，玩家都能尝试新组合
- 玩家会主动思考"我想要哪只生物来凑XX组合"

## 四、15分钟纸面测试流程

### 测试材料

**需要准备：**
1. 生物卡（6张）：正面是蛋+碎片进度（0/3到3/3），背面是生物+单体效果
2. 组合效果卡（10张）：列出所有二生物组合效果
3. 隐藏组合卡（2张）：三生物组合效果，初始不展示
4. 战斗结果卡（3张）：每场战斗的结果表
5. 碎片代币（20个）：用于标记进度

### 测试流程

**阶段1：引导（0-3分钟）**

测试员：
1. "你是牧场守护者，收集生物组成队伍战斗"
2. 展示6张生物卡（蛋状态）："每只生物需要3个碎片孵化"
3. 展示组合效果卡："不同生物组合会产生特殊效果"
4. "你的目标是找到最强的组合，通关3场战斗"

**阶段2：第一轮收集+孵化（3-6分钟）**

测试员：
1. "第一场战斗，获得兔子碎片x2，小鹿碎片x1"
2. 玩家标记进度：兔子2/3，小鹿1/3
3. "第二场战斗，获得兔子碎片x1，羊碎片x1"
4. 玩家孵化兔子（3/3），标记羊1/3
5. 测试员翻转兔子卡，展示单体效果："兔子-快速移动，攻击力低"
6. **观察：玩家是否查看组合效果卡，寻找兔子的组合？**

**阶段3：第一次组合实验（6-9分钟）**

测试员：
1. "第三场战斗，获得小鹿碎片x2"
2. 玩家孵化小鹿（3/3）
3. 测试员："你现在有兔子和小鹿，可以尝试组合"
4. **等待玩家主动查看组合效果卡**
5. 玩家发现："兔子+小鹿 = 闪电突袭（★★★）"
6. 测试员："你想用这个组合挑战战斗1吗？"
7. 玩家选择组合，测试员翻开战斗1结果卡："胜利，HP剩余60%"
8. **观察：玩家是否表现出"找到正确组合"的兴奋？**

**阶段4：组合探索（9-12分钟）**

测试员：
1. "第四场战斗，获得羊碎片x2"
2. 玩家孵化羊（3/3）
3. **不主动提示，等待玩家自己查看组合效果卡**
4. 玩家发现：兔子+羊（★★）、小鹿+羊（未列出，无组合）
5. 测试员："你想尝试哪个组合？"
6. **观察：玩家是否会尝试不同组合，对比效果？**

**阶段5：追求强力组合（12-15分钟）**

测试员：
1. "第五场战斗，获得熊碎片x2，狼碎片x1"
2. 玩家孵化熊（3/3），狼1/3
3. 玩家查看组合效果卡，发现：羊+熊（铁壁防线★★★）
4. 测试员："你想用羊+熊挑战战斗2吗？"
5. 玩家尝试，测试员："胜利，HP剩余70%"
6. "第六场战斗，获得狼碎片x2"
7. 玩家孵化狼（3/3）
8. 玩家查看组合效果卡，发现：羊+熊+狼（？？？）
9. 测试员拿出隐藏组合卡："野兽军团（★★★★★）-羊持续治疗，熊吸引火力，狼输出"
10. **观察：玩家是否表现出"发现隐藏组合"的惊喜？**
11. 玩家用野兽军团挑战战斗3，测试员："完美胜利！"

**阶段6：询问（15分钟）**

测试员询问：
1. "你在什么时候最想继续收集生物？"
2. "你发现强力组合时的感受是什么？"
3. "你会为了凑齐某个组合而继续玩吗？"
4. "如果继续玩，你下一步想凑哪个组合？"
5. "你觉得'实验组合'这个玩法有意思吗？"

## 五、验证清单

### 核心假设验证

- [ ] 玩家孵化新生物后，会主动查看组合效果卡
- [ ] 玩家发现强力组合时，表现出兴奋（语言/表情/行为）
- [ ] 玩家会主动尝试不同组合，对比效果
- [ ] 玩家会为了凑齐想要的组合而继续收集
- [ ] 玩家发现隐藏组合时，表现出惊喜

### 体验验证

- [ ] 玩家能在15分钟内孵化4-5只生物
- [ ] 玩家能尝试3-5种不同组合
- [ ] 玩家能明确说出"我想凑XX组合"
- [ ] 玩家在15分钟结束时仍想继续玩

### 失败信号

- [ ] 玩家孵化新生物后，不查看组合效果卡
- [ ] 玩家发现强力组合时，没有明显反应
- [ ] 玩家只用一个组合，不尝试其他
- [ ] 玩家说"组合效果看不出差别"
- [ ] 玩家在10分钟后失去兴趣

## 六、测试后决策标准

### 强通过（进入开发Demo）

**3-5人测试中，满足以下条件：**
- 至少80%的玩家主动查看组合效果卡
- 至少80%的玩家发现强力组合时表现出兴奋
- 至少60%的玩家主动尝试3种以上组合
- 至少60%的玩家明确说出"我想凑XX组合"
- 至少60%的玩家在15分钟后仍想继续玩

**决策：进入Godot Demo开发，实现组合战斗系统**

### 弱通过（修改后再测）

**3-5人测试中，满足以下条件：**
- 至少60%的玩家主动查看组合效果卡
- 至少60%的玩家发现强力组合时有反应
- 至少40%的玩家尝试2种以上组合

**决策：修改组合设计，调整组合效果的表现力**

**可能的修改方向：**
- 组合效果更夸张（伤害x3而不是+50%）
- 增加组合的视觉描述（"鹰抓起熊空投"）
- 减少组合数量，聚焦最有趣的5个

### 不通过（退回重构）

**3-5人测试中，出现以下情况：**
- 少于60%的玩家主动查看组合效果卡
- 少于60%的玩家发现强力组合时有反应
- 超过50%的玩家在10分钟后失去兴趣

**决策：退回重构，重新思考核心玩法**

**需要分析的问题：**
1. 组合效果不够吸引人？
2. 战斗验证不够直观？
3. 收集节奏太慢？
4. "组合实验"这个方向本身不成立？

## 七、与之前版本的对比

| 维度 | 之前版本（v1-v3） | 当前版本（组合核心） |
|------|-----------------|-------------------|
| 核心玩法 | 收集碎片→孵化生物 | 实验组合→发现强力组合 |
| 验证重点 | "差1个"是否驱动再来一局 | 组合实验是否有乐趣 |
| 战斗作用 | 获取碎片 | 验证组合效果 |
| 牧场作用 | 展示生物 | 选择组合 |
| 生物价值 | 用时-3秒（感知弱） | 解锁新组合（目标明确） |
| 测试时长 | 10分钟 | 15分钟 |
| 孵化阈值 | 5个碎片 | 3个碎片（加快节奏） |
| 生物数量 | 3只 | 6只（更多组合可能） |
