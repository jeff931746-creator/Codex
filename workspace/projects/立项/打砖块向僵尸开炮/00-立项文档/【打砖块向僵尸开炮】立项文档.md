# 打砖块向僵尸开炮

整理时间：2026-04-14

## 一句话定义

玩家操控底线炮台，通过弹射炮弹击碎砖墙、改写弹道，并在僵尸压线前把整条战线打穿。

## 立项结论

这不是纯打砖块换皮，也不是纯《向僵尸开炮》式守线，而是一个把“弹射破砖”和“压线清僵尸”缝在同一张战场上的轻中度爽游。

更准确的项目定位是：

- 前台：弹射轨迹、连锁破砖、僵尸压线、满屏击碎反馈
- 中台：角度瞄准、路径改写、局内三选一强化
- 后台：炮台/炮弹/砖块三条成长线

核心判断：

- 比单纯打砖块更有目标压力
- 比单纯向僵尸开炮更有轨迹变化
- 比把两者简单叠加更像一个新产品

## 核心体验目标

- 我能一眼看懂这一局在打什么
- 我能感受到炮弹撞砖后弹道被改变
- 我能在打砖块的同时感受到僵尸在逼近
- 我能通过强化把这一局的弹射方式做出来
- 我能在 Boss 波前形成明显的 Build 差异

## 核心玩法模型

### 主循环：弹射破砖

1. 玩家选择角度开炮
2. 炮弹在砖墙中反弹、穿透或分裂
3. 砖块被打碎后改变后续弹道和可打击区域
4. 砖块掉落强化与资源
5. 玩家继续做下一次发射与路径优化

### 中循环：僵尸压线

1. 僵尸从上方或侧边持续推进
2. 玩家需要在破砖的同时处理僵尸威胁
3. 打得不够快就会形成压线
4. 压线越强，玩家越需要优先切换火力目标

### 高潮循环：局内成型与 Boss 收束

1. 关键强化出现
2. 炮弹形态发生变化
3. 僵尸潮加速压线
4. Boss 带着特殊砖层或护甲入场
5. 玩家用成型 Build 完成收束

## 角色与系统分工

### 炮台

定位：

- 主控核心
- 发射节奏源
- 角度与 Build 的承载点

职责：

- 选择发射方向
- 处理高优先级僵尸
- 触发连锁破砖

### 炮弹

定位：

- 轨迹主体
- 反馈主体
- 伤害链路主体

职责：

- 反弹
- 穿透
- 分裂
- 触发砖块联动效果

### 砖块

定位：

- 路径材料
- 伤害调度器
- 战场结构本身

原则：

- 不只是障碍物
- 必须影响弹道
- 必须影响局面变化

### 僵尸

定位：

- 目标压力
- 时间压力
- 收束压力

职责：

- 持续压线
- 逼迫玩家切目标
- 作为终局 Boss 的前置威胁

## 打砖块的重要性判断

结论：如果没有“砖块改写弹道”，这条方向就会退化成普通守线射击。

因此本项目保留的不是纯粹的“消灭砖块”表达，而是：

- 砖块决定炮弹怎么走
- 炮弹决定僵尸怎么死
- 僵尸决定玩家什么时候必须改变发射策略

这三者要在同一局里互相咬住。

## 僵尸压线系统

### 设计原则

- 压力必须明确
- 威胁必须可读
- 不能只是一层皮
- 不打砖块也会输，但打砖块必须能显著改善局面

### 表现形式

- 直线压进的普通僵尸
- 带护甲的砖盾僵尸
- 破坏砖墙的冲锋僵尸
- Boss 型压线单位

### 目标作用

- 提供时间压力
- 提供优先级切换
- 让打砖块不只是“刷分”

## 三选一系统

### 普通三选一

内容方向：

- 炮弹分裂
- 炮弹穿透
- 反弹次数提升
- 对僵尸增伤
- 对砖块增伤

### 高质量三选一

内容方向：

- 砖块击碎后产生新炮弹
- 炮弹撞砖后触发范围爆炸
- 僵尸死亡后回弹一段炮弹
- 终局炮弹改变全局弹道规则

## 打砖块和向僵尸开炮的关系

不是两个系统并列，而是同一套压力结构里的两个可见层：

- 打砖块负责“路径变化”
- 向僵尸开炮负责“压力目标”

如果只剩打砖块，局面会偏空；
如果只剩僵尸压线，局面会偏直。

把两者绑在一起，才有这条方向最重要的识别点。
