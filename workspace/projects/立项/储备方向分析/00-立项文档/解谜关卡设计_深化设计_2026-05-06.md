# 解谜/关卡设计方向 — 深化设计

## 核心定位

**底层玩法**：设计关卡布局 → 放置陷阱机关 → 战斗中操控角色配合 → 优化设计 → 挑战更难关卡

**独立玩法验证**：关卡编辑器本身就是独立游戏（马里奥制造、传送门2编辑器）

**与操控防守结合**：我设计的关卡，我来守卫，战斗中看到自己的设计发挥作用

## 与塔防的核心区别

| 维度 | 传统塔防 | 解谜/关卡设计 + 操控角色 |
|---|---|---|
| 核心体验 | "我布置的防线挡住了敌人" | "我设计的关卡+我的操作守住了阵地" |
| 玩家角色 | 上帝视角，只布置不操控 | 既是设计师，也是执行者 |
| 战斗参与 | 观察为主，偶尔技能 | 操控角色是主要防守力量 |
| 设计重点 | 防御塔的位置和升级 | 陷阱机关与角色操作的配合 |
| 成就感来源 | "我的布局很完美" | "我的设计+我的操作配合无间" |

**核心区别总结**：
- 塔防：布置是核心，观察是主要体验
- 本方向：设计是准备，操控角色配合设计是核心体验

## 核心循环

```
设计阶段（局外/波次间）
├─ 查看地形和敌人路线
├─ 规划陷阱和机关位置
├─ 设置触发条件和连锁
└─ 预览设计效果
    ↓
战斗准备
├─ 确认设计方案
├─ 选择角色装备
└─ 进入战斗
    ↓
战斗阶段（局内）
├─ 操控角色防守
├─ 引导敌人进入陷阱
├─ 手动触发机关
└─ 配合陷阱战斗
    ↓
战斗结束
├─ 查看设计效果统计
├─ 发现设计问题
├─ 获得新陷阱/机关
└─ 解锁新地形/敌人
    ↓
优化设计
├─ 调整陷阱位置
├─ 改进触发逻辑
├─ 尝试新组合
└─ 挑战更高难度
    ↓
回到设计阶段
```

## 陷阱机关系统（核心玩法）

**设计原则**：陷阱机关必须和角色操作配合，不是"布置完就自动运行"

### 陷阱类型

**伤害型陷阱**：

1. **尖刺陷阱**
   - 效果：敌人踩中受到伤害
   - 触发：自动触发（敌人经过）
   - 配合：玩家击退敌人到陷阱上
   - 升级：伤害提升、范围扩大、添加减速效果

2. **火焰喷射器**
   - 效果：持续喷射火焰
   - 触发：手动触发（玩家按键）或定时触发
   - 配合：玩家引导敌人到喷射范围
   - 升级：伤害提升、持续时间延长、添加点燃效果

3. **落石机关**
   - 效果：从上方落下巨石
   - 触发：手动触发或压力板触发
   - 配合：玩家引导敌人到落石区域
   - 升级：落石数量增加、范围扩大、添加眩晕效果

4. **毒气陷阱**
   - 效果：释放毒气持续伤害
   - 触发：自动触发或手动触发
   - 配合：玩家将敌人困在毒气区域
   - 升级：伤害提升、持续时间延长、添加减速效果

**控制型陷阱**：

1. **捕兽夹**
   - 效果：固定敌人位置
   - 触发：自动触发（敌人经过）
   - 配合：玩家集火被固定的敌人
   - 升级：固定时间延长、范围扩大、添加伤害

2. **冰冻陷阱**
   - 效果：冰冻敌人
   - 触发：自动触发或手动触发
   - 配合：玩家攻击冰冻敌人造成额外伤害
   - 升级：冰冻时间延长、范围扩大、添加易伤效果

3. **推力机关**
   - 效果：将敌人推向指定方向
   - 触发：手动触发或定时触发
   - 配合：玩家将敌人推入其他陷阱
   - 升级：推力增强、范围扩大、冷却时间缩短

4. **传送门**
   - 效果：将敌人传送到另一位置
   - 触发：自动触发（敌人经过）
   - 配合：玩家将敌人传送到陷阱密集区
   - 升级：传送距离增加、冷却时间缩短

**辅助型机关**：

1. **治疗泉**
   - 效果：玩家站在上面恢复生命
   - 触发：玩家进入范围
   - 配合：玩家战术撤退到治疗泉
   - 升级：恢复速度提升、范围扩大、添加护盾

2. **弹药补给站**
   - 效果：补充玩家弹药/技能冷却
   - 触发：玩家互动
   - 配合：玩家规划战斗路线经过补给站
   - 升级：补给速度提升、范围扩大、添加增益

3. **屏障发生器**
   - 效果：生成阻挡敌人的屏障
   - 触发：手动触发
   - 配合：玩家控制敌人行进路线
   - 升级：屏障耐久提升、持续时间延长

4. **视野塔**
   - 效果：提供视野和预警
   - 触发：被动效果
   - 配合：玩家提前准备应对
   - 升级：视野范围扩大、添加减速光环

**召唤型机关**：

1. **炮塔**
   - 效果：自动攻击敌人
   - 触发：敌人进入范围
   - 配合：玩家引导敌人到炮塔射程
   - 升级：伤害提升、射程扩大、攻速提升

2. **召唤阵**
   - 效果：召唤友军单位
   - 触发：手动触发
   - 配合：玩家和友军协同作战
   - 升级：单位数量增加、单位强度提升

### 机关组合系统

**连锁触发**：

1. **压力板连锁**
   - 敌人踩中压力板 → 触发多个机关
   - 例如：压力板 → 火焰喷射器 + 落石机关

2. **延迟触发**
   - 第一个机关触发后，延迟触发第二个
   - 例如：冰冻陷阱 → 3秒后 → 火焰喷射器

3. **条件触发**
   - 满足特定条件才触发
   - 例如：敌人数量≥5 → 触发毒气陷阱

**组合效果**：

1. **控制+伤害组合**
   - 捕兽夹固定 + 火焰喷射器
   - 冰冻陷阱 + 落石机关
   - 推力机关 + 尖刺陷阱

2. **范围控制组合**
   - 屏障发生器 + 毒气陷阱（困住敌人）
   - 推力机关 + 传送门（强制路线）

3. **多重伤害组合**
   - 火焰喷射器 + 毒气陷阱（双重持续伤害）
   - 落石机关 + 尖刺陷阱（多段伤害）

### 地形改造系统

**可改造地形**：

1. **墙壁**
   - 建造墙壁阻挡敌人
   - 墙壁可以被摧毁
   - 玩家可以利用墙壁卡位

2. **高低差**
   - 建造高台获得视野和射程优势
   - 挖掘低地形成天然陷阱区

3. **路径**
   - 铺设路径引导敌人
   - 路径上可以放置陷阱

4. **障碍物**
   - 放置障碍物改变敌人路线
   - 障碍物可以被摧毁

**地形与陷阱配合**：

1. **狭窄通道 + 范围陷阱**
   - 建造墙壁形成狭窄通道
   - 在通道中放置火焰喷射器/毒气陷阱

2. **高台 + 推力机关**
   - 建造高台
   - 用推力机关将敌人推下高台

3. **迷宫 + 控制陷阱**
   - 建造迷宫路线
   - 在关键位置放置捕兽夹/冰冻陷阱

## 设计工具系统

### 设计界面

**俯视图模式**：

1. **地形预览**
   - 显示完整地形和敌人路线
   - 显示已放置的陷阱和机关
   - 显示触发范围和效果范围

2. **陷阱放置**
   - 拖拽放置陷阱
   - 实时显示放置是否合法
   - 显示陷阱覆盖范围

3. **连锁设置**
   - 连接陷阱设置触发关系
   - 设置触发条件和延迟
   - 预览连锁效果

**模拟测试**：

1. **快速模拟**
   - 快进模拟敌人路线
   - 显示陷阱触发情况
   - 不需要实际战斗

2. **问题标注**
   - 标注陷阱未触发的位置
   - 标注敌人未受伤害的路径
   - 提示优化建议

### 设计约束

**资源限制**：

1. **建造点数**
   - 每个陷阱消耗建造点数
   - 高级陷阱消耗更多点数
   - 通过战斗获得更多点数

2. **空间限制**
   - 地图有限的可建造区域
   - 陷阱之间需要保持距离
   - 某些位置不可建造

3. **类型限制**
   - 某些陷阱有数量上限
   - 某些组合不可同时使用
   - 某些地形限制陷阱类型

**设计挑战**：

1. **多路径敌人**
   - 敌人从多个方向进攻
   - 需要平衡各路径的防守

2. **特殊敌人**
   - 飞行敌人无视地面陷阱
   - 重甲敌人免疫控制
   - 需要针对性设计

3. **时间压力**
   - 波次间设计时间有限
   - 需要快速决策和调整

## 战斗配合系统

### 角色与陷阱配合

**引导型配合**：

1. **风筝战术**
   - 玩家吸引敌人追击
   - 引导敌人进入陷阱区域
   - 陷阱触发后玩家补刀

2. **卡位战术**
   - 玩家站在关键位置
   - 迫使敌人绕路经过陷阱
   - 配合墙壁和障碍物

3. **诱饵战术**
   - 玩家假装撤退
   - 敌人追击进入陷阱密集区
   - 玩家反击配合陷阱

**触发型配合**：

1. **手动触发时机**
   - 玩家判断最佳触发时机
   - 等待敌人聚集后触发
   - 配合角色技能同时释放

2. **连锁触发控制**
   - 玩家触发第一个机关
   - 观察连锁效果
   - 根据情况触发后续机关

3. **紧急触发**
   - 玩家被围困时触发逃生机关
   - 敌人突破防线时触发备用陷阱

**补刀型配合**：

1. **陷阱削弱+玩家击杀**
   - 陷阱降低敌人生命值
   - 玩家补刀击杀
   - 提升击杀效率

2. **控制+集火**
   - 陷阱控制敌人
   - 玩家集火被控制目标
   - 快速清理威胁

### 战斗反馈

**视觉反馈**：

1. **陷阱触发特效**
   - 明显的触发动画
   - 伤害数字显示
   - 控制效果图标

2. **连锁效果展示**
   - 连锁触发的视觉连线
   - 组合效果的特殊特效
   - 高伤害的震屏效果

**数据反馈**：

1. **战斗统计**
   - 每个陷阱的触发次数
   - 每个陷阱的伤害贡献
   - 陷阱击杀数量

2. **效率分析**
   - 陷阱利用率
   - 资源消耗效率
   - 设计优化建议

## 长线驱动

### 收集目标

1. **陷阱图鉴**：解锁所有陷阱类型
2. **机关图鉴**：解锁所有机关类型
3. **组合图鉴**：发现所有有效组合
4. **地形图鉴**：解锁所有地形改造选项

### 挑战目标

1. **完美防守**：不受伤通关
2. **最少陷阱**：用最少陷阱通关
3. **最高效率**：最短时间通关
4. **创意设计**：发现独特组合

### 分享目标

1. **关卡分享**：分享自己的设计
2. **挑战他人**：挑战他人设计的关卡
3. **设计竞赛**：参加设计比赛
4. **教学关卡**：制作教学关卡

## 包装变体

同样的底层玩法，可以包装成：

1. **要塞防守**：设计要塞防御，守卫城堡
2. **地下城设计**：设计地下城陷阱，守护宝藏
3. **实验室防御**：设计实验室机关，保护研究
4. **太空站防御**：设计太空站系统，抵御入侵
5. **魔法塔防御**：设计魔法机关，守护法师塔

**包装差异**：
- 要塞防守：中世纪陷阱（滚木、落石、箭塔）
- 地下城设计：经典陷阱（尖刺、毒箭、机关门）
- 实验室防御：科技机关（激光、电网、机器人）
- 太空站防御：未来科技（力场、传送、AI炮塔）
- 魔法塔防御：魔法机关（元素陷阱、召唤阵、结界）

| 核心驱动 | 创造成就感 | 情感绑定 | 制作成就感 | 收集完成度 |

## 潜在问题与解决方案

**问题1：学习曲线会不会太陡？**
- 解决：渐进式教学，从简单陷阱开始
- 解决：提供预设模板，玩家可以直接使用
- 解决：模拟测试功能，快速验证设计

**问题2：设计会不会变成"唯一解"？**
- 解决：多种设计都能通关，没有唯一最优解
- 解决：不同陷阱组合有不同优势
- 解决：玩家操作可以弥补设计不足

**问题3：设计和战斗的时间比例？**
- 解决：波次间设计时间有限，避免过度设计
- 解决：战斗中可以手动触发，保持参与感
- 解决：战斗后可以优化设计，形成循环

**问题4：会不会变成纯塔防？**
- 解决：陷阱伤害不足以单独击杀敌人
- 解决：玩家操控角色是主要防守力量
- 解决：陷阱是辅助，配合角色操作才有效

**问题5：设计自由度和平衡性的矛盾？**
- 解决：资源限制避免无限堆陷阱
- 解决：陷阱有数量和类型限制
- 解决：敌人有针对性设计（飞行、重甲等）

## 为什么是第一梯队

1. **独立玩法验证**：关卡编辑器本身就是独立游戏（马里奥制造）
2. **创造成就感强**：玩家完全自定义设计，成就感极强
3. **策略深度高**：陷阱组合+地形改造+角色配合，策略空间大
4. **重玩价值高**：每次可以尝试不同设计，探索新组合
5. **社交潜力大**：关卡分享、挑战他人设计，社交驱动强
6. **与战斗结合紧密**：设计直接影响战斗，战斗验证设计，循环紧密
