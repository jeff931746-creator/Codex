操作说明

  • 移动主角WASD / 手机虚拟摇杆

  • 重开R

核心机制

  • 护送目标自动沿路线前进
  • 尸潮四周围城压迫
  • 主角走位 + 自动锁敌
  • 成长利息 + 驻点买卡包
