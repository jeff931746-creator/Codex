# 题材分类参考库

配套文档：[题材选择方法论](题材选择方法论_2026-05-08.md)

本库是对常见游戏题材的结构化分类，供立项阶段快速筛选候选题材使用。每条题材给出三要素结构、主情绪、四象限倾向和主要风险，不替代方法论中的完整评分。

---

## 分类逻辑

题材按**主情绪**分类，而不是按美术风格或文化标签分类。情绪是玩家选择题材时真正购买的预期，所有其他要素（环境、玩法、付费）都应服务主情绪。

同一题材会因战斗体系不同落入不同情绪区。「末日」带枪械 → 爽感释放；带资源管理 → 求生压力；带日常建设 → 治愈放松。因此分类时，同一世界背景可能出现在多个情绪区下。

---

## 一、求生压力类

玩家进入危险处境，需要用有限资源维持生存。主情绪是压迫、紧张与短暂的喘息。

### 1-1 末日丧尸求生

```text
环境/时代：现代城市或荒野，文明崩溃后
文化范式/战斗体系：枪械 + 近战 + 资源收集
叙事：玩家是幸存者，在尸潮中保住自己和队伍

核心辨识词：尸潮、幸存者、据点
主情绪：求生压力 + 短暂爽感
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右上（高有效获量 + 高 ROI 承接）— 但大泡泡，竞争密度高 |
| 主要风险 | 素材同质化，必须有差异切口（新地图、新机制、新人群） |
| 常见玩法接口 | 塔防、策略建设、Roguelite、放置、MMO |

---

### 1-2 极寒末日资源生存

```text
环境/时代：冰雪废土、核冬天、气候崩溃后的地球
文化范式/战斗体系：资源调度 + 建筑御寒 + 社群管理
叙事：玩家带领幸存群体度过漫长寒冬

核心辨识词：冰雪、炉火、幸存营地
主情绪：求生压力 + 成长满足
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右上 — 泡泡中等，竞争适中，策略玩家受众清晰 |
| 主要风险 | 内容制作成本较高，需要大量可循环内容撑长留存 |
| 常见玩法接口 | 城市建设、策略经营、资源管理、塔防 |

---

### 1-3 荒野求生探索

```text
环境/时代：无人森林、荒岛、沙漠或海洋
文化范式/战斗体系：采集 + 制作 + 地图探索
叙事：玩家独自或与伙伴在原始环境中活下去

核心辨识词：荒野、营地、采集、制作
主情绪：探索敬畏 + 求生压力
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右侧中等 — 获量中等，LTV 依赖成长和建造内容深度 |
| 主要风险 | 容易变成内容消费型，缺少循环付费场景 |
| 常见玩法接口 | 沙盒建造、Roguelite、放置、动作RPG |

---

### 1-4 地牢深渊探索

```text
环境/时代：地下城、废弃迷宫、克苏鲁神庙
文化范式/战斗体系：战斗闯关 + 随机生成 + 永久死亡
叙事：玩家是冒险者，一次次深入危险核心

核心辨识词：地牢、随机、死亡代价
主情绪：求生压力 + 成长满足
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右上 — Roguelite 品类认知成熟，买量成本相对可控 |
| 主要风险 | 高品质竞争集中，中低品质难以脱颖而出 |
| 常见玩法接口 | Roguelite、Roguelike、动作、卡牌构筑 |

## 二、爽感释放类

玩家通过打击反馈获得即时快感。主情绪是强烈反馈、控制感和持续的能力展示。

### 2-1 现代都市枪械防守

```text
环境/时代：现代城市，危机爆发的街道或建筑
文化范式/战斗体系：枪械技能体系 + 防守阵地
叙事：玩家是精英防御者，守住人类最后阵地

核心辨识词：枪械、僵尸潮、防守、爆头
主情绪：爽感释放 + 求生张力
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右上 — 题材获量能力极强，但需要高质量打击反馈 |
| 主要风险 | 对美术和手感要求高，轻量产品难以达标 |
| 常见玩法接口 | 塔防、射击、动作 |

---

### 2-2 修仙破境爽感

```text
环境/时代：东方玄幻世界，仙门、宗门、秘境
文化范式/战斗体系：境界 + 功法 + 法宝 + 宗门体系
叙事：玩家从凡人成长为绝世强者

核心辨识词：修炼、破境、仙术、宗门
主情绪：成长满足 + 爽感释放
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 左上（高 ROI 承接 + 低有效获量）— 默认状态，因为买量红海 |
| 主要风险 | 仙侠 CPI 高、素材疲劳，必须有新切口才能移入右上 |
| 进入条件 | 新情绪反差（喜剧/恐怖/治愈）、新玩法（塔防/合成/肉鸽）、新人群 |

---

### 2-3 赛博朋克暗黑动作

```text
环境/时代：近未来霓虹都市，企业统治的地下世界
文化范式/战斗体系：义体改造 + 黑科技武器 + 暗网网络
叙事：玩家是雇佣兵或叛客，在权力缝隙中求生与反抗

核心辨识词：霓虹、义体、黑客、企业
主情绪：爽感释放 + 敬畏探索
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右侧中等 — 题材热度高，但美术成本高，LTV 依赖养成深度 |
| 主要风险 | 世界观建设成本大，轻量产品撑不住题材期待 |
| 常见玩法接口 | 动作 RPG、策略、放置、收集养成 |

---

### 2-4 超级英雄能力释放

```text
环境/时代：现代都市或架空世界，超能力者并存
文化范式/战斗体系：超能力技能 + 身份/势力体系
叙事：玩家拥有特殊能力，在人类与超级力量之间寻找位置

核心辨识词：超能力、技能爆发、特殊身份
主情绪：爽感释放 + 成长满足
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右侧中等 — IP 依赖度高，原创版本需要强技能反馈支撑 |
| 主要风险 | 无 IP 时题材区分度弱 |
| 常见玩法接口 | 动作、RPG、塔防、Roguelite |

---

### 2-5 割草清版大乱斗

```text
环境/时代：各类世界背景，以视觉冲击为主
文化范式/战斗体系：AOE 技能 + 大量敌人密度 + 短周期成长
叙事：玩家扮演强力角色，用技能清空敌人

核心辨识词：割草、密度、技能爆发
主情绪：爽感释放（即时）
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右下（高有效获量 + 低 ROI 承接）— 素材跑量强，但长线留存弱 |
| 主要风险 | 内容消耗快，付费场景有限，需要成长线或收集线补充 LTV |
| 常见玩法接口 | 幸存者类、清版动作、塔防混合 |

## 三、敬畏探索类

玩家进入未知空间，情绪是好奇、恐惧与对未知的敬畏。主情绪是探索欲和发现感。

### 3-1 克苏鲁/洛夫克拉夫特恐怖

```text
环境/时代：现代或 20 世纪初，隐藏于日常背后的宇宙恐惧
文化范式/战斗体系：调查推理 + 理智值 + 禁忌知识
叙事：玩家是调查者，越深入越接近发疯或毁灭

核心辨识词：古神、理智崩溃、禁忌、未知
主情绪：敬畏恐惧 + 紧张破局
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右侧中等小泡泡 — 受众垂直，但核心用户忠诚度高 |
| 主要风险 | 美术和剧情投入大，轻量化难度高 |
| 常见玩法接口 | 卡牌、策略、动作 RPG、文字冒险 |

---

### 3-2 太空深渊探索

```text
环境/时代：遥远未来，深空、外星星球、太空站
文化范式/战斗体系：科技升级 + 宇宙物理 + 外星生命体系
叙事：玩家是宇航员或先驱者，在宇宙中寻找出路或真相

核心辨识词：深空、孤独、未知星球、宇宙
主情绪：敬畏探索 + 求生压力
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右侧中等 — 科幻受众稳定，但视觉期待高，制作成本压力大 |
| 主要风险 | 如果无法达到科幻美术水准，会打破情绪预期 |
| 常见玩法接口 | 沙盒建造、策略、4X、Roguelite |

---

### 3-3 古文明考古神话

```text
环境/时代：古埃及、玛雅、美索不达米亚或架空上古文明
文化范式/战斗体系：神祇力量 + 古代机关 + 遗迹探索
叙事：玩家揭开古文明的真相，或借助其力量对抗更大威胁

核心辨识词：神庙、诅咒、古神、遗迹、考古
主情绪：敬畏探索 + 成长满足
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右侧中等 — 视觉符号辨识度高，受众在冒险/RPG 人群中有稳定渗透 |
| 主要风险 | 题材边界相对有限，扩展内容需要借助叙事或玩法横向延伸 |
| 常见玩法接口 | 解谜、动作 RPG、卡牌、策略 |

---

### 3-4 深海未知生物

```text
环境/时代：深海、海沟、水下遗迹
文化范式/战斗体系：潜水器 + 深海生物 + 资源与压力管理
叙事：玩家探索深海，对抗未知生物或解开海底秘密

核心辨识词：深海、巨型生物、黑暗、水压
主情绪：敬畏恐惧 + 求生压力
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右侧中小泡泡 — 画面辨识度独特，但受众相对垂直 |
| 主要风险 | 美术渲染成本高，光影和生物设计是核心卖点 |
| 常见玩法接口 | 沙盒、Roguelite、恐怖生存 |

## 四、治愈放松类

玩家进入温和的环境，情绪是安全感、满足感与日常节奏。主情绪是放松与治愈。

### 4-1 田园农场经营

```text
环境/时代：乡村、牧场、田野、小岛
文化范式/战斗体系：种植 + 收获 + 社区互动
叙事：玩家经营一块土地，建立属于自己的田园生活

核心辨识词：农场、种植、季节、收获、邻居
主情绪：治愈放松 + 成长满足
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右上 — 品类验证充分，受众明确（女性向、轻度玩家），但头部竞争激烈 |
| 主要风险 | Stardew Valley / 星露谷定义品质天花板，需要差异化切口 |
| 常见玩法接口 | 模拟经营、放置、解谜、社交 |

---

### 4-2 宠物收集养育

```text
环境/时代：现代生活或架空世界
文化范式/战斗体系：收集 + 养成 + 情感绑定
叙事：玩家收集、培育宠物，建立情感连接

核心辨识词：宠物、可爱、收集、养成
主情绪：治愈放松 + 成长满足
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右上中等 — 女性向/泛用户受众，收集付费点自然，IP 化潜力高 |
| 主要风险 | 宠物设计成本高，情感设计决定留存上限 |
| 常见玩法接口 | 放置、模拟经营、卡牌收集、策略 |

---

```text
环境/时代：餐厅、食堂、街边小摊或幻想烹饪世界

核心辨识词：料理、餐厅、食材、顾客

```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右上中等 — 受众清晰，但买量素材容易同质化 |

| 常见玩法接口 | 放置、模拟、休闲解谜 |

---

### 4-4 花园/岛屿装扮建设

```text
环境/时代：私家花园、孤岛、虚拟家园
文化范式/战斗体系：收集装饰元素 + 布置 + 社交展示
叙事：玩家打造属于自己的美丽空间

核心辨识词：花园、装扮、布置、美化
```

| 主要风险 | 无社交的纯外观产品 LTV 有上限，需要持续内容注入 |

## 五、成长满足类

玩家持续变强，情绪来自可见的进度和对比。主情绪是积累感、控制感和远期期待。

### 5-1 奇幻英雄养成

```text
环境/时代：中世纪奇幻大陆，魔法与剑并存
文化范式/战斗体系：职业 + 技能树 + 装备成长
叙事：玩家成为传奇英雄，在史诗征程中不断突破

核心辨识词：英雄、成长、装备、史诗
主情绪：成长满足 + 爽感释放
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右上大泡泡 — 品类验证最充分，付费体系完整，但竞争密度最高 |
| 主要风险 | 无差异化的奇幻 RPG 在当前市场获量成本极高 |
| 常见玩法接口 | RPG、卡牌、放置、策略、MMO |

---

### 5-2 基地建设成长

```text
环境/时代：各类题材皆可作为外壳（末日、奇幻、现代）
文化范式/战斗体系：建筑升级 + 资源循环 + 势力扩张
叙事：玩家从零开始建立强大基地，成为最强领主

核心辨识词：基地、升级、资源、扩张、领地
主情绪：成长满足 + 求生压力（初期）
```

| 项 | 说明 |
|---|---|

| 主要风险 | 买量依赖题材外壳，本体竞争极其激烈，题材差异是唯一切口 |
| 常见玩法接口 | SLG、策略、放置、经营 |

---

## 六、紧张破局类

玩家身陷困境，需要在压力下做出判断。主情绪是紧张、悬疑与解脱的快感。

### 6-1 轮回逃生

```text
环境/时代：循环异常空间（密室、废弃楼层、诡异学校）
文化范式/战斗体系：线索 + 死亡重置 + 记忆积累
叙事：玩家在无限循环中用每次死亡积累信息，最终找到出口

核心辨识词：轮回、循环、死亡、记忆、出口
主情绪：紧张破局 + 成长满足
```

| 项 | 说明 |
|---|---|
| 四象限倾向 | 右侧中等 — 小众但用户忠诚，素材表达有独特性 |
| 主要风险 | 受众垂直，放量上限有限；叙事密度要求高 |
| 常见玩法接口 | 文字冒险、解谜、Roguelite、卡牌 |

---

### 6-2 推理解谜悬疑

```text
环境/时代：现代都市、历史场景或架空世界
文化范式/战斗体系：线索采集 + 逻辑推断 + 证据对比
叙事：玩家作为侦探或当事人，解开一系列谜题或案件

核心辨识词：推理、线索、真相、悬疑
主情绪：紧张破局 + 认知成就
```

| 项 | 说明 |

## 使用说明

1. 这里的倾向判断基于**当前市场状态（2026 年）的通用估计**，不代表你的产品必然落在这个位置。
2. 快查结果只是起点。进入正式判断前，必须先过**三层判断体系**（题材天然能力 → 当前市场状态 → 我方进入方式），再用方法论中的四象限评分表做逐项校正。**题材天然能力成立，不等于市场成立。**
3. 评分时横轴用"有效获量能力"（含竞争密度、素材疲劳、CPI 校正），纵轴用"ROI 承接能力"（LTV 能否覆盖买量成本）。两者与"吸量"和"商业化"是不同的判断层级。
4. 快查矩阵里的"默认不立项"意味着**需要额外证明进入切口**，不代表绝对不能做。
5. 题材可以叠加，叠加时以方法论中的**题材融合四条原则**为准，不能只堆标签。
6. 第二篇来源文档读取完成后，此库的评分倾向可能需要局部更新。
