## 一句话结论

题材选择不是挑一个流行标签，而是选择一个玩家一眼能理解、能产生情绪预期，并且能自然导向玩法的体验承诺。

```text
题材 = 环境/时代 + 文化范式/战斗体系 + 叙事
有效题材 = 能承接情绪 + 能展开玩法 + 能被卖相表达
题材融合 = 主导题材 + 辅助题材 + 统一情绪与逻辑桥梁
```

## 核心定义

### 题材

题材是游戏依托的文化传统、美学呈现与叙事模型的集合。它决定玩家第一次看到游戏时，会如何判断这个世界、这个角色、这个冲突和自己将要获得的体验。

题材不是单个标签：

```text
弱表达：维多利亚时代、二次元、末日、修仙
强表达：末日丧尸潮 + 枪械 + 求生
```

单个标签只能说明外观或文化来源，不能说明玩家要做什么、为什么做、用什么方式做。

### 题材三要素

一个可用于立项判断的题材，至少要同时回答三个问题：

| 要素 | 回答的问题 | 作用 |
|---|---|---|
| 环境/时代 | 事情发生在哪里、什么时候、世界长什么样 | 建立第一眼识别与空间想象 |
| 文化范式/战斗体系 | 这个世界如何运作，冲突如何解决 | 建立能力规则、符号系统与玩法接口 |
| 叙事 | 玩家是谁，要完成什么，核心矛盾是什么 | 建立目标、身份、情绪和长期牵引 |

题材表达公式：

```text
在【环境/时代】中，通过【文化范式/战斗体系】，完成【叙事目标】的【体验旅程】
```

例：

```text
在现代都市中，通过枪械技能体系防御阵地，对抗僵尸潮的射击爽游。
在冰雪末世中，通过炉火资源体系建设家园，带领社群活下去的策略生存之旅。
在中式民俗诡谲空间中，通过轮回因果体系破解谜题，逃离异常空间的破局之旅。
```

## 为什么题材选择重要

题材承担的是立项早期的三层任务：

| 层级 | 题材要解决的问题 | 失败表现 |
|---|---|---|
| 认知 | 玩家能不能一眼看懂这是什么 | 只有美术风格，没有清晰体验 |
| 情绪 | 玩家是否立刻知道自己会获得什么感受 | 设定很多，但没有明确期待 |
| 玩法 | 题材是否自然导向可玩的系统 | 题材和玩法分离，只是换皮 |

题材弱，常见问题不是“不新”，而是：

- 玩家看不出自己要扮演谁。
- 玩家看不出核心冲突是什么。
- 玩家看不出玩法为什么必须发生在这个题材里。
- 美术元素很丰富，但没有统一的情绪方向。
- 题材标签很热，但无法形成截图、短视频、广告素材里的明确卖相。

## 题材选择流程

### 步骤 1：先定体验承诺

不要先问“现在什么题材火”，先问玩家要获得什么体验。

可选起点：

| 起点 | 典型问题 | 示例 |
|---|---|---|
| 身份幻想 | 玩家想成为谁 | 幸存者领袖、猎魔人、粮草官、太空列车成员 |
| 情境幻想 | 玩家想进入什么处境 | 极寒末世、尸潮围城、轮回逃生、海底探索 |
| 冲突幻想 | 玩家想解决什么矛盾 | 生存 vs 道德、秩序 vs 混乱、自由 vs 宿命 |
| 行为幻想 | 玩家想反复做什么 | 防守、建设、探索、成长、收集、经营 |

输出一句话：

```text
这是一款让玩家体验【身份/情境/冲突/行为】的游戏。
```

### 步骤 2：拆成题材三要素

把体验承诺拆成可执行的题材结构：

```text
环境/时代：
文化范式/战斗体系：
叙事：
核心辨识词：
一句话题材定义：
```

检查标准：

- 环境/时代能不能形成画面。
- 文化范式/战斗体系能不能形成系统。
- 叙事能不能形成目标。
- 核心辨识词能不能出现在封面、截图、短视频和标题里。

### 步骤 3：判断题材是否承上情绪

题材必须服务主情绪。情绪不是额外包装，而是玩家选择题材时真正购买的预期。

常见匹配：

| 主情绪 | 适配题材方向 | 风险 |
|---|---|---|
| 求生压力 | 末日、灾难、荒野、深海、地牢 | 如果资源压力不可见，会变成普通冒险 |
| 爽感释放 | 枪械、技能、割草、超能力、复仇 | 如果敌人和反馈弱，爽感不成立 |
| 敬畏探索 | 克苏鲁、神话、太空、古文明、深海 | 如果未知感不足，会变成普通设定 |
| 治愈放松 | 田园、花园、料理、宠物、自然 | 如果目标压力过强，会破坏放松 |
| 成长满足 | 修仙、英雄养成、职业进阶、基地建设 | 如果成长不可见，会只剩数值 |
| 紧张破局 | 轮回、密室、无限流、潜入、逃生 | 如果失败代价不清晰，紧张感不足 |

判断句：

```text
这个题材是否让玩家在看到第一张图时，就预期到对应情绪？
```

### 步骤 4：判断题材是否启下玩法

题材不能只停留在视觉层。它要自然生成玩法规则。

| 题材要素 | 应导出的玩法问题 |
|---|---|
| 环境/时代 | 地形、空间、资源、天气、时间是否影响玩法 |
| 文化范式/战斗体系 | 能力来源、敌我关系、成长方式、战斗方式是否明确 |
| 叙事 | 玩家目标、失败代价、阶段推进、长期追求是否明确 |

反向检查：

```text
如果把题材换成另一个皮，玩法是否几乎不变？
```

如果答案是“几乎不变”，题材只是包装，不是立项支柱。

### 步骤 5：验证卖相表达

题材要能被市场素材表达出来。

最小验证：

| 表达物 | 必须回答的问题 |
|---|---|
| 标题 | 能不能让玩家知道大概题材和体验 |
| 封面 | 能不能出现核心辨识词 |
| 第一张截图 | 能不能看出玩家在做什么 |
| 5 秒短视频 | 能不能展示题材带来的冲突和反馈 |
| 一句话简介 | 能不能说清三要素 |

失败信号：

- 题材需要长篇背景解释才成立。
- 核心卖点无法截图化。
- 广告素材只能展示角色或氛围，看不出玩法。
- 玩家评论会集中问“这游戏到底干嘛的”。

## 题材四象限

题材定义解决的是“它是什么”，四象限解决的是“它在当前买量市场里是否值得选”。

买量项目不能只看题材天然是否吸量、是否商业化。一个题材可能天然商业化很强，但在当前市场里已经竞争过度、素材疲劳、CPI 过高，导致 ROI 不成立。仙侠就是典型：题材能力成立，但如果没有新的买量表达或进入切口，市场层面不成立。

所以题材四象限的底层不是：

```text
吸量能力 × 商业化能力
```

而是：

```text
有效获量能力 × ROI 承接能力
```

也就是两段转化：

```text
外部买量转化：看到素材 → 产生兴趣 → 点击 / 下载，且获量成本可接受
内部价值转化：进入游戏 → 形成目标 → 留存 / 付费，且 LTV 能覆盖买量成本
```

```text
横轴：有效获量能力
纵轴：ROI 承接能力
泡泡大小：市场盘子 / 内容供给 / 竞争拥挤度
```

### 坐标定义

#### 横轴：有效获量能力

有效获量能力不是“题材看起来吸不吸引人”，而是这个题材在当前买量环境下，能否以可接受成本拿到目标用户。

公式：

```text
有效获量能力 =
题材需求规模
× 第一眼刺激
× 素材新鲜度
× 目标人群匹配
÷ 竞争密度
÷ 素材疲劳
÷ 买量成本
```

高有效获量题材通常具备：

- 有真实需求：已有用户在消费类似内容，不是团队自嗨。
- 有低成本表达：第一眼素材能说清冲突、身份或爽点。
- 有新鲜角度：不是和同题材素材同质化竞争。
- 有可触达人群：素材能命中明确用户，而不是泛泛“所有玩家”。
- 有可接受 CPI：预估获量成本没有高到 LTV 无法覆盖。

低有效获量题材常见问题：

- 题材天然吸引，但买量红海，素材高度同质化。
- 题材需要长解释，素材 3 秒内说不清。
- 玩家见过太多类似表达，点击意愿下降。
- 目标人群太窄，或者可触达人群和产品实际受众不一致。
- CPI 过高，即使有量也很难回本。

#### 纵轴：ROI 承接能力

ROI 承接能力不是“题材能不能商业化”，而是产品进入游戏后，能否用留存和 LTV 覆盖买量成本。

公式：

```text
ROI 承接能力 =
LTV 上限
× 付费自然度
× 留存深度
× 内容扩展效率
÷ 产品制作成本
÷ 买量回本压力
```

高 ROI 承接题材通常具备：

- LTV 上限清楚：能自然支撑成长、收集、养成、外观、社交或策略付费。
- 留存目标清楚：玩家有持续追求，而不是只体验一次刺激。
- 内容可持续：角色、敌人、关卡、地图、皮肤、剧情章节能稳定扩展。
- 付费不破坏题材承诺：付费服务体验，而不是和题材情绪冲突。
- 制作成本可控：内容生产效率能匹配预期回收周期。

低 ROI 承接题材常见问题：

- 只适合短期猎奇或一次性观看。
- 有点击，但缺少长期目标。
- 题材边界窄，内容很快做尽。
- 付费点只能硬塞，容易破坏体验。
- 买量成本高于题材可承接的 LTV。

### 三层判断体系

题材进入四象限前，必须先过三层判断。

| 层级 | 判断问题 | 作用 |
|---|---|---|
| 题材天然能力 | 这个题材本身是否有吸引、成长、内容和付费空间？ | 判断题材原始潜力 |
| 当前市场状态 | 这个题材现在是否红海、疲劳、CPI 过高？ | 校正真实买量环境 |
| 我方进入方式 | 我们是否有新表达、新人群、新玩法承接或成本优势？ | 判断是否有进入资格 |

硬规则：

```text
题材天然能力成立，不等于市场成立。
市场红海不等于不能做，但必须有明确进入切口。
无法说明进入切口时，不进入立项，只能进入素材小测或继续找题材。
```

### 市场校正变量

每个候选题材必须做市场校正，避免把过往成功误判成当前机会。

| 变量 | 判断问题 | 对结论的影响 |
|---|---|---|
| 竞争密度 | 同题材、同人群、同素材表达是否已经大量存在？ | 竞争越高，有效获量越低 |
| 素材疲劳 | 用户是否已经反复见过类似首图、桥段、卖点？ | 疲劳越高，点击成本越高 |
| CPI 压力 | 当前题材的人群价格是否高于产品可承接范围？ | CPI 越高，ROI 门槛越高 |
| 人群错配 | 点击人群和实际付费人群是否一致？ | 错配越高，下载后转化越差 |
| 表达差异 | 我们的题材表达是否有一眼可见的新角度？ | 差异越强，越能抵消红海 |
| 内容成本 | 题材是否要求高成本美术、剧情、角色或世界观生产？ | 成本越高，回本压力越大 |

### 四个象限

| 象限 | 特征 | 适合策略 | 主要风险 |
|---|---|---|---|
| 高有效获量 + 高 ROI 承接 | 当前市场能拿量，产品内能回本 | 优先进入立项验证 | 竞争会迅速跟进，必须保留迭代素材能力 |
| 高有效获量 + 低 ROI 承接 | 素材能跑，但产品内承接弱 | 适合轻量产品、广告变现、短周期验证或副题材 | 下载后留存和付费不足 |
| 低有效获量 + 高 ROI 承接 | 内核能赚钱，但当前题材外壳买量困难 | 换更强题材外壳，或找细分人群和新表达 | 产品可能好，但买不进用户 |
| 低有效获量 + 低 ROI 承接 | 当前市场拿量难，产品内也难回本 | 不适合作为商业立项 | 团队容易用“题材品味”替代市场判断 |

### 象限解释

#### 1. 高有效获量 + 高 ROI 承接：当前可立项题材

这是最理想的区域，但前提是“当前市场仍然能以合理成本获量”。不是题材大、商业化强就自动进入右上角。

判断重点：

```text
这个题材现在还能不能买进目标用户？
买进来的用户能不能被产品 LTV 接住？
```

进入要求：

- 有明确竞品和市场数据。
- 有可截图化、可短视频化的差异表达。
- 有可控 CPI 假设。
- 有产品内 LTV 承接路径。
- 有素材持续迭代能力，避免首轮素材跑通后快速疲劳。

#### 2. 高有效获量 + 低 ROI 承接：流量外壳题材

这类题材能让用户点进来，但产品内难以形成长期价值。很多热点、猎奇、反常识内容会落在这里。

它的问题不是“不吸引”，而是“赚不回来”。

判断重点：

```text
这个题材带来的点击人群，是否能被现有产品循环转化为留存和付费？
```

适合用法：

- 广告变现产品。
- 短周期轻产品。
- 素材测试外壳。
- 作为副题材、活动题材、关卡题材。

转化方式：

- 给猎奇题材加成长线：收集、升级、图鉴、基地、角色关系。
- 给短期爽点加目标线：关卡推进、Boss 挑战、赛季任务。
- 给强情绪加身份线：玩家不是旁观刺激，而是在扮演一个有目标的人。

#### 3. 低有效获量 + 高 ROI 承接：商业内核题材

这类题材的产品内核可能很能赚钱，但当前题材表达买量困难。典型问题是题材老化、竞争过密、素材疲劳，或者第一眼不够强。

它的问题不是“不赚钱”，而是“买不进人”。

判断重点：

```text
这个高 LTV 内核，能不能换一个更低成本、更强点击的题材外壳？
```

常见做法：

- 内核不变，换高有效获量外壳。
- 把复杂系统压缩成一眼能懂的冲突。
- 用强身份替代抽象系统，例如“经营资源循环”改成“带领幸存者活过极寒夜晚”。
- 用强敌人、强危机或强反差提高首图张力。
- 避开最贵人群，找更细分、更便宜的买量入口。

仙侠如果没有新切口，通常应先放在这一类，而不是直接放在右上角：它商业化强，但传统仙侠买量竞争激烈、素材疲劳高、CPI 压力大。只有当团队能提出新的素材表达、新玩法承接、新人群切口或成本优势时，才可能重新进入右上角。

#### 4. 低有效获量 + 低 ROI 承接：不可立项题材

这类题材在当前市场拿量难，产品内也难回本。它不一定没有审美价值，但不适合作为买量商业立项的主轴。

常见信号：

- 团队能讲很多设定，但用户第一眼没有反应。
- 没有明确竞品，也没有替代消费场景。
- 题材不产生玩法，只产生背景故事。
- 付费点和题材承诺互相冲突。
- 即使小量点击不错，也看不到放量和回本空间。

处理方式：

- 放弃作为主导题材。
- 拆出其中一个强元素，作为辅助题材。
- 回到体验承诺，重新寻找身份、冲突或行为幻想。

### 泡泡大小的含义

图里的泡泡大小不等于“越大越好”。在买量视角下，大泡泡同时意味着机会和压力。

| 含义 | 大泡泡意味着 | 小泡泡意味着 |
|---|---|---|
| 市场盘子 | 潜在人群更大，验证材料更多 | 人群更窄，需要更精确命中 |
| 内容供给 | 素材、竞品、用户认知更充分 | 可借鉴少，教育成本高 |
| 竞争拥挤 | 红海程度更高，买量更贵 | 可能是蓝海，也可能是伪需求 |

所以象限判断不能只看位置，还要看泡泡大小：

- 右上大泡泡：市场最大，但必须证明差异化和成本优势。
- 右上小泡泡：可能是细分机会，也可能是样本不足，需要快速素材验证。
- 左上大泡泡：产品内核被验证，但买量外壳老化，需要重包装。
- 右下大泡泡：短期流量强，但要警惕“热闹不赚钱”。

### 题材迁移策略

四象限不是给题材判死刑，而是告诉团队应该怎么改。

| 起点 | 目标 | 调整方式 |
|---|---|---|
| 高有效获量 + 低 ROI 承接 | 往上移动 | 增加成长线、收集线、基地线、角色线、社交线，提高 LTV |
| 低有效获量 + 高 ROI 承接 | 往右移动 | 换外壳、换情绪、换人群、换素材表达，降低获量成本 |
| 低有效获量 + 低 ROI 承接 | 先找单点 | 保留最强情绪或最强玩法，重组三要素 |
| 高有效获量 + 高 ROI 承接 | 找防守位 | 降低 scope，避开头部正面竞争，建立素材迭代优势 |

### 四象限使用流程

```text
1. 先用题材三要素定义候选题材。
2. 判断题材天然能力：是否有视觉符号、情绪刺激、成长空间、付费承接。
3. 做市场校正：竞争密度、素材疲劳、CPI、人群错配、内容成本。
4. 说明我方进入方式：新表达、新人群、新玩法承接或成本优势。
5. 计算有效获量能力和 ROI 承接能力。
6. 把候选题材放入四象限。
7. 对低项做迁移，不直接用总分平均。
8. 迁移后回到题材定义卡，检查题材是否仍然自洽。
```

### 四象限评分表

每项按 1 到 5 分评分。

| 维度 | 评分问题 |
|---|---|
| 获量-需求规模 | 当前市场是否有真实可触达需求？ |
| 获量-第一眼刺激 | 第一张图是否有强识别、强情绪、强冲突？ |
| 获量-素材新鲜度 | 和现有买量素材相比，是否有一眼可见的新表达？ |
| 获量-竞争压力 | 同题材、同人群、同表达是否已经高度拥挤？ |
| 获量-CPI 风险 | 预估买量成本是否仍在产品可承接范围内？ |
| ROI-LTV 上限 | 题材是否支撑足够高的长期价值？ |
| ROI-留存深度 | 题材是否能形成持续目标和重复行为？ |
| ROI-付费自然度 | 付费是否自然服务成长、效率、收集、外观或身份？ |
| ROI-内容效率 | 题材内容是否能持续生产，且成本可控？ |
| ROI-回本压力 | 预期回本周期是否匹配团队资金和制作能力？ |

判定：

```text
有效获量均分 >= 4：高有效获量
ROI 承接均分 >= 4：高 ROI 承接
任一均分 < 3：对应轴存在硬伤，不能直接进入立项
题材天然能力高但市场校正后低：题材成立，市场不成立
```

### 仙侠判断示例

不能这样判断：

```text
仙侠商业化强，所以值得做。
```

应该这样判断：

```text
仙侠天然商业化强：境界、功法、法宝、灵宠、宗门、外观、战力都能付费。
但当前买量竞争强：素材同质化、用户疲劳、CPI 高。
所以默认不能直接立项。
除非我们能证明新的有效获量入口。
```

可进入条件：

- 仙侠 + 新情绪：反常识、喜剧、恐怖、治愈、爽感反差。
- 仙侠 + 新玩法承接：塔防、经营、肉鸽、放置、合成、割草。
- 仙侠 + 新人群：更轻量、更女性向、更休闲、更年轻或更垂直。
- 仙侠 + 新素材叙事：不是飞剑渡劫老桥段，而是一眼不同的新冲突。
- 仙侠 + 成本优势：美术、内容、投放或迭代效率明显优于同类。

结论：仙侠不是不能做，而是不能用“仙侠商业化强”作为立项理由。必须证明“当前买量仍能进人，并且进来后能回本”。

## 题材融合方法

当单一题材不足以形成新鲜感时，可以做题材融合。但融合不是堆标签，而是用辅助题材增强主导题材的表现力。

### 四条原则

| 原则 | 要求 | 失败表现 |
|---|---|---|
| 主次分明 | 必须有一个主导题材，另一个只是增强变量 | 两套审美互相抢戏 |
| 情绪兼容 | 核心情绪一致，或形成可控反差 | 玩家不知道该紧张、爽、害怕还是放松 |
| 逻辑自洽 | 两个题材之间有底层桥梁 | 只是把名词硬贴在一起 |
| 受众重叠 | 两边受众有交集或互补 | 原受众排斥，新受众不认可 |

### 四种融合方式

#### 1. 保持内核换皮

逻辑不变，美学替换。适合降低理解门槛，同时增加新鲜感。

```text
宫廷斗争 → 宫廷修仙
侦探推理 → 魔法侦探
武侠江湖 → 赛博武侠
```

判断重点：换皮后，原本的冲突和目标是否仍然成立。

#### 2. 机制置换

功能不变，实现方式替换。适合把题材转化成玩法差异。

```text
悟道升级 → 系统任务刷经验
单一古墓 → 无限流副本
商业谈判 → 读心/预知博弈
```

判断重点：置换是否改变玩家行为，而不只是改名词。

#### 3. 情绪反差

让两个基调发生可控冲突，制造记忆点。

```text
僵尸 + 喜剧
童话 + 暗黑
末日 + 治愈日常
```

判断重点：反差是否服务主情绪，而不是打断主情绪。

#### 4. 世界观扩容

在单层世界外增加第二层解释，制造深度和反转。

```text
历史神话 → 外星文明干预
现实娱乐 → 虚拟程序
驱魔事件 → 科学异常或心理投射
```

判断重点：第二层解释是否增强玩法和叙事，而不是只增加设定复杂度。

### 融合步骤

```text
1. 定锚点：确认主导题材、核心受众、主要情绪
2. 找变量：选择能增加亮点的辅助题材
3. 建桥梁：解释两个题材为什么能够共存
4. 换元素：把辅助题材植入关键玩法、角色、敌人、资源或目标
5. 验情绪：确认融合后的情绪比例和表达顺序
```

建议比例：

```text
主导题材 70% + 辅助题材 30%
```

这个比例不是数学规则，而是提醒团队：辅助题材应该放大主体验，不应该抢走主体验。

## 题材选择评分表

四象限用于判断题材在当前买量市场里的进入位置，评分表用于判断题材方案是否完整。每个维度按 1 到 5 分评分。低于 3 分的维度必须重做，不建议用总分掩盖单项硬伤。

| 维度 | 1 分 | 3 分 | 5 分 |
|---|---|---|---|
| 第一眼识别 | 看不出题材 | 能看出大类 | 能看出具体体验与冲突 |
| 情绪清晰度 | 情绪混乱 | 有主情绪但不稳定 | 一眼能预期核心感受 |
| 玩法导向 | 只是美术包装 | 能解释部分玩法 | 自然生成核心玩法 |
| 叙事牵引 | 没有明确目标 | 有目标但不强 | 身份、目标、冲突都清楚 |
| 有效获量能力 | 市场红海且素材无差异 | 有题材识别但获量成本不确定 | 需求明确、表达新鲜、CPI 可承接 |
| ROI 承接能力 | 难以设计长线和付费 | 有部分成长或内容扩展 | LTV、留存、付费、内容效率都能承接买量成本 |
| 受众验证 | 不知道给谁 | 有模糊受众 | 有明确竞品与购买人群 |
| Scope 可控 | 需要大量内容才成立 | 可以砍到中等版本 | 最小可售版本也能表达 |
| 融合自洽 | 缝合感强 | 有桥梁但需解释 | 形异神同，逻辑自然 |

决策规则：

```text
单项低于 3：不通过
有效获量能力低于 3：先做市场进入切口或卖相迁移，不进入完整立项
ROI 承接能力低于 3：先做长线承接和 LTV 假设，不进入完整立项
总分 27 以下：不建议立项
总分 28-35：可继续补验证
总分 36 以上：进入原型与素材验证
```

## 常见陷阱

### 1. 把文化标签当题材

```text
错误：维多利亚、二次元、赛博朋克、修仙
正确：维多利亚哥特都市 + 血疗猎杀 + 揭开古神真相
```

标签只能告诉玩家“像什么”，不能告诉玩家“玩什么”。

### 2. 题材和玩法断开

如果题材换成别的皮，玩法仍然完全成立，说明题材没有参与系统设计。

### 3. 融合没有主次

两个题材都想当主角，结果会导致情绪、视觉、叙事和玩法都不稳定。

### 4. 用复杂设定掩盖卖相不足

玩家在点击前看不到完整世界观，只会看标题、封面、截图和短视频。

### 5. 受众只存在于想象中

题材选择必须找到已有受众、竞品或内容消费习惯。没有验证时，只能说“假设有人喜欢”，不能说“市场成立”。

## 最终检查清单

- [ ] 题材能用“环境/时代 + 文化范式/战斗体系 + 叙事”说清。
- [ ] 玩家一眼能知道自己是谁、在哪里、要做什么。
- [ ] 题材能自然导出核心玩法，而不是只做美术包装。
- [ ] 主情绪清晰，辅助情绪不会打断它。
- [ ] 核心辨识词能进入标题、封面、截图和短视频。
- [ ] 已完成四象限判断，明确有效获量能力与 ROI 承接能力的短板。
- [ ] 已做市场校正：竞争密度、素材疲劳、CPI、人群错配和内容成本都已评估。
- [ ] 已说明我方进入切口：新表达、新人群、新玩法承接或成本优势。
- [ ] 有明确受众和可参考竞品。
- [ ] 最小可售版本也能表达题材承诺。
- [ ] 若做融合，主导题材、辅助题材和逻辑桥梁都清楚。
