## 一句话结论

题材选择不是挑一个流行标签，而是选择一个玩家一眼能理解、能产生情绪预期，并且能自然导向玩法的体验承诺。

```text
题材 = 环境/时代 + 文化范式/战斗体系 + 叙事
有效题材 = 能承接情绪 + 能展开玩法 + 能被卖相表达
题材融合 = 主导题材 + 辅助题材 + 统一情绪与逻辑桥梁
```

## 核心定义

### 题材

题材是游戏依托的文化传统、美学呈现与叙事模型的集合。它决定玩家第一次看到游戏时，会如何判断这个世界、这个角色、这个冲突和自己将要获得的体验。

题材不是单个标签：

```text
弱表达：维多利亚时代、二次元、末日、修仙
强表达：末日丧尸潮 + 枪械 + 求生
```

单个标签只能说明外观或文化来源，不能说明玩家要做什么、为什么做、用什么方式做。

### 题材三要素

一个可用于立项判断的题材，至少要同时回答三个问题：

| 要素 | 回答的问题 | 作用 |
|---|---|---|
| 环境/时代 | 事情发生在哪里、什么时候、世界长什么样 | 建立第一眼识别与空间想象 |
| 文化范式/战斗体系 | 这个世界如何运作，冲突如何解决 | 建立能力规则、符号系统与玩法接口 |
| 叙事 | 玩家是谁，要完成什么，核心矛盾是什么 | 建立目标、身份、情绪和长期牵引 |

题材表达公式：

```text
在【环境/时代】中，通过【文化范式/战斗体系】，完成【叙事目标】的【体验旅程】
```

例：

```text
在现代都市中，通过枪械技能体系防御阵地，对抗僵尸潮的射击爽游。
在冰雪末世中，通过炉火资源体系建设家园，带领社群活下去的策略生存之旅。
在中式民俗诡谲空间中，通过轮回因果体系破解谜题，逃离异常空间的破局之旅。
```

## 为什么题材选择重要

题材承担的是立项早期的三层任务：

| 层级 | 题材要解决的问题 | 失败表现 |
|---|---|---|
| 认知 | 玩家能不能一眼看懂这是什么 | 只有美术风格，没有清晰体验 |
| 情绪 | 玩家是否立刻知道自己会获得什么感受 | 设定很多，但没有明确期待 |
| 玩法 | 题材是否自然导向可玩的系统 | 题材和玩法分离，只是换皮 |

题材弱，常见问题不是“不新”，而是：

- 玩家看不出自己要扮演谁。
- 玩家看不出核心冲突是什么。
- 玩家看不出玩法为什么必须发生在这个题材里。
- 美术元素很丰富，但没有统一的情绪方向。
- 题材标签很热，但无法形成截图、短视频、广告素材里的明确卖相。

## 题材选择流程

### 步骤 1：先定体验承诺

不要先问“现在什么题材火”，先问玩家要获得什么体验。

可选起点：

| 起点 | 典型问题 | 示例 |
|---|---|---|
| 身份幻想 | 玩家想成为谁 | 幸存者领袖、猎魔人、粮草官、太空列车成员 |
| 情境幻想 | 玩家想进入什么处境 | 极寒末世、尸潮围城、轮回逃生、海底探索 |
| 冲突幻想 | 玩家想解决什么矛盾 | 生存 vs 道德、秩序 vs 混乱、自由 vs 宿命 |
| 行为幻想 | 玩家想反复做什么 | 防守、建设、探索、成长、收集、经营 |

输出一句话：

```text
这是一款让玩家体验【身份/情境/冲突/行为】的游戏。
```

### 步骤 2：拆成题材三要素

把体验承诺拆成可执行的题材结构：

```text
环境/时代：
文化范式/战斗体系：
叙事：
核心辨识词：
一句话题材定义：
```

检查标准：

- 环境/时代能不能形成画面。
- 文化范式/战斗体系能不能形成系统。
- 叙事能不能形成目标。
- 核心辨识词能不能出现在封面、截图、短视频和标题里。

### 步骤 3：判断题材是否承上情绪

题材必须服务主情绪。情绪不是额外包装，而是玩家选择题材时真正购买的预期。

常见匹配：

| 主情绪 | 适配题材方向 | 风险 |
|---|---|---|
| 求生压力 | 末日、灾难、荒野、深海、地牢 | 如果资源压力不可见，会变成普通冒险 |
| 爽感释放 | 枪械、技能、割草、超能力、复仇 | 如果敌人和反馈弱，爽感不成立 |
| 敬畏探索 | 克苏鲁、神话、太空、古文明、深海 | 如果未知感不足，会变成普通设定 |
| 治愈放松 | 田园、花园、料理、宠物、自然 | 如果目标压力过强，会破坏放松 |
| 成长满足 | 修仙、英雄养成、职业进阶、基地建设 | 如果成长不可见，会只剩数值 |
| 紧张破局 | 轮回、密室、无限流、潜入、逃生 | 如果失败代价不清晰，紧张感不足 |

判断句：

```text
这个题材是否让玩家在看到第一张图时，就预期到对应情绪？
```

### 步骤 4：判断题材是否启下玩法

题材不能只停留在视觉层。它要自然生成玩法规则。

| 题材要素 | 应导出的玩法问题 |
|---|---|
| 环境/时代 | 地形、空间、资源、天气、时间是否影响玩法 |
| 文化范式/战斗体系 | 能力来源、敌我关系、成长方式、战斗方式是否明确 |
| 叙事 | 玩家目标、失败代价、阶段推进、长期追求是否明确 |

反向检查：

```text
如果把题材换成另一个皮，玩法是否几乎不变？
```

如果答案是“几乎不变”，题材只是包装，不是立项支柱。

### 步骤 5：验证卖相表达

题材要能被市场素材表达出来。

最小验证：

| 表达物 | 必须回答的问题 |
|---|---|
| 标题 | 能不能让玩家知道大概题材和体验 |
| 封面 | 能不能出现核心辨识词 |
| 第一张截图 | 能不能看出玩家在做什么 |
| 5 秒短视频 | 能不能展示题材带来的冲突和反馈 |
| 一句话简介 | 能不能说清三要素 |

失败信号：

- 题材需要长篇背景解释才成立。
- 核心卖点无法截图化。
- 广告素材只能展示角色或氛围，看不出玩法。
- 玩家评论会集中问“这游戏到底干嘛的”。

## 题材四象限

题材定义解决的是“它是什么”，四象限解决的是“它值不值得选”。

题材选择不能只看吸量，也不能只看商业化。吸量代表玩家是否愿意点进来，商业化代表玩家是否愿意留下、付费或长期投入。两者需要分开判断，再合并决策。

```text
横轴：吸量能力
纵轴：商业化能力
泡泡大小：题材热度 / 内容供给量 / 竞争拥挤度
```

### 坐标定义

#### 横轴：吸量能力

吸量能力指题材在第一眼素材中的点击能力，主要看玩家是否能快速理解、是否有情绪刺激、是否有传播钩子。

高吸量题材通常具备：

- 核心图像强：丧尸、怪物、美女、神仙、机甲、恐龙、巨大灾害等。
- 情绪刺激强：恐惧、爽感、猎奇、复仇、逆袭、性张力、危机感等。
- 认知门槛低：不需要解释世界观，玩家能立刻知道“发生了什么”。
- 短视频可演示：3 到 5 秒内能看到冲突、反转或反馈。

低吸量题材常见问题：

- 题材理解依赖长文本。
- 第一眼只看到氛围，看不到冲突。
- 视觉符号弱，无法和同类内容拉开距离。
- 玩家需要先懂某个文化背景才会感兴趣。

#### 纵轴：商业化能力

商业化能力指题材能否支撑付费点、长线目标、内容扩展和持续消费。

高商业化题材通常具备：

- 成长空间清楚：角色、装备、基地、宠物、境界、阵营、城市等可以持续升级。
- 内容可扩展：敌人、关卡、地图、角色、皮肤、剧情章节能稳定生产。
- 身份投入强：玩家愿意经营、收集、养成或炫耀自己的结果。
- 付费承接自然：付费点不是硬塞，而是能服务成长、效率、外观、策略或社交身份。

低商业化题材常见问题：

- 只适合一次性观看或短期猎奇。
- 爽点很强，但没有长期追求。
- 题材边界窄，内容很快做尽。
- 付费会破坏题材承诺，例如治愈题材被强行做成高压数值竞争。

### 四个象限

| 象限 | 特征 | 适合策略 | 主要风险 |
|---|---|---|---|
| 高吸量 + 高商业化 | 既容易获得点击，又能承接长期付费和内容扩展 | 优先立项，但必须找差异化切口 | 竞争拥挤，买量成本高，容易被头部压制 |
| 高吸量 + 低商业化 | 素材容易爆，但游戏内难以沉淀长期价值 | 适合轻量产品、短周期验证或作为副题材 | 留存和付费弱，热度消耗快 |
| 低吸量 + 高商业化 | 系统和长线强，但第一眼不容易点击 | 需要强化卖相包装，寻找高吸量外壳 | 做出来可能好玩但没人点 |
| 低吸量 + 低商业化 | 既难获量，也难承接付费 | 通常不建议立项 | 团队容易用“品味”替代市场判断 |

### 象限解释

#### 1. 高吸量 + 高商业化：主战场题材

这是最理想也最拥挤的区域。图中的“僵尸”“仙侠题材”“魔兽世界”都接近这个区域：它们有强视觉符号，也有天然成长、收集、装备、阵营或世界扩展空间。

这类题材适合优先考虑，但不能只因为它大就进入。判断重点不是“这个题材火不火”，而是：

```text
我们能不能在高竞争题材中，找到一个低成本可表达的细分切口？
```

可用切口：

- 换情绪：僵尸不只恐怖，也可以喜剧、治愈、割草、策略防守。
- 换玩法承接：仙侠不只 RPG，也可以经营、塔防、肉鸽、放置。
- 换规模：大世界题材可以缩成一个据点、一条路线、一次轮回、一个职业。
- 换人群：把重度题材做成轻量版本，或把大众题材做成更垂直的深度版本。

立项要求：

- 必须有明确竞品。
- 必须有可截图化差异。
- 必须说明我们不和头部正面硬撞的原因。

#### 2. 高吸量 + 低商业化：爆点题材

这类题材适合拿点击，但不一定适合做长线。很多“气泡内容”会落在这里：看起来新奇、刺激、容易被点开，但玩家看完或玩完后缺少持续目标。

它的问题不是“不吸引”，而是“承接不住”。

判断重点：

```text
这个题材的吸量点，能不能转化为可重复的玩家行为？
```

如果不能，它更适合作为：

- 广告素材外壳。
- 副题材或关卡题材。
- 短周期活动。
- 轻量玩法的前期冷启动包装。

转化方式：

- 给猎奇题材加成长线：收集、升级、图鉴、基地、角色关系。
- 给短期爽点加目标线：关卡推进、Boss 挑战、赛季任务。
- 给强情绪加身份线：玩家不是旁观刺激，而是在扮演一个有目标的人。

不建议做法：

- 只因为素材点击高，就把它当成完整立项。
- 用大量系统强行延长一个本质短促的题材。

#### 3. 低吸量 + 高商业化：内核题材

这类题材的系统价值高，但第一眼不够强。它可能适合长期经营、收集、养成、策略或社交，但题材符号不够刺激，广告和首图吃亏。

它的问题不是“不赚钱”，而是“没人进门”。

判断重点：

```text
这个题材能不能外接一个高吸量表达层，同时保留原本的商业化内核？
```

常见做法：

- 内核不变，换高吸量题材外壳。
- 把复杂系统压缩成一个一眼能懂的冲突。
- 用强身份替代抽象系统，例如“经营资源循环”改成“带领幸存者活过极寒夜晚”。
- 用强敌人或强危机提高首图张力。

这类题材适合用“保持内核换皮”和“机制置换”处理。比如一个资源经营内核，本身吸量弱，但套入冰雪末世、僵尸围城、太空殖民后，点击能力会明显提升。

#### 4. 低吸量 + 低商业化：自嗨题材

这类题材通常只有团队内部兴趣，缺少外部点击，也缺少长期承接。它不一定没有艺术价值，但不适合作为商业立项的主轴。

常见信号：

- 团队能讲很多设定，但玩家第一眼没有反应。
- 没有明确竞品，也没有替代消费场景。
- 题材不产生玩法，只产生背景故事。
- 付费点和题材承诺互相冲突。

处理方式：

- 放弃作为主导题材。
- 拆出其中一个强元素，作为辅助题材。
- 回到体验承诺，重新寻找身份、冲突或行为幻想。

### 泡泡大小的含义

图里的泡泡大小不等于“越大越好”。泡泡应该同时理解为三件事：

| 含义 | 大泡泡意味着 | 小泡泡意味着 |
|---|---|---|
| 需求规模 | 受众更大，验证材料更多 | 受众更窄，需要更精确命中 |
| 内容供给 | 素材、竞品、用户认知更充分 | 可借鉴少，教育成本高 |
| 竞争拥挤 | 红海程度更高，买量更贵 | 可能是机会，也可能是伪需求 |

所以象限判断不能只看位置，还要看泡泡大小：

- 右上大泡泡：市场最大，但必须找差异化和成本优势。
- 右上小泡泡：可能是蓝海机会，也可能是样本不足，需要快速素材验证。
- 左上大泡泡：商业内核已被证明，但题材表达老化，需要重包装。
- 右下大泡泡：短期流量强，但要警惕“热闹不赚钱”。

### 题材迁移策略

四象限不是给题材判死刑，而是告诉团队应该怎么改。

| 起点 | 目标 | 调整方式 |
|---|---|---|
| 高吸量 + 低商业化 | 往上移动 | 增加成长线、收集线、基地线、角色线、社交线 |
| 低吸量 + 高商业化 | 往右移动 | 增加强视觉符号、强敌人、强危机、强身份、强冲突 |
| 低吸量 + 低商业化 | 先找单点 | 保留最强情绪或最强玩法，重组三要素 |
| 高吸量 + 高商业化 | 找切口 | 降低 scope，避开头部正面竞争，做细分表达 |

### 四象限使用流程

```text
1. 先用题材三要素定义候选题材。
2. 分别给吸量能力和商业化能力打分。
3. 把候选题材放入四象限。
4. 看泡泡大小，判断市场规模与竞争压力。
5. 对低项做迁移，不直接用总分平均。
6. 迁移后再回到题材定义卡，检查题材是否仍然自洽。
```

### 四象限评分表

每项按 1 到 5 分评分。

| 维度 | 评分问题 |
|---|---|
| 吸量-视觉符号 | 第一张图是否有强识别物？ |
| 吸量-情绪刺激 | 玩家是否能立刻感到危险、爽感、猎奇、欲望或好奇？ |
| 吸量-理解成本 | 不看说明时，玩家能否知道大概在发生什么？ |
| 吸量-短视频表达 | 3 到 5 秒内能否展示冲突和反馈？ |
| 商业化-成长空间 | 是否天然支持角色、装备、基地、境界、阵营等成长？ |
| 商业化-内容扩展 | 是否能持续生产敌人、关卡、地图、角色、皮肤或剧情？ |
| 商业化-付费承接 | 付费是否能自然服务效率、外观、策略、收集或身份？ |
| 商业化-长期目标 | 玩家是否有持续追求，而不是只体验一次刺激？ |

判定：

```text
吸量均分 >= 4：高吸量
商业化均分 >= 4：高商业化
任一均分 < 3：对应轴存在硬伤，需要先迁移再评估
```

## 题材融合方法

当单一题材不足以形成新鲜感时，可以做题材融合。但融合不是堆标签，而是用辅助题材增强主导题材的表现力。

### 四条原则

| 原则 | 要求 | 失败表现 |
|---|---|---|
| 主次分明 | 必须有一个主导题材，另一个只是增强变量 | 两套审美互相抢戏 |
| 情绪兼容 | 核心情绪一致，或形成可控反差 | 玩家不知道该紧张、爽、害怕还是放松 |
| 逻辑自洽 | 两个题材之间有底层桥梁 | 只是把名词硬贴在一起 |
| 受众重叠 | 两边受众有交集或互补 | 原受众排斥，新受众不认可 |

### 四种融合方式

#### 1. 保持内核换皮

逻辑不变，美学替换。适合降低理解门槛，同时增加新鲜感。

```text
宫廷斗争 → 宫廷修仙
侦探推理 → 魔法侦探
武侠江湖 → 赛博武侠
```

判断重点：换皮后，原本的冲突和目标是否仍然成立。

#### 2. 机制置换

功能不变，实现方式替换。适合把题材转化成玩法差异。

```text
悟道升级 → 系统任务刷经验
单一古墓 → 无限流副本
商业谈判 → 读心/预知博弈
```

判断重点：置换是否改变玩家行为，而不只是改名词。

#### 3. 情绪反差

让两个基调发生可控冲突，制造记忆点。

```text
僵尸 + 喜剧
童话 + 暗黑
末日 + 治愈日常
```

判断重点：反差是否服务主情绪，而不是打断主情绪。

#### 4. 世界观扩容

在单层世界外增加第二层解释，制造深度和反转。

```text
历史神话 → 外星文明干预
现实娱乐 → 虚拟程序
驱魔事件 → 科学异常或心理投射
```

判断重点：第二层解释是否增强玩法和叙事，而不是只增加设定复杂度。

### 融合步骤

```text
1. 定锚点：确认主导题材、核心受众、主要情绪
2. 找变量：选择能增加亮点的辅助题材
3. 建桥梁：解释两个题材为什么能够共存
4. 换元素：把辅助题材植入关键玩法、角色、敌人、资源或目标
5. 验情绪：确认融合后的情绪比例和表达顺序
```

建议比例：

```text
主导题材 70% + 辅助题材 30%
```

这个比例不是数学规则，而是提醒团队：辅助题材应该放大主体验，不应该抢走主体验。

## 题材选择评分表

四象限用于判断题材的市场位置，评分表用于判断题材方案是否完整。每个维度按 1 到 5 分评分。低于 3 分的维度必须重做，不建议用总分掩盖单项硬伤。

| 维度 | 1 分 | 3 分 | 5 分 |
|---|---|---|---|
| 第一眼识别 | 看不出题材 | 能看出大类 | 能看出具体体验与冲突 |
| 情绪清晰度 | 情绪混乱 | 有主情绪但不稳定 | 一眼能预期核心感受 |
| 玩法导向 | 只是美术包装 | 能解释部分玩法 | 自然生成核心玩法 |
| 叙事牵引 | 没有明确目标 | 有目标但不强 | 身份、目标、冲突都清楚 |
| 吸量能力 | 第一眼无刺激 | 有题材识别但传播弱 | 视觉符号、情绪刺激、短视频表达都强 |
| 商业化能力 | 难以设计长线和付费 | 有部分成长或内容扩展 | 成长、内容、付费、长期目标都自然 |
| 受众验证 | 不知道给谁 | 有模糊受众 | 有明确竞品与购买人群 |
| Scope 可控 | 需要大量内容才成立 | 可以砍到中等版本 | 最小可售版本也能表达 |
| 融合自洽 | 缝合感强 | 有桥梁但需解释 | 形异神同，逻辑自然 |

决策规则：

```text
单项低于 3：不通过
吸量能力低于 3：先做卖相迁移，不进入完整立项
商业化能力低于 3：先做长线承接，不进入完整立项
总分 27 以下：不建议立项
总分 28-35：可继续补验证
总分 36 以上：进入原型与素材验证
```

## 常见陷阱

### 1. 把文化标签当题材

```text
错误：维多利亚、二次元、赛博朋克、修仙
正确：维多利亚哥特都市 + 血疗猎杀 + 揭开古神真相
```

标签只能告诉玩家“像什么”，不能告诉玩家“玩什么”。

### 2. 题材和玩法断开

如果题材换成别的皮，玩法仍然完全成立，说明题材没有参与系统设计。

### 3. 融合没有主次

两个题材都想当主角，结果会导致情绪、视觉、叙事和玩法都不稳定。

### 4. 用复杂设定掩盖卖相不足

玩家在点击前看不到完整世界观，只会看标题、封面、截图和短视频。

### 5. 受众只存在于想象中

题材选择必须找到已有受众、竞品或内容消费习惯。没有验证时，只能说“假设有人喜欢”，不能说“市场成立”。

## 最终检查清单

- [ ] 题材能用“环境/时代 + 文化范式/战斗体系 + 叙事”说清。
- [ ] 玩家一眼能知道自己是谁、在哪里、要做什么。
- [ ] 题材能自然导出核心玩法，而不是只做美术包装。
- [ ] 主情绪清晰，辅助情绪不会打断它。
- [ ] 核心辨识词能进入标题、封面、截图和短视频。
- [ ] 已完成四象限判断，明确吸量能力与商业化能力的短板。
- [ ] 有明确受众和可参考竞品。
- [ ] 最小可售版本也能表达题材承诺。
- [ ] 若做融合，主导题材、辅助题材和逻辑桥梁都清楚。
