# 方法论目录

已沉淀、可复用的方法论文档。只有能指导后续立项、分析或复盘的稳定结论才放入本目录。

| 目录 | 内容 |
|---|---|
| `立项/` | 通用立项框架：发现方向、评估卖相、选题材、拆解外部方法论 |
| `分析/` | 游戏拆解与产品分析框架 |
| `推演案例/` | 方法论在具体方向上的应用实例，可作为推演参考 |
