# 养成/培育方向 — 深化设计

## 核心定位

**底层玩法**：获取生物 → 训练培养 → 进化成长 → 战斗配合 → 获得资源 → 继续培养

**独立玩法验证**：怪物牧场、宝可梦本身就是独立游戏，养成本身有深度

**与操控防守结合**：我养成的生物和我一起守基地，伙伴配合作战

## 两个子方向对比

| 维度 | 方向A：单一深度养成 | 方向B：多生物收集养成 |
|---|---|---|
| 核心体验 | "我养大的那只" | "我的队伍" |
| 情感绑定 | 极强，单一对象 | 分散，但有收集成就感 |
| 养成深度 | 深，单一生物的细节培养 | 广，多种生物的组合 |
| 获取方式 | 孵化/领养一只 | 捕获/召唤多只 |
| 长线驱动 | 进化路线探索 | 收集图鉴完成度 |
| 战斗策略 | 单一伙伴的深度配合 | 多伙伴的组队策略 |
| 参考对标 | 怪物牧场、驯龙高手 | 宝可梦、数码宝贝 |

## 方向A：单一深度养成（怪物牧场模式）

### 核心循环

```
获取阶段
├─ 孵化蛋/领养幼崽
├─ 选择初始属性倾向
└─ 命名建立情感连接
    ↓
日常养成阶段（局外）
├─ 训练小游戏（提升属性）
├─ 喂养选择（影响进化方向）
├─ 互动培养（提升亲密度）
└─ 休息作息（影响成长速度）
    ↓
进化阶段
├─ 达到进化条件
├─ 选择进化分支
└─ 解锁新能力
    ↓
战斗阶段（局内）
├─ 我操控角色防守
├─ 生物伙伴独立战斗
├─ 配合技能触发
└─ 生物能力直接体现
    ↓
战斗结束
├─ 获得训练道具
├─ 获得食物材料
├─ 生物获得经验
└─ 解锁新训练项目
    ↓
回到日常养成
```

### 训练系统（核心玩法）

**设计原则**：训练本身必须可玩，不是"点击→等待→数值增长"

**训练项目类型**：

1. **力量训练**（小游戏：击打木桩）
   - 玩法：按节奏点击，连击提升效果
   - 属性：提升攻击力
   - 副作用：消耗体力，降低敏捷成长

2. **敏捷训练**（小游戏：障碍跑）
   - 玩法：躲避障碍，完成度影响效果
   - 属性：提升速度和闪避
   - 副作用：消耗体力，降低防御成长

3. **智力训练**（小游戏：记忆配对）
   - 玩法：记忆卡牌配对，正确率影响效果
   - 属性：提升技能威力和冷却
   - 副作用：消耗精神，降低体力成长

4. **体质训练**（小游戏：耐力跑）
   - 玩法：保持节奏不断奔跑，坚持时间影响效果
   - 属性：提升生命值和防御
   - 副作用：消耗体力，降低速度成长

5. **综合训练**（小游戏：实战模拟）
   - 玩法：模拟战斗，操作表现影响多项属性
   - 属性：平衡提升多项属性
   - 副作用：消耗大量体力和精神

**训练反馈**：
- 即时反馈：训练完成后立刻看到属性变化
- 视觉反馈：生物体型、颜色、特征随属性变化
- 战斗反馈：下次战斗能直接感受到训练效果

**训练约束**：
- 体力系统：每天有限的训练次数
- 疲劳系统：连续训练同一项目效果递减
- 成长倾向：不同训练影响进化方向

### 喂养系统

**食物类型**：

1. **肉类**：提升力量倾向，影响进化为力量型
2. **果蔬**：提升敏捷倾向，影响进化为速度型
3. **矿物**：提升体质倾向，影响进化为防御型
4. **魔法植物**：提升智力倾向，影响进化为魔法型
5. **综合营养**：平衡提升，影响进化为平衡型

**喂养机制**：
- 饥饿度：定期需要喂养，否则影响训练效果
- 喜好度：不同生物有不同食物偏好
- 成长影响：长期喂养某类食物影响进化方向
- 特殊食物：稀有食物可以解锁特殊进化路线

**食物获取**：
- 战斗掉落：击败敌人获得食物
- 种植系统：可以种植食物（可选）
- 商店购买：用战斗奖励购买

### 互动系统

**互动类型**：

1. **抚摸**：提升亲密度，降低压力
2. **对话**：了解生物状态，获得提示
3. **玩耍**：小游戏，提升心情和亲密度
4. **清洁**：保持卫生，影响健康状态

**亲密度影响**：
- 战斗配合：亲密度高，配合技能威力更强
- 进化选择：亲密度影响某些进化路线解锁
- 特殊能力：高亲密度解锁专属技能

### 进化系统

**进化条件**：
- 等级要求：达到特定等级
- 属性要求：特定属性达到阈值
- 亲密度要求：亲密度达到一定程度
- 特殊条件：使用特殊道具、在特定地点

**进化分支**（以初始形态为例）：

```
初始形态（幼崽）
    ↓
第一次进化（Lv10）
├─ 力量型（攻击力>50）
│   ├─ 战士型（Lv20，力量>100）
│   └─ 狂战士型（Lv20，力量>100 + 低亲密度）
│
├─ 敏捷型（速度>50）
│   ├─ 刺客型（Lv20，速度>100）
│   └─ 游侠型（Lv20，速度>100 + 高亲密度）
│
├─ 体质型（防御>50）
│   ├─ 守护型（Lv20，防御>100）
│   └─ 坦克型（Lv20，防御>100 + 特殊食物）
│
├─ 智力型（魔力>50）
│   ├─ 法师型（Lv20，魔力>100）
│   └─ 召唤师型（Lv20，魔力>100 + 特殊道具）
│
└─ 平衡型（属性均衡）
    ├─ 全能型（Lv20，所有属性>60）
    └─ 特化型（Lv20，根据战斗风格）
```

**进化不可逆**：选择后无法回退，增加决策权重

**进化视觉**：形态、颜色、特效明显变化

### 战斗配合系统

**配合方式**：

1. **伙伴独立战斗**
   - 生物有自己的AI和行为模式
   - 根据进化类型有不同战斗风格
   - 玩家可以下达简单指令（守这里/跟着我/攻击那个）

2. **配合技能**
   - 玩家和生物同时攻击同一目标触发连携
   - 玩家使用技能时，生物自动配合
   - 高亲密度解锁专属配合技

3. **保护机制**
   - 生物生命值降低时会撤退
   - 玩家可以召回生物避免受伤
   - 生物受伤影响下次战斗表现

**战斗中的成长体现**：
- 训练的属性直接影响战斗表现
- 进化形态决定战斗风格
- 亲密度影响配合默契度

### 长线驱动

1. **进化图鉴**：解锁所有进化形态
2. **完美培养**：培养出满属性的生物
3. **重置重养**：重置生物重新探索不同进化路线
4. **多周目**：养成多只不同进化路线的生物

### 包装变体

同样的底层玩法，可以包装成：

1. **怪物牧场**：养成怪物，参加比赛
2. **驯龙高手**：孵化龙蛋，训练飞龙
3. **机甲改造**：组装机甲，调试性能
4. **英雄训练**：培养英雄，提升能力
5. **宠物养成**：领养宠物，日常互动

**包装差异**：
- 怪物牧场：训练是小游戏，进化是生物形态
- 驯龙高手：训练是飞行课程，进化是龙的成长
- 机甲改造：训练是性能测试，进化是部件升级
- 英雄训练：训练是技能修炼，进化是职业转职
- 宠物养成：训练是日常活动，进化是成长阶段

## 方向B：多生物收集养成（宝可梦模式）

### 核心循环

```
收集阶段
├─ 战斗中捕获敌人
├─ 完成任务获得
└─ 特殊事件召唤
    ↓
养成阶段（局外）
├─ 选择培养哪些生物
├─ 投入资源升级
├─ 选择进化方向
└─ 配置队伍组合
    ↓
战斗阶段（局内）
├─ 我操控角色防守
├─ 部署养成的生物队伍
├─ 不同生物协同作战
└─ 生物能力组合效果
    ↓
战斗结束
├─ 获得新生物
├─ 获得养成资源
├─ 解锁新生物类型
└─ 图鉴完成度提升
    ↓
回到收集阶段
```

### 捕获系统

**捕获机制**：

1. **战斗中捕获**
   - 敌人生命值降低到一定程度可以捕获
   - 使用捕获道具（捕获球/封印卷轴）
   - 捕获成功率受敌人状态影响

2. **捕获时机**
   - 不打断战斗节奏：捕获是快速操作
   - 捕获失败不惩罚：可以继续战斗
   - 捕获成功有即时反馈：特效+音效

3. **捕获策略**
   - 稀有敌人捕获难度高
   - 特定条件下捕获解锁特殊形态
   - 捕获道具有限，需要选择优先级

### 收集系统

**生物分类**：

1. **按属性分类**
   - 火系：高攻击，克制冰系
   - 水系：高防御，克制火系
   - 风系：高速度，克制土系
   - 土系：高生命，克制雷系
   - 雷系：高爆发，克制水系

2. **按稀有度分类**
   - 普通：容易捕获，基础能力
   - 稀有：较难捕获，特殊能力
   - 史诗：很难捕获，强力能力
   - 传说：极难捕获，独特能力

3. **按功能分类**
   - 输出型：高伤害，低防御
   - 坦克型：高生命，低伤害
   - 辅助型：提供增益/减益
   - 控制型：限制敌人行动

**图鉴系统**：
- 记录所有捕获过的生物
- 显示未捕获生物的剪影
- 图鉴完成度是长线目标
- 完成特定图鉴解锁奖励

### 养成系统（简化版）

**与方向A的区别**：
- 方向A：单一生物的深度养成（训练小游戏、喂养、互动）
- 方向B：多生物的广度养成（升级、进化、装备）

**养成方式**：

1. **升级**：投入经验道具提升等级
2. **进化**：达到条件后进化（每种生物1-2条进化路线）
3. **装备**：为生物配置装备提升能力
4. **技能**：学习新技能或替换技能

**养成深度**：
- 单个生物的养成深度低于方向A
- 但养成多个生物的组合策略有深度

### 队伍组合系统

**队伍配置**：
- 战斗前选择3-5只生物组成队伍
- 不同生物组合产生不同效果
- 属性克制关系影响战斗表现

**组合效果**：

1. **属性共鸣**：同属性生物组队提升该属性伤害
2. **功能互补**：输出+坦克+辅助的经典组合
3. **特殊组合**：特定生物组合解锁特殊效果

**战斗中的队伍管理**：
- 生物自动战斗，玩家操控角色
- 玩家可以切换生物出战顺序
- 生物受伤可以替换

### 长线驱动

1. **图鉴完成度**：收集所有生物
2. **稀有生物**：捕获传说级生物
3. **完美队伍**：组建最强组合
4. **多套队伍**：针对不同关卡配置不同队伍

### 包装变体

同样的底层玩法，可以包装成：

1. **宝可梦模式**：捕获宝可梦，组队战斗
2. **卡牌召唤**：召唤卡牌，组建卡组
3. **英雄召唤**：召唤英雄，组建队伍
4. **异兽图鉴**：捕获异兽，收集图鉴
5. **机甲收集**：收集机甲，组建舰队

### 循环框架设计（基于 CM 底层逻辑）

**参考案例**：Coin Master 的核心循环设计

**CM 底层逻辑提取**：
1. 核心资源双循环：体力 → Slot → 金币 → 建岛 → 体力
2. 反比控制新鲜感：机台反比（攻击/掠夺/突袭）打破单一循环
3. 复玩性来源：数值膨胀 + 反比事件随机性 + 社交互动
4. 系统设计目的：解决核心循环运转中的具体问题

**宝可梦养成线的适配**：

| CM 设计 | 底层逻辑 | 宝可梦养成适配 | 设计目的 |
|---------|---------|---------------|---------|
| Slot | 核心体验触发器 + 资源转化 | 防守战斗（操控宝可梦） | 让玩家在战斗中获得培育资源 |
| 体力 → 金币 | 资源流转 | 精力 → 培育点 | 建立基础资源循环 |
| 建岛 | 资源投入 + 成长 | 培育/选择宝可梦 | 让玩家投入资源提升战斗能力 |
| 金币 → 体力 | 循环闭合 | 培育点 → 解锁新宝可梦 → 提升防守效率 → 精力回流 | 形成正反馈循环 |
| 机台反比 | 打破单一循环 | 敌人波次反比（普通波/精英波/Boss波） | 提供新鲜感和挑战感 |
| 卡牌收集 | 解决"不愿建岛"问题 | 图鉴收集 + 进化树 | 解决"不愿培育"问题，提供长期目标 |
| 攻击/掠夺选择 | 策略深度 | 宝可梦选择 + 技能释放时机 | 提供策略决策空间 |

**核心循环**：

```
精力 → 防守关卡（操控宝可梦战斗）→ 培育点 → 培育/选择宝可梦 → 解锁新技能/新宝可梦 → 提升防守能力 → 精力回流
```

**反比控制层**（对应 CM 的机台反比）：

| 波次类型 | 概率 | 特征 | 培育点收益 | 设计目的 |
|---------|------|------|-----------|---------|
| 普通波 | 60% | 3-5 只弱敌人，容易击败 | 5 培育点 | 保底体验，确保循环不断 |
| 精英波 | 30% | 1-2 只强敌人，需要技能配合 | 15 培育点 | 提供挑战感，奖励技能使用 |
| Boss波 | 10% | 1 只Boss，需要多只宝可梦配合 | 50 培育点 + 新宝可梦 | 提供惊喜感，奖励团队配合 |

**循环闭合机制**：
- 培育到一定等级的宝可梦解锁"精力回流"能力
- 例如：妙蛙草等级 10 → 每次防守战斗后返还 1 精力
- 高等级宝可梦可以更高效地完成防守，获得更多培育点

**复玩性设计**：

| 维度 | CM 做法 | 宝可梦养成适配 |
|------|---------|---------------|
| 数值膨胀 | 金币/体力指数增长 | 培育点需求递增，但防守收益同步增长 |
| 随机性 | 机台反比 | 敌人波次反比 + 宝可梦技能暴击 |
| 策略深度 | 攻击/掠夺选择 | 宝可梦选择 + 技能释放时机 |
| 长期目标 | 建岛收集 | 图鉴完成度 + 进化树解锁 |

### 10 分钟体验设计（宝可梦模式）

**设计原则**：
- 让玩家完整走完一次核心循环
- 体验到反比事件带来的新鲜感
- 建立"防守战斗 → 培育 → 提升防守能力"的正反馈

#### 时间线

**0:00 - 2:00 | 引导：建立核心循环认知**

- 玩家获得初始宝可梦（妙蛙种子，等级 1）+ 10 点精力
- 引导：消耗 3 精力进入第一次防守关卡
- **防守战斗**：
  - 3 只小拉达从左侧进攻
  - 引导玩家点击妙蛙种子 → 选择"撞击"技能 → 点击小拉达
  - 妙蛙种子自动攻击，击败 3 只小拉达
  - 战斗结束，获得 5 培育点
- **反馈**：培育点入账动画，显示"可以培育妙蛙种子了"

**2:00 - 3:30 | 培育：建立成长正反馈**

- 引导：消耗 5 培育点培育妙蛙种子 → 等级 1 → 2
- **反馈**：
  - 妙蛙种子立绘变化（体型变大）
  - 解锁新技能"藤鞭"（远程攻击，冷却 3 秒）
  - 属性提升：HP 20 → 30，攻击 5 → 8
- 引导：再次消耗 3 精力进入第二次防守关卡

**3:30 - 5:00 | 第一次反比事件：精英波**

- **防守战斗**：
  - 触发精英波：1 只穿山鼠（HP 50，防御高）
  - 玩家尝试用"撞击"攻击 → 伤害很低（只打掉 5 HP）
  - 引导：使用"藤鞭"（远程攻击，伤害 15）→ 3 次攻击击败穿山鼠
  - 战斗结束，获得 15 培育点
- **反馈**：大量培育点入账动画，强化"精英波 = 高回报"认知

**5:00 - 6:30 | 培育深化：解锁进化路径**

- 玩家消耗 15 培育点继续培育妙蛙种子 → 等级 2 → 5
- 等级 5 时触发进化提示："妙蛙种子可以进化为妙蛙草，需要 20 培育点"

#### 数值平衡验证

**资源流转**：
- 总精力消耗：3 + 3 + 3 + 3 = 12 精力（初始 10 + 自然恢复 2）
- 总培育点获得：5 + 15 + 50 + 8 = 78 培育点
- 总培育点消耗：5 + 15 + 20 + 5 = 45 培育点
- 剩余培育点：33 培育点（实际设计为 13，需要调整）

**调整建议**：
- 进化消耗从 20 提升到 30 培育点
- 或者降低 Boss 波收益从 50 到 40 培育点
- 确保玩家在 10 分钟后有"还需要继续战斗"的动力

**战斗次数**：
- 4 次防守战斗，平均 2.5 分钟一次
- 符合"防守是核心体验"的定位
- 培育环节不抢占过多时间

**进化时机**：
- 第 6.5 分钟进化，时机合适
- 玩家在进化前体验了基础战斗
- 进化后立刻体验到能力提升

## 两个方向的选择建议

**选择方向A（单一深度养成）如果**：
- 想要强情感绑定体验
- 愿意投入深度养成系统开发
- 目标受众喜欢养成类游戏
- 有能力做好训练小游戏

**选择方向B（多生物收集养成）如果**：
- 想要收集驱动的长线目标
- 愿意投入大量生物内容制作
- 目标受众喜欢收集类游戏
- 有能力做好捕获和组合系统

**混合方案**：
- 核心是收集（方向B），但对少数核心生物有深度养成（方向A）
- 例如：主角有一只深度养成的伙伴，其他生物是收集养成
- 平衡情感绑定和收集驱动
