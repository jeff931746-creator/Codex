# 收集/图鉴方向 — 深化设计

## 核心定位

**底层玩法**：捕获敌人 → 收集图鉴 → 组建队伍 → 战斗使用 → 捕获新敌人

**独立玩法验证**：收集本身就是驱动力（宝可梦图鉴、怪物猎人图鉴）

**与操控防守结合**：我捕获的敌人为我而战，我操控角色+部署收集单位

## 与养成/培育的核心区别

| 维度 | 养成/培育 | 收集/图鉴 |
|---|---|---|
| 核心驱动 | 情感绑定（"我养大的那只"） | 收集完成度（"我收集了所有种类"） |
| 深度 vs 广度 | 深度养成少数单位 | 广度收集大量单位 |
| 单位获取 | 孵化/领养（少量） | 捕获/召唤（大量） |
| 单位培养 | 深度训练、进化、互动 | 简单升级、装备 |
| 长线目标 | 完美培养、进化路线探索 | 图鉴完成度、稀有收集 |
| 情感强度 | 强（单一对象深度绑定） | 弱（批量收集） |
| 策略深度 | 单位深度配合 | 队伍组合策略 |

**核心区别总结**：
- 养成/培育：少而精，深度情感，单一伙伴
- 收集/图鉴：多而全，收集驱动，队伍组合

## 核心循环

```
捕获阶段（战斗中）
├─ 战斗中遇到敌人
├─ 降低敌人生命值
├─ 使用捕获道具
└─ 捕获成功加入图鉴
    ↓
收集整理阶段（局外）
├─ 查看图鉴完成度
├─ 选择培养哪些单位
├─ 简单升级和装备
└─ 组建战斗队伍
    ↓
队伍配置阶段
├─ 选择出战单位（3-5只）
├─ 配置单位装备
├─ 规划队伍组合
└─ 设置单位行为
    ↓
战斗阶段（局内）
├─ 我操控角色防守
├─ 部署收集的单位
├─ 单位自动战斗
└─ 队伍协同效果
    ↓
战斗结束
├─ 遇到新敌人类型
├─ 获得捕获道具
├─ 获得升级资源
└─ 图鉴完成度提升
    ↓
回到捕获阶段
```

## 捕获系统（核心玩法）

**设计原则**：捕获不能打断战斗节奏，必须是流畅的战斗操作

### 捕获机制

**方案A：生命值捕获**

1. **捕获条件**
   - 敌人生命值低于30%可以捕获
   - 生命值越低，捕获成功率越高
   - 稀有敌人捕获阈值更低（10%）

2. **捕获操作**
   - 按捕获键（快捷键）
   - 投掷捕获道具（捕获球/封印卷轴）
   - 捕获动画短暂（1-2秒）
   - 成功/失败即时反馈

3. **捕获成功率**
   - 基础成功率 = (1 - 敌人当前生命值%) × 100%
   - 稀有度修正：普通×1.0，稀有×0.7，史诗×0.5，传说×0.3
   - 道具加成：高级捕获道具提升成功率
   - 状态加成：敌人异常状态提升成功率

4. **捕获失败**
   - 失败不惩罚，可以继续战斗
   - 失败消耗捕获道具
   - 可以再次尝试捕获

**方案B：击杀捕获**

1. **捕获条件**
   - 使用捕获道具击杀敌人
   - 击杀后自动捕获
   - 不需要判断成功率

2. **捕获操作**
   - 装备捕获武器（捕获网枪/封印法杖）
   - 使用捕获武器击杀敌人
   - 击杀后自动加入图鉴

3. **捕获限制**
   - 捕获武器伤害较低
   - 捕获武器有弹药限制
   - 稀有敌人需要特殊捕获武器

4. **优势**
   - 不打断战斗节奏
   - 不需要判断成功率
   - 操作更流畅

**推荐方案B**：击杀捕获更流畅，不打断战斗节奏

### 捕获道具

**道具类型**：

1. **基础捕获网**
   - 捕获普通敌人
   - 无限使用但冷却时间长
   - 伤害低

2. **高级捕获网**
   - 捕获稀有敌人
   - 有使用次数限制
   - 伤害中等

3. **传说封印**
   - 捕获传说敌人
   - 稀有道具，难以获得
   - 伤害高

**道具获取**：
- 战斗奖励
- 商店购买
- 任务奖励
- 制作系统

### 捕获反馈

**视觉反馈**：
- 捕获成功：特效+音效+图鉴弹窗
- 新种类捕获：特殊动画+图鉴解锁提示
- 稀有捕获：更华丽的特效

**即时奖励**：
- 捕获后立刻可以使用
- 图鉴完成度即时更新
- 解锁奖励即时发放

## 图鉴系统

### 图鉴分类

**按属性分类**：

1. **火系**：高攻击，克制冰系
2. **水系**：高防御，克制火系
3. **风系**：高速度，克制土系
4. **土系**：高生命，克制雷系
5. **雷系**：高爆发，克制水系
6. **光系**：治疗辅助
7. **暗系**：控制减益

**按稀有度分类**：

1. **普通**（白色）
   - 容易捕获
   - 基础能力
   - 图鉴占比60%

2. **稀有**（蓝色）
   - 较难捕获
   - 特殊能力
   - 图鉴占比25%

3. **史诗**（紫色）
   - 很难捕获
   - 强力能力
   - 图鉴占比10%

4. **传说**（金色）
   - 极难捕获
   - 独特能力
   - 图鉴占比5%

**按功能分类**：

1. **输出型**：高伤害，低防御
2. **坦克型**：高生命，低伤害
3. **辅助型**：提供增益/治疗
4. **控制型**：限制敌人行动
5. **召唤型**：召唤小单位
6. **爆发型**：短时间高伤害

### 图鉴界面

**图鉴展示**：

1. **已捕获**
   - 显示完整模型和信息
   - 显示属性和技能
   - 显示捕获时间和地点

2. **未捕获**
   - 显示剪影
   - 显示基础信息（属性、稀有度）
   - 显示捕获提示（在哪里遇到）

3. **图鉴完成度**
   - 总体完成度百分比
   - 分类完成度（火系、水系等）
   - 稀有度完成度（普通、稀有等）

### 图鉴奖励

**完成度奖励**：

1. **10%完成度**：解锁基础奖励
2. **25%完成度**：解锁稀有捕获道具
3. **50%完成度**：解锁传说捕获道具
4. **75%完成度**：解锁特殊单位
5. **100%完成度**：解锁终极奖励

**分类完成奖励**：

1. **火系全收集**：火系单位属性+10%
2. **水系全收集**：水系单位属性+10%
3. **稀有全收集**：稀有捕获成功率+20%
4. **传说全收集**：解锁隐藏传说单位

## 单位培养系统（简化版）

**设计原则**：培养深度低于养成/培育，重点是收集和组合

### 培养方式

**升级系统**：

1. **经验升级**
   - 战斗获得经验
   - 投入经验道具升级
   - 升级提升基础属性

2. **等级上限**
   - 普通：Lv30
   - 稀有：Lv50
   - 史诗：Lv70
   - 传说：Lv100

**装备系统**：

1. **装备槽位**
   - 武器槽：提升攻击
   - 防具槽：提升防御
   - 饰品槽：提升特殊属性

2. **装备获取**
   - 战斗掉落
   - 商店购买
   - 制作系统

**技能系统**：

1. **固定技能**
   - 每个单位有固定的技能组
   - 升级解锁新技能
   - 技能不可更换

2. **技能升级**
   - 投入资源升级技能等级
   - 技能等级提升效果

**进化系统**（可选）：

1. **简单进化**
   - 达到特定等级进化
   - 进化提升属性和外观
   - 每个单位1-2次进化

2. **进化不分支**
   - 不像养成/培育有多分支
   - 进化路线固定
   - 重点是收集不同单位

## 队伍组合系统

### 队伍配置

**队伍规模**：

1. **出战单位**：3-5只
2. **替补单位**：2-3只
3. **战斗中切换**：可以替换单位

**配置策略**：

1. **属性克制**：针对敌人属性配置
2. **功能互补**：输出+坦克+辅助
3. **稀有度平衡**：传说单位消耗高

### 组合效果

**属性共鸣**：

1. **单属性共鸣**
   - 3只同属性：该属性伤害+20%
   - 5只同属性：该属性伤害+50%

2. **双属性共鸣**
   - 火+雷：爆发伤害+30%
   - 水+冰：控制时间+30%
   - 风+光：移动速度+30%

**功能组合**：

1. **经典组合**：输出+坦克+辅助
2. **爆发组合**：多输出+爆发型
3. **控制组合**：控制型+输出型
4. **召唤组合**：多召唤型

**特殊组合**：

1. **传说组合**：特定传说单位组合解锁特殊效果
2. **主题组合**：同一主题单位组合（龙族、恶魔族等）
3. **稀有组合**：全稀有以上单位组合

### 单位行为设置

**行为模式**：

1. **进攻模式**：主动攻击敌人
2. **防守模式**：守在特定位置
3. **跟随模式**：跟随玩家角色
4. **自由模式**：自主判断行动

**优先级设置**：

1. **攻击优先级**：优先攻击哪类敌人
2. **保护优先级**：优先保护哪个目标
3. **技能使用**：何时使用技能

## 战斗应用

### 单位部署

**部署方式**：

1. **战斗前部署**
   - 选择出战单位
   - 设置初始位置
   - 设置行为模式

2. **战斗中召唤**
   - 消耗资源召唤单位
   - 召唤有冷却时间
   - 单位阵亡可以重新召唤

**部署限制**：

1. **资源消耗**
   - 普通单位：消耗少
   - 稀有单位：消耗中
   - 史诗单位：消耗高
   - 传说单位：消耗极高

2. **同时在场限制**
   - 最多3-5只单位同时在场
   - 单位阵亡后可以召唤新的

### 单位战斗

**战斗方式**：

1. **自动战斗**
   - 单位根据AI自动战斗
   - 玩家可以下达简单指令
   - 玩家专注操控自己角色

2. **协同效果**
   - 玩家和单位攻击同一目标触发连携
   - 单位之间有协同技能
   - 组合效果自动触发

**单位状态**：

1. **生命值**：单位生命值降低会撤退
2. **能量值**：技能消耗能量
3. **状态异常**：单位可能被控制/减益

### 战斗反馈

**视觉反馈**：

1. **单位效果**：单位攻击有明显特效
2. **组合效果**：组合触发有特殊特效
3. **属性克制**：克制关系有视觉提示

**数值反馈**：

1. **伤害数字**：显示单位造成的伤害
2. **治疗数字**：显示辅助单位的治疗
3. **状态图标**：显示单位当前状态

## 长线驱动

### 收集目标

1. **图鉴完成度**：收集所有单位
2. **稀有收集**：收集所有传说单位
3. **属性收集**：收集每个属性的所有单位
4. **功能收集**：收集每个功能的所有单位

### 挑战目标

1. **单属性通关**：只用单一属性单位通关
2. **低稀有度通关**：只用普通单位通关
3. **最少单位通关**：用最少单位通关
4. **完美组合**：找到最强组合

### 社交目标

1. **图鉴展示**：向其他玩家展示图鉴
2. **队伍分享**：分享队伍配置
3. **稀有炫耀**：炫耀稀有单位

## 包装变体

同样的底层玩法，可以包装成：

1. **宝可梦模式**：捕获宝可梦，收集图鉴
2. **卡牌召唤**：召唤卡牌，收集卡组
3. **英雄召唤**：召唤英雄，收集英雄
4. **异兽图鉴**：捕获异兽，收集图鉴
5. **机甲收集**：收集机甲，组建舰队
6. **灵魂收集**：收集灵魂，召唤战斗

**包装差异**：
- 宝可梦模式：捕获球捕获，宝可梦战斗
- 卡牌召唤：抽卡获得，卡牌召唤
- 英雄召唤：召唤阵召唤，英雄战斗
- 异兽图鉴：封印卷轴捕获，异兽战斗
- 机甲收集：收集机甲，机甲出战
- 灵魂收集：灵魂石捕获，灵魂附身
