# 卡牌养成包装（放置产出）

## 核心定位

**底层经营**：放置产出卡牌 → 养成卡牌 → 战斗使用 → 获得资源 → 加速产出

**表层包装**：卡牌工坊、英雄训练营、召唤阵、卡牌农场

**认知负荷**：低（放置自动产出，养成是简单选择）

## 四要素验证

| 要素 | 验证 |
|---|---|
| 时间分离 | ✓ 局外放置产出+养成，局内战斗 |
| 低操作频次 | ✓ 放置自动产出，养成是低频选择 |
| 本能目标 | ✓ "我养成的卡牌"有收集和培养双重驱动 |
| 成长/策略 | ✓ 养成路线、卡组构筑、放置优化 |

## 核心循环

```
局外放置阶段
├─ 放置系统自动产出卡牌
├─ 定期收获新卡牌
└─ 选择养成路线（进化/强化/融合）
    ↓
卡牌养成
├─ 投入资源养成卡牌
├─ 选择进化方向
└─ 构筑卡组
    ↓
进入战斗
    ↓
局内战斗阶段
├─ 操控角色防守
├─ 使用养成的卡牌（召唤/技能/增益）
└─ 卡牌效果直接体现
    ↓
战斗结束
├─ 获得养成资源
├─ 解锁新卡牌产出
└─ 加速放置效率
    ↓
回到局外放置
```

## 具体设计方案

### 方向A：放置产出 + 卡牌养成（收集向）

**核心定位**：放置系统自动产出多种卡牌，玩家收集并养成

**核心机制**：
- 放置系统每隔一段时间产出随机卡牌
- 不同稀有度的卡牌产出概率不同
- 玩家收集卡牌，选择养成哪些
- 养成后的卡牌在战斗中使用

**放置系统**：
- **卡牌农场**：自动产出基础卡牌
- **召唤阵**：消耗资源召唤稀有卡牌
- **离线产出**：离线时也能产出卡牌（有上限）
- **产出加速**：战斗获得的资源可以加速产出

**养成系统**：
- **升级**：投入资源提升卡牌等级
- **进化**：达到条件后进化成高级形态
- **融合**：多张相同卡牌融合提升品质
- **技能解锁**：养成过程中解锁新技能

**战斗应用**：
- 战斗前选择携带的卡组（10-15张）
- 战斗中使用卡牌：召唤单位、释放技能、获得增益
- 卡牌有冷却时间，不能无限使用
- 养成等级直接影响卡牌效果

**长线驱动**：
- 收集所有卡牌（图鉴完成度）
- 养成多套卡组应对不同关卡
- 优化放置产出效率
- 追求满级满进化

**参考对标**：
- 剑与远征（放置产出英雄 + 养成）
- AFK Arena（放置 + 英雄收集）
- 少女前线（放置产出 + 养成）

**双约束验证**：
- **约束一（结构合法性）**：✓ 放置产出是独立玩法（放置类游戏），卡牌养成是独立玩法（卡牌养成游戏）
- **约束二（体验闭环）**：✓ 放置产出卡牌→养成卡牌→战斗使用→获得资源→加速产出，循环完整

---

### 方向B：放置产出 + 卡牌养成（深度养成向）

**核心定位**：放置系统产出少量核心卡牌，玩家深度养成

**核心机制**：
- 放置系统产出固定的几张核心卡牌
- 每张卡牌有深度的养成路线
- 玩家选择养成方向，影响卡牌形态和能力
- 重点是深度养成，不是广度收集

**放置系统**：
- **英雄训练营**：自动产出养成资源（经验、材料）
- **资源产出**：不是产出新卡牌，而是产出养成资源
- **离线产出**：离线时也能产出资源
- **产出优化**：战斗获得的奖励可以提升产出效率

**养成系统**：
- **多分支进化**：每张卡牌有3-5条进化路线
- **属性培养**：选择培养方向（力量/敏捷/魔法）
- **技能树**：解锁不同技能组合
- **装备系统**：为卡牌配置装备

**战斗应用**：
- 战斗前选择3-5张核心卡牌
- 每张卡牌都是深度养成的结果
- 战斗中卡牌能力强大且独特
- 养成路线直接决定战斗风格

**长线驱动**：
- 探索所有进化路线
- 养成完美属性的卡牌
- 优化养成资源产出
- 尝试不同养成组合

**参考对标**：
- 怪物牧场（深度养成 + 放置产出资源）
- 宝可梦（深度养成，但不是放置产出）
- 数码宝贝（多分支进化）

**双约束验证**：
- **约束一（结构合法性）**：✓ 放置产出资源是独立玩法，深度养成是独立玩法（怪物牧场）
- **约束二（体验闭环）**：✓ 放置产出资源→深度养成→战斗使用→获得奖励→加速产出，循环完整

---

### 两个方向的对比

| 维度 | 方向A：收集向 | 方向B：深度养成向 |
|---|---|---|
| 放置产出内容 | 产出卡牌本身 | 产出养成资源 |
| 养成深度 | 浅，但卡牌种类多 | 深，单张卡牌有多条路线 |
| 收集驱动 | 强，图鉴完成度 | 弱，卡牌数量少 |
| 情感绑定 | 分散，但有收集成就感 | 强，深度养成单张卡牌 |
| 长线驱动 | 收集所有卡牌 | 探索所有养成路线 |
| 战斗策略 | 卡组搭配策略 | 深度养成的少数核心卡牌 |
| 参考对标 | 剑与远征、AFK Arena | 怪物牧场 + 放置产出 |
| 体验承诺 | "我收集养成的卡牌军团，守住了基地" | "我深度养成的几张核心卡牌，展现了强大力量" |

## 与操控防守的结合

**方向A（收集向）**：
- 我的卡牌农场自动产出卡牌
- 我收集并养成这些卡牌
- 战斗中使用养成的卡牌召唤单位、释放技能
- 卡牌效果直接体现在战斗中

**方向B（深度养成向）**：
- 我的训练营自动产出养成资源
- 我深度养成几张核心卡牌
- 战斗中使用深度养成的卡牌
- 每张卡牌都是独特的战斗力量

**资源循环**：
- 放置产出（卡牌/资源） → 养成 → 战斗使用 → 获得奖励 → 加速产出
- 循环清晰，互相喂养

**成长可见**：
- 方向A：卡牌种类越来越多，图鉴完成度提升
- 方向B：单张卡牌能力越来越强，进化形态越来越高级

## 长线驱动

**方向A（收集向）**：
- 收集所有卡牌（图鉴完成度）

- 优化放置产出效率（提升稀有卡产出概率）

**方向B（深度养成向）**：
- 探索所有进化路线（每张卡牌3-5条路线）
- 养成完美属性的卡牌

## 参考对标

**成功案例**：
- 剑与远征（放置产出英雄 + 养成 + 战斗）
- AFK Arena（放置 + 英雄收集养成）

- 放置降低操作负担，养成提供情感驱动

## 潜在问题与解决方案

**问题1：放置会不会太无聊？**
- 解决：放置只是产出环节，核心乐趣在养成和战斗
- 解决：养成选择提供策略深度（进化路线、技能树）
- 解决：放置产出有即时反馈（新卡牌/资源到手）

**问题2：会不会变成纯数值游戏？**
- 解决：战斗是操控的，不是自动的
- 解决：卡牌在战斗中有明显效果（召唤、技能、增益）
- 解决：养成不只是数值，还有进化形态、技能解锁

**问题3：放置和战斗的关联够强吗？**
- 解决：战斗直接使用放置产出的卡牌
- 解决：战斗奖励直接用于加速放置产出
- 解决：循环紧密，互相依赖

**问题4：离线收益会不会破坏平衡？**
- 解决：离线收益有上限（最多8小时）
- 解决：在线战斗的奖励更高
- 解决：离线只是补偿，不是主要玩法

**问题5：卡牌养成和卡牌构筑的区别？**
- 解决：卡牌养成是放置产出+培养，卡牌构筑是战斗中临时组牌
- 解决：养成是长线投入（升级、进化），构筑是局内即时选择
- 解决：养成的卡牌是战斗前准备好的，构筑的卡牌是战斗中临时获得的

## 为什么是第一梯队

1. **认知负荷极低**：放置几乎不需要操作，完美互补战斗的高负荷
2. **节奏对比极强**：放置（极慢）vs 战斗（快），对比度最大
3. **市场验证充分**：放置游戏有成熟受众和成功案例（剑与远征、AFK Arena）
4. **循环清晰**：放置产出 → 战斗消耗 → 战斗奖励 → 升级放置
5. **实现成本低**：放置系统相对简单，不需要复杂AI
