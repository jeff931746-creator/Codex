# 工坊经营包装

## 核心定位

**底层经营**：投入原料 → 加工制作 → 产出成品 → 出售或使用

**表层包装**：铁匠铺、炼金工坊、魔法工坊、武器作坊

**认知负荷**：低-中（制作有简单决策）

## 四要素验证

| 要素 | 验证 |
|---|---|
| 时间分离 | ✓ 局外制作，局内战斗 |
| 低操作频次 | ✓ 制作是低频操作 |
| 本能目标 | ✓ "我打造的武器"有成就感 |
| 成长/策略 | ✓ 配方解锁、材料搭配、品质追求 |

## 核心循环

```
局外工坊阶段
├─ 收集材料（战斗掉落）
├─ 选择配方
├─ 制作装备/道具
└─ 强化/附魔
    ↓
进入战斗
    ↓
局内战斗阶段
├─ 使用自己打造的装备/道具
├─ 感受装备/道具效果
└─ 获得新材料
    ↓
战斗结束
├─ 材料入库
├─ 解锁新配方
└─ 发现稀有材料
    ↓
回到局外工坊
```

## 具体设计方案

### 方向A：局外制作，局内消耗（炼金工坊模式）

**核心定位**：制作消耗品，战斗中使用后消失

**核心机制**：
- 收集草药、矿石、怪物素材
- 炼制药水、炸弹、增益道具
- 实验新配方
- 批量生产

**炼金系统**：
- **配方实验**：尝试不同材料组合发现新配方
- **效果调整**：同一配方可以调整效果倾向
- **批量生产**：解锁后可以批量制作常用道具
- **特殊炼金**：稀有材料炼制传说道具

**材料系统**：
- 草药（治疗、增益）
- 矿石（爆炸、强化）
- 怪物素材（特殊效果）

**实验乐趣**：
- 不同材料组合产生不同效果
- 发现新配方有成就感
- 失败也有产出（失败品也能用）

**战斗应用**：
- 治疗药水（恢复生命）
- 增益药水（提升攻击/防御）
- 炸弹（范围伤害）
- 特殊道具（召唤、控制）

**战斗中的体验**：
- 我炼制的药水在关键时刻救命
- 我制作的炸弹清空了一波敌人
- 道具消耗后需要回工坊继续制作

**参考对标**：
- Potion Craft（炼金模拟）
- Little Alchemy（元素组合）
- Atelier系列（炼金+冒险）

**双约束验证**：
- **约束一（结构合法性）**：✓ 炼金本身是独立玩法，Potion Craft可以不战斗只炼金
- **约束二（体验闭环）**：✓ 实验配方→制作道具→战斗消耗→获得材料→继续实验，循环完整

---

### 方向B：局外建造，局内协助（铁匠铺+防御单位模式）

**核心定位**：制作防御单位/装置，战斗中持续协助

**核心机制**：
- 收集金属、宝石等材料
- 打造防御塔、机器人、陷阱装置
- 战斗中这些单位/装置自动协助
- 损坏后需要修理或重建

**制作系统**：
- **防御塔**：自动攻击敌人（箭塔、炮塔、魔法塔）
- **机器人**：自动巡逻战斗（近战机器人、远程机器人）
- **陷阱装置**：触发式伤害（地刺、火焰陷阱、冰冻陷阱）
- **辅助装置**：提供增益（治疗光环、护盾发生器）

**材料获取**：
- 战斗掉落基础材料
- Boss掉落稀有材料
- 特定关卡产出特定材料

**制作过程**：
- 选择配方
- 投入材料
- 简单小游戏（打铁节奏）
- 产出单位/装置

**战斗应用**：
- 战斗前部署防御单位/装置
- 战斗中它们自动协助
- 玩家操控角色配合防御单位作战
- 单位/装置损坏后需要修理

**战斗中的体验**：
- 我打造的防御塔在自动开火
- 我制作的机器人在前线战斗
- 我和我的造物一起守住基地

**参考对标**：
- Orcs Must Die（陷阱+操控角色）
- Dungeon Defenders（塔防+操控角色）
- Sanctum（塔防+FPS）

**双约束验证**：
- **约束一（结构合法性）**：⚠️ 建造防御单位接近塔防/经营，独立玩法性弱于方向A
- **约束二（体验闭环）**：✓ 打造单位→部署→战斗协助→修理/重建，循环完整
- **与Thronefall区分度**：⚠️ 虽然是制作单位而非建造建筑，但本质上仍是"局外建造，局内防守"，区分度不如方向A

---

### 两个方向的对比

| 维度 | 方向A：局外制作局内消耗 | 方向B：局外建造局内协助 |
|---|---|---|
| 独立玩法性 | 强，炼金本身是独立品类 | 弱，接近塔防/经营 |
| 与Thronefall区分度 | 高，消耗品不是建筑 | 低，建造单位接近建造建筑 |
| 战斗中的体验 | 主动使用道具，决策感强 | 被动协助，自动化感强 |
| 制作深度 | 配方实验、效果调整 | 单位类型、材料搭配 |
| 长线驱动 | 配方收集、批量生产 | 单位图鉴、组合策略 |
| 参考对标 | Potion Craft、Atelier | Orcs Must Die、Dungeon Defenders |
| 梯队评级 | 第一梯队 | 第二梯队 |
| 体验承诺 | "我炼制的药水和炸弹，在关键时刻扭转战局" | "我打造的防御单位，和我一起守住了基地" |

## 与操控防守的结合

**方向A（局外制作局内消耗）**：
- 我炼制的药水在关键时刻救命
- 我制作的炸弹清空了一波敌人
- 道具消耗后需要回工坊继续制作

**方向B（局外建造局内协助）**：
- 我打造的防御塔在自动开火
- 我制作的机器人在前线战斗
- 我和我的造物一起守住基地

**材料循环**：
- 战斗获得材料
- 材料用于制作
- 制作产出道具/单位
- 道具/单位用于战斗

**成长可见**：
- 方向A：配方越来越多，道具效果越来越强
- 方向B：单位越来越强，组合越来越复杂

## 长线驱动

**方向A（局外制作局内消耗）**：
- 配方收集：解锁所有配方、发现隐藏配方、实验新组合
- 效果优化：调整配方效果倾向、批量生产常用道具
- 工坊升级：解锁新工作台、增加制作槽位、提升制作效率

**方向B（局外建造局内协助）**：
- 单位图鉴：收集所有单位类型、解锁稀有单位
- 组合策略：发现单位协同效果、优化部署配置
- 工坊升级：解锁新单位类型、提升单位品质上限

## 参考对标

**成功案例**：
- Moonlighter（商店+地牢）
- Recettear（道具店+地牢）
- Atelier系列（炼金+冒险）
- Potion Craft（炼金模拟）

**独特性**：
- 工坊制作 + 操控战斗（不是自动战斗）
- 制作的装备直接在战斗中使用
- 材料循环紧密

## 潜在问题与解决方案

**问题1：制作会不会变成重复劳动？**
- 方向A解决：配方实验有探索乐趣，失败品也能用
- 方向A解决：批量生产减少重复，效果调整提供深度
- 方向B解决：单位类型多样，组合策略有深度

**问题2：材料收集会不会太肝？**
- 解决：基础材料容易获得
- 解决：稀有材料是长线目标，不是必需
- 解决：可以用基础材料做出够用的道具/单位

**问题3：工坊和Thronefall经营的区别？**
- 方向A：炼金是实验配方，不是空间布局；产出是消耗品，不是建筑
- 方向B：虽然都是建造单位，但工坊是制作过程（打铁节奏小游戏），不是放置建筑

**问题4：制作和战斗的时间比例？**
- 解决：制作是低频操作，不需要长时间
- 方向A：批量生产减少重复
- 方向B：单位制作后持续使用，不需要频繁重建
- 解决：战斗是主要时间消耗

## 为什么是第一梯队

**方向A（局外制作局内消耗）**：
1. **本能目标清晰**："我炼制的药水和炸弹"有成就感
2. **独立玩法性强**：炼金本身是独立品类（Potion Craft）
3. **制作过程有可玩性**：配方实验、效果调整，不是点击等待
4. **成果直接体现**：道具在战斗中立刻能感受到
5. **长线驱动充足**：配方收集、效果优化、批量生产
6. **与Thronefall区分明确**：消耗品不是建筑，实验不是布局

**方向B（局外建造局内协助）**：
1. **本能目标清晰**："我打造的防御单位"有成就感
2. **独立玩法性中等**：建造单位接近塔防/经营，但有制作过程
3. **制作过程有可玩性**：打铁节奏小游戏，不是纯放置
4. **成果直接体现**：单位在战斗中自动协助
5. **长线驱动充足**：单位图鉴、组合策略
6. **与Thronefall区分度中等**：虽然都是建造，但工坊是制作单位不是放置建筑

**梯队评级**：
- 方向A：第一梯队（独立玩法性强，区分度高）
- 方向B：第二梯队（独立玩法性弱于方向A，与Thronefall区分度不如方向A）

## 一句话体验承诺

**方向B（铁匠铺+防御单位）**："我打造的防御单位，和我一起守住了基地"
