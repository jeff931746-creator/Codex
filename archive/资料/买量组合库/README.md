# 买量组合库

## 目的

为买量游戏立项提供 **题材 × 画风** 组合评估体系，支持市场买量效果评估和题材选型决策。

## 核心理念

单一题材或画风无法准确评估买量效果，必须以 **组合** 为评估单位。

例如：
- "暗黑童话 × 第五人格画风" 可能表现优异
- "暗黑童话 × 像素复古" 可能完全失效

## 目录结构

- `题材库/`：题材基础信息（受众、情感钩子、市场热度），按母题→具体题材两级索引
- `画风库/`：画风基础信息（视觉特征、制作门槛、适用场景）
- `组合案例库/`：题材 × 画风的完整评估报告
- `评估模板/`：各类评估的标准字段模板
- [`题材评估标准.md`](题材评估标准.md)：题材四象限评分、市场信息来源、证据等级和推荐结论规则
- [`题材总表.md`](题材总表.md)：133 个题材的四象限排序表（含母题ID、人群IDs 字段）

## 与三位一体框架的关系

本库的 [题材库](题材库/题材索引.md) 是 [三位一体框架](../../../reference/部门标准/立项/三位一体框架/README.md) 中**题材母题库**的具体落地。

题材评估时需要同时引用：

- [题材库](题材库/题材索引.md) → 母题 ID（THM-*）
- [人群簇库](../人群簇库/README.md) → 人群 ID（AUD-*）
- [玩法承接库](../玩法承接库/README.md) → 玩法 ID（PLY-*）

详见 [评估模板](评估模板/题材评估模板.md) 的"三位一体关联"章节。

## 使用流程

1. **收集阶段**：通过 Skill 收集题材和画风到对应库
2. **证据阶段**：按 [`题材评估标准.md`](题材评估标准.md) 补齐市场来源、竞品证据、素材证据或内部测试证据
3. **评估阶段**：运行"组合评估" Skill，生成组合报告
4. **决策阶段**：基于组合报告的评分矩阵做立项选型

## 评估维度

组合评估包含以下核心维度：

- **受众契合度**：组合是否符合目标用户审美/情感预期
- **素材表现力**：视觉冲击力、情感钩子、广告素材可行性
- **竞品密度**：成功案例、失败案例、市场饱和度
- **长线潜力**：素材迭代空间、IP 延展性
- **团队适配度**：美术门槛、内容生产成本（面向 20 人以内小团队）
- **创新空间**：未被充分挖掘的组合角度
- **风险点**：政策风险、文化敏感度、技术难点

## 数据来源优先级

详见 [`题材评估标准.md`](题材评估标准.md)。简要优先级：

1. 自有投放数据、内部留存付费数据
2. 投放素材库、商店榜单、下载收入估算、竞品拆解
3. 内容平台热度、社区讨论、KOL 观点
4. 单点灵感或主观判断

## 适用场景

- 立项前的题材选型
- 买量素材方向评估
- 竞品分析与差异化定位
