# 平台策略变化：应用包/资产 × 素材（2026 Q2）

更新时间：2026-05-11
来源：巨量引擎《内容消费商品×素材策略投放方法论_for广告主》（飞书 wiki，审批编号 2026011530456）
状态：平台公告策略，预估 2026 Q2 逐步上线，对除 DPA 外的所有竞价广告（信息流、直播等）生效

## 一句话结论

```text
平台把【商品 × 素材】（游戏非端游场景下等价于【应用包/资产 × 素材】）作为参竞单元，
重复堆同一个参竞单元的广告不再能稳定换来更多跑量；流量会向竞争力强的少数广告集中。
```

这是平台对参竞规则的变更，不是题材评估底层逻辑的变更。
本文档作为既有买量框架（人群簇 → 题材母题 → 玩法承接）的修正层，不替代题材评估方法论。

## 关键事实（来自原文）

1. **参竞单元升级**：相同【商品 × 素材】在多个广告中重复投放，消耗会集中到少量竞争力强的广告上，弱广告减少跑量或无法跑量
2. **生效范围**：除 DPA 外的所有竞价广告，所有广告主
3. **生效时间**：预估 2026 年 Q2 逐步上线
4. **游戏行业的"商品"定义**：
   - 游戏非端游：**只绑资产，无需绑商品**（即应用包 / 小程序 / 小游戏 ID 即为参竞主体）
   - 游戏端游：需绑端游商品
5. **素材差异度标准**：
   - 低标：素材之间差异画面 ≥ 60%（规避同质化）
   - 高标：与大盘素材差异 ≥ 90%（命中首发素材，获得平台流量扶持）

## 对既有买量框架的修正

### 原链路

```text
用户群 → 题材母题 → 素材表达 → 买量分发 → 玩法承接 → 回收
```

### 修正后链路

```text
用户群
→ 题材母题
→ 素材表达（每个内容方向需具备差异度产能）
→ 应用包/资产 × 素材 × 载体 × 转化目标（= 平台参竞单元）
→ 多载体 × 多转化目标组合分发
→ 玩法承接（素材承诺与产品内玩法一致性）
→ 回收
```

修正点只有一处：**在"素材表达"和"买量分发"之间显式插入"参竞单元"层**。
这一层不是新概念，而是把过去隐藏在"买量分发"内部的平台理解逻辑显式化。

## 对题材选择的影响

不替换既有 `题材评估标准.md` 的双轴结构（有效获量 × ROI 承接），但需要在"有效获量"轴下补一个子维度：**参竞单元覆盖度**。

参竞单元覆盖度回答三件事：

1. **载体覆盖度**：这个题材能否在 ≥ 2 个载体（APP / 抖音小游戏 / 微信小游戏 / 端原生 / 小程序）下都有合理承接？
2. **转化目标组合可用性**：能否同时跑 IAP 组合（如 7 日 ROI + 每次付费 + 激活且付费 + 激活）和 IAA 组合（如 变现 7R + 变现 ROI + 关键行为 + 激活）？还是只能跑单一目标？
3. **素材差异度产能**：6 个月内能否持续产出 ≥ 90% 差异度（首发级）和 ≥ 60% 差异度（常规级）两档素材？还是钩子单一、只能反复换皮？

详细评分维度见 `题材评估标准.md` "有效获量评分标准"中的"参竞单元覆盖度"行。

## 投放层操作变化（不直接影响题材评估，但影响买量团队 SOP）

以下是原文给出的投放层动作，团队执行 SOP 调整时参考，不是题材评估输入：

| 阶段 | 过去做法 | 新做法 |
|---|---|---|
| 投前 | 同一应用包 / 资产下小预算多账号矩阵探索 | **同应用包/资产 × 素材集中预算**，不分散到多条广告 |
| 投前 | 重复 / 相似商品创建 | **不要重复创建**，重复创建会加剧相互挤压跑量 |
| 投前 | nobid + 常规广告并行作为探索 | **谨慎使用 nobid**，同应用包/资产 × 素材下不建议同时搭建 nobid 和常规广告 |
| 投中 | 老素材衰退时复制广告或加预算 | **P0：补新素材 + 用素材起量工具**；P1：自查同素材是否已搭建不同转化产品广告 + 在不同载体上同时搭建；P2：孵化抖音号矩阵（同素材绑不同抖音号） |
| 投中 | 同素材 = 一条广告 | **同素材跨载体 + 跨转化产品组合并行**才能最大化跑量 |
| 投中扩量 | 加预算 / 复制广告 | **直播间直投 + 直播混投 + 关键行为多行为组合 + 付费面板分层** |

## 后续追踪点

1. **2026 Q2 实际上线节奏**：平台会提前通知，关注是否有 ABtest 阶段、是否对部分行业先生效
2. **首发素材识别规则细化**：原文只给了 ≥ 90% 差异度的标准，但具体识别口径、扶持力度需要持续跟进
3. **AIGC / 试玩 / 直玩 / 星广联投 等素材产能工具**：原文将这些工具列为"补新素材"的 P0 手段，需要评估这些工具对我方"素材差异度产能"的实际加成
4. **小游戏赛道适配性**：原文明确小游戏素材"双主线 + 试玩"是独特形式，与 APP 游戏素材逻辑不同，团队评估小游戏题材时要区别对待
