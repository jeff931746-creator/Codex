# 菇勇者传说·垂直养成底盘拆解

## 这个专题要回答什么

11个互不干扰的垂直养成管线为什么能撑起极高的ARPU，而不是让玩家觉得系统太碎？

## 总体判断

这11大系统（坐骑、神器、遗物、技能、伙伴、家园、祭祀、武魂、称号、公会等级、图鉴）本质上是11条独立的数值沉淀管道。它们互不干扰，每一条都有自己的获取方式和付费点。这种设计让大R可以通过全管线拉满获得维度碾压，而0氪也能在单管线慢慢积累。底盘的深度不是靠一两条管线挖得很深，而是靠多条管线并行堆出来的。

## 怎么判断某个东西属于垂直养成底盘

- 它如果主要提供长期数值沉淀且跨局保留，就属于底盘层
- 它如果拥有独立的获取方式和升级材料，就属于底盘层
- 它如果只在单局内生效或可以随时更换，就不属于底盘层

## 垂直养成底盘的关键结构

### 1. 11条管线各自独立沉淀

- 触发条件：玩家获得当前养成管线所需的碎片或材料（如坐骑碎片、神器融合石、遗物碎片、武魂碎片等）
- 具体结果：对应系统的属性提升，且不影响其他系统
- 设计原因：独立管线让付费点极度分散，每个系统都可以单独做付费活动，不会互相干扰

### 2. 碎片扭蛋与直接抽卡双轨制

- 触发条件：玩家参与技能/伙伴/武魂等扭蛋
- 具体结果：碎片扭蛋产出碎片需万次抽取合成，直接抽卡产出成品有保底
- 设计原因：碎片系统服务大R的极致追求，直接抽卡服务中R的保底需求，双轨覆盖不同付费层

### 3. 搜寻成本递增的遗物系统

- 触发条件：玩家在遗物系统进行搜寻
- 具体结果：7个槽位装配遗物，但搜寻消耗的金币随当日搜寻次数递增
- 设计原因：递增成本制造了天然的"停手点"，让玩家在资源有限时必须选择优先哪个槽位

### 4. 公会等级限制免费资源上限

- 触发条件：玩家参与公会活动提升公会等级
- 具体结果：免费资源的获取上限被抬高
- 设计原因：把个人底盘成长和社交系统绑定，用社交压力逼玩家投入公会活动

## 这个层级不负责什么

- 不负责具体装备的随机产出，那是神灯系统的职责
- 不负责战斗过程中的表现，那是执行层的职责
- 不负责材料从哪里购买，那是付费与Pass矩阵的职责

## 可迁移结论

- If 游戏把养成拆成十几个互不干扰的独立管线，Then 付费点会极度分散，大R可以通过全管线拉满获得绝对优势，但0氪会面临全线平庸
- If 碎片扭蛋系统没有掉率保护且需要万次抽取，Then 它只服务极少数大R的极致追求，中低付费玩家会被劝退转向有保底的直接抽卡
