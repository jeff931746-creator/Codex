# 菇勇者传说·付费与Pass矩阵拆解

## 这个专题要回答什么

为什么每个系统都有自己的Battle Pass，这种矩阵式付费为什么能撑起极高ARPU？

## 总体判断

这套付费体系的核心不是单个礼包的性价比，而是"每个系统都有独立Pass"带来的全盘覆盖。玩家在任何一条养成管线上想加速，都会遇到对应的付费门槛。首日转化促销拉入门，Pass矩阵做日常付费，扭蛋双轨吃大R，三层结构把不同付费层的需求都接住了。

## 怎么判断某个东西属于付费与Pass矩阵

- 它如果主要改变养成速度而不是改变战斗机制，就属于奖励层
- 它如果需要玩家花钱或看广告才能获得，就属于奖励层
- 它如果可以通过纯时间投入获得，就不属于纯奖励层

## 付费体系的关键结构

### 1. 首日转化促销拉入门

- 触发条件：玩家创角首日
- 具体结果：起价$0.99逐步提升至$9.99的装备、技能、伙伴捆绑礼包
- 设计原因：极低门槛让玩家完成首次付费破冰，逐步提价制造"现在买最划算"的紧迫感

### 2. Battle Pass矩阵覆盖全系统

- 触发条件：每个养成系统解锁
- 具体结果：通常3条通道：免费($0) / 标准($4.99) / 高级($9.99)
- 设计原因：单个Pass的价格不高，但11个系统叠加后月均消费价值约$100，用低单价高频次做高ARPU

### 3. 扭蛋双轨制分层吃大R

- 触发条件：玩家追求技能/伙伴/武魂的极致属性
- 具体结果：碎片系统无掉率保护需万次抽取，直接抽卡系统有保底但免费距离保底过远
- 设计原因：碎片系统服务极致大R，直接抽卡服务中R，双轨让不同付费层都能找到自己的消费点

### 4. 前7天每日解锁配套促销

- 触发条件：前7天每日解锁新功能
- 具体结果：推出对应内购捆绑包
- 设计原因：在玩家最兴奋的新功能解锁时刻推送对应礼包，转化率最高

## 付费和养成的关系

- 付费层决定"能多快拿到材料"
- 养成层决定"拿到材料后能变多强"
- 只给付费不给养成深度，大R会觉得没地方花钱；只给养成深度不给付费，0氪会觉得差距太大

## 这个模块不负责什么

- 不负责战斗执行和推关结果，那是执行层的职责
- 不负责装备的随机产出机制，那是神灯系统的职责
- 不负责社交竞争的压力来源，那是排行榜与公会战的职责

## 可迁移结论

- If 付费矩阵为每条独立养成管线配置阶梯定价的Battle Pass（$0/$4.99/$9.99），Then 玩家的单次付费门槛会被$4.99锚定，但月均消费上限会被11条管线叠加推高至$100以上
- If 扭蛋系统同时存在无保底的碎片系统和有保底的直接抽卡，Then 极致大R会被碎片系统吸干，中R会被直接抽卡的保底吸引，但0氪会因为免费距离保底过远而放弃追求
