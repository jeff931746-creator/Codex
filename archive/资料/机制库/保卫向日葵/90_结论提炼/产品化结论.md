# 保卫向日葵·产品化结论

## 这一层的用途
这里的结论带适用前提与反例，用于产品化决策。本文件将《保卫向日葵》的塔防与Roguelike构筑特征转化为可落地的设计数值与机制判断，明确在何种用户层、变现形态与局内时长下这些判断生效，并指出跨品类时的失效边界，避免设计思路的滥用。

## 结论列表

**结论 1：局内合成即路线声明**

If 局内合成能改变后续资源投放与升级选项池，Then 玩家单局留存与重玩意愿会提升 20%[推测]，因为构筑感替代了纯数值堆叠。

*适用前提*：微信小游戏平台，单局时长 10-15分钟[推测]的轻量塔防，且局内需提供至少 3条[推测]差异化路线。
*反例游戏*：《原神》（开放世界），其开放探索重环境交互与长线驱动，局内无合成路线声明，战斗系统不依赖局内动态构筑，此结论不适用。
*太宽测试*：若替换为 开放世界 类型，结论不成立。

**结论 2：冗余资源转化底盘**

If 局外养成底盘能消耗重复掉落的植物碎片并转化为全局属性，Then 玩家对卡关的负反馈会降低 30%[推测]，冗余掉落从负资产变为长线养料。

*适用前提*：IAP+IAA混合变现，依赖重复刷关投放资源的塔防游戏，且卡关期超过 2-3天[推测]。
*反例游戏*：《英雄联盟》（MOBA），单局结算重置，无局外长线底盘养成消耗冗余资源，单局得失不累积。
*太宽测试*：若替换为 MOBA 类型，结论不成立。

**结论 3：短波次切割与微决策付费**

If 关卡节奏将压力切分为 15-30秒[推测]的短波次并在波次间插入合成/升级节点，Then 玩家在波次间隙的微决策付费转化率会高于纯长线防守塔防。

*适用前提*：碎片化场景，用户单次连续游戏时长不足 5分钟[推测]的微信小游戏，且IAA激励点需高频嵌入。
*反例游戏*：《巫师3》（单机RPG），重沉浸与长线叙事，短波次切割会严重破坏心流与叙事连贯性。
*太宽测试*：若替换为 单机RPG 类型，结论不成立。

**结论 4：偏科检定驱动即时补缺**

If Boss检定专门针对阵容偏科（如高物防低法防），Then 商店中对应短板植物的限时礼包购买率会提升 40%[推测]，检定直接转化为付费动力。

*适用前提*：拥有多流派构筑且商业化深度绑定流派补齐的塔防游戏，玩家已形成 1-2个[推测]主力流派。
*反例游戏*：《炉石传说》（卡牌对战），卡组构筑在局外完成，局内Boss检定无法直接驱动即时付费补短板，只能驱动局外卡包购买。
*太宽测试*：若替换为 卡牌对战 类型，结论不成立。

**结论 5：奖励系统作为流派修正器**

If 庄园与日常奖励定向投放特定流派的植物源液，Then 0氪玩家会在 3-5天[推测]内自然形成1个可通关的主流派，大R则通过全流派补齐获得验证优势。

*适用前提*：需要维持大R与0氪共存生态，且局外系统（庄园/商店）资源产出占单日总产出 30%[推测]以上的游戏。
*反例游戏*：《艾尔登法环》（单机RPG），无日常奖励修正，完全靠玩家探索获取流派装备，不存在0氪大R生态区分。
*太宽测试*：若替换为 单机RPG 类型，结论不成立。

**结论 6：不可逆杂交提升策略深度**

If 杂交合成需要 2个[推测]同等级特定植物且产出不可逆，Then 玩家在局内合成前的思考时长会增加 5-10秒[推测]，构筑决策权重显著提升。

*适用前提*：核心用户追求策略深度，且局内有足够容错空间允许玩家试错的Roguelike塔防。
*反例游戏*：《万智牌》（卡牌对战），卡牌组合在构筑期完成，局内无不可逆的合成机制，策略重心在赛前的套牌构建。
*太宽测试*：若替换为 卡牌对战 类型，结论不成立。

## 局限性
1. **重度SLG品类**：本结论依赖高频重开与局内Roguelike构筑，对于单局长达数月、强调持续对抗与联盟政治的SLG不适用。
2. **纯竞技对战品类**：本结论的付费与养成逻辑建立在PvE偏科检定与数值验证上，对于强调绝对公平竞技、零数值累积的对战游戏不适用。
3. **强叙事驱动品类**：短波次切割与合成节点会打断叙事沉浸，对于以剧情演出和情感体验为核心的产品形态不适用。
