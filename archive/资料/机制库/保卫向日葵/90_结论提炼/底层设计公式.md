# 保卫向日葵·底层设计公式

## 这一层的用途
这里的结论脱离本游戏语境仍应成立。它们揭示了“塔防+Roguelike构筑”品类在底层结构上的因果关系——当游戏试图将复杂策略拆解为短循环决策、并让局外养成与局内构筑深度绑定时，必须遵守的机制传导逻辑。这些命题不依赖具体的植物题材或小游戏平台，而是该设计范式成立的基石。

## 结论列表

**结论 1：合成跃迁律**

If 局内合成结果能动态改变后续资源投放与选项池，Then 合成系统就会跃迁为路线构筑层而非单纯的收集层。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 2：偏科检定律**

If 检定节点专门针对构筑偏科设置高压门槛，Then 玩家受挫后的行为会转向补齐阵容短板而非纯堆数值。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 3：冗余转化律**

If 局外成长底盘能无损吸收局内重复掉落物，Then 冗余资源产出将转化为长线养成驱动力而非负反馈。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 4：决策颗粒律**

If 塔防决策被拆散至高频短波次的间隙中执行，Then 每次局部补强对战局的影响权重都会被显著放大。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 5：奖励修正律**

If 奖励系统同时耦合即时战力补缺与长期流派供给，Then 奖励投放就会演变为流派修正器而非单纯的福利发放。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 6：直觉防线律**

If 执行层单位的运作逻辑可被独立且直觉地理解，Then 新手就能在低认知负荷下建立有效防线。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

## 边界
这套公式在缺乏局外长线养成或局内选项不随进程动态变化的纯策略塔防中不适用；在重操作反应轻策略构筑的动作类塔防中也不适用。当游戏的核心乐趣不再依赖于“局内构筑路线的衍生与局外底盘的沉淀互相反哺”时，上述因果链便会断裂。
