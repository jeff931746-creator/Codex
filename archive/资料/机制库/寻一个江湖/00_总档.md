# 《寻一个江湖》总档

## 基本信息

- 游戏名：寻一个江湖
- 类型标签：半开放武侠 / 策略克制战斗 / 生活模拟 / 跨代际传承
- 平台：移动端及PC端
- 日期：2024-05-20
- 资料来源：官方介绍文本、公开平台资料

## 一句话定义

一款用“策略克制战斗”和“双线养成”把武侠体验从数值比拼改成路线选择的半开放世界游戏：局内用刚柔阴阳的武功克制关系和破绽状态触发执行战斗，游历与生活双线提供资源与战斗前置条件，跨代际传承机制把单周目未解决的短板变成下一周目的构筑起点。

## 核心结构

- 核心循环：自由游历探索 → 触发奇遇或进行生活劳作 → 获取武学、资源与生活材料 → 通过策略克制战斗验证构筑 → 角色归隐留下传承 → 新角色继承武学与遗愿继续游历
- 主要资源：武学秘籍/经历、生活材料（烹饪食材、行医草药）、生存资源（金创药、饱食度）、战斗状态资源（破绽、内伤）
- 主要系统：战斗执行层、武功与生活养成层、游历探索层、传承构筑层、资源循环
- 主要节奏节点：游历奇遇触发点、生活劳作周期、战斗克制检定点、归隐传承节点
- 主要检定节点：策略克制战斗（破绽触发与属性克制）、高级武学强敌挑战（缺乏对应克制手段的卡点）
