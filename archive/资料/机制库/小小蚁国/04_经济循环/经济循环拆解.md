# 专题名称：经济循环拆解

## 这个专题要回答什么
《小小蚁国》如何通过七资源体系与仓库保护机制，将SLG典型的"大R碾压、小R流失"结构，转化为"资源暴露风险"驱动的主动消耗循环？硬通货（钻石）的注入点和消耗点如何设计？

## 总体判断
游戏的经济核心是"资源溢出即风险"：仓库保护上限使超额资源必须被消耗（养成）或转移（联盟），否则成为被掠夺的目标。这使得每个玩家的资源存量维持在一个动态的"刚好够用"区间，避免了传统SLG中大量中小玩家囤积资源却无法转化为实际战力的沉积问题。付费的核心价值不是"买更多资源"，而是"买更快消化资源的速度"（建筑加速、孵化加速），与资源保护上限机制形成了良好的配合。

## 拆解正文

### 1. 资源宏观供需结构

**产出端**（按稳定性降序）：
1. 建筑持续生产（最稳定）：24小时产出量 = 建筑等级 × 每小时产出速率
2. 完成任务/活动奖励（可预测）：日常任务、周任务、赛季活动
3. 打野怪（波动高）：依赖兵力效率，奖励随野怪级别差异大
4. 掠夺他人（不稳定）：依赖目标选择、兵种克制、兵力规模，受保护上限约束

**消耗端**（按优先级降序）：
1. 建筑升级（刚性消耗，主要消耗湿土+菌+其他）
2. 兵蚁孵化（半刚性消耗，取决于出征频率和损耗）
3. 进化树研究（主要消耗蜜露，刚性但可延迟）
4. 特化蚁培养（蜜露高消耗，可选）

**资源黑洞设计**：蜜露同时是进化树研究和特化蚁培养的消耗品，而蜜露只能由建筑产出（不可野外采集），形成独立的稀缺资源池。畜牧习性通过+蜜露产量增益，成为最依赖蜜露消耗的特化蚁发展路线的首选习性。

### 2. 仓库保护上限的经济意义

| 资源存量区间 | 玩家行为 | 经济效果 |
|---|---|---|
| < 保护上限 | 安全积累，可挂机等待 | 资源沉积，无消耗驱动 |
| = 保护上限 | 安全但无法继续积累（超出部分暴露） | 触发决策节点 |
| > 保护上限 | 超出部分随时可被掠夺，风险随差值增大 | 强制消耗或在线防护 |

**关键设计约束**：保护上限随建筑等级提升而提高，但提升速度慢于生产速度——玩家在中后期必然面临"产量持续超过保护上限"的状态，从而保持对消耗的持续需求。

**防护罩付费点**：在线时有战败风险（联盟战/被攻击），离线时有掠夺风险。防护罩提供一定时间（8小时/24小时）的免疫状态，是主动 PvP 参与障碍高、资源又超出保护上限时的关键付费场景。

### 3. 硬通货体系

| 货币 | 主要获取方式 | 主要消耗方向 |
|---|---|---|
| 钻石 | 充值、日常任务、月卡、礼包 | 建筑/研究加速、战争道具（防护罩、行军加速）、异化卵孵化 |
| 粉钻 | 礼包、兑换码 | 粉钻商店专属道具（高级特化蚁材料等） |

**加速道具的经济逻辑**：建筑升级时间在中后期单次可达数天，研究时间同样。加速道具（30分钟/8小时/24小时）的边际价值在"最后一步"时最高（剩余1小时却要等24小时）。游戏常在此时弹出加速道具礼包，匹配玩家的即时焦虑。

### 4. 联盟商店的经济功能

联盟商店是游戏内的"第二货币兑换层"：玩家通过联盟活动（集结/远征/进化贡献）获得联盟积分，兑换联盟商店道具。

高价值道具（习性转职令、高阶特化蚁材料）只能通过联盟商店或直接充值获得，形成双通道：
- 免费玩家：积极参与联盟活动，缓慢积累
- 付费玩家：直接购买，或把联盟活动奖励投向更稀缺的道具

这使联盟活动参与对免费玩家产生强经济驱动，间接提升服务器活跃度。

### 5. 免费-付费用户的资源竞争结构

- **免费用户优势**：大量在线时间 → 精准掠夺 → 弥补生产差距
- **付费用户优势**：加速道具 → 更早达到建筑等级上限 → 更高兵力规模 → 掠夺成功率更高

两者的竞争终点仍在服务器资源总量：掠夺系统使资源在玩家间再分配，大R的资源领先会吸引更多免费用户的反掠夺，形成PvP博弈张力。习性分层（种植=不进攻的免费玩家聚集、掠夺=大R聚集）进一步把免费和付费用户分隔在不同行为层，降低直接对撞导致的免费用户流失率。

## 这个模块不负责什么
本模块不负责联盟战争的具体积分结算规则（属于联盟系统模块）；不负责野怪掉落物的具体种类和概率；不负责各建筑等级对应的升级资源消耗精确数值（数据不足）。

## 关键判断
《小小蚁国》经济设计的核心杠杆是仓库保护上限，而非资源总量的丰裕度。通过让超量资源变成风险敞口，游戏将"挂机等待"的放置型玩家行为，转化为"持续消耗或主动防护"的主动参与行为，付费点自然嵌入在"消化速度不够"的瓶颈处。

## 可迁移结论
If SLG类游戏的仓库保护上限被设计为随建筑等级提升，But 提升速度慢于产量增速，Then 中后期玩家将持续面临资源超出保护的压力，加速类付费的需求不会随玩家成熟度上升而自然消失，长线付费耐力优于固定产量设计。

If 一款SLG把高价值消耗型道具同时设置"联盟积分兑换"和"钻石直购"两种获取路径，Then 免费玩家的联盟活跃度会成为付费玩家利益的一部分（活跃免费玩家让服务器生态更健康），形成正向的服务器黏性。
