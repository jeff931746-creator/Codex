# 《小丑牌》专题目录

这是机制库里一个按**单个游戏**组织的专题拆解样例。

## 本目录结构

```text
小丑牌/
  00_总档.md
  01_系统结构/
  02_模块拆解/
  03_流派设计/
  04_经济循环/
  05_关卡节奏/
  06_商店与奖励/
  07_Boss与检定/
  90_结论提炼/
  99_归档/
```

## 当前约束

- 先写总档，再拆专题
- 专题只能回答一个主问题
- 每个专题必须回挂到总档和具体模块
- 参数不单独成专题
- 流派、经济、关卡、检定必须分开写
- 一个专题只保留一个主文件，其他内容只能作为引用
