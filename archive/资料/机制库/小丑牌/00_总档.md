# 《小丑牌》总档

## 基本信息

- 游戏名：《小丑牌》
- 类型标签：Roguelike / Roguelite / 构筑
- 资料来源：迁移自 `research/资料/机制库/肉鸽/` 的历史专题内容
- 当前状态：专题拆解样例

## 一句话定义

- 这是一款用低理解成本的短循环对局，承载高方向性的构筑组件，并通过高频修正节点与周期性检定制造流派成型快感的肉鸽构筑游戏。

## 核心结构

- 核心循环：出牌结算 -> 过关奖励 -> 商店修正 -> 下一关检定
- 主要资源：手牌、弃牌、金币、Joker 槽位、牌库结构、牌型成长
- 主要系统：扑克牌、Joker、星球牌、塔罗牌、幻灵牌、商店、Boss Blind、经济、关卡结构
- 主要节奏节点：普通盲注、大盲注、Boss Blind、商店
- 主要检定节点：Boss Blind、关卡门槛、资源是否足以继续押注当前路线

## 系统总览

| 系统 | 层级 | 职责边界 |
| --- | --- | --- |
| 扑克牌 | 执行层 | 实际出牌与分数结算，是所有其他层的兑现地基 |
| Joker | 路线与构筑层 | 声明流派方向，提供触发条件和倍率乘区；5 槽限制制造取舍 |
| 星球牌 | 底盘成长层 | 永久提升特定牌型的基础筹码和倍率，让倍率有意义地兑现 |
| 塔罗牌 | 编辑与塑形层 | 对单张扑克牌施加增强、印章、花色改造，让手牌分布趋向目标形态 |
| 幻灵牌 | 破格风险层 | 高收益高副作用的越界改造，拉高极端 build 的上限 |
| 商店 | 修正节点 | 每次 Boss Blind 后的路线重评估和补强窗口 |
| Boss Blind | 检定节点 | 每轮终点，针对不同偏科方式施加定向压力 |
| 经济系统 | 节奏杠杆 | 利息机制让"省钱"成为主动策略，调控成型速度 |
| 关卡结构 | 压力框架 | 8 Ante × 3 盲注的短循环，分数要求指数增长 |

系统间的核心咬合：扑克牌是结算地基，Joker 定路线并放大倍率，星球牌抬高筹码让倍率真正有效，塔罗牌让手牌向目标靠拢，经济决定成型速度，关卡结构决定成型必须在何时完成，Boss Blind 检验成型是否足够鲁棒。

## 本游戏最值得拆的专题

- 牌种体系
- 流派设计
- 经济循环
- 关卡节奏
- 商店与奖励
- Boss 与检定

## 总判断

- 这游戏靠什么成立：牌种分工清楚、构筑方向明确、修正节点密集、检定压力稳定
- 最大机会点：把“这局到底要做什么机器”讲得足够清楚
- 最大风险点：如果系统边界不清，容易把构筑感冲淡成单纯数值堆叠
