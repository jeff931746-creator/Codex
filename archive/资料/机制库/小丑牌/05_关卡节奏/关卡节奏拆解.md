# 小丑牌关卡节奏拆解

## 这个专题要回答什么

- 《小丑牌》的关卡、盲注和商店是怎么一起形成节奏的，以及压力如何在 8 个 Ante 内持续累积

## 总体判断

《小丑牌》的节奏强点不在于地图复杂，而在于它用**固定的三段结构（小盲→大盲→Boss Blind）×8 轮**，在非常短的循环里不断切换压力状态，让玩家始终处于"刚过线、需要调整、又被下一个节点追着跑"的状态。

---

## 基本节奏结构

每个 Ante（轮）由三个盲注组成：

```
[小盲注] → [大盲注] → [Boss Blind] → 商店 → 下一个 Ante
```

共 8 个 Ante，总计 24 个盲注节点。

| 节点类型 | 职责 | 可跳过？ |
| --- | --- | --- |
| 小盲注 | 低压力，用来验证当前构筑是否正常运作 | 可以，获得标签 |
| 大盲注 | 中等压力，抬高分数门槛 | 可以，获得标签 |
| Boss Blind | 高压力 + 特殊负面效果，构筑检定 | 不可跳过 |
| 商店 | 修正节点，买卖、刷新、调整路线 | 自动进入 |

---

## 分数门槛的压力曲线

每个 Ante 的分数要求都在前一轮基础上大幅提升，通常是**倍数增长而非线性增长**。

早期（Ante 1–3）：分数要求低，主要给玩家时间建立 build 基础。  
中期（Ante 4–6）：分数要求开始快速上升，未成型的 build 会开始卡关。  
后期（Ante 7–8）：分数要求极高，只有充分成型且有乘区的 build 才能过。

这意味着在 Ante 4 前还没有稳定流派方向的 build，通常很难撑到终局。

---

## 手数和弃牌数

每个盲注默认分配：
- 4 次出手机会（hand）
- 3 次弃牌机会（discard）

回合结束（过关或失败）后全部重置。

**弃牌的战略价值**：弃牌不只是筛牌，它还是：
- 紫色印章的触发条件（弃牌→生成塔罗牌）
- 部分 Joker 的触发条件（每次弃牌获得+X 倍率）
- 调整手牌分布的主动工具

手数通过 Voucher 或特定 Joker 可以增加，但弃牌数的增加更为稀缺。

---

## 跳过盲注的节奏意义

跳过普通盲注（小盲/大盲）会立即获得一个随机标签（Tag），常见标签包括：

- **Joker 标签**：商店里下一个 Joker 免费
- **Boss 标签**：可重置下一个 Boss Blind 的负面效果
- **Polychrome 标签**：随机一张 Joker 获得 Polychrome 版本
- **经济标签**：直接获得金币

跳过节奏判断：
- 当前 build 已经够强，该盲注没有挑战性 → 跳过换标签更划算
- 当前 build 需要磨合 → 打完获取过关金币更安全

---

## 三段节奏的心理模型

每个 Ante 内部的节奏切换会对玩家的心理状态产生明确的压力信号：

1. **小盲注**：验证阶段——"我的 build 今天能用吗？"
2. **大盲注**：压力阶段——"我有没有在上一关商店里做好修正？"
3. **Boss Blind**：检定阶段——"我的 build 在最苛刻的条件下还成立吗？"
4. **商店**：喘息+调整——"下两关我需要补什么？"

这个四阶段循环每隔约 3–5 分钟重复一次，频率高但每次都有意义。

---

## 关卡节奏和经济的交叉

商店出现在每个 Boss Blind 之后，意味着：
- 过 Boss Blind 本身就会产生成就感
- 商店是奖励节点，而不是惩罚前的准备节点

这个顺序很关键：**先检定，再修正**，而不是先买好再去打。这让商店里的每次购买都带有"基于刚才那一关的判断"的决策感。

---

## 结论

关卡节奏的关键不是"关卡多"，而是每个节奏节点的功能不同，且三段一组的循环周期短到让玩家始终感受到进展。8 个 Ante 的压力是指数级的，这意味着在中期就必须完成 build 的核心成型，而不能靠后期再补救。
