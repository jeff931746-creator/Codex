# 小丑牌幻灵牌设计拆解

## 这个专题要回答什么

- 幻灵牌为什么承担"破格风险层"，以及它的具体效果类型和使用时机

## 总体判断

幻灵牌不是为了平滑成长，而是让玩家在关键节点做出高风险、高戏剧性的越界决策。它负责把"局内能走多远"的上限拉到常规系统之外的范围，代价是通常附带不可忽视的副作用。

---

## 幻灵牌的效果分类

### 版本提升类：给牌加 Edition

| 牌名（示例） | 效果 |
| --- | --- |
| Aura | 给选定扑克牌加 Foil/Holo/Polychrome 版本（随机）|
| Ectoplasm | 给随机 Joker 加 Negative（+1槽），但手牌上限永久 -1 |
| Immolate | 销毁 5 张随机手牌，获得 $20 |

**Ectoplasm** 是幻灵牌里最典型的"破格"设计——额外的 Joker 槽可能让 build 突破上限，但永久减少手牌数意味着后续每局打出有效牌型的难度都会上升。这是一个真正意义上的双向取舍。

---

### 批量改造类：大范围修改牌库

| 牌名（示例） | 效果 |
| --- | --- |
| Familiar | 销毁一张随机手牌，创建 3 张带有随机增强的花色牌 |
| Grim | 销毁一张随机手牌，创建 2 张带有随机增强的 A |
| Incantation | 销毁一张随机手牌，创建 4 张带有随机增强的牌 |
| Talisman | 给所有 Gold 印章的牌额外 +1 Gold 印章 |

批量改造类幻灵牌的特点是**以销毁换取大量带增强的新牌**。这一类在前期可能适得其反（销毁了核心牌），但在中后期牌库已经稳定时，是快速批量增强牌库的高效手段。

---

### Joker 改造类：直接操作 Joker 系统

| 牌名（示例） | 效果 |
| --- | --- |
| Hex | 给随机 Joker 加 Polychrome，销毁所有其他消耗品 |
| Ouija | 随机选一种花色，牌库里所有牌改为该花色（但手牌上限 -1）|
| Ankh | 复制一张随机 Joker，销毁所有其他 Joker |

**Ouija** 是同花流的终极加速件——把整副牌库强制统一花色，但代价是手牌数减少，意味着每次打出同花的难度不变但组合余地减少。

**Ankh** 的逻辑更极端：把最强的那张 Joker 复制一份（两倍强度），但清掉其余所有 Joker，彻底改变现有触发链结构。这是赌局面已经足够只靠一张 Joker 撑下去时的孤注一掷。

---

### 牌库结构类：重塑整副牌

| 牌名（示例） | 效果 |
| --- | --- |
| Black Hole | 将所有星球牌升级一次 |
| The Soul | 创建一张传奇 Joker（极稀有）|

**Black Hole** 是少数纯正向的幻灵牌——全体星球牌各升一级，没有副作用，但需要消耗一张幻灵牌槽位。在路线确认、已有多张星球牌时使用价值极高。

---

## 副作用的设计逻辑

幻灵牌的副作用不是随机惩罚，而是针对该效果的核心收益做出对等代价：

| 效果强度 | 典型副作用 |
| --- | --- |
| Joker 槽位增加 | 手牌永久 -1 |
| 整副牌库花色统一 | 手牌永久 -1 |
| 复制最强 Joker | 清空其余所有 Joker |
| 批量创建增强牌 | 随机销毁一张现有手牌 |

这个设计模式确保幻灵牌永远是"判断当前局面值不值得赌"的问题，而不是无脑使用的强化件。

---

## 使用时机判断

| 局面 | 是否适合用幻灵牌 |
| --- | --- |
| 路线未定，核心 Joker 还没齐 | 通常不适合，副作用可能伤害还未成型的底盘 |
| 路线已成型，在某个关键方向缺临门一脚 | 适合，赌一次可能直接封顶 |
| 分数紧，当前 build 已经跑不过关卡门槛 | 可以赌，反正不赌也会输 |
| 正在稳步推进，不缺任何东西 | 不适合，引入副作用没有必要 |

---

## 结论

幻灵牌是《小丑牌》里负责制造"这局有点不一样"的资源层。它的存在让每局都有可能出现常规路线之外的极端 build，也让玩家在面对它时必须做真正有赌注的决策，而不是无脑使用。

它不是流派的底盘，也不是流派的方向——它是在底盘稳了、方向定了之后，用来把上限推向不可预期地带的最后一个工具。
