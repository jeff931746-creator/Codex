# 小丑牌扑克牌执行层拆解

## 这个专题要回答什么

- 扑克牌系统在《小丑牌》里承担什么，以及它具体如何和其他模块咬合

## 总体判断

扑克牌是整套系统的执行层——它不决定流派方向，但决定每次结算能产生多少分。Joker 和星球牌最终都要落在扑克牌的结算上才能兑现。理解扑克牌的结构，是理解所有其他模块如何联动的前提。

---

## 结算公式

```
得分 = (参与结算的牌面值之和 + 牌型基础筹码) × (牌型基础倍率 + 加成倍率)
```

关键：**只有构成牌型的牌才参与筹码结算**。

- 出一对 5，手里还有 3 张其他牌 → 只有两张 5 的牌面值计入筹码
- 出同花顺 → 五张牌全部参与结算

这个机制直接决定了塔罗牌应该强化哪些牌、哪些牌值得留在手里。

---

## 牌面值

| 牌面 | 筹码值 |
| --- | --- |
| 2–10 | 面值本身 |
| J / Q / K | 10 |
| A | 11 |

大牌（A/K/Q/J）的筹码值并不比 10 高多少，但它们在某些牌型（如皇家同花顺）里是构成条件，对特定流派有结构价值。

---

## 花色系统

四种花色（黑桃、红心、梅花、方块）本身没有数值差异，但花色是大量 Joker 的触发条件。

- 同花系 Joker：只计同一花色的牌
- 特定花色 Joker：仅对红心/黑桃等触发
- 花色不影响基础结算，但影响 Joker 触发率

这意味着**牌库的花色分布是隐性构筑变量**——不是一张牌的个体问题，而是整副牌的花色比例问题。塔罗牌"Wild"改花色，就是在直接修改这个变量。

---

## 牌面增强（Enhancement）

增强是通过塔罗牌施加在单张扑克牌上的永久属性，是执行层最直接的成长手段。

| 增强类型 | 效果 | 适用场景 |
| --- | --- | --- |
| Bonus | 结算时额外 +30 筹码 | 底盘不够时补量 |
| Mult | 结算时额外 +4 倍率 | 倍率不足的流派 |
| Wild | 可视为任意花色 | 同花流派的修正件 |
| Glass | 结算时 ×2 倍率，有 1/4 概率破碎 | 高风险高回报 |
| Steel | 留在手中（未打出）时持续 ×1.5 倍率 | 低频牌型 + 高频持有 |
| Stone | 无论是否构成牌型始终 +50 筹码，无花色/牌面 | 厚筹码底盘 |
| Gold | 回合结束时若在手中得 $3 | 经济流 |
| Lucky | 1/5 概率 +20 倍率，1/15 概率 +$20 | 幸运触发流 |

**Steel 牌的特殊性**：它不需要出牌就能触发，这意味着牌库越薄、这张牌越频繁留在手里，效益越高。它和塔罗牌"拆牌"操作有天然联动。

**Glass 牌的特殊性**：破碎是永久销毁，是《小丑牌》里少数真正有下行风险的决策。在有"复制牌"或"重制牌"能力前，Glass 牌需要保守使用。

---

## 牌面版本（Edition）

版本是施加在单张扑克牌上的稀有加成，可以叠加在增强之上。

| 版本 | 效果 |
| --- | --- |
| Foil（箔面） | +50 筹码 |
| Holographic（全息） | +10 倍率 |
| Polychrome（多彩） | ×1.5 倍率 |
| Negative（负片） | +1 Joker 槽位 |

Negative 最为特殊——它不提升单次结算，而是永久增加 Joker 容量，直接改变构筑上限。

---

## 印章（Seal）

印章是施加在单张牌上、触发额外行为的属性，效果不是即时的而是条件触发的。

| 印章 | 触发时机 | 效果 |
| --- | --- | --- |
| 金色印章 | 打出时 | 得 $3 |
| 红色印章 | 打出时 | 该牌结算再触发一次 |
| 蓝色印章 | 回合结束时若在手中 | 生成对应牌型的星球牌 |
| 紫色印章 | 弃牌时 | 生成一张塔罗牌 |

红色印章是整个印章系统里强度最高的，因为它让一张牌的结算翻倍触发，等同于把 Glass/Steel/Mult 等增强的收益直接加倍。紫色印章则把"弃牌"这个通常是损耗的动作转换成了塔罗牌来源。

---

## 执行层如何和其他模块联动

```
扑克牌（执行层）
  ← 塔罗牌施加增强和印章（改造执行层）
  ← 星球牌提升牌型基础值（抬高执行层天花板）
  ← Joker 的触发条件大多依赖执行层（如特定花色、特定牌面、特定牌型）
  ← 幻灵牌的改造通常影响整副牌库（批量修改执行层分布）
```

---

## 结论

扑克牌系统是整套机制的结算地基。其他四种牌种最终都要通过影响"哪些牌参与结算、每张牌结算多少"来实现收益。

设计上值得注意的是：增强、版本、印章三层属性彼此独立叠加，意味着**一张被充分改造的单牌可以独立撑起一套流派**，而不需要大量同类牌的组合。这是《小丑牌》能够孕育出"单卡流"的结构基础。
