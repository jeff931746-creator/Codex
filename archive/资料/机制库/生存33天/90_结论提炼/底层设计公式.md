# 生存33天·底层设计公式

## 这一层的用途
这里的结论脱离本游戏语境仍应成立。它们揭示了“末日生存+阵营克制”品类在底层结构上的因果关系——当游戏试图将生存压力转化为策略抉择、并让阵营搭配与探索路线深度绑定时，必须遵守的机制传导逻辑。这些命题不依赖具体的废土题材或移动端平台，而是该设计范式成立的基石。

## 结论列表

**结论 1：探索风险权衡跃迁律**

If 迷雾探索需要权衡路线风险（敌人等级浮动）、时间成本（消耗速率变化）和小队状态，Then 探索系统就会跃迁为路线构筑层而非单纯的地图移动层。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立。若探索无需权衡风险、成本与状态，它就退化为单纯的地图移动。

**结论 2：生存消耗对冲律**

If 时间推移会自动消耗基础物资且探索是物资唯一主动来源，Then 探索行为就从可选变成了强制，玩家无法通过停滞来规避风险。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立。若无自动消耗或探索非唯一来源，玩家可通过囤积或停滞规避风险。

**结论 3：阵营检定补短律**

If 遭遇战采用纯阵营敌人进行克制检定，Then 玩家受挫后的行为会转向补齐阵容短板而非纯堆数值。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立。若无纯阵营检定，玩家受挫后更可能通过堆叠数值（升级/强化）来解决问题。

**结论 4：倒计时决策压缩律**

If 生存游戏采用固定倒计时作为终局节点，Then 资源分配就从无限养成变成了限量时间内的最优化问题。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立。若无固定终局，资源分配会趋向于无限长线养成，不存在最优化时间窗口。

**结论 5：战损止损变现律**

If 广告加速绑定在战损恢复和探索返回节点，Then IAA收入会与生存高压期的高频需求强绑定。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立。若广告加速绑定在非紧急节点（如装饰购买），则IAA收入与生存压力脱钩。

## 边界
这套公式在缺乏生存消耗压力或阵营克制不构成硬检定的纯模拟经营游戏中不适用；在重操作反应轻策略构筑的动作类生存游戏中也不适用。当游戏的核心乐趣不再依赖于“探索路线的风险权衡与生存消耗的对冲”时，上述因果链便会断裂。
