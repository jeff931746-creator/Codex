# 生存33天·商店与奖励拆解

## 这个专题要回答什么

《生存33天》的商店、兑换和日常奖励为什么不是附属系统，而是直接参与阵容成型的修正器？

## 总体判断

这套奖励系统的作用不是“发福利”，而是把玩家的阵容缺口和生存危机转成可被修补的节点。商店解决当下缺哪个阵营，日常签到解决长期缺核心幸存者，广告加速解决紧急缺状态。

## 奖励节点是怎么工作的

### 1. 商店兑换负责定向补缺阵营

- 触发条件：玩家进入商店，消耗500友情点或20核心元件
- 具体结果：直接获取物理、神圣或魔法阵营的指定幸存者碎片x5
- 为什么有效：商店是最直接的“把阵营短板变成选择题”的装置，玩家可按需补缺

### 2. 日常签到负责保底幸存者获取

- 触发条件：每日登录触发签到结算
- 具体结果：累计签到7天获得神圣阵营角色“圣光守护者”，累计签到30天获得魔法阵营核心角色
- 为什么有效：确保不付费玩家也能在33天内凑齐基础阵容，维持生存推进

### 3. 广告加速负责紧急状态恢复

- 触发条件：幸存者战损生命值低于30%或探索返回时间剩余超过2小时时，点击免费加速按钮
- 具体结果：观看30秒广告后，立即恢复生命值20%或缩短等待时间50%
- 为什么有效：在高压生存节点提供IAA止损手段，把时间紧迫感转化为变现点

## 奖励层和经济层的分工

- 奖励层决定“现在给你什么幸存者和物资”（如签到送角色、商店兑换碎片）
- 经济层决定“这些东西怎么被生存消耗吃掉、循环到哪里去”（如每4小时消耗食物）
- 只给奖励不建消耗，会让物资溢出；只建消耗不给补足，会让玩家绝望

## 这个模块不负责什么

- 不负责探索路线风险，那是迷雾探索与路线的职责
- 不负责战斗中怎么利用阵营克制，那是战斗执行层的职责
- 不负责33天的时间压力，那是生存节奏与倒计时的职责

## 可迁移结论

- If 商店能直接兑换特定阵营幸存者碎片，Then 阵容偏科就能通过付费或囤积资源快速修正
- If 广告加速绑定在战损恢复（生命值<30%）和探索返回节点，Then IAA收入会与生存高压期的高频需求强绑定
