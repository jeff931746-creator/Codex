# 《弹壳特攻队》总档

## 基本信息

- 游戏名：弹壳特攻队
- 类型标签：App手游 / Roguelite生存割草 / 装备养成
- 平台：iOS + Android
- 全球版本：Survivor.io
- 资料来源：TapTap游戏页、百度百科、知乎攻略、小米游戏中心攻略、玩家社区帖子

## 一句话定义

一款用"竖屏割草局内 + 装备神铸局外"把Roguelite生存与长线养成结合起来的手游：局内靠3选1技能和武器合成声明本局强化路线，局外靠六件装备的神铸升级、天赋树、特工技能把底盘数值持续拉高，付费和免费玩家共用同一个成长终点但速度差距明显。

## 核心结构

- 核心循环：进入关卡（8或15分钟）→ 自动射击击杀敌人 → 击杀获经验 → 局内升级时3选1技能 → 生存至时间结束 → 结算奖励（装备/金币/图纸） → 局外强化装备（合成/神铸）→ 升级天赋、强化特工 → 推进主线章节解锁更高难度

- 主要资源：金币（天赋解锁/装备升级）、钻石（高级货币/广告获取）、图纸（装备合成）、S特工碎片（特工解锁/升级）、永恒/破坏/混乱之石（神铸专用材料）、体力（进入关卡消耗）

- 主要系统：战斗执行层、技能构筑层（局内Roguelite）、装备神铸层（局外核心养成）、特工系统、天赋树、经济循环、关卡结构

- 主要节奏节点：每次局内升级（3选1技能）、装备品阶跨越（金→红）、神铸等级突破（8级解锁神炼）、特工获取/升阶、主线章节门槛

- 主要检定节点：主线每隔20~30章的难度跨越节点（强制数值门槛）、回响之战赛季（定期难度上调检定当前构筑）、月矿远征副本

## 本游戏最值得拆的专题

- 局内3选1技能为什么是路线决策而不是纯力量堆叠——答：武器合成要求两个特定技能同时持有，迫使玩家在升级时主动锁定方向而非随机选强
- 装备神铸线（永恒/破坏/混乱）为什么会形成不同的构筑偏向——答：三条线对应不同的增伤乘区，叠乘规则决定了同一套装备的不同神铸方向会产出截然不同的战力天花板
- 无排行榜+无社交的付费设计为什么还能跑通——答："毕业制"成长路线让玩家为缩短到达终点的时间买单，而非为超越他人竞争买单
- 回响之战赛季为什么是检定层而不是内容层——答：它检定当前装备/特工搭配能否通过动态难度上调，不提供新内容，只提供当前构筑的横向对比压力

## 总判断

- 这游戏靠什么成立：把Roguelite局内"15分钟技能构筑"的即时爽感和"半年毕业"的长线养成深度接在同一条成长轴上——每局局内有取舍，每天局外有进度，两套反馈钟表频率互不干扰
- 最大机会点：神铸多线（永恒/破坏/混乱/异星）天然提供构筑分叉，不同神铸路线对应不同技能组合优先级，使装备养成本身具有构筑语言
- 最大风险点：局内Roguelite决策深度随局外数值积累而下降——当装备强到能无脑通关时，3选1的取舍感会消失，游戏退化成纯刷资源的放置游戏
