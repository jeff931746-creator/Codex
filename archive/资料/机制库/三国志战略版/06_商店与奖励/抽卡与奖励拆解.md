# 三国志战略版·抽卡与奖励拆解

## 这个专题要回答什么

《三国志战略版》的抽卡为什么不是“发福利”，而是“构筑的原材料分配器”？

## 总体判断

这套抽卡与奖励系统的作用不是给玩家发变强的道具，而是分配构筑阵容的“原材料”。因为战法传承需要消耗武将卡，觉醒和升红需要同名卡，抽卡直接决定了玩家能走哪条路线、能走多远。官方宣称无VIP不卖资源，但高级武将与S级战法的获取高度依赖金铢抽卡。

## 怎么判断某个东西属于奖励层

- 它如果在“缺什么卡就补什么卡”这个逻辑里工作，就属于奖励层
- 它如果主要决定武将和战法进入速度，而不是直接决定战斗胜负，就属于奖励层
- 它如果可以被玩家按需抽取、兑换或领取，就更像奖励层而不是战斗层

## 奖励节点是怎么工作的

### 1. 金铢抽卡是构筑原材料的唯一稳定来源

- 触发条件：玩家消耗金铢或玉璧进行抽卡
- 具体结果：获得名将、良将等武将卡，核心橙将（如张角、陆逊）抽卡概率约为0.06%
- 为什么有效：用低概率卡住核心构筑件的产出，把付费动力直接绑定到阵容成型上

### 2. 重复抽取的冗余转成红度底盘

- 触发条件：抽到已拥有的武将
- 具体结果：转化为红度提升，提供每红度4%全属性加成
- 为什么有效：把抽卡歪掉的负反馈转成长线底盘成长，让每次抽卡都不算完全浪费

### 3. 赛季签到与福利降低入门门槛

- 触发条件：新版本或赛季开启，玩家登录领取
- 具体结果：获得核心橙将（如S2赛季签到送庞统）或A级战法
- 为什么有效：给0氪玩家提供基础构筑件，保证底层玩家有最低限度的阵容参与感

### 4. 不可逆拆解让抽卡结果产生质变

- 触发条件：玩家拆解武将获取传承战法
- 具体结果：武将卡消失，战法解锁。例如拆解张角获得太平道法
- 为什么有效：让抽到的卡不只是数值提升，而是构筑路线的关键开关

## 奖励层和经济层的分工

- 奖励层决定“现在给你什么卡”
- 经济层决定“这些卡怎么被花掉、循环到哪里去”
- 只给奖励不建流向，会让抽卡显得像赌博；只建流向不发奖励，会让玩家觉得系统空转

## 这个模块不负责什么

- 不负责战斗怎么打，那是执行层的职责
- 不负责战法怎么配，那是构筑层的职责
- 不负责赛季什么时候开，那是节奏层的职责

## 可迁移结论

- If 抽卡系统产出的卡牌同时承担“构筑路线开关（传承战法）”和“觉醒底盘材料（红度）”双重职责，Then 玩家对抽卡的需求会从“想要更强”变成“必须要有”，付费意愿会显著提升
- If 游戏通过签到赠送核心构筑件（如S2赛季签到送庞统），Then 0氪玩家的基础参与感能得到保底，但与氪金玩家的红度差距依然明显
