# 三国志战略版·付费与招募奖励层拆解

## 这个专题要回答什么

《三国志战略版》的付费与招募为什么不是"卖数值"，而是"卖构筑核心件"？

## 总体判断

这套付费系统的核心不是直接卖属性，而是卖战法与阵容构筑的关键组件——橙将和S级战法。重复武将可转化为战法点和进阶材料，让抽卡不只是赌概率，而是规划构筑路线的投入。玉璧与金铢的双轨制在大小R之间划出进度鸿沟，但同盟分工让微氪也能找到职能位置。

## 怎么判断某个东西属于付费与招募奖励层

- 它如果主要改变武将和战法的获取速度，就属于奖励层
- 它如果用充值货币购买关键构筑件，就属于奖励层
- 它如果只改变资源产出而不直接提供武将，就不属于招募层，而是经济层

## 核心结构

### 1. 招募抽卡获取核心武将

- 触发条件：玩家消耗金铢或玉璧进行招募
- 具体结果：随机获得名将（橙将）、良将（紫将）等，橙将概率极低但构筑价值极高
- 设计原因：用低概率高价值的卡池驱动核心付费，橙将的战法和适性是顶级阵容的门槛

### 2. 重复武将转化为战法经验与进阶材料

- 触发条件：抽到重复武将卡
- 具体结果：可用于拆解高级战法（如拆解庞统获太平道法），或作为进阶材料提升武将属性，也可转化为战法经验升级战法
- 设计原因：消除抽到重复卡的负反馈，让抽卡永远有转化出口，驱动持续付费

### 3. 玉璧与金铢双轨制区分进度

- 触发条件：充值获得玉璧，日常任务获得金铢
- 具体结果：玉璧可高频次招募，金铢只能低频次招募，大小R的构筑成型速度差距极大
- 设计原因：用进度差驱动付费，同时保留免费玩家的基本生存空间

### 4. 限时礼包定向补缺

- 触发条件：玩家达到主城7级升级瓶颈、武将20级觉醒瓶颈时弹出限时礼包
- 具体结果：用小额付费获取加速道具、资源或战法点
- 设计原因：在玩家最需要补缺的节点提供付费出口，转化率高

## 这个模块不负责什么

- 不负责抽到武将后怎么配队，那是战法与阵容构筑层的职责
- 不负责武将升级和建筑升级，那是武将与建筑底盘层的职责
- 不负责大地图行军和攻城，那是沙盘与行军节奏层的职责

## 可迁移结论

- If 抽卡产出的重复武将卡能1:1无损转化为S/A级战法（如拆解庞统获太平道法）或进阶材料，Then 玩家的抽卡意愿会显著高于重复卡纯废弃的设计，付费深度大幅扩展
- If 付费抽卡产出的是具有特定战法联动的武将（构筑核心件）而非直接的属性数值包，Then 大R的优势体现在阵容多样性和克制覆盖面上，而不是单队绝对战力碾压
