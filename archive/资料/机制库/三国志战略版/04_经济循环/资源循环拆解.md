# 三国志战略版·资源循环拆解

## 这个专题要回答什么

《三国志战略版》的资源循环为什么是“从被动等待到主动规划的杠杆”，而不是“挂机收菜”？

## 总体判断

这款游戏的资源循环本质上是“领地产出+提案加速”的双轨制：基础四资（木材、铁矿、石料、粮食）和铜币由领地按时产出，支撑日常征兵与建筑；而提案系统允许玩家按需选择资源获取倍率，把被动等待变成了主动规划。资源的核心出口是征兵，征兵速度直接决定了玩家在沙盘上的出击频率。

## 怎么判断某个元素属于经济循环

- 它如果主要改变“拿到什么材料、花在哪、什么时候花”，就属于经济循环
- 它如果只改变战斗中谁打得更快，但不改变资源流向，就不属于经济循环
- 它如果能把一次性掉落转成长期价值，就一定在经济循环里占有位置

## 资源分层怎么工作

### 1. 领地定时产出支撑基础运转

- 触发条件：玩家占领木材、铁矿、石料、粮食地及铜矿
- 具体结果：按小时产出对应资源，资源地等级越高产出越快
- 设计原因：用领地面积和品质控制玩家的基础发育速度，制造开荒期的抢地压力

### 2. 提案系统把被动收菜变成主动规划

- 触发条件：玩家在提案系统中选择特定政策（如征调兵粮、因粮于敌）
- 具体结果：选择“征调兵粮”提升40%木牛流马资源量，或“因粮于敌”通过交战每日获取40万资源
- 设计原因：给玩家提供资源缺口的自救通道，让资源循环从纯挂机变成按需调节的杠杆

### 3. 征兵是最大的资源消耗出口

- 触发条件：玩家在前线征兵
- 具体结果：大量消耗粮食、木材、铁矿和铜币，补充战损兵力
- 设计原因：用战损与征兵的循环消耗把战斗频率与资源储备绑定，打不起仗比打不赢仗更致命

### 4. 金铢与玉璧作为跨层硬通货

- 触发条件：充值、任务或活动发放
- 具体结果：用于抽卡、购买策书、加速建筑等
- 设计原因：用硬通货把付费与核心资源流转打通，付费可以直接转化为征兵速度与武将获取

## 经济循环的两个关键出口

- 一个出口是“征兵相关消耗”，直接决定玩家在沙盘上的持续作战能力
- 另一个出口是“建筑与战法升级消耗”，直接决定玩家城建设施与战法强度的成长速度
- 如果一个资源不能明确流向这两个出口之一，它就更像装饰性奖励，不像核心经济

## 这个模块不负责什么

- 不负责具体战斗胜负，那是执行层与构筑层的职责
- 不负责武将怎么配，那是战法与阵容构筑的职责
- 不负责发什么武将，那是抽卡与奖励的职责

## 可迁移结论

- If 资源循环在领地定时产出之外，允许玩家通过提案或政策按需选择资源获取倍率（如提升40%木牛流马资源量），Then 资源瓶颈会从“被动等待时间”变成“主动规划选择”，玩家的留存意愿与策略参与度会提升
- If 征兵消耗远大于日常产出，且战损兵力不可自动回复，Then 战损恢复速度将成为比单场战斗胜负更核心的战略制约点
