# 三国志战略版·资源与征兵循环拆解

## 这个专题要回答什么

《三国志战略版》的资源循环为什么能把"占地"变成"生存压力"，而不是简单的收菜？

## 总体判断

这游戏的经济循环本质是"领土即产能，战损即消耗"。占地定时产出木铁石粮，征兵消耗资源补充兵量。战损越高，征兵压力越大，资源地不够就会断兵。玩家不是在单纯攒资源，而是在维持一条"扩张-战损-补兵-再扩张"的战争机器。

## 怎么判断某个东西属于资源与征兵循环

- 它如果主要改变木铁石粮铜的产出和消耗速度，就属于经济层
- 它如果能把领土面积转化为兵力补充速度，就属于经济层
- 它如果只改变战斗伤害而不改变资源流向，就不属于经济层

## 核心结构

### 1. 占地定时产出，领土即产能

- 触发条件：部队成功占领空地或玩家资源地
- 具体结果：该地块开始每小时产出木材、铁矿、石料或粮食，产量随地块等级提升
- 设计原因：把领土扩张直接转化为经济收益，打地不只是为了过图，而是为了维持战争机器

### 2. 征兵消耗资源，战损即经济压力

- 触发条件：部队兵力受损，玩家在兵营发起征兵
- 具体结果：消耗木材、铁矿、石料、粮食和铜币补充兵量
- 设计原因：用战损消耗控制进攻节奏，打赢了但资源跟不上，也无法持续扩张

### 3. 屯田与掠夺提供短期冲刺手段

- 触发条件：玩家对已占领地块发起屯田，或对敌军领地发起掠夺
- 具体结果：屯田在一段时间后获得大量指定资源，掠夺直接抢夺敌方部分仓库资源
- 设计原因：在常规产出不足时提供短期冲刺或报复手段，同时增加玩家间互动摩擦

### 4. 仓库容量限制资源囤积上限

- 触发条件：资源产出达到仓库容量上限
- 具体结果：超出部分不再产出，玩家必须消耗资源升级仓库或升级建筑
- 设计原因：防止无限囤积，逼迫玩家持续进行建筑升级和征兵，保持经济循环的流动性

## 这个模块不负责什么

- 不负责具体战斗怎么打和战损比例，那是战斗执行层的职责
- 不负责武将怎么配和战法怎么选，那是战法与阵容构筑层的职责
- 不负责大地图行军和攻城分工，那是沙盘与行军节奏层的职责

## 可迁移结论

- If 征兵需要等量消耗木材铁矿石料粮食且战损直接扣除兵量，Then 玩家的进攻决策就会极度谨慎，"穷兵黩武"会被资源链断裂惩罚
- If 沙盘上占领的地块是木材铁矿石料粮食的唯一被动产出源且产能与地块等级/数量直接绑定，Then 寸土必争就不是口号，而是经济生存的硬需求
