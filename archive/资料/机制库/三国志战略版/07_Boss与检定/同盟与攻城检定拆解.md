# 三国志战略版·同盟与攻城检定拆解

## 这个专题要回答什么

《三国志战略版》的同盟战与攻城为什么是“结构检定器”，而不是“纯战力比拼”？

## 总体判断

这套检定系统不是只看个人战力高低，而是看阵容克制与同盟分工有没有短板。只会配一队主力的玩家可能在开荒期很舒服，但一到攻城战和洛阳决战，就会暴露兵种被克制、体力耗尽或缺乏铺路卡位能力的缺口。同盟若缺乏执行力，即使全员满红也无法攻下核心城池。

## 怎么判断某个节点属于检定层

- 它如果会强迫玩家暴露阵容或同盟协作短板，而不是单纯给更多奖励，就属于检定层
- 它如果会改变玩家对“要不要继续补同一条线”的判断，就属于检定层
- 它如果只是在沙盘上平滑加难，而不会导致构筑修正，就更像节奏层而不是检定层

## 检定是怎么发生的

### 1. 高级地守军检定前期开荒构筑

- 触发条件：玩家攻打7级以上资源地
- 具体结果：战法未成型或兵种被克制的队伍极易战败，战损极高
- 为什么有效：把“开荒队怎么配”这件事显性化，逼迫玩家在前期就规划好开荒阵容

### 2. NPC城池守军检定同盟协作与铺路能力

- 触发条件：同盟向NPC城池发起攻城
- 具体结果：需要玩家先铺路到城下，再卡位阻截敌方援军，最后主力集火守军
- 为什么有效：检定同盟是否具备分工执行力，散沙同盟无法完成铺路与卡位的协同

### 3. 敌方满红主力队检定阵容克制链

- 触发条件：在沙盘遭遇敌方高红度主力队
- 具体结果：单纯靠战力值平推极易被克制链碾压，必须出动兵种和战法克制对方的队伍
- 为什么有效：让后期不是纯数值比拼，而是构筑克制链的博弈

### 4. 洛阳决战检定同盟的终极执行力

- 触发条件：赛季末期同盟发起洛阳争夺战
- 具体结果：极其考验同盟的在线率、铺路速度、卡位坚韧度和主力集结密度
- 为什么有效：让同盟的强弱不只看总战力，更看组织度与战略纪律

## 检定层和节奏层的分界

- 节奏层回答“什么时候开战”
- 检定层回答“打赢到底需要什么能力”
- 同样是攻城，只有能指出阵容或同盟短板类型的节点才算检定

## 这个模块不负责什么

- 不负责长期资源怎么来，那是资源循环的职责
- 不负责赛季开服的时机，那是节奏层的职责
- 不负责发什么武将，那是抽卡与奖励的职责

## 可迁移结论

- If 攻城检定需要同盟分工（主力冲锋与平民铺路）且大地图存在空间阻隔机制，Then 检定对象就从个人阵容偏科扩展为同盟结构偏科，平民玩家也能在检定中找到不可替代的位置，0氪流失率会降低
- If 阵容检定专门针对兵种与战法克制而非纯战力数值，且存在不可逆构筑约束，Then 玩家受挫后的行为会转向补齐克制链阵容而非继续堆单一队伍的数值
