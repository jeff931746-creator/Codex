# 三国志战略版·武将与底盘养成拆解

## 这个专题要回答什么

《三国志战略版》的武将养成为什么是“底盘”，而不是“路线”？

## 总体判断

武将养成系统负责把单赛季里看得到的强度变成可持续积累的底盘。它不直接告诉玩家该走哪条流派（那是战法搭配的职责），但它决定了这条流派能不能稳定兑现，以及兑现速度有多快。红度（重复抽取）和觉醒是这个底盘最核心的沉淀物。

## 武将养成由哪些部分组成

### 1. 等级与觉醒是基础兑现

- 触发条件：武将上阵战斗获取经验，或消耗同名武将卡/材料进行觉醒
- 具体结果：武将等级提升增加基础面板，觉醒解锁第三战法格，使战法联动成为可能
- 设计原因：第三战法格是构筑成型的硬性门槛，未觉醒的武将无法承担核心路线职责

### 2. 红度（重复抽取）把付费转成底盘压制

- 触发条件：抽到重复的武将卡
- 具体结果：武将红度提升，获得每红度4%全属性加成。满红（5次重复抽取）武将相比白板（0次重复）拥有约20%全属性压制
- 设计原因：用红度机制把抽卡的冗余负反馈转成底盘成长的正反馈，同时拉开付费玩家与免费玩家的数值兑现速度

### 3. 装备与洗练提供长线微调

- 触发条件：武将20级解锁装备栏，穿戴武器、防具、坐骑、宝物
- 具体结果：装备提供固定属性加成，洗练可重铸附加属性
- 设计原因：给长线玩家提供日常可追求的微小成长点，平滑赛季中后期的养成曲线

### 4. 主公历练与幕僚改变战法形态

- 触发条件：主公在沙盘历练解锁城池、获得幕僚，将幕僚上阵至城池
- 具体结果：获得治军战法，使部分核心传承战法产生进化效果（如太平道法进化为太平道法·治军）
- 设计原因：用局外轻度系统为特定战法提供额外乘区，让老战法在新赛季有新的底盘兑现方式

## 这个层级不负责什么

- 不负责战法怎么搭配，那是战法与阵容构筑的职责
- 不负责战斗中的兵种克制与行军，那是战斗执行层的职责
- 不负责提供觉醒和升红所需的同名卡，那是抽卡与奖励的职责

## 可迁移结论

- If 重复抽取卡牌能提供百分比属性加成（如每红度4%全属性），且战斗结算高度依赖面板数值乘区，Then 付费玩家的底盘兑现速度将远超0氪，冗余抽卡从负反馈变为长线驱动力
- If 养成系统存在“解锁关键功能位”（如第三战法格）的觉醒门槛，Then 玩家的构筑节奏会被强制卡点，未觉醒前无法验证完整路线
