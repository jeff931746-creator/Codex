# 三国志战略版·武将与建筑底盘层拆解

## 这个专题要回答什么

《三国志战略版》的武将养成和建筑升级为什么是"底盘"，而不是"路线"？

## 总体判断

武将升级、进阶和建筑升级负责把战前构筑的路线兑现成可用的数值和带兵量。它不直接告诉玩家该走兵刃还是谋略，但它决定了这条路线能不能在高级地上站稳，以及战损能不能控制在可接受范围内。

## 怎么判断某个东西属于武将与建筑底盘层

- 它如果主要提升基础面板、带兵量上限或解锁系统功能，就属于底盘层
- 它如果跨局保留且长期沉淀，就属于底盘层
- 它如果改变伤害结算方式或战法形态，就不属于底盘层，而是构筑层

## 核心结构

### 1. 升级直接挂钩带兵量，让成长等于战力

- 触发条件：武将通过扫荡、练兵获取经验升级
- 具体结果：武将属性提升，带兵量上限增加，战力直接上涨
- 设计原因：让成长反馈极度直观，升一级就能多带兵，打地就更轻松

### 2. 进阶强化属性与兵种适性

- 触发条件：消耗重复武将卡进行进阶
- 具体结果：武将基础属性提升，部分武将兵种适性提升（如A升S）
- 设计原因：给重复卡找第二个消耗出口，同时微调武将的适性边界

### 3. 建筑升级解锁队伍队列与高级兵种

- 触发条件：消耗木铁石粮升级主城、兵营等建筑
- 具体结果：主城升级解锁更多部队队列，兵营升级解锁高级兵种（如3级兵升5级兵）
- 设计原因：用建筑时间锁死赛季节奏，控制全服战力膨胀速度，防止玩家过早满编高级兵

### 4. 装备系统在20级解锁，提供额外属性乘区

- 触发条件：武将达到20级
- 具体结果：解锁武器、防具、坐骑、宝物四个装备栏，直接提升属性
- 设计原因：在中期提供新的养成深度，拉开同期武将的数值差距

## 这个模块不负责什么

- 不负责战法怎么搭配和阵容怎么组，那是战法与阵容构筑层的职责
- 不负责具体战斗的克制和战损结算，那是战斗与兵种执行层的职责
- 不负责资源怎么产出，那是资源与征兵循环的职责

## 可迁移结论

- If 武将升级的唯一收益是带兵量上限的提升且带兵量等于战力，Then 玩家的养成目标会极度聚焦于经验获取，任何能加速升级的资源（如练兵经验）和道具付费意愿都会极高
- If 在赛季制SLG中，主城与兵营的建筑升级是解锁高级兵种和第三支队伍队列的唯一途径，Then 建筑升级时间就会成为全服战力节奏的硬阀门
