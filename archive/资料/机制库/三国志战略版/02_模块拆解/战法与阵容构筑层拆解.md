# 三国志战略版·战法与阵容构筑层拆解

## 这个专题要回答什么

《三国志战略版》的战法搭配为什么能把"抽卡收集"变成"战前路线声明"？

## 总体判断

战法系统不只是给武将加数值，而是决定这支部队走兵刃爆发、谋略输出还是辅助控制路线的关键开关。拆解闲置武将获取战法，让重复卡变成构筑组件；兵种适性筛选，让阵容搭配不能随意散搭，必须围绕S适性收束方向。

## 怎么判断某个东西属于战法与阵容构筑层

- 它如果会改变一支部队的输出方式和伤害类型，就属于构筑层
- 它如果让玩家必须在物理和谋略之间做排他性选择，就属于构筑层
- 它如果只是提升基础面板而不改变技能形态，就不属于构筑层，而是底盘成长层

## 核心结构

### 1. 拆解闲置武将获取战法，把重复卡变构筑件

- 触发条件：玩家消耗闲置武将卡进行战法拆解
- 具体结果：获得S/A/B级战法（如拆解庞统获太平道法、拆解关羽获横扫千军），可供其他武将学习
- 设计原因：解决抽卡重复的负反馈，让非核心武将也有拆解价值，同时扩大战法池深度

### 2. 兵种适性强制约阵容搭配方向

- 触发条件：武将上阵时，携带兵种适性评级（S/A/B/C）
- 具体结果：S适性享受120%属性加成，C适性只有70%，玩家必须优先选择S适性武将组队
- 设计原因：防止玩家随意混搭，强制收束阵容方向，让"桃园队""魏法骑"等固定流派有存在基础

### 3. 觉醒解锁战法栏，制造构筑阈值

- 触发条件：武将达到20级并消耗同名将或觉醒材料
- 具体结果：武将解锁第三个战法栏，从自带战法+1学习战法，变为自带战法+2学习战法
- 设计原因：用关键阈值拉开武将上下限，觉醒前是半成品，觉醒后才是完整构筑容器

### 4. 流派体系分化为兵刃、谋略与混合

- 触发条件：组合具体武将与战法（如刘备关羽张飞组桃园队，曹操贾诩组魏法骑）
- 具体结果：队伍输出重心明确，兵刃队吃武力加成和物理连击，谋略队吃智力加成和技能发动率
- 设计原因：让构筑有明确终点，不同流派对战法碎片和装备的需求完全不同，避免全服只养一种队

## 这个模块不负责什么

- 不负责战斗中的伤害结算和克制计算，那是战斗与兵种执行层的职责
- 不负责武将怎么升级和进阶，那是武将与建筑底盘层的职责
- 不负责大地图上的行军和占位，那是沙盘与行军节奏层的职责

## 可迁移结论

- If 在卡牌SLG中，拆解重复武将卡是获取S/A级战法的唯一途径，Then 玩家就不会把重复卡当成纯废品，而是会规划拆解优先级（如优先拆解庞统拿太平道法），付费抽卡的挫败感会降低
- If 武将的兵种适性加成在S级与C级之间存在50%的属性鸿沟（120%对70%），Then 阵容搭配就会被强制收束到S适性组合，散搭策略会被迅速淘汰
