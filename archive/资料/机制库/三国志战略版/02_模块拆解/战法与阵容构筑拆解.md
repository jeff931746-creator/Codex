# 三国志战略版·战法与阵容构筑拆解

## 这个专题要回答什么

《三国志战略版》的战法搭配为什么是“有去无回的路线声明”，而不是随时可换的技能栏？

## 总体判断

战法搭配系统是这款游戏最核心的路线层。它不是简单的装备穿脱，而是通过“传承拆解不可逆”的机制，把玩家的每一次战法获取都变成沉重的路线声明。一旦决定拆解张角换太平道法，就等于宣布这赛季要走包含太平道法的路线，且再也拿不回张角本体。

## 战法与阵容构筑到底怎么工作

### 1. 自带战法与传承战法联动形成流派核心

- 触发条件：武将上阵并配置战法
- 具体结果：武将自带战法与传承战法产生化学反应，例如陆逊自带战法火烧连营配合太平道法提升发动率，形成吴枪、魏法骑等流派
- 设计原因：用战法间的联动效应制造构筑深度，让阵容强度来源于组合而非单卡

### 2. 传承拆解的不可逆性锁死路线方向

- 触发条件：玩家决定拆解核心橙将（如张角、孙权）获取其传承战法
- 具体结果：该武将卡消失，玩家获得对应战法，且操作不可撤销。错拆核心橙将会导致该武将相关的所有阵容无法成型
- 设计原因：用极高的试错成本把“随便试试”变成“深思熟虑的路线声明”，构筑深度直接转化为沉没成本

### 3. 阵容克制循环打破单一最强阵容

- 触发条件：不同流派的队伍在沙盘上相遇
- 具体结果：虎臣弓克制魏法骑，魏法骑克制吴枪，吴枪又能在特定环境下反制虎臣弓。不存在绝对无敌的阵容，只有优势对局与劣势对局
- 设计原因：用模糊的克制链逼迫玩家不能只养一队，必须构筑多支不同路线的队伍应对不同检定

### 4. 战法分级制造构筑门槛

- 触发条件：玩家获取S/A/B级战法
- 具体结果：S级战法（如太平道法、抚辑军民）获取难度极高，A级战法作为下位替代，B级战法过渡。缺乏S级战法的阵容在后期检定中极难生存
- 设计原因：用战法品质拉开付费与免费的构筑完整度差距，付费玩家能组建更完善的克制链

## 这个层级不负责什么

- 不负责武将的基础面板数值，那是武将与底盘养成的职责
- 不负责战斗中的具体伤害结算顺序，那是战斗执行层的职责
- 不负责提供拆解所需的武将卡，那是抽卡与奖励的职责

## 可迁移结论

- If 构筑系统允许通过不可逆地消耗核心卡牌来解锁关键战法，且该卡牌同时承担路线开关与底盘养成双重职责，Then 玩家的构筑选择会从“随时切换”变成“沉重的路线声明”，试错成本会直接转化为沉没成本与付费动力
- If 阵容间存在基于战法联动的克制循环而非纯数值碾压，Then 玩家会被迫构筑多支不同路线的队伍，游戏的策略纵深与养成广度会同时提升
