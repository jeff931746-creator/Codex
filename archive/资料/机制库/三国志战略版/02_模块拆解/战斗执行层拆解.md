# 三国志战略版·战斗执行层拆解

## 这个专题要回答什么

《三国志战略版》的战斗执行层为什么是“战前配置的兑现器”，而不是“操作的舞台”？

## 总体判断

这款游戏的战斗执行层不提供战中操作空间，玩家能做的所有决策都在战前完成。战斗执行层的核心职责是把兵种克制、地形阻隔和战法触发顺序兑现成战斗结果。它决定了“这队能不能打赢”，但不决定“这队怎么配”，配将逻辑来自构筑层。

## 战斗执行层到底在做什么

### 1. 兵种克制循环决定基础伤害倍率

- 触发条件：骑、盾、弓、枪四种常规兵种相遇，或器械兵种参战
- 具体结果：骑>盾>弓>枪>骑形成克制循环，器械被所有兵种克制。克制方获得约15%伤害加成，被克制方受到约15%受伤增加
- 设计原因：用极简的循环规则确保玩家在战前必须考虑兵种搭配，避免单一兵种无脑平推

### 2. 真实地形与自由行军制造空间博弈

- 触发条件：部队在沙盘上移动，遭遇山脉、河流、关卡、码头等地形，或玩家下达行军指令
- 具体结果：行军路线受地形阻隔，玩家可在山谷、码头等狭窄地形卡位，实现围点打援或阻截援军
- 设计原因：把战斗从纯数值对撞扩展为空间站位博弈，让执行层拥有非数值的胜负手

### 3. 自动战斗与战报回放把操作权归零

- 触发条件：两支队伍在沙盘上接触
- 具体结果：战斗自动进行，战中不可操作，玩家只能通过战报回放查看战法触发顺序与伤害结算
- 设计原因：强制把玩家的注意力从微操转移到战前构筑与大局调度，拉开大R与平民在操作上的公平性

### 4. 部队可随意切换兵种修正克制关系

- 触发条件：玩家在非战斗状态下切换部队兵种
- 具体结果：同一队武将可以根据敌方阵容调整兵种，以获取克制优势
- 设计原因：让兵种克制成为可被利用的战术工具，而不是锁死配队的一次性绑定

## 这个层级的边界在哪里

- 它不负责武将怎么配、战法怎么选，那是战法与阵容构筑的职责
- 它不负责武将面板数值怎么堆，那是武将与底盘养成的职责
- 它不负责资源怎么征兵，那是资源循环的职责

## 典型识别标准

如果一个元素满足下面任意两条，就更接近执行层：

- 在两军接触时立刻参与结算
- 受地形和站位直接影响
- 其结果只反映当前单场战斗的胜负

如果它主要满足下面这些条件，就不是执行层：

- 需要局外长期投入才能改变
- 本身不直接参与战斗结算
- 更像路线选择或奖励修正

## 这个模块不负责什么

- 不负责战法搭配逻辑，那是战法与阵容构筑的职责
- 不负责武将属性养成，那是武将与底盘养成的职责
- 不负责战斗发生的战略时机，那是赛季与同盟节奏的职责

## 可迁移结论

- If 战斗系统采用完全自动且不可操作的模式，且战法触发存在概率与联动效应，Then 战前构筑与兵种克制的权重将远大于战中微操，玩家的策略重心会前置到配卡与阵容搭配
- If 地形系统能直接阻断行军路线，且大地图存在自由行军机制，Then 空间站位就会成为独立于数值之外的胜负手，大地图的调度价值会显著提升
