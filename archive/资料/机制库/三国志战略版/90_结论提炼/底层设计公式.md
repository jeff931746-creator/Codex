# 三国志战略版·底层设计公式

## 这一层的用途
这里的公式揭示了"SLG+战前构筑"品类在底层结构上的机制传导逻辑——当游戏试图将策略重心从操作转移到战前规划与同盟协作、并将领土扩张与经济生存深度绑定时，必须遵守的代数与因果约束。这些命题不依赖具体的三国题材或沙盘地图，而是该设计范式成立的基石。

## 公式列表

**公式 1：战力等价与策略前置传导**

`战力 = 兵量 × 兵种适性系数 × 战法联动系数`

当且仅当 `兵量` 为可消耗资源且战斗过程不可操作时，策略重心前置成立。
*推导*：兵量作为可消耗资源，使得战损直接转化为经济压力（`战损 = 兵力消耗 = 征兵资源消耗`），迫使玩家在战前通过提升 `兵种适性系数`（针对性出兵）和 `战法联动系数`（配将）来降低战损比。若兵量不可消耗或战斗可操作，策略重心将分散至战中微操。

**公式 2：冗余转化与付费深度传导**

`构筑件获取 = 抽卡产出 + 重复卡转化(拆解/进阶)`

当且仅当 `重复卡转化率 > 0` 且转化物为核心构筑件（如S/A级战法）时，抽卡边际收益不递减。
*推导*：若 `重复卡转化 = 0`，抽卡期望收益随核心卡收集度提升而急剧下降（废卡惩罚），付费深度受限；当 `重复卡转化` 指向核心战法时，每次抽卡均有稳定收益（`收益 = 核心武将 或 核心战法 或 进阶材料`），抽卡意愿随构筑池深度线性增长。

**公式 3：互斥分工与组织度传导**

`攻城效率 = f(主力清守军效率, 器械拆耐久效率) 且 主力拆耐久 ≈ 0, 器械清守军 ≈ 0`

当且仅当两项效率函数互斥（全能单体效率趋近于0）时，组织度成为胜负手。
*推导*：若存在全能单元（`主力拆耐久 > 0 且 器械清守军 > 0`），单体战力可覆盖攻城全流程，组织协同非必要；互斥约束使得 `攻城效率` 必须依赖多单元叠加（`同盟组织度`），单体战力优势无法绕过机制门槛。

**公式 4：领土产能与生存压力传导**

`资源产能 = ∑(占领地块等级 × 产出率)`

当且仅当 `资源产能` 强绑定于领土面积且 `战争消耗（征兵资源）` 巨大时，领土争夺从排名驱动升级为生存驱动。
*推导*：若 `资源产能` 与领土无关（如内城自产足够），则争夺地块仅为荣誉/排名收益；当产能全量依赖地块且征兵消耗极高时，失地 = 断产能 = 无法补兵 = 灭国，生存压力强制寸土必争。

**公式 5：时空精度与战术纵深传导**

`战术纵深 = f(行军时间精度, 地形阻隔度)`

当且仅当 `行军时间精度` 达到秒级且 `地形阻隔度` 强制改变路线时，战术重心从数值延伸至卡位与压秒。
*推导*：若行军时间粗放（如按分钟计）或地形可无视，则空间与时间不可被精确操作，战术退化为纯数值对撞；精度与阻隔度的提升，使得 `到达时间差` 和 `地形卡位` 成为可计算的战术变量（如压秒合击、关卡卡位），策略维度增加。

## 边界
这套公式在缺乏战前构筑深度或战中可操作空间大的动作类游戏中不适用；在无领土概念或战争消耗极低的和平经营类游戏中不适用。当游戏的核心乐趣不再依赖于"战前构筑的衍生与同盟协作的互相反哺"时，上述因果链便会断裂。
