# 三国志战略版·产品化结论

## 这一层的用途
这里的结论带适用前提与反例，用于产品化决策。本文件将《三国志战略版》的SLG与策略卡牌特征转化为可落地的设计数值与机制判断，明确在何种用户层、变现形态与局内时长下这些判断生效，并指出跨品类时的失效边界，避免设计思路的滥用。

## 结论列表

**结论 1：构筑前置律**

If 在自动战斗的SLG中，战斗力完全等价于可消耗的兵量且战斗过程不可操作，Then 玩家的策略重心会完全前置于战前配将与战法联动，战中焦虑感极低。

*适用前提*：长线SLG，单局战斗结果影响领土与资源归属，用户群体对操作反应无要求，更看重战前规划与战后复盘。
*反例游戏*：《王者荣耀》（MOBA），核心乐趣在于实时操作与反应，若战斗自动进行则完全丧失体验。
*太宽测试*：若替换为 MOBA 类型，结论不成立。

**结论 2：冗余转化抽卡律**

If 在卡牌SLG中，拆解重复武将卡是获取S/A级战法的唯一途径，Then 玩家的抽卡意愿会显著高于重复卡纯废弃的设计，付费深度大幅扩展。

*适用前提*：卡牌驱动型游戏，武将/角色池深度大，且构筑需要大量不同战法/技能支撑，抽卡是核心变现手段。
*反例游戏*：《原神》（开放世界RPG），命座系统虽消耗重复卡，但核心战法（技能）不依赖拆解角色获取，重复卡边际收益递减明显。
*太宽测试*：若替换为开放世界RPG类型，结论不成立。

**结论 3：分工强制组织律**

If 攻城伤害结算强制拆分为"消灭守军"与"拆除耐久"两个互斥阶段，且器械兵种对战斗兵种极弱，Then 同盟内会出现专门的攻城职能分工，个人战力无法替代同盟组织度。

*适用前提*：强同盟绑定的SLG，以城池攻防为核心赛季目标，需要通过机制强制玩家协作。
*反例游戏*：《万国觉醒》（SLG），统帅带队集结对城池造成伤害，虽有兵种差异但无强制拆城/战斗分工，单兵战力强仍可单刷部分城池。
*太宽测试*：若替换为无强制分工的SLG类型，结论不成立。

**结论 4：领土产能绑定律**

If 沙盘上占领的地块是木材铁矿石料粮食的唯一被动产出源且产能与地块等级/数量直接绑定，Then 寸土必争会成为玩家的自发行为，资源地争夺战会比纯排名争夺更激烈。

*适用前提*：战争损耗大、资源不可无限囤积的沙盘SLG，玩家生存与扩张高度依赖持续产出。
*反例游戏*：《部落冲突》（策略塔防），资源靠自身采集器与单局掠夺，领土面积不影响产能，无寸土必争的沙盘压力。
*太宽测试*：若替换为塔防类型，结论不成立。

**结论 5：适性收束律**

If 武将的兵种适性加成在S级与C级之间存在50%的属性鸿沟（120%对70%），Then 阵容搭配会被强制收束到S适性组合，散搭策略会被迅速淘汰。

*适用前提*：需要固定流派体系支撑卡牌深度的策略游戏，且武将存在明确的兵种归属。
*反例游戏*：《炉石传说》（卡牌对战），无固定兵种适性约束，卡组构筑依赖法力曲线与卡牌联动，散搭（宇宙流）亦可成立。
*太宽测试*：若替换为卡牌对战类型，结论不成立。

**结论 6：行军时间制造战术纵深**

If 沙盘行军时间精确到秒级且山脉/关卡等地形强制阻隔路线，Then 战术重心就会从配将延伸到卡位和压秒，空间和时间成为可操作的战术变量。

*适用前提*：大地图沙盘SLG，地图足够大且存在关卡/河流等战略要地，同盟间对抗频繁。
*反例游戏*：《梦幻模拟战》（SRPG），战棋地图小，无行军时间概念，战术重心在走位与技能范围，无时间纵深。
*太宽测试*：若替换为SRPG类型，结论不成立。

## 局限性
1. **重度动作/RPG品类**：本结论依赖战前构筑与自动战斗，对于强调实时操作与动作反馈的游戏完全不适用。
2. **休闲模拟经营品类**：本结论的生存压力与战损惩罚建立在战争消耗上，对于无强对抗、无战损的模拟经营游戏不适用。
3. **快节奏卡牌对战品类**：本结论的行军时间与攻城分工依赖长线沙盘，对于单局快速结算的卡牌对战不适用。
