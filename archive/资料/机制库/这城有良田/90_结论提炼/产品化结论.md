# 这城有良田·产品化结论

## 这一层的用途
这里的结论带适用前提与反例，用于产品化决策。本文件提炼了《这城有良田》在“模拟经营+SLG+卡牌”混合品类下的核心设计逻辑与数值推断，当你在做同品类或跨品类融合产品时，可直接对照前提条件判断是否可复用，避免生搬硬套导致的系统崩溃或用户体验断层。

## 结论列表

**结论 1：经济-晋升双轨绑定**

If 将核心经济产出（如铜钱）直接设为进度（如官职）的唯一衡量标准 Then 玩家在前期会极度聚焦于产能优化，使得建筑升级决策的优先级提升约 40%[推测]，避免纯数值堆叠导致的经营与养成玩法割裂。

*适用前提*：模拟经营向SLG过渡的早期阶段，且存在明确的职位/权限解锁节点，局内时长>15分钟[推测]/天。
*反例游戏*：《暗黑破坏神3》（ARPG）——其进度靠的是等级与装备掉落，金币只是附庸修复与重铸费用，若用单一货币卡进度会导致刷装驱动力被枯燥的攒钱行为取代。
*太宽测试*：若替换为 单机 RPG 类型，结论不成立（单机RPG强调多线程探索与战斗掉落，单一货币门槛会扼杀战斗与成长乐趣）。

**结论 2：阵容羁绊的路线互斥**

If 阵容系统采用“3+2”同势力羁绊且养成材料绑定势力 Then 玩家会在开服前 3-5天[推测]内确定单势力深耕或双势力混搭路线，使得抽卡目标感增强，付费转化率提升约 15%[推测]。

*适用前提*：卡牌养成深度绑定SLG战斗检定，且卡池存在明确的势力划分，单局战斗时长<5分钟[推测]，玩家需要极简的组队决策。
*反例游戏*：《英雄联盟》（MOBA）——无长线养成材料羁绊，单局独立，不需要在阵容搭配上做跨局的长线资源互斥投入，若加入长线势力羁绊会严重破坏竞技公平性。
*太宽测试*：若替换为 MOBA 类型，结论不成立（MOBA单局重置，无长线资源投入互斥，羁绊只能做局内临时天赋）。

**结论 3：赛季重置的局部清零**

If 赛季重置仅清零竞争结果（领地）而保留养成底盘（僚属数据） Then 可缓解老服死水效应，使新玩家入局留存率提升约 20%[推测]，同时保住付费沉没成本。

*适用前提*：SLG品类且具备赛季制框架，单赛季时长约 6周[推测]，玩家间存在强零和博弈与地缘压制。
*反例游戏*：《文明6》（4X策略）——单局完全重置，无长线账号养成，若保留部分养成会导致多局滚雪球彻底破坏4X策略的起跑线公平。
*太宽测试*：若替换为 卡牌对战 类型，结论不成立（卡牌对战如炉石，构筑卡组是核心，若保留卡组只重置天梯排名，无法解决老玩家对新手的环境压制）。

**结论 4：双轴成长解决付费与活跃焦虑**

If 长线付费点采用双轴设计（等级重置提供活跃度，星级保留提供付费价值） Then 既能保证每赛季约 30天[推测]的活跃追赶体验，又能维持大R玩家跨赛季约 80%[推测]的付费优越感。

*适用前提*：具备赛季重置机制的混合品类游戏，且存在高价值赛季Battle Pass或累充道具，单次付费门槛>100元[推测]。
*反例游戏*：《CS:GO》（FPS）——皮肤无数值属性，纯外观付费，不存在等级重置与数值保留的矛盾，若引入双轴数值会破坏竞技绝对公平。
*太宽测试*：若替换为 开放世界 类型，结论不成立（开放世界无赛季重置，数值只增不减，双轴重置毫无意义）。

**结论 5：前置休闲外壳对冲后置重度的流失**

If 用轻度模拟经营作为前期 1-7天[推测]的新手引导外壳，掩盖中后期重度SLG博弈 Then 可大幅降低SLG的获客成本，使首周留存率提升约 25%[推测]，但会在第 14天[推测]左右出现因玩法预期不符的断崖式流失。

*适用前提*：目标用户包含泛休闲女性或轻度模拟经营玩家，且核心变现依赖后期的GVG战争与抽卡，LTV回收周期>30天[推测]。
*反例游戏*：《守望先锋》（英雄射击）——纯竞技射击，无长线经营与数值博弈，若用休闲外壳掩盖核心射击玩法，会导致玩家操作预期完全错位。
*太宽测试*：若替换为 单机 RPG 类型，结论不成立（单机RPG无需掩盖核心玩法，前后体验一致是沉浸感基础，外壳欺骗只会导致退款）。

## 局限性
1. **纯竞技对称博弈产品**：本结论高度依赖非对称的数值碾压与长线养成，对于绝对公平的对称竞技游戏（如格斗、战术竞技）完全不适用。
2. **无赛季制单机体验**：结论3与4基于周期性重置框架，对于一次性买断、无长线运营更新的纯单机产品无法发挥重置与双轴的复用价值。
3. **UGC驱动生态产品**：若游戏核心寿命由玩家创作内容（如沙盒建造）驱动，则用SLG战争消耗资源的逻辑会破坏创作生态，本结论的变现与留存推演失效。
