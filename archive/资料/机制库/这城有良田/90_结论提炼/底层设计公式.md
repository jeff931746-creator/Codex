# 这城有良田·底层设计公式

## 这一层的用途
这里的结论脱离本游戏语境仍应成立。它们揭示了“模拟经营+卡牌养成+SLG”这一复合品类在底层资源流转、跨系统复用、赛季生态维持上的结构性规律。无论外皮是古风县城、星际殖民地还是末日避难所，只要遵循“前置经营供养后置战争+卡牌底盘串联+周期性重置竞争”的架构，这些命题即为系统成立的决定性杠杆。

## 结论列表

**结论 1：经济晋升双重锚定**

If 经济产出指标直接作为核心晋升门槛，Then 玩家在扩张期的资源分配决策必然产生双重收益预期（即经济积累与权限解锁同步达成）。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 2：养成底盘跨层复用**

If 卡牌养成同时提供经济产出加成与战斗检定数值，Then 玩家在非竞争期的养成沉没成本可平滑过渡至竞争期。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 3：赛季重置的切割守恒**

If 赛季重置仅清除地缘竞争结果而保留个人养成底盘，Then 长期生态中的先发优势壁垒会被周期性打破且不引发剧烈资产流失。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 4：长线道具双轨制**

If 长线付费道具采用“跨周期保留的稀有度轴”与“周期内重置的成长轴”双轨制，Then 付费玩家的优越感与每周期的活跃驱动力可同时被满足。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 5：阵容路线的资源锁闭**

If 阵容羁绊的深度收益与特定阵营的稀缺材料消耗强绑定，Then 玩家的阵容构建必然从全阵营均摊转向聚焦式路线声明。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 6：前置层对竞争层的门锁效应**

If 前置层的核心进度指标直接作为后置竞争层的解锁门锁，Then 玩家在竞争层开启前的行为必然被前置层的最优解所规训。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

## 边界
这套公式在不包含“跨系统复用养成线”（即经营与战斗完全割裂、资源不可互通）的纯单线模拟经营游戏中不适用；在无周期性重置机制、强调绝对单向滚服的纯数值比拼SLG中不适用；在无前置经济建造层、开局即进入零和博弈的纯竞技对抗游戏中亦不适用。
