# 《这城有良田》总档

## 基本信息

- 游戏名：这城有良田
- 类型标签：App手游 / 古风模拟经营 / 卡牌养成 / SLG
- 平台：iOS + Android
- 发行商：益世界网络科技（广东）有限公司（YGame）
- 开发商：开元工作室
- 上线日期：2023年7月20日
- 资料来源：TapTap游戏页、知乎攻略、小米游戏中心攻略、搞趣网分析文章、游研社

## 一句话定义

一款把"古风小城建设经营"作为前台叙事外壳、把"卡牌僚属养成+SLG九州征战"作为中后期留存核心的分层手游：玩家在前期用铜钱数值推进城市经营，中期把铜钱和元宝转化为僚属战力，后期用僚属+军队在九州地图进行周期性赛季竞争，赛季约6周重置一次领地状态。

## 核心结构

- **核心循环**：修复废旧建筑 → 开粥棚招募流民 → 将流民安置入民居转化为居民 → 建设西/南/北三市产业（农田/鱼塘/盐池/木坊等）→ 任命僚属到对应职位提升产出 → 铜钱税收积累 → 税收额达标升官职 → 解锁九州SLG战争层 → 在九州地图采集资源/占领城池/联盟协作 → 约6周赛季结束/九州领地重置 → 携带保留数据进入下一赛季

- **主要资源**：
  - 铜钱（主货币，建筑升级+官职晋升衡量标准）
  - 元宝（高级货币，用于僚属抽卡）
  - 粮草（军事层资源，训练军队+采集消耗）
  - 强化石/橙色强化石（僚属养成强化材料）
  - 星辰石（赛季Battle Pass产出，僚属培养）
  - 彩玉（高级赛季材料，累充1.2万元宝获取）
  - 玄石（赛季专属，从永业地采集，用于兵种进阶）
  - 宝具（赛季性道具，等级每赛季重置，星级保留）

- **主要系统**：城市建设执行层、政务/经济层、僚属养成层（卡牌底盘）、九州SLG战争层、课业精研层（科技树）、赛季制节奏框架

- **主要节奏节点**：每次建筑升级解锁新产能、官职晋升（税收额门槛）、僚属星级突破（同势力材料消耗）、九州城池攻略开放（赛季第3天）、赛季重置（约6周）

- **主要检定节点**：九州城池攻略（军队战力检定）、赛季末战力排名（联盟规模和个人军事实力双重检定）

## 本游戏最值得拆的专题

- 铜钱税收额为什么同时是货币和官职晋升衡量标准——答：把经济产出直接绑定为升级门槛，使建筑升级和职位分配决策具有双重收益，避免玩家只堆数值不注重布局
- 僚属"3+2 阵容"为什么是路线声明而不是纯力量堆叠——答：同势力3人激活羁绊加成使两条阵容路线（本势力深耕vs全势力覆盖）产生明确收益对立，材料消耗（同势力材料）也和路线方向绑定
- 赛季重置"仅重置领地不重置人"为什么能维持长期留存——答：玩家养成数据全部保留，赛季重置只删除竞争结果（领地占有权），这解决了SLG常见的"强者恒强导致新玩家无力入局"问题，同时又保住了玩家的养成沉没成本
- 宝具等级赛季重置而星级保留——答：等级重置提供赛季内成长感（每赛季重新升宝具给出新鲜体验），星级保留保住付费价值（付费获取高星宝具跨赛季有效），两个数值轴解决了不同玩家群体的不同焦虑

## 总判断

- **这游戏靠什么成立**：用"古风县令经营城市"的低门槛叙事把玩家带入游戏，让经营层提供安全感和掌控感，再用僚属卡牌和赛季SLG竞争逐层暴露付费深度——前期玩家感知的是"种田养城"，后期付费用户竞争的是"九州霸权"，两套体验在同一个游戏里服务不同留存阶段的玩家
- **最大机会点**：赛季重置机制（仅重置九州领地+宝具等级，保留全部角色数据）天然解决了SLG品类的强者恒强问题，每赛季提供相对公平的新竞争起点，使轻度玩家有动力持续参与而不是被碾压后流失
- **最大风险点**：经营层和SLG层的受众重叠度低——喜欢种田经营的玩家往往回避竞争性SLG，而SLG玩家对城市建设的耐心有限；游戏用分层解锁（后期才暴露SLG）缓解了这个问题，但也意味着核心付费用户（SLG玩家）需要在经营层长时间教学后才能进入真正的竞争核心
