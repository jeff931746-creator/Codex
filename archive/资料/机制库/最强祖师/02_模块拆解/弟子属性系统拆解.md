# 弟子属性系统拆解

这个专题要回答：弟子的哪些属性决定了培养上限，天命系统在什么条件下激活，以及为什么"天命>灵根>体质>六维"的筛选优先级是合理的而不是随意排序？

---

## 品质层级（五档）

弟子出生时品质随机确定，品质不可升级，只能通过遣散低品质弟子、保留高品质弟子来实现整体质量提升。

| 品质 | 颜色标识 | 天命系统 | 说明 |
|------|---------|---------|------|
| 平庸 | 绿 | 不激活 | 基础属性低，仅用于前期补充后勤岗位 |
| 出众 | 蓝 | 不激活 | 可作为中期战斗弟子使用 |
| 傲人 | 紫 | 不激活 | 中期主力选择 |
| 天资 | 橙 | **激活天命系统** | 后期核心培养对象 |
| 逆天 | 红 | **激活天命系统** | 最高品质，天命成长上限最高 |

**品质识别边界**：品质决定是否激活天命系统——橙色（天资）以上才能解锁天命阶段，这是品质筛选的硬门槛，不是软性优先级。不满足这个条件的弟子无论六维多高，后期天花板都低于激活天命的弟子。

---

## 灵根系统

灵根是五行属性的组合，影响功法学习效率。

**关键规则**：灵根数量越少，每条灵根的威力越强。

| 灵根数量 | 效果 |
|---------|------|
| 单灵根（如纯火） | 对应五行功法修炼效率最高 |
| 双灵根 | 中等效率 |
| 三灵根及以上 | 效率最低 |

**灵根的边界**：灵根只影响功法学习效率（修为获取速率的倍率系数），不影响战斗技能威力本身，也不影响后勤生产效率。后勤弟子不需要关注灵根。

**筛选场景**：在招募到天资/逆天弟子后，优先看其灵根数量，同等品质下单灵根弟子优于多灵根弟子。

---

## 天命系统

**激活条件**：弟子品质必须是天资（橙）或逆天（红）。

天命分为五个阶段（从低到高）：黄1 → 黄2 → 黄3 → 仙1 → 仙2 → 仙3

- 后期战力目标：天命≥仙3
- 天命系统是修炼上限的主要来源，不是短期战力的主要来源
- 天命阶段提升需要消耗专属材料，具体材料来源于大世界稀有探索和活动奖励

**天命的边界**：天命只提升该弟子自身的成长上限，不影响其他弟子，也不影响后勤建筑产出。

---

## 六维属性

六维：资质、根骨、机缘、聪慧、魅力、潜力

**功能分类**：

| 维度 | 影响 |
|------|------|
| 资质 | 修炼速度（吃丹药的修为获取效率）、技能伤害 |
| 根骨 | 修炼速度、技能伤害 |
| 潜力 | 修炼速度、技能伤害 |
| 机缘 | 具体效果文档不详，推测为掉落/运气类 |
| 聪慧 | 推测为学习功法速度相关 |
| 魅力 | 推测为招募/社交相关 |

**六维的边界**：六维是"放大器"而非"决定者"——天命、灵根是决定上限的变量，六维是在上限内的效率倍率。高六维的低品质弟子仍然无法达到天资/逆天弟子的天花板。这是"天命>灵根>体质>六维"排序成立的原因。

---

## 体质系统

- 每名弟子仅有一种体质（唯一性）
- 体质提供特定属性加成和特效（如某些体质可能提升特定技能触发率或生产效率）
- 体质优先级低于天命、灵根，但高于六维——因为体质带来的特效有时是特定玩法的关键

**体质的边界**：体质不可更改，是弟子出生时的固有属性。体质不影响弟子品质判断，也不影响灵根效率。

---

## 天赋系统

- 每名弟子最多拥有六种天赋
- 天赋分为战斗天赋和生产天赋两类
  - 战斗天赋：增强技能效果、提升战斗中的触发概率
  - 生产天赋：提升对应后勤岗位的产出效率（如"炼丹奇才"天赋使炼丹房产出+X%）
- 天赋数量上限为六种，多于六种时需要取舍

**天赋的边界**：天赋不影响弟子品质，不影响灵根，也不影响天命。天赋是后勤弟子分配优化的主要变量（把有对应生产天赋的弟子分配到对应建筑），而不是战斗弟子培养的优先考量。

---

## 巧匠特质（庶务弟子专属）

- 专属于后勤弟子，分为四个等级
- 等级越高，该弟子在后勤岗位上的产出效率越高
- 战斗弟子不具备也不需要巧匠特质

**识别标准**：如果一名弟子有巧匠特质，优先考虑其后勤用途；如果一名弟子有高品质天命+单灵根，优先培养为战斗弟子，不应分配后勤。二者互斥。

---

## 弟子筛选决策树

```
新弟子入门
    ├─ 品质是天资/逆天？
    │       ├─ 是 → 进入精细筛选（灵根>体质>六维）
    │       └─ 否 → 检查是否有高等级巧匠特质或高价值生产天赋
    │                   ├─ 是 → 分配后勤岗位
    │                   └─ 否 → 考虑遣散（返还心德）
    └─ 保留天资/逆天弟子后，优先检查灵根数量
            ├─ 单灵根 → 最优战斗弟子候选
            └─ 多灵根 → 次优，可作为填充或后勤备用
```
