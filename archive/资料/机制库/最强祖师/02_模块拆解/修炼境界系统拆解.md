# 修炼境界系统拆解

这个专题要回答：弟子修为从哪里来、丹药饱食度机制如何迫使玩家维持多条修为通道，以及传功系统为什么能大幅压缩境界推进时间？

---

## 境界阶梯（十级）

凡人 → 炼气 → 筑基 → 结丹 → 元婴 → 化神 → 炼虚 → 合体 → 大乘 → 渡劫

**关键规则**：每个境界达到圆满后，弟子**自动**触发渡劫，突破到下一境界。渡劫无需玩家手动干预，属于纯被动推进。

**边界**：境界系统只负责把弟子属性转化为当前战力，不改变弟子属性本身（属性固定在出生时，见弟子属性系统拆解）。

---

## 修为获取三大通道

### 通道一：自主修炼（被动，永久开启）

- 弟子拜入宗门后立即开始自动挂机修炼，每秒获得固定修为值
- 修为获取速率受以下因素叠加影响（以下数值为明文加成，非估算）：

| 加成来源 | 加成幅度 |
|---------|---------|
| 建造洞府（优于普通房舍） | 具体加成取决于洞府等级 |
| 芜青堂（宗门建筑） | 全宗门弟子修炼效率+30% |
| 传道圣简（造化至宝） | 修为效率加成（等级依赖） |
| 掌门任职加成 | +15% |
| 长老任职加成 | +10% |
| 内门弟子任职加成 | +5% |

**复合叠加规则**：各项加成是否乘算或加算未明文说明，按普遍放置游戏惯例推测为加算叠加后再乘以基础修炼速率。

**识别边界**：自主修炼通道不受饱食度限制，是唯一无饱食度压制的修为来源，也是中后期的主要修为来源。

### 通道二：消耗丹药/灵酒（主动，受饱食度限制）

- 修为丹由炼丹房生产（需分配执事弟子）
- 灵酒和灵丹可从大世界冒险中获取
- **饱食度机制**：弟子喂食丹药会累积饱食度，饱食度满后修为获取效率显著下降（具体阈值未明文，社区攻略描述为"满了就停"）
- 弟子资质直接影响"吃丹药的修为获取效率"——高资质弟子同等丹药量获取更多修为

**饱食度的设计意图**：限制玩家无限堆丹药刷修为，迫使维持通道一（被动挂机）的持续运作，不能只依赖通道二快速推进。

### 通道三：研读心得（主动，无饱食度）

- 心得来源：
  - 遣散高阶弟子时返还修炼心德（弟子境界越高，返还越多）
  - 悟道堂建筑缓慢产出心德
- 弟子研读心得获取修为，**不累积饱食度**
- 心得供给有限，是补充性通道而非主力通道

**三通道联动**：心得通道的设计使"遣散培养到一定境界的低品质弟子"产生额外价值，而不是纯粹的资源损失——这形成了弟子筛选的动态平衡（见弟子属性系统拆解）。

## 传功系统（境界时间压缩机制）

**触发位置**：【传功阁】建筑

**运作规则**：
1. 选择一名弟子作为传功者（捐出者）
2. 选择另一名弟子作为受功者（接受者）
3. 传功完成后，受功弟子直接获得传功弟子的**修为境界**和**炼体等级**

**传功的战略意义**：
- 场景一：有一名低品质但境界很高的弟子 A，和一名高品质但境界很低的弟子 B → 把 A 传给 B，B 立刻获得高境界，省去数天/数周的慢速修炼时间
- 场景二：活动限时产出大量培养资源，集中培养一名"传功专用"弟子到高境界，再传给主力弟子

**传功系统的边界**：传功只转移境界和炼体，不转移弟子品质、灵根、天命、体质、天赋。传出者传完后境界归零（相当于牺牲）。传功阁本身需要宗门建设解锁（具体等级未明文说明）。

---

## 淬体系统（炼体境界）

- 从二转金身升至六转金身，是后期战力的主要来源之一
- 后期战力目标：淬体≥四转金身
- 淬体材料通过大世界探索、活动产出获取
- 淬体进度可以通过传功系统一并转移给受功弟子

---

## 境界推进的关键瓶颈

| 瓶颈 | 触发条件 | 解决路径 |
|------|---------|---------|
| 修为不足 | 弟子境界停滞 | 投喂丹药+心德+检查修炼效率建筑 |
| 丹药饱食度满 | 修为丹投喂效率下降 | 停止投喂，依赖被动修炼，等饱食度自然下降 |
| 高品质弟子境界追不上战斗需求 | 新招募的逆天弟子等级太低 | 使用传功系统把高境界低品质弟子的境界转移过来 |
| 炼丹房产能不足 | 修为丹储量用尽 | 增加炼丹弟子数量或升级炼丹房 |

## 这个模块不负责什么

- 不负责弟子品质和属性的确定（见 `弟子属性系统拆解.md`）
- 不负责修为丹的生产和材料来源（见 `后勤生产系统拆解.md`）
- 不负责传功阁等建筑的解锁条件（见 `宗门建设系统拆解.md`）
