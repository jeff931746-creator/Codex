# 《最强祖师》总档

## 基本信息

- 游戏名：最强祖师
- 类型标签：修仙主题 / 宗门模拟经营 / 弟子养成放置
- 平台：iOS / Android
- 上线日期：2024年2月1日
- 发行：夜猫子发行工作室
- 资料来源：TapTap 游戏介绍、TapTap 论坛攻略、MuMu 模拟器官方攻略、百度百科（2024-2025年）

## 一句话定义

一款以"宗门祖师"视角经营修仙门派的放置模拟游戏：玩家负责招募弟子、分配后勤岗位、建设宗门建筑，弟子自动挂机修炼、渡劫飞升，强化循环核心是弟子品质筛选和修为获取效率最大化。

## 核心结构

- 核心循环：招募弟子（随机属性）→ 筛选保留高品质弟子 → 分配战斗/后勤岗位 → 消耗令牌大世界探索 → 弟子自动修炼渡劫升境界 → 消耗材料升级宗门建筑 → 宗门等级提升解锁新上限 → 招募并筛选更高天命弟子
- 主要资源：令牌（体力）、灵石、灵草、灵木、矿石、玄铁、符纸、阵石、灵玉（高级货币）、修为丹、心德
- 主要系统：弟子属性与筛选系统、修炼境界系统、宗门建设系统、后勤生产系统、战斗阵容系统、大世界探索系统、社交/仙盟系统、付费/商店系统
- 主要节奏节点：每日令牌上限、丹药饱食度周期、弟子渡劫触发点、宗门等级门槛、七日任务周期
- 主要检定节点：大世界猎妖关卡、九层妖塔、章节推进战斗

## 系统总览

每个系统：负责什么 / 不负责什么 / 和谁联动最紧密。

| 系统 | 层级 | 职责边界（是什么 + 不是什么） | 联动最紧密 |
|------|------|-------------------------------|------------|
| 弟子属性与筛选系统 | 核心资产层 | 决定每个弟子的战斗上限和生产效率上限；不负责弟子的实际修炼进度，只负责确定天花板 | 修炼境界系统、战斗阵容系统 |
| 修炼境界系统 | 成长层 | 决定弟子当前修为进度和境界；不负责弟子属性，只负责把属性转化为实际战力 | 弟子属性系统、宗门建设系统 |
| 宗门建设系统 | 底盘放大层 | 决定弟子修炼效率上限和后勤产出上限；不负责弟子属性和修炼自动进行，只负责放大效率天花板 | 后勤生产系统、修炼境界系统 |
| 后勤生产系统 | 资源供给层 | 决定八大材料的产出速率；不负责战斗结果和弟子修炼，只负责保证建设和养成的材料供给 | 宗门建设系统、经济循环 |
| 战斗阵容系统 | 执行层 | 决定大世界和关卡的输出结果；不负责弟子修炼速度，只负责把弟子当前战力转化为闯关能力 | 弟子属性系统、修炼境界系统 |
| 大世界探索系统 | 内容消耗层 | 决定令牌如何转化为战斗次数和材料产出；不负责挂机产出，只负责主动消耗令牌换取稀有材料 | 战斗阵容系统、后勤生产系统 |
| 社交/仙盟系统 | 辅助资源层 | 决定灵玉等货币的非付费获取渠道；不负责弟子培养的核心流程，只提供边际增益资源 | 经济循环 |
| 付费/商店系统 | 变现层 | 决定付费加速的具体手段和性价比梯度；不负责免费核心流程，只提供加速通道和品质天花板提前触达 | 弟子属性系统、宗门建设系统 |

## 本游戏最值得拆的专题

- 弟子品质筛选为什么是核心决策点，而不是修炼效率叠加？
- 传功系统为什么能压缩境界养成时间，它改变了哪条关键路径？
- 丹药饱食度机制如何迫使玩家维持多条修为获取通道，而不是只堆单一渠道？
- 付费体系（尤其是30元芜青堂）如何在修炼效率这条核心路径上直接放大回报而不是绕过它？

## 总判断

- 这游戏靠什么成立：弟子属性的随机生成+高品质弟子的稀缺性，让"招募到好弟子"成为一种持续的低频惊喜，掩盖了挂机养成本身的被动属性
- 最大机会点：所有系统（修炼、后勤、建设、付费）最终都在为"提升弟子修为获取效率"服务，形成清晰的目标漏斗
- 最大风险点：弟子随机性导致中期进度完全依赖运气，没有足够的弟子筛选/替换机制时，玩家会陷入无法推进的停滞感
