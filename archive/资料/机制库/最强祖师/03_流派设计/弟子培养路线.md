# 弟子培养路线

这个专题要回答：最强祖师是否存在真正的"路线"选择，还是只有一条数值最优路径？以及"战斗弟子路线"和"后勤弟子路线"在资源分配层面有何结构性分歧？

---

## 路线识别问题

**先回答核心问题**：最强祖师不存在像 Roguelike 构筑游戏那样的"路线声明"机制（选了一条路就关闭另一条路）。游戏的主要养成方向是线性的：高品质弟子 → 高境界 → 高战力。

但在以下两个维度存在"配置选择"，构成有限的路线差异：

---

## 维度一：弟子数量的战斗/后勤比例

**选择场景**：宗门容量有限时，新弟子应该充实战斗还是后勤？

| 路线配置 | 优势 | 劣势 |
|---------|------|------|
| 偏战斗（多弟子修炼战斗） | 战力提升快，关卡推进快 | 后勤弟子少，材料产出慢，建设卡资源 |
| 偏后勤（多弟子生产） | 建设材料充足，宗门升级快 | 战力成长慢，关卡推进受阻 |

**识别标准**：这是否构成真正路线？
- 是，当：宗门容量刚够，一个弟子的增加战斗/后勤会显著改变短期推进速度
- 否，当：宗门容量足够，可以同时覆盖足够的战斗弟子和后勤弟子

**结论**：前期容量有限时存在短期路线张力，中后期随宗门扩容逐渐消解。

---

## 维度二：功法五行路线

弟子灵根与功法五行对应，不同灵根弟子选择对应五行功法时效率最高：

| 灵根属性 | 对应功法 | 适合位置 |
|---------|---------|---------|
| 火 | 火系功法 | 通常对应输出类技能 |
| 水 | 水系功法 | 辅助/控制类 |
| 金 | 金系功法 | 攻击/破防类 |
| 土 | 土系功法 | 防御/坦克类 |
| 木 | 木系功法 | 治愈/辅助类 |

**路线约束**：拥有单火灵根弟子学习火系功法效率最高，如果主力选择单火灵根弟子，则阵容将有较强的输出倾向，但辅助和坦克弟子需要用不同灵根的弟子补充。

**识别标准**：功法五行路线是否是真正的路线选择？
- 是，当：随机生成的弟子灵根分布决定了阵容的五行构成，玩家需要根据已有弟子灵根来配置阵容，而不是自由选择
- 否，当：玩家可以自由选择任意灵根弟子进入阵容（实际上弟子属性随机生成，玩家不能自由指定灵根）

**结论**：功法五行路线是被动适应型选择，而非主动构建型选择。玩家的主要决策是"用已有弟子的灵根来最优化阵容"，不是"预先规划功法路线再招募对应弟子"（因为弟子品质随机，品质优先级高于灵根，在逆天品质弟子中再挑灵根）。

## 弟子发展生命周期

每名弟子的完整生命周期：

```
招募入门 → 评估品质
    ├─ 天资/逆天 → 培养战斗弟子路线
    │       ├─ 自主修炼（洞府+芜青堂）
    │       ├─ 投喂丹药（炼丹房产出）
    │       ├─ 渡劫升境界（自动触发）
    │       ├─ 淬体进阶（材料消耗）
    │       └─ 天命升阶（高阶稀有材料）
    ├─ 傲人/出众 + 有生产天赋 → 后勤弟子路线
    │       └─ 分配对应生产建筑岗位，长期驻留
    └─ 品质低且无生产天赋 → 遣散（返还心德）
```

**遣散高境界战斗弟子的时机**：
- 当有更高品质（天资→逆天）的新弟子入门时
- 遣散返还的心德可以立即用于新弟子修为提升
- 传功阁中：先把老弟子境界传给新弟子，再遣散老弟子

## 这个模块不负责什么

- 不负责每个功法的具体技能效果（运营数据，非机制架构层）
- 不负责特定弟子（元润、蝶衣等命名角色）的属性分析（角色性能随版本变化）
- 不负责多服务器或竞技场的对抗性路线选择（缺乏资料）
