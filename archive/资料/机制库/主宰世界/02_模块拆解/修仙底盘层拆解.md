# 主宰世界·修仙底盘层拆解

## 这个专题要回答什么

元婴、真灵和异兽为什么是底盘，而不是路线？

## 总体判断

修仙底盘系统负责把单局里看得到的强度变成可持续积累的底盘。它不直接告诉玩家该走哪条流派，但它决定了这条流派能不能稳定兑现，以及兑现速度有多快。元婴前万剑弱于御天，正是底盘没有到位导致路线无法兑现的典型表现。

## 成长底盘由哪些部分组成

### 1. 元婴是改变职业兑现效率的质变点

- 触发条件：角色达到元婴突破条件并完成突破
- 具体结果：职业强度发生反转，如万剑在元婴前秒伤与御天差距大，元婴后则可能反超
- 为什么有效：用阶段性质变把“弱势期”变成“等待期”，给玩家长期追求

### 2. 真灵提供定向属性加成

- 触发条件：激活和培养真灵
- 具体结果：角色获得暴击、攻击力等定向属性加成，补足职业或神通路线的短板
- 设计原因：让底盘不只是数值放大，也能做微调修正

### 3. 异兽系统提供额外战斗力承接

- 触发条件：获得并培养异兽
- 具体结果：异兽在战斗中提供协同攻击伤害或控制效果
- 为什么有效：把养成维度从角色自身扩展到宠物，增加底盘厚度

### 4. 底盘数值决定神通路线是否成立

- 触发条件：神通搭配完美但底盘数值不足
- 具体结果：秒伤依然无法达到检定门槛
- 设计原因：底盘是路线的兑现器，没有底盘的路线只是空壳

## 这个层级不负责什么

- 不负责单局里具体怎么放神通，那是职业执行层的职责
- 不负责哪个神通跟哪个神通搭配，那是神通构筑层的职责
- 不负责奖励从哪里来，那是资源循环和商店奖励的职责

## 可迁移结论

- If 一个养成系统（如元婴）能改变职业的乘区结算逻辑并导致强度反转（万剑元婴后反超御天），Then 这个系统就不只是数值放大器，而是路线兑现的开关
- If 底盘数值（真灵暴击率、异兽协同伤害）不足会导致完美神通搭配无法通过秒伤检定，Then 底盘就是路线的承重墙，没有底盘的路线只是空壳
