# 主宰世界·职业执行层拆解

## 这个专题要回答什么

职业在《主宰世界》里为什么是执行层，而不是构筑层？

## 总体判断

职业执行层负责把基础攻击方式和战斗节奏打出来。它决定“这个角色动作是什么样的”，但不决定“这局到底走爆发还是持续”，路线声明来自神通构筑层。

## 职业执行层到底在做什么

### 1. 职业上场后立即定调战斗动作

- 触发条件：玩家创建角色或切换职业（如万剑、凌霄、御天）
- 具体结果：角色使用对应武器和基础技能进行攻击，万剑是高爆发灵活动作，御天是稳定输出动作，凌霄是控制动作
- 为什么有效：执行层必须是低解释成本的，玩家选了职业就知道自己要怎么打

### 2. 职业天赋决定神通的可承接范围

- 触发条件：选择神通时
- 具体结果：万剑职业能最大化伏明剑典的收益，御天职业则更适合生存和稳定输出型神通
- 识别边界：只要这个职业的核心价值是“让爆发型或生存型神通更强”，它就是执行层容器；如果它自己就能声明路线，它就越界了

### 3. 职业强度随底盘阶段发生偏移

- 触发条件：达到元婴等关键底盘突破节点
- 具体结果：万剑在元婴前秒伤与御天差距大，元婴后则可能反转
- 为什么有效：让职业不是一成不变的强度，而是和成长底盘联动的动态容器

## 这个层级的边界在哪里

- 它不负责声明“这局要玩什么流派”，那是神通构筑层的职责
- 它不负责长期解锁新上限，那是修仙底盘层的职责
- 它不负责产出资源和礼包材料，那是资源循环与商店奖励的职责

## 典型识别标准

如果一个元素满足下面任意两条，就更接近执行层：

- 选定后立刻改变角色的基础攻击动作
- 决定了哪些神通可以被高效承接
- 其强度主要体现为“当前阶段好不好打”

如果它主要满足下面这些条件，就不是执行层：

- 需要靠局外资源长期养成
- 本身不直接参与战斗结算
- 更像路线选择或奖励修正

## 可迁移结论

- If 职业只负责基础攻击动作（如万剑的高频连击）和神通承接，而不负责路线声明，Then 新手就能靠“选个顺手的动作”上手，构筑深度交给神通系统
- If 职业强度受底盘突破节点（如元婴）影响发生反转（万剑元婴前弱势，元婴后爆发），Then 玩家对前期弱势职业的容忍度会提高，因为期待后期的质变
