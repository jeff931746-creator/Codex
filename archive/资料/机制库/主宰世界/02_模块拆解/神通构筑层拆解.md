# 主宰世界·神通构筑层拆解

## 这个专题要回答什么

《主宰世界》的神通为什么不是“技能收集”，而是路线声明？

## 总体判断

神通系统不是单纯的技能池，而是把“凑技能”变成“确定本阶段打法”的关键开关。玩家一旦决定带伏明剑典和阳极爆，就等于在确认本局的爆发路线；而下掉万钧定化导致秒伤从上限跌至88亿/秒下限，证明神通不是微调，而是路线本身。

## 神通构筑到底怎么工作

### 1. 主动神通定打法，被动神通放乘区

- 触发条件：玩家在神通界面装备主动神通（如混元决、剑心神通、牵星剑）和被动神通（如阳极爆、罡阳授力、分光化影）
- 具体结果：主动神通决定输出手段，被动神通决定伤害放大方式
- 为什么有效：把“怎么打”和“怎么放大”拆开，让构筑有明确的组合逻辑

### 2. 核心神通直接声明路线方向

- 触发条件：装备伏明剑典等核心神通
- 具体结果：后续神通选择必须围绕伏明剑典的特效和触发条件展开
- 识别边界：如果一个神通的出现改变了后续所有神通的选择优先级，它就是核心路线件；如果只是数值微调，它就是填充件

### 3. 替换单个神通会导致秒伤断崖式下跌

- 触发条件：下掉万钧定化这样的关键被动神通
- 具体结果：整体伤害刷新下限，平均秒伤跌至88亿/秒，与前面搭配的伤害表现形成巨大落差
- 为什么有效：用极端的数值落差证明这个位置不是可替的填充，而是路线的承重墙

### 4. 神通搭配会排斥其他路线

- 触发条件：选择了阳极爆、罡阳授力等爆发型被动
- 具体结果：持续输出型或控制型神通的收益大幅下降，玩家不再考虑兼顾生存与输出的均衡搭配
- 为什么有效：真正的路线声明必须让其他选择变差，否则只是收集

## 这个层级不负责什么

- 不负责职业的基础攻击动作，那是职业执行层的职责
- 不负责元婴、真灵等底盘数值的提供，那是修仙底盘层的职责
- 不负责给你稳定掉落神通升级材料，那是资源循环的职责

## 可迁移结论

- If 替换一个占据独立乘区的被动神通（如下掉万钧定化）会导致秒伤从上限跌至下限（88亿/秒），Then 这个技能位就是构筑放大件而非可替填充件，构筑系统已经从收集变成路线声明
- If 主动神通（如混元决）定打法、被动神通（如阳极爆）放乘区，Then 玩家的构筑逻辑会从“全都要”变成“先定核心再补配套”
