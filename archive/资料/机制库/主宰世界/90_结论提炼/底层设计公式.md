# 主宰世界·底层设计公式

## 这一层的用途
这里的结论脱离本游戏语境仍应成立。它们揭示了“修仙RPG+神通构筑”品类在底层结构上的因果关系——当游戏试图将复杂策略拆解为神通路线声明、并让局外底盘与局内构筑深度绑定时，必须遵守的机制传导逻辑。这些命题不依赖具体的修仙题材或职业名称，而是该设计范式成立的基石。

## 结论列表

**结论 1：构筑承重墙律**

If 构筑系统中，替换单个构筑件会导致输出效能断崖式下跌（因占据独立乘区），Then 该件就是构筑的承重墙而非可替填充件，系统已从收集跃迁为路线声明。

*反面测试*：把 If 取反后（替换单个构筑件不会导致输出效能断崖式下跌），Then 是否仍成立？不成立，若替换无断崖反馈则仅为可替填充件，系统仍是收集。

**结论 2：底盘反转律**

If 底盘突破系统能改变职业/流派的底层结算逻辑导致强度反转，Then 该系统就不只是数值放大器，而是路线兑现的开关，玩家对前期弱势的容忍度会因期待质变而提升。

*反面测试*：把 If 取反后（底盘突破系统只做纯数值放大不改变结算逻辑导致强度反转），Then 是否仍成立？不成立，若仅放大数值则弱势依旧弱势，不会成为路线兑现开关。

**结论 3：秒伤显性律**

If 检定系统将输出能力转化为精确的秒伤等数值指标，Then 玩家的构筑调优行为会从“感觉验证”转向“数值验证”，优化频率和深度显著增加。

*反面测试*：把 If 取反后（检定系统不提供精确秒伤指标，只提供通关与否或模糊评价），Then 是否仍成立？不成立，缺乏精确指标则无法定位构筑件优劣，调优仍依赖感觉。

**结论 4：主被动分工律**

If 主动技能定义输出手段、被动技能提供独立乘区放大，Then 玩家的构筑逻辑会从“全都要”变成“先定核心再补配套”，构筑的路线感会强于纯技能池收集。

*反面测试*：把 If 取反后（主被动技能不分工，均提供同质化输出手段与数值），Then 是否仍成立？不成立，同质化技能无分工则无法形成核心与配套关系，构筑仍是全都要的收集。

**结论 5：缺口补齐律**

If 资源循环中，高稀缺质变突破材料与轻量级付费/广告节点绑定以补足单次缺口，Then 玩家在卡门槛时的付费意愿会高于纯长线等待，且不会显著破坏免费玩家的进度感。

*反面测试*：把 If 取反后（高稀缺质变突破材料无轻量级付费/广告补缺渠道，只能长线刷取），Then 是否仍成立？不成立，缺乏轻量补缺渠道则卡门槛时玩家只能长线等待或重氪，微决策付费意愿不会提升。

## 边界
这套公式在缺乏局外长线养成或局内选项不随进程动态变化的纯动作RPG中不适用；在重操作反应轻策略构筑的动作类游戏中也不适用。当游戏的核心乐趣不再依赖于“局内构筑路线的衍生与局外底盘的沉淀互相反哺”时，上述因果链便会断裂。
