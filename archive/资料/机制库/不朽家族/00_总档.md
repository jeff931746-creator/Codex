# 《不朽家族》总档

## 基本信息

- 游戏名：不朽家族
- 类型标签：手游 / 放置合成 / 卡牌RPG / 家族经营
- 平台：iOS / Android
- 日期：2026-04-28
- 资料来源：TapTap 游戏介绍页、TapTap 新手攻略帖、TapTap 养成篇攻略帖、知乎攻略

## 一句话定义

一款把"放置资源积累"和"成员合成+种族阵容构筑"绑定在一起的手游：玩家以家族祖宗身份管理凯斯世界里龙裔、矮人、精灵等六族成员，核心决策是把挂机时间兑换成突破石，再通过阵容路线（土系荆棘/水系寒冷）和图书馆传承逐步拓宽战力天花板。

## 核心结构

- 核心循环：庄园挂机产出资源 → 召集石/突破石积累 → 成员突破等级阈值（40/80/120/140/160/200级）→ 推主线图 → 无尽塔/深渊检定 → 商店购买补充材料 → 图书馆研习传承 → 再次挂机
- 主要资源：突破石（魔力精粹）、成员召集石、凯斯火种、传承结晶、钻石、庄园收益
- 主要系统：战斗执行层（不朽者+小兵）、阵容路线层、成员成长底盘、图书馆传承、庄园挂机经济、关卡节奏与推图、无尽塔检定、不朽召唤商店
- 主要节奏节点：每日庄园收益结算、突破节点（40/80级等）、图章/Blood Pass周期、版本活动（机械国韵版本）
- 主要检定节点：无尽塔、深渊/极限挑战、不朽回廊（新增）

## 系统总览

| 系统 | 层级 | 职责边界（是什么 + 不是什么） | 联动最紧密 |
| --- | --- | --- | --- |
| 战斗执行层（不朽者+小兵） | 执行层 | 决定单局里哪些角色自动战斗、小兵何时合成升阶；不负责路线声明，不负责资源流转 | 阵容路线层、成员成长底盘 |
| 阵容路线层（土系/水系流派） | 路线层 | 决定这局要走荆棘叠层还是寒冷控制；不负责底盘数值，只负责把技能和种族搭配成协同结构 | 战斗执行层、无尽塔检定 |
| 成员成长底盘 | 底盘成长层 | 决定成员等级突破、血脉觉醒、星级碎片积累；不负责阵容选择，只负责让某条路线的数值达到推图门槛 | 庄园挂机经济、图书馆传承 |
| 图书馆传承系统 | 底盘成长层 | 决定传承秘法研习顺序（先射手技能→守护者→牧师）；不负责资源产出源头，只负责放大已有数值效率 | 成员成长底盘、庄园挂机经济 |
| 庄园挂机经济 | 经济层 | 决定离线时间换算成多少突破石与资源；不负责战斗结果，只负责把时间投入转成养成进度 | 成员成长底盘、商店置换 |
| 关卡节奏与推图 | 节奏框架层 | 决定章节推进速度和战力门槛节奏；不负责资源种类，只负责规定什么阶段需要多少战力 | 无尽塔检定、成员成长底盘 |
| 不朽召唤商店 | 编辑层/奖励层 | 决定钻石换哪系不朽者、小礼包买什么；不负责战斗执行，只负责补充特定路线缺失的核心角色 | 庄园挂机经济、阵容路线层 |
| 无尽塔与不朽回廊 | 检定层 | 决定当前阵容能否通过持续压力测试；不负责推进主线，只负责验证荆棘/寒冷路线有没有短板 | 阵容路线层、关卡节奏 |

## 本游戏最值得拆的专题

- 土系荆棘流为什么能同时作为推塔主力又作为防御构筑，其他放置游戏的阵容路线为什么做不到这一点？
- 图书馆传承天赋为什么跳过前期露娜而优先守护者，这个顺序约束如何决定了整个成长节奏？
- 突破石（魔力精粹）作为核心瓶颈资源，无尽塔和日常任务两条供给渠道如何塑造了玩家行为？
- 不朽召唤四系轮换池如何让"仅水火两系3技能卡值得抽"成立，这套筛选逻辑如何影响付费决策？

## 总判断

- 这游戏靠什么成立：用"挂机时间积累突破石"把推图解锁和成员养成绑定成同一条进度线，玩家每个突破节点都能感受到战力台阶，配合土系/水系双路线给构筑决策提供分叉感
- 最大机会点：六族种族标签和职业专属宝石共享同一套阵容语言，所有成员最终都在回答"哪条流派路线的协同更完整"
- 最大风险点：如果无尽塔不能持续在塔楼高层卡住偏科构筑，整个检定层就失去意义，游戏退化为单纯的数值堆叠放置
