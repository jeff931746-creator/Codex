# 口袋奇兵·产品化结论

## 这一层的用途
这里的结论带适用前提与反例，用于产品化决策。本文件将《口袋奇兵》的合成策略与轻量化SLG特征转化为可落地的设计数值与机制判断，明确在何种用户层、变现形态与局内时长下这些判断生效，并指出跨品类时的失效边界，避免设计思路的滥用。

## 结论列表

**结论 1：合成即产能律**

If 基地建筑升级通过即时合成完成且直接提升资源产出上限，Then 玩家对基建的投入意愿会提升 30%[推测]，因为合成消除了等待疲劳，把基建从被动等待变成主动加速。

*适用前提*：轻量化SLG，单局操作时长 3-5分钟[推测]的碎片化场景，且基建产出是核心循环的发动机。
*反例游戏*：《万国觉醒》（重度SLG），其基建升级依赖长时间等待和加速道具，合成即时完成会破坏重度SLG的时间付费模型。
*太宽测试*：若替换为 重度SLG 类型，结论不成立。

**结论 2：英雄绑定兵种驱动抽卡**

If 英雄技能加成高度绑定特定兵种（如马克西莫绑定陆军），Then 玩家在招募到T0英雄后的 24小时[推测]内，会将对应该兵种的资源投入提升 50%[推测]，抽卡直接转化为路线声明。

*适用前提*：拥有多兵种分支且商业化深度绑定英雄招募的SLG游戏，玩家需在 3条[推测]以上路线中抉择。
*反例游戏*：《皇室战争》（即时对战），卡牌无局外长线兵种养成，英雄/卡牌强度不驱动长线基建资源分配。
*太宽测试*：若替换为 即时对战 类型，结论不成立。

**结论 3：克制检定驱动多线养成**

If PVP检定强制引入三军克制循环（陆克空、空克海、海克陆），Then 单一兵种极限养成的玩家在跨服争霸中的败率会超过 70%[推测]，检定直接转化为多线养成动力。

*适用前提*：拥有PVP争霸玩法且强调阵容深度的SLG游戏，玩家已形成 1个[推测]主力兵种流派。
*反例游戏*：《崩坏：星穹铁道》（回合制RPG），其弱点机制仅在单局内生效，不驱动局外长线多阵营养成，只需切换角色即可应对。
*太宽测试*：若替换为 回合制RPG 类型，结论不成立。

**结论 4：合成路线锁定律**

If 合成操作直接决定单位等级和战力跃迁且消耗不可逆，Then 玩家在合成高阶单位前的思考时长会增加 3-5秒[推测]，构筑决策权重显著提升。

*适用前提*：核心用户追求策略深度，且合成素材存在稀缺性的合成策略游戏。
*反例游戏*：《2048》（休闲益智），合成结果无长线构筑意义，只追求单局分数，不涉及路线锁定。
*太宽测试*：若替换为 休闲益智 类型，结论不成立。

**结论 5：社交援助修正资源曲线**

If 联盟援助系统能稳定提供 10%-15%[推测]的日常资源缺口补充，Then 0氪玩家的留存率会提升 20%[推测]，社交从装饰变成经济循环的补丁。

*适用前提*：需要维持大R与0氪共存生态，且联盟活跃度占单日资源产出 15%[推测]以上的游戏。
*反例游戏*：《单机SLG》，无社交援助，完全靠玩家单机发育，不存在0氪大R生态区分。
*太宽测试*：若替换为 单机SLG 类型，结论不成立。

## 局限性
1. **重度MOBA品类**：本结论依赖长线养成与路线锁定，对于单局结算、无局外累积的MOBA不适用。
2. **纯休闲合成品类**：本结论的付费与检定逻辑建立在PVP偏科检定与多线养成上，对于无对抗、纯分数追求的休闲合成游戏不适用。
3. **强叙事驱动品类**：合成即时反馈与PVP争霸会打断叙事沉浸，对于以剧情演出和情感体验为核心的产品形态不适用。
