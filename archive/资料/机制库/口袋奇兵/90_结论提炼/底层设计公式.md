# 口袋奇兵·底层设计公式

## 这一层的用途
这里的结论脱离本游戏语境仍应成立。它们揭示了"合成+轻量化SLG"品类在底层结构上的因果关系——当游戏试图将重度养成压缩为即时反馈、并让路线声明与兵种克制深度绑定时，必须遵守的机制传导逻辑。这些命题不依赖具体的军事题材或小游戏平台，而是该设计范式成立的基石。

## 结论列表

**结论 1：合成跃迁律**

If 合成操作直接决定单位等级和战力跃迁且产出不可逆，Then 合成系统就会跃迁为路线构筑层而非单纯的整理减负层。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 2：克制检定律**

If 检定节点专门针对三军克制循环设置高压门槛，Then 玩家受挫后的行为会转向补齐阵容短板而非纯堆单一数值。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 3：产能耦合律**

If 建筑合成直接改写资源产出公式，Then 基建系统就会从被动等待的计时器变成经济循环的主动发动机。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 4：英雄绑定律**

If 英雄技能加成高度绑定特定兵种，Then 英雄招募就会从战力叠加演变为路线声明器而非单纯的收集要素。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 5：社交补丁律**

If 联盟援助能稳定填补日常资源缺口，Then 社交系统就会演变为经济循环的修正器而非单纯的聊天室。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

**结论 6：双心跳律**

If PVE提供稳定产出而PVP提供爆发掠夺，Then 节奏层就能成功把发育和对抗拆成两个独立且互补的心跳周期。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立

## 边界
这套公式在缺乏多兵种克制循环或合成不产生战力跃迁的纯策略游戏中不适用；在重操作反应轻策略构筑的动作类游戏中也不适用。当游戏的核心乐趣不再依赖于"即时合成的反馈与长线养成的路线声明互相反哺"时，上述因果链便会断裂。
