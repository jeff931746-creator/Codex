# 口袋奇兵·英雄与阵容路线层拆解

## 这个专题要回答什么

《口袋奇兵》里的英雄为什么会从"谁强上谁"变成"先决定兵种路线，再配对应英雄"？

## 总体判断

阵容形成不是靠一个终局大招，而是靠英雄梯队、兵种克制、科技树和合成方向共同把玩家推向某种稳定打法。换句话说，这游戏的流派是"被英雄的兵种绑定逼出来的"，不是玩家凭空想出来的。

## 怎么判断一个东西属于阵容路线层

- 它会改变后续资源怎么投，而不是只改变当前一个英雄有多强
- 它会让玩家开始排斥另一类选择，出现"走这条线就不值得再平均养"的效果
- 它至少会和一个路线层系统、一个底盘层系统发生联动

## 路线是怎么形成的

### 1. 先有核心英雄，再围绕它补兵种

- 触发条件：玩家通过招募拿到T0级别英雄（如马克西莫、莎琳娜）或T1英雄（如红蜘蛛、威震天）
- 具体结果：后续合成和科技选择开始向该英雄加成的兵种属性倾斜。马克西莫绑定陆军，莎琳娜绑定海军
- 为什么有效：核心件先确定，配套件才有存在意义

### 2. 陆军流、海军流和空军流不是平行选择，而是互相克制

- 触发条件：玩家把英雄和兵种锁定某一军种后开始养成
- 具体结果：主战兵种负责穿关，克制兵种负责修正短板。陆军流主战单位是坦克，克制空军
- 识别边界：能独立终结关卡压力的是主战流；不能独立终结，但能显著提高克制胜率的是辅助流

### 3. 极限单兵流的本质是"把资源压在一个克制循环上"

- 触发条件：平民玩家和微氪玩家选择死磕一个兵种（如全陆军）
- 具体结果：该兵种战力远超同水平玩家，但在遇到克制兵种（空军）时极易被击穿
- 为什么有效：单兵流减少了阵容里的解释成本，也减少了成型所需的资源量

### 4. 流派成型依赖"停手点"

- 触发条件：辅助兵种、科技树或英雄星级达到某个临界值（如英雄升至3星解锁核心被动）
- 具体结果：玩家会停止继续平均分配资源，转而集中给主力兵种
- 设计原因：流派不是无限强化，而是识别"够用了"之后开始补短板

### 5. 流派的真假区分在于"是否会排斥别的方向"

- 触发条件：玩家拿到某个核心英雄或解锁关键科技
- 具体结果：其他路线的资源投入开始变得低效。投入海军资源会稀释陆军英雄的加成收益
- 为什么有效：如果一个构筑不会让别的选择变差，它更像临时增益，不像真正的流派

## 不同流派之间的边界

- 只堆一个高战英雄，不叫流派，只叫单点强化
- 只有基础兵种没有英雄加成，不叫流派，只叫人海战术
- 同时拥有核心英雄、对应兵种专精和科技兑现，才更像真正的流派

## 这个模块不负责什么

- 不负责兵种具体数值公式，那是科技底盘和资源循环的职责
- 不负责关卡何时变难，那是节奏层的职责
- 不负责PVP具体检定规则，那是PVP与高难检定的职责

## 可迁移结论

- If 英雄的被动技能仅对特定兵种（如陆军）生效且加成比例显著（如提升30%攻击力），Then 玩家的阵容构筑就会被强制锁定在单一兵种路线上，形成"抽到什么英雄就玩什么流派"的强导向
- If 辅助件（如副英雄/副兵种）能显著修正主战兵种的克制短板，流派会比单纯堆输出更稳定，也更耐玩
