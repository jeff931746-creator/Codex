# 口袋奇兵·基地与资源循环拆解

## 这个专题要回答什么

《口袋奇兵》的资源循环为什么不是"简单发资源"，而是把合成、产能和养成速度绑在一起？

## 总体判断

这款游戏的经济循环本质上是"合成即产能"：低门槛资源负责日常造兵，高稀缺资源负责英雄招募，建筑合成直接改变资源产出上限。玩家不是在单一货币里做选择，而是在多个层级的资源里做取舍。

## 怎么判断某个元素属于经济循环

- 它如果主要改变"拿到什么材料、花在哪、什么时候花"，就属于经济循环
- 它如果只改变战斗中谁打得更快，但不改变资源流向，就不属于经济循环
- 它如果能把一次性掉落转成长期价值，就一定在经济循环里占有位置

## 资源分层怎么工作

### 1. 即时资源负责单局推进

- 触发条件：基地建筑产出、PVE关卡结算或日常任务完成
- 具体结果：玩家拿到金币、石油、电力等即时资源
- 作用：支撑当前关卡里的造兵和合成

### 2. 高稀缺资源负责路线解锁

- 触发条件：钻石、高级招募券、加速道具被使用
- 具体结果：玩家可以抽取英雄、补缺或快速完成科技研究
- 作用：控制路线成型速度，而不是单纯提高数值

### 3. 建筑合成直接改写资源产出公式

- 触发条件：将基地内的资源建筑（如金矿、油井）进行合成升级（两个1级金矿合成1个2级金矿）
- 具体结果：建筑等级跃迁，单次产出量和存储上限提升约50%-100%[推测]
- 作用：让"基建"从被动等待变成主动加速

### 4. 联盟协作把外部收益接到主循环里

- 触发条件：联盟成员互相援助、联盟领地收益发放
- 具体结果：获得额外资源、加速和战斗增益。联盟援助可缩短建筑升级时间约10%-20%[推测]
- 作用：让"社交"也能推动养成

## 经济循环的两个关键出口

- 一个出口是"战斗相关材料"，直接喂给兵种合成和科技底盘
- 另一个出口是"路线相关材料"，直接喂给英雄招募和升星
- 如果一个资源不能明确流向这两个出口之一，它就更像装饰性奖励，不像核心经济

## 购买和消耗的真实位置

- 造兵消耗决定玩家当前能合成多少高阶单位
- 钻石消耗决定玩家今天能补多少英雄缺口
- 科技材料消耗决定玩家今天能把哪条兵种线推进到哪个阈值
- 建筑合成消耗决定玩家中长期资源产出速度

## 这个模块不负责什么

- 不负责具体关卡难度，那是节奏层和检定层的职责
- 不负责单个兵种的战斗表现，那是执行层和底盘成长的职责
- 不负责英雄技能加成，那是路线层的职责

## 可迁移结论

- If 资源建筑的产出上限由建筑等级决定，且建筑升级依赖合成操作而非时间倒计时，Then 资源获取速度就直接挂钩玩家的活跃度与合成策略，而非单纯的时间积累
- If 资源系统同时存在即时造兵消耗、建筑合成投入和联盟援助补充，玩家会更容易把短线失败理解为阶段性亏损，而不是彻底白打
