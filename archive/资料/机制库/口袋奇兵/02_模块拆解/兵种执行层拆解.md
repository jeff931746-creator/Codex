# 口袋奇兵·兵种执行层拆解

## 这个专题要回答什么

兵种在《口袋奇兵》里为什么是执行层，而不是路线层？

## 总体判断

兵种执行层负责把海陆空三军的伤害和克制关系兑现成战斗结果。它决定"这波能不能打赢"，但不决定"这局到底走哪条兵种路线"，路线声明来自英雄招募和科技树选择。

## 兵种执行层到底在做什么

### 1. 三军单位上场后立即承担战斗职责

- 触发条件：玩家将合成的兵种单位派入战场或防守阵容
- 具体结果：陆军、海军、空军单位开始按各自攻击逻辑输出，并触发克制关系（陆克空、空克海、海克陆）。克制方对被克制方造成约130%伤害[推测]，被克制方造成约70%伤害[推测]
- 为什么有效：执行层必须是低解释成本的，玩家看得见克制反馈才会继续投入对应兵种

### 2. 输出型和防御型兵种分工明确

- 触发条件：兵种被生产和解锁时
- 具体结果：重型坦克（陆军）负责前排承伤，狙击手（陆军）负责后排输出；驱逐舰（海军）负责反潜输出，护卫舰（海军）负责防空掩护
- 识别边界：只要这个单位的核心价值是"直接消灭敌军"，它就是输出；只要它的核心价值是"吸收伤害保护后排"，它就是防御

### 3. 兵种等级直接决定执行层强度

- 触发条件：低级兵种通过合成变为高阶兵种（如两个1级坦克合成1个2级坦克）
- 具体结果：单位基础属性产生阶梯式提升。1级单位战力为100基准，2级单位战力跃迁至300[推测]，3级单位战力跃迁至900[推测]
- 为什么有效：让执行层强度与合成操作强绑定，玩家每次合成都能立刻看到战力变化

### 4. 高阶兵种的关键不在名字，而在"能否承接更多乘区"

- 触发条件：兵种等级提升且搭配对应英雄时
- 具体结果：高阶单位拥有更高的基础面板，能更有效地承接英雄技能加成（如"攻击力+30%"）和科技树加成（如"生命值+20%"）
- 为什么有效：让强度来源从"造更多兵"转为"围绕高阶兵种继续加乘"

## 这个层级的边界在哪里

- 它不负责声明"这局要主玩陆军还是海军"，那是英雄与阵容路线层的职责
- 它不负责长期解锁新上限，那是科技与底盘成长层的职责
- 它不负责产出资源和加速道具，那是基地与资源循环的职责

## 典型识别标准

如果一个元素满足下面任意两条，就更接近执行层：

- 上场后立刻参与伤害结算
- 受到三军克制关系（陆克空、空克海、海克陆）直接影响
- 其强度主要体现为"当前关卡/对战能不能赢"

如果它主要满足下面这些条件，就不是执行层：

- 需要靠局外资源长期养成
- 本身不直接参与战斗结算
- 更像路线选择或奖励修正

## 可迁移结论

- If 战斗结果由三军克制关系（陆克空、空克海、海克陆）主导而非纯数值碾压，Then 玩家的兵种搭配决策权重就会高于单纯的数量堆叠，克制关系成为策略核心
- If 执行层单位的强度被拆分为"基础面板"和"外加成乘区"两部分，Then 英雄和科技系统才能找到明确的数值投放位置，避免数值膨胀失控
