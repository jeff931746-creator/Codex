# 口袋奇兵·科技与底盘成长层拆解

## 这个专题要回答什么

《口袋奇兵》的科技树为什么是"底盘"，而不是"路线"？

## 总体判断

科技树系统负责把单局里看得到的合成强度变成可持续积累的底盘。它不直接告诉玩家该走哪条兵种路线，但它决定了这条路线能不能稳定兑现，以及兑现速度有多快。

## 成长底盘由哪些部分组成

### 1. 兵种独立科技树是基础兑现

- 触发条件：消耗金币、石油等资源投入陆军、海军或空军科技树研究
- 具体结果：对应兵种全单位基础攻击、生命、防御上升。每级科技提升约5%-10%基础属性[推测]
- 为什么有效：最容易理解的成长方式，适合承接SLG用户的长期沉淀预期

### 2. 特殊技能解锁把长期投入转成战术优势

- 触发条件：兵种科技研究达到5级、10级等关键节点
- 具体结果：解锁该兵种的特殊技能或被动增益（如陆军科技5级解锁"坦克攻速+10%"）
- 设计原因：把"枯燥点数值"的挫败转成"获得新战术"的长期期待

### 3. 资源分层控制成长速度

- 触发条件：玩家获取不同等级的科技材料、加速道具
- 具体结果：普通材料（金币）支撑日常升级，稀有材料（科技芯片）支撑关键节点突破。稀有材料产出受限，控制成长节奏
- 为什么有效：用材料层级把长期追赶和短期冲刺分开

### 4. 科技树等级会改变实际决策

- 触发条件：某条兵种科技树等级远高于其他两条（如陆军科技20级，海军空军10级）
- 具体结果：玩家会主动放弃另外两条兵种的日常投入，转而集中补强主力兵种
- 设计原因：阈值型成长比线性成长更容易制造"养成选择题"

## 这个层级不负责什么

- 不负责单次战斗里具体怎么摆阵，那是兵种执行层的职责
- 不负责谁跟谁合成，那是合成升级层的职责
- 不负责奖励从哪里来，那是基地与资源循环的职责

## 可迁移结论

- If 科技树提供的属性加成对所有同类型单位永久生效而非单局临时Buff，Then 玩家的养成动力就会从"应对单局压力"升级为"积累账号资产"，从而支撑起长线留存
- If 科技树等级存在明显的边际收益递增节点（如5级、10级解锁关键技能），玩家会自然形成"死磕一个兵种"的优化行为，设计者也更容易控制兵种生态
