# 口袋奇兵·合成升级层拆解

## 这个专题要回答什么

《口袋奇兵》的合成为什么不是"简化操作"，而是局内路线声明？

## 总体判断

合成系统不是单纯的拖拽合并，而是把"造兵/建房"变成"确定本阶段主攻方向"的关键开关。玩家一旦决定把资源集中合成某条兵种线的高阶单位，就等于在确认当前的战力结构和后续科技升级方向。

## 合成到底怎么工作

### 1. 相同单位拖拽合并后，直接产出高阶单位

- 触发条件：将两个相同等级和类型的兵种或建筑拖拽到一起
- 具体结果：合成为1个更高等级的单位。战力呈阶梯式跃迁：1级单位战力100，2级单位战力300，3级单位战力900[推测]，而非线性增长
- 为什么有效：把"等待升级"的动作变成"即时质变"的动作，消除传统SLG的等待疲劳

### 2. 合成后的核心收益不是"省空间"，而是战力跃迁

- 触发条件：合成完成
- 具体结果：高阶单位解锁更强的攻击表现和生存能力。一个3级单位（战力900）的战斗力远超三个1级单位（总战力300）[推测]
- 识别边界：如果一个合并操作只是减少数量而不改变战力评级（如背包整理），它就不算路线层；能改变战力评级和后续决策，才算

### 3. 合成会改变后续产能分配逻辑

- 触发条件：玩家已经合出某条兵种线的高阶单位（如陆军坦克升到5级）
- 具体结果：后续产能更愿意投给该兵种线的低级单位作为合成素材，而不是平均分配给三军
- 为什么有效：合成不只是结果，它会反向塑造资源分配决策

### 4. 局内先合成出高阶兵种的那一刻，路线就锁定了大半

- 触发条件：战斗中率先合出某条兵种线的高阶核心单位
- 具体结果：后续英雄搭配、科技研究都会围绕这条线展开
- 为什么有效：用"先成型者优先"制造局内策略压力

## 这个层级不负责什么

- 不负责兵种的基础克制关系，那是兵种执行层的职责
- 不负责给你稳定产出合成素材，那是基地与资源循环的职责
- 不负责英雄技能加成，那是英雄与阵容路线层的职责

## 可迁移结论

- If 合成操作是单位升级的唯一途径且合成结果即时生效，Then 玩家在局内的操作重心就会从"等待进度条"转向"优化合成路径"，操作反馈从延时满足变为即时满足
- If 合成素材的消耗会反向塑造产能分配（如集中生产某一兵种），Then 合成系统就不只是操作减负，而是经济循环的节流阀，直接影响资源流向
