# 永远的蔚蓝星球·Boss与合作检定拆解

## 这个专题要回答什么

《永远的蔚蓝星球》的Boss与合作模式为什么更像“偏科检定器”，而不是单纯的数值墙？

## 总体判断

这套检定系统不是只看战力高低，而是看构筑有没有短板。只会堆单点输出的阵容可能在普通波次很舒服，但一到合作Boss，就会暴露清场、续航、控制或成型速度的缺口。

## 怎么判断某个节点属于检定层

- 它如果会强迫玩家暴露短板，而不是单纯给更多奖励，就属于检定层
- 它如果会改变玩家对“要不要继续补同一条线”的判断，就属于检定层
- 它如果只是在主线中平滑加难，而不会导致构筑修正，就更像节奏层而不是检定层

## 检定是怎么发生的

### 1. 合作Boss检定阵容完整度

- 触发条件：进入合作模式，与好友或随机队友共同抵御Boss
- 具体结果：玩家需要在有限时间内处理更强目标，单靠单人阵容无法覆盖所有克制需求
- 为什么有效：Boss把“当前阵容够不够完整”这件事显性化，强制要求职业互补

### 2. 职业克制检定偏科程度

- 触发条件：遇到如重甲巨兽（高物防低法防）或法术幽灵（高法防低物防）Boss波次
- 具体结果：纯射手暴伤流可能打不动重甲巨兽，纯法师爆发流可能对法术幽灵刮痧
- 为什么有效：逼玩家区分“持续输出”和“短时爆发”，补齐毒系持续伤害或战士击退

### 3. 高波次生存检定后期鲁棒性

- 触发条件：进入更高波次、更快出怪或更重压力的玩法
- 具体结果：阵容短板被快速放大，缺乏控制或辅助的阵容容易崩盘
- 为什么有效：让后期不是纯数值比拼，而是构筑稳定性比拼

### 4. 检定的本质是逼玩家修正偏科

- 触发条件：某个流派在普通关卡里效率很高，但在合作Boss里失效
- 具体结果：玩家必须补控制、补群伤、补生存或补成型速度
- 识别边界：只要一个节点不是在检验“你有没有某类缺口”，它就不算检定层

## 检定层和节奏层的分界

- 节奏层回答“什么时候更难”
- 检定层回答“难的到底是哪一类问题”
- 同样是加压，只有能指出短板类型的节点才算检定

## 这个模块不负责什么

- 不负责长期资源怎么来，那是资源循环的职责
- 不负责关卡中间的平滑节奏，那是节奏层的职责
- 不负责奖励节点如何发材料，那是商店与任务奖励的职责

## 可迁移结论

- If 合作Boss（如重甲巨兽）具有极高的单抗偏科特性，且需要两人阵容互补（一人破甲一人输出）才能在时限内击杀，Then 玩家就会被强制要求补齐自己不擅长的职业，而不是只练一队
- If 检定节点专门针对职业偏科（如重甲巨兽高物防低法防），Then 玩家就会在失败后自然补短板，而不是只刷更久
