# 永远的蔚蓝星球·底层设计公式

## 这一层的用途
这里的结论脱离本游戏语境仍应成立。它们揭示了“竖屏塔防+Roguelike合成”品类在底层结构上的因果关系——当游戏试图将高方差随机性拆解为短循环决策、并让局外构筑与局内合成深度绑定时，必须遵守的机制传导逻辑。这些命题不依赖具体的7职业设定或3分钟单局，而是该设计范式成立的基石。

## 结论列表

**结论 1：波动经济律**

If 局内核心资源的获取/消耗采用非线性波动机制，Then 资源管理就会从被动积累跃迁为主动的时机博弈。

*反面测试*：把 If 取反后（资源获取/消耗为固定值），Then 是否仍成立？不成立，固定值只会产生最优解挂机。

**结论 2：合成对冲律**

If 局内存在高方差的随机获取机制，Then 必须提供确定性的合成进阶路径（如2合1倍率跃迁）来对冲随机挫败感，否则玩家会因连续霉运流失。

*反面测试*：把 If 取反后（获取机制完全确定），Then 是否仍成立？不成立，确定获取不需要对冲。

**结论 3：强制社交衰减律**

If 单人刷关收益存在疲劳衰减，Then 必须提供收益衰减豁免的合作模式，否则疲劳衰减只会导致玩家直接下线。

*反面测试*：把 If 取反后（单人刷关无衰减），Then 是否仍成立？不成立，无衰减则无需豁免。

**结论 4：互斥构筑律**

If 天赋/技能树采用互斥的双分支设计，Then 同一基础单位的复用价值会翻倍，且构筑多样性从收集数量转向排列组合。

*反面测试*：把 If 取反后（天赋/技能可全部点亮），Then 是否仍成立？不成立，全点亮则无互斥无复用。

**结论 5：共鸣锚定律**

If 装备共鸣需要固定件数（如3件套）且绑定特定职业/元素才激活质变羁绊，Then 共鸣系统就会跃迁为构筑锚点而非通用数值加成。

*反面测试*：把 If 取反后（共鸣只需2件且不绑定职业），Then 是否仍成立？不成立，低门槛无绑定只会产生通用最优解。

## 边界
这套公式在缺乏局内随机性或单局时长极长的纯策略塔防中不适用；在重操作反应轻策略构筑的动作类塔防中也不适用。当游戏的核心乐趣不再依赖于“用确定性对冲随机性”与“用互斥约束制造构筑方向”时，上述因果链便会断裂。
