# 永远的蔚蓝星球·资源循环拆解

## 这个专题要回答什么

《永远的蔚蓝星球》的资源循环为什么不是“简单发钻石”，而是把局内波动经济、局外疲劳衰减和养成材料绑在一起？

## 总体判断

这款游戏的经济循环本质上是“双轨分流”：局内金币负责即时召唤与合成决策，局外钻石与招募券负责长线英雄获取，炼器石与疲劳衰减则控制单日产出上限。玩家不是在单一货币里做选择，而是在局内外的资源层级里做取舍。

## 怎么判断某个元素属于经济循环

- 它如果主要改变“拿到什么材料、花在哪、什么时候花”，就属于经济循环
- 它如果只改变战斗中谁打得更快，但不改变资源流向，就不属于经济循环
- 它如果能把单局时间转化成养成材料产出，就一定在经济循环里占有位置

## 资源分层怎么工作

### 1. 局内金币负责即时召唤决策

- 触发条件：局内击杀怪物或时间推移获得金币
- 具体结果：玩家花费10-40金币波动费用召唤英雄
- 作用：支撑当前关卡里的召唤与合成，用波动费用制造微决策

### 2. 钻石与招募券负责长线英雄获取

- 触发条件：任务结算、商店购买或活动兑换
- 具体结果：玩家获取英雄碎片、整卡或图腾
- 作用：控制路线成型速度，决定玩家这周能养哪条职业线

### 3. 炼器石与锻造书控制装备成长上限

- 触发条件：合作模式、主线关卡或商店兑换产出
- 具体结果：用于装备强化与锻造，提升英雄底盘数值
- 作用：分离英雄等级与装备等级，让资源消耗有两个独立出口

### 4. 疲劳衰减控制单日总产出

- 触发条件：单日进行20局以上游戏
- 具体结果：金币与炼器石收益下降
- 作用：防止无限刷关，把单日游戏时长控制在健康范围内

## 经济循环的两个关键出口

- 一个出口是“英雄相关材料”，直接喂给英雄成长底盘
- 另一个出口是“装备相关材料”，直接喂给装备强化与披风洗练
- 如果一个资源不能明确流向这两个出口之一，它就更像装饰性奖励，不像核心经济

## 这个模块不负责什么

- 不负责具体关卡难度，那是节奏层和检定层的职责
- 不负责单个英雄的战斗表现，那是执行层和成长底盘的职责
- 不负责合成结果本身，那是路线层的职责

## 可迁移结论

- If 游戏经济将局内金币（高频波动消耗）与局外钻石/招募券（低频稳定获取）完全隔离，且局内金币无法带出局外，Then 局内的随机性就不会侵蚀局外的养成确定性，玩家能接受局内的高方差霉运
- If 游戏设置20局后的疲劳衰减机制，且衰减只影响主线金币与炼器石而不影响合作Boss掉落，Then 玩家的单日单人刷关收益会被软性封顶，且单人意愿会被强制导向社交合作
