# 永远的蔚蓝星球·英雄执行层拆解

## 这个专题要回答什么

英雄在《永远的蔚蓝星球》里为什么是执行层，而不是路线层？

## 总体判断

英雄执行层负责把局内召唤、摆位和7职业克制效果兑现成战斗结果。它决定“这局能不能顶住”，但不决定“这局走哪条合成路线”，路线声明来自合成强化和图腾天赋系统。

## 英雄执行层到底在做什么

### 1. 英雄上场后立即承担7职业之一的战斗职责

- 触发条件：玩家花费10-40金币波动费用召唤英雄并放置到战场
- 具体结果：该英雄开始执行射手暴伤、战士击退、毒系持续伤害、法师爆发、召唤物抗压、控制打断或辅助回血等职能
- 设计原因：7职业克制让执行层必须是低解释成本的，玩家看得见职业相克结果才会继续投入召唤

### 2. 非线性召唤费用强制执行经济管理

- 触发条件：玩家在局内点击召唤栏位（最多12个）
- 具体结果：召唤费用在10-40金币间非线性波动，玩家无法无脑点击，必须判断当前金币量是否值得赌这一次召唤
- 设计原因：把“抽卡”从纯爽感变成资源分配决策，防止前期无脑铺场导致后期金币断裂

### 3. 英雄星级直接决定执行层强度上限

- 触发条件：两个相同英雄2合1进阶
- 具体结果：4级英雄的攻击力达到1级的4倍，技能冷却大幅缩减
- 为什么有效：让执行层不是死板单位，而是可被合成系统持续改写强度的容器

### 4. 职业克制提供最高10%的伤害加成修正

- 触发条件：上阵不同职业组合触发克制关系
- 具体结果：阵容伤害获得最高10%的加成
- 为什么有效：用小幅度加成引导玩家补全职业，而不是只堆单一职业的数量

## 这个层级的边界在哪里

- 它不负责声明“这局要玩法师流还是战士流”这种路线判断，那是三重随机与合成层和图腾天赋的职责
- 它不负责长期解锁新上限，那是英雄成长底盘的职责
- 它不负责产出资源和礼包材料，那是资源循环与商店奖励的职责

## 典型识别标准

如果一个元素满足下面任意两条，就更接近执行层：

- 上场后立刻参与7职业之一的伤害结算
- 在单局内频繁被召唤、合成或替换
- 其强度主要体现为“当前波次能不能过”

如果它主要满足下面这些条件，就不是执行层：

- 需要靠局外资源长期养成
- 本身不直接参与战斗结算
- 更像路线选择或奖励修正

## 这个模块不负责什么

- 不负责英雄合成的随机性规则，那是三重随机与合成层的职责
- 不负责英雄局外的基础属性成长，那是英雄成长底盘的职责
- 不负责职业克制的深层构筑加成，那是图腾与天赋构筑的职责

## 可迁移结论

- If 塔防游戏的局内召唤费用采用10-40金币的非线性波动机制，Then 玩家的召唤行为就会从无脑点击转向时机与经济管理
- If 英雄升阶采用2合1且4级攻击力达到1级的4倍，Then 玩家就会极度优先寻找合成机会而不是铺场低级单位
