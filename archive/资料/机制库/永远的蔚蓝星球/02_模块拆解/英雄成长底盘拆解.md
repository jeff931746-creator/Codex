# 永远的蔚蓝星球·英雄成长底盘拆解

## 这个专题要回答什么

《永远的蔚蓝星球》的英雄成长为什么是“底盘”，而不是“路线”？

## 总体判断

英雄成长系统负责把单局里看得到的强度变成可持续积累的底盘。它不直接告诉玩家该走哪条流派，但它决定了这条流派能不能稳定兑现，以及兑现速度有多快。

## 成长底盘由哪些部分组成

### 1. 英雄碎片与升级是基础兑现

- 触发条件：投入对应的英雄碎片与金币
- 具体结果：英雄基础攻击、生命、技能强度上升
- 为什么有效：最容易理解的成长方式，适合承接多平台用户的短反馈预期

### 2. 装备强化与锻造提供数值杠杆

- 触发条件：使用锻造书与炼器石对装备进行强化
- 具体结果：装备主属性大幅提升，提供额外的数值乘区
- 设计原因：把“重复刷关”的挫败转成“材料积累”的长期沉淀

### 3. 披风洗练把随机性转成长期追求

- 触发条件：使用炼器石对披风进行洗练
- 具体结果：随机获得不同词条与数值的披风属性
- 为什么有效：用随机词条制造“极品属性”的追求，延长局外养成周期

### 4. 材料分层控制成长速度

- 触发条件：玩家获取不同层级的金币、锻造书、炼器石或英雄碎片
- 具体结果：同样是升级，普通金币与稀有炼器石会产生不同速度和上限
- 为什么有效：用材料层级把长期追赶和短期冲刺分开

## 这个层级不负责什么

- 不负责单局里具体怎么摆阵，那是执行层的职责
- 不负责谁跟谁合成，那是三重随机与合成层的职责
- 不负责奖励从哪里来，那是资源循环和商店奖励的职责

## 可迁移结论

- If 局外成长系统将披风洗练作为独立于英雄等级和装备强化的第三随机乘区，且洗练消耗与局内经济解耦，Then 玩家的局外长线留存会显著增加，但同时会产生词条焦虑导致的重氪倾向
- If 装备强化使用独立于英雄升级的锻造书与炼器石，且这两种材料只能通过合作Boss与疲劳衰减后的关卡产出，Then 材料获取的社交性与衰减机制就会成为控制养成进度的核心阀门
