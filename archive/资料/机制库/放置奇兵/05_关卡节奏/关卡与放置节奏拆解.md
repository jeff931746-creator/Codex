# 放置奇兵·关卡与放置节奏拆解

## 这个专题要回答什么

《放置奇兵》的关卡节奏为什么能把“放置挂机”拆成一串极低频、但极关键的卡点检定？

## 总体判断

它的关卡节奏不是靠持续操作，而是靠“自动推图-卡点-提升-再推图”的循环。玩家绝大部分时间在看着数值增长，一旦卡住，就必须做一次高权重的阵容或养成决策。

## 怎么判断某个东西属于关卡节奏

- 它如果主要控制压力何时升高、何时给喘息，就属于关卡节奏
- 它如果主要决定挂机收益的档位提升，就属于关卡节奏
- 它如果直接判断构筑是否过关，那就更接近检定挑战而不是纯节奏

## 关卡节奏的关键结构

### 1. 挂机阶段让数值自动积累

- 触发条件：玩家通关某一关键节点后
- 具体结果：挂机收益的每小时产出档位提升，推图自动进行
- 设计原因：先给无脑成长的爽感，才能让后面的卡点显得有打破必要

### 2. 战力卡点逼出养成决策

- 触发条件：推图遇到战力压制或带有沉默技能的暗影怪
- 具体结果：自动战斗失败，玩家必须手动调整阵容、升级英雄或更换装备
- 设计原因：节奏不是线性加速，而是用卡点逼迫玩家从“挂机”转入“构筑”

### 3. 收益上限逼出回流节拍

- 触发条件：离线时间达到收益存储上限12小时
- 具体结果：资源停止增长，玩家必须上线收菜
- 设计原因：放置游戏如果没有回流节拍，日活就会失控

## 节奏和养成的关系

- 关卡卡点越硬，升星和羁绊的必要性越强
- 如果卡点只看等级不看阵容，玩家会觉得只需要挂机更久
- 如果卡点需要特定策略，构筑的价值才会显现

## 这个模块不负责什么

- 不负责具体奖励内容，那是抽卡与商店奖励的职责
- 不负责英雄如何自动战斗，那是执行层的职责
- 不负责偏科是否被判失败，那是检定挑战的职责

## 可迁移结论

- If 关卡Boss同时具备高防御（检定数值底盘）和沉默技能（检定阵容是否带解控），Then 玩家卡关时就必须同时提升底盘星级并修正构筑缺口
- If 每推过50关，挂机每小时的资源产出档位就提升5%，Then 推图动力就会与日常收益深度绑定，卡关等于每天损失基础收入
