# 放置奇兵·阵营职业构筑拆解

## 这个专题要回答什么

《放置奇兵》的阵营和职业为什么是构筑声明，而不是简单的属性加成？

## 总体判断

阵营职业构筑系统不是单纯的收集加成，而是把"凑5个英雄"变成"确定本队伍核心方向"的关键开关。玩家一旦决定走同源阵营（如黑暗系、光明系），就等于确认了本队的核心输出结构和后续碎片投入方向。阵营羁绊的边际递增效应，使得混编阵容在后期天然弱于纯色阵容，从而逼迫玩家做出路线取舍。

## 怎么判断一个东西属于阵营职业构筑

- 它会改变后续碎片资源怎么投，而不是只改变当前一场战斗的属性
- 它会让玩家开始排斥另一类选择，出现"走这条线就不值得再养其他阵营"的效果
- 它至少会和一个执行层系统、一个底盘层系统发生联动

## 构筑是怎么形成的

### 1. 阵营光环是路线锁定的核心推手

- 触发条件：上阵同阵营多名英雄（如3名、5名）
- 具体结果：触发阵营光环，获得大幅属性加成（资料未覆盖具体加成比例，需后续补充）
- 为什么有效：光环加成随同阵营英雄数量递增，使得"5同阵营"的收益远高于"3+2混搭"，直接锁死了最优构筑为纯色队

### 2. 职业分工是阵容的骨架

- 触发条件：编队时配置不同职业（战士、法师、牧师等）
- 具体结果：形成前排承伤、后排输出、辅助治疗的基础战斗结构
- 识别边界：职业决定英雄在战斗中的职责（执行层），而阵营决定英雄属于哪条养成路线（构筑层）

### 3. 光暗阵营是高上限的稀有路线

- 触发条件：获取光明或黑暗阵营的英雄
- 具体结果：光暗英雄通常拥有更强技能与更高基础数值，但碎片获取难度远高于普通阵营
- 为什么有效：制造阶梯式的路线追求，平民走普通阵营，大R追求光暗阵营，拉开花费差距

### 4. 构筑成型依赖"停手点"与"纯化"

- 触发条件：玩家在某一阵营投入大量同名卡碎片升星至质变
- 具体结果：其他阵营的英雄因星级落后而无法上场，阵容彻底纯化
- 设计原因：构筑不是无限收集，而是识别"主力成型"后停止分散投资

## 不同构筑之间的边界

- 只堆同职业不凑阵营，不叫构筑，只叫功能编队
- 凑齐5同阵营但无核心输出，不叫构筑，只叫光环挂件
- 同时拥有核心输出、光环加成与职业分工，才更像真正的构筑

## 这个模块不负责什么

- 不负责英雄具体数值公式，那是英雄养成底盘的职责
- 不负责关卡何时变难，那是关卡放置节奏的职责
- 不负责战斗中技能怎么放，那是英雄战斗执行的职责

## 可迁移结论

- If 阵营羁绊的加成比例随同阵营英雄数量呈指数递增（5同源远超3+2混搭），Then 构筑路线会从广度收集转向深度纯化，玩家会自发排斥混编阵容
- If 存在获取概率极低且只能靠抽卡获取的光暗稀有阵营，Then 付费用户的构筑路线会与只能靠时间积累普通阵营碎片的免费用户产生显著分层
