# 放置奇兵·检定挑战拆解

## 这个专题要回答什么

《放置奇兵》的PvP与高难PvE为什么更像“构筑检定器”，而不是单纯的数值墙？

## 总体判断

这套检定系统不是只看战力高低，而是看构筑有没有短板。只会堆单核输出的阵容可能在推图很舒服，但一到PvP或破碎虚空，就会暴露缺乏控制、续航或爆发速度的缺口。

## 怎么判断某个节点属于检定层

- 它如果会强迫玩家暴露短板，而不是单纯给更多奖励，就属于检定层
- 它如果会改变玩家对“要不要继续补同一条线”的判断，就属于检定层
- 它如果只是在主线中平滑加难，而不会导致构筑修正，就更像节奏层而不是检定层

## 检定是怎么发生的

### 1. 推图卡点检定当前阵容数值是否达标

- 触发条件：关卡进度遇到战力压制
- 具体结果：自动战斗失败，逼迫玩家提升底盘数值
- 为什么有效：最基础的检定，把“挂机够不够久”显性化

### 2. PvE高难副本检定阵容功能完整性

- 触发条件：进入破碎虚空等高难玩法
- 具体结果：纯输出阵容无法生存，必须补充控制、减伤或治疗
- 为什么有效：逼玩家区分“推图队”和“攻坚队”

### 3. PvP竞技场检定出手速度与爆发协同

- 触发条件：参与竞技场对战
- 具体结果：先手控制或爆发秒杀成为关键，慢速阵容被克制
- 为什么有效：让构筑不是纯比战力，而是比速度与协同

### 4. 检定的本质是逼玩家修正偏科

- 触发条件：某个流派在推图高效但在PvP被秒杀
- 具体结果：玩家必须培养第二套构筑或补齐功能辅助
- 识别边界：只要一个节点不是在检验“你有没有阵容功能缺口”，它就不算检定层

## 检定层和节奏层的分界

- 节奏层回答“什么时候更难”
- 检定层回答“难的到底是哪一类问题”
- 同样是卡关，只有能指出阵容短板类型的节点才算检定

## 这个模块不负责什么

- 不负责长期资源怎么来，那是放置资源循环的职责
- 不负责关卡中间的平滑节奏，那是关卡与放置节奏的职责
- 不负责奖励节点如何发英雄，那是抽卡与商店奖励的职责

## 可迁移结论

- If 封印之地副本强制要求同阵营英雄上阵，且Boss具备群体控制，Then 玩家就会从纯堆数值转向补齐该阵营的解控辅助
- If 竞技场结算只看1回合胜负且无战斗回复，Then 玩家对速度属性和先手爆发技能的权重认知会显著提升，慢速续航阵容被淘汰
