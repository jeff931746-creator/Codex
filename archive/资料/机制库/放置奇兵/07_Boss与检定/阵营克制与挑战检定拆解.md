# 放置奇兵·阵营克制与挑战检定拆解

## 这个专题要回答什么

《放置奇兵》的高难挑战为什么是阵营检定器，而不是单纯的战力墙？

## 总体判断

这套检定系统不是只看战力高低，而是看构筑有没有短板。只会堆纯色输出的阵容可能在推图中很舒服，但一到高难副本或PVP，就会暴露缺乏控制、续航或被克制的缺口。阵营克制机制的存在，使得没有任何一套阵容能通吃所有检定。

## 怎么判断某个节点属于检定层

- 它如果会强迫玩家暴露阵容短板，而不是单纯给更多奖励，就属于检定层
- 它如果会改变玩家对"要不要继续只养一个阵营"的判断，就属于检定层
- 它如果只是在推图中平滑加难，而不会导致构筑修正，就更像节奏层而不是检定层

## 检定是怎么发生的

### 1. 阵营克制检定纯色阵容的鲁棒性

- 触发条件：遇到克制己方阵营的敌人或Boss
- 具体结果：己方受到额外伤害或对敌伤害降低（资料未覆盖具体克制比例，需后续补充）
- 为什么有效：打破"一队通关全游戏"的惰性，逼迫玩家培养副队应对克制

### 2. 高难Boss检定阵容功能完备性

- 触发条件：进入异域之门等高难PVE副本
- 具体结果：纯输出阵容无法在长线战斗中生存，必须配备控制、减益与治疗
- 为什么有效：检定"只有矛没有盾"的偏科构筑，验证阵容的深度而非单一强度

### 3. PVP竞技场检定阵容的泛化性与针对性

- 触发条件：防守方阵容固定，攻击方针对性出击
- 具体结果：攻击方利用阵营克制或站位针对（如刺客切后排核心）击破高战力防守
- 为什么有效：让战力不再是唯一标准，策略针对也能跨战力取胜

### 4. 检定的本质是逼玩家拓展广度

- 触发条件：主力阵容在特定检定中反复失败
- 具体结果：玩家必须投入资源培养第二阵营或控制/治疗位英雄
- 识别边界：只要一个节点不是在检验"你有没有功能缺失"，它就不算检定层

## 检定层和节奏层的分界

- 节奏层回答"什么时候更难"
- 检定层回答"难的到底是哪一类问题"
- 同样是加压，只有能指出短板类型的节点才算检定

## 这个模块不负责什么

- 不负责长期资源怎么来，那是放置资源循环的职责
- 不负责关卡中间的平滑节奏，那是关卡放置节奏的职责
- 不负责奖励节点如何发材料，那是商店与充值奖励的职责

## 可迁移结论

- If 高难Boss对克制阵营造成额外30%伤害，且免疫纯输出阵容的控制，Then 玩家受挫后的行为会转向培养副队与补齐控制/治疗位，而非纯堆主力输出数值
- If PVP防守方阵容固定且攻击方可提前查看阵容进行针对出击，Then 阵容的泛化防御能力（如免控、多恢复）会比极限输出能力更有战略价值
