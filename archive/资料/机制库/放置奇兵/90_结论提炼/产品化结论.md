# 放置奇兵·产品化结论

## 这一层的用途
这里的结论带适用前提与反例，用于产品化决策。本文件将《放置奇兵》的放置策略RPG特征转化为可落地的设计判断，明确在何种用户层、变现形态与局内时长下这些判断生效，并指出跨品类时的失效边界，避免设计思路的滥用。

## 结论列表

**结论 1：时长货币律**

If 基础经验与金币的产出唯一绑定于挂机时长且设有12小时离线收益上限，Then 玩家的付费点会自然集中在“购买时间（立即获取12小时收益）”上，日活回流节点会被收益上限严格锚定至每天至少2次。

*适用前提*：主打“护肝”的低操作负担放置RPG，玩家碎片化时间特征明显，单日离线时长常超过8小时，IAP变现以加速为主。
*反例游戏*：《魔兽世界》（MMORPG），资源产出依赖主动打怪与采集，离线无收益，时长不作为底层货币，付费不买时间买服务。
*太宽测试*：若替换为 MMORPG 类型，结论不成立。

**结论 2：冗余升星律**

If 英雄升星必须消耗同名英雄本体（9星升10星需1个同名本体），Then 抽卡系统的冗余产出会转化为长线养成刚需，玩家对“抽歪/重复”的负反馈会显著降低。

*适用前提*：以英雄收集与高星升阶为核心的卡牌RPG，且抽卡池包含大量重复获取概率，长线养成周期极长。
*反例游戏*：《云顶之弈》（自走棋），单局英雄合成只需金币购买，无局外卡牌冗余问题，抽卡负反馈机制不存在。
*太宽测试*：若替换为 自走棋 类型，结论不成立。

**结论 3：阵营锁构筑律**

If 阵营羁绊加成在6人满编时提供全属性+30%的指数级收益（远超3人+10%），Then 玩家的阵容会迅速从“混搭最强单卡”转向“纯色阵营构筑”，资源投入产生强排他性。

*适用前提*：拥有6英雄上阵槽位且设有明确阵营划分的RPG，构筑深度依赖协同而非单卡，付费深度随阵容纯度提升。
*反例游戏*：《暗黑破坏神》（ARPG），装备与技能构筑不依赖阵营协同，无排他性路线锁死。
*太宽测试*：若替换为 ARPG 类型，结论不成立。

**结论 4：自动执行律**

If 战斗过程完全托管于速度属性与技能CD自动循环且不可干预，Then 策略重心会100%转移到战前编队、英雄养成与阵容克制上。

*适用前提*：核心用户追求“护肝”与数值验证，不愿意承担局内高频操作负担的放置游戏，局内时长极短。
*反例游戏*：《星际争霸》（RTS），胜负完全依赖局内即时操作与多线调度，战前构筑只决定起点。
*太宽测试*：若替换为 RTS 类型，结论不成立。

**结论 5：定向兜底律**

If 抽卡系统在随机召唤外提供先知球定向兑换指定阵营英雄的途径，Then 玩家构筑流派的成功率会大幅提升，避免因纯随机导致路线无法兑现。

*适用前提*：长线养成周期且强调阵营流派配合的卡牌游戏，纯随机会导致大量无效碎片，用户流失风险高。
*反例游戏*：《原神》（开放世界RPG），角色主要通过限定卡池获取，无常规代币定向兑换角色途径，命座提升依赖重复抽卡。
*太宽测试*：若替换为 开放世界RPG 类型，结论不成立。

## 局限性
1. **重度动作品类**：本结论依赖自动战斗与战前构筑，对于强调局内操作反应与即时闪避的动作类游戏不适用。
2. **纯竞技对战品类**：本结论的付费与养成逻辑建立在PvE时长购买与数值验证上，对于零数值累积的公平竞技游戏不适用。
3. **沙盒创造品类**：阵营锁死与定向兑换逻辑在自由度极高的沙盒游戏中会严重限制玩家的创造力。
