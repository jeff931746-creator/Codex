# 放置奇兵·放置资源循环拆解

## 这个专题要回答什么

《放置奇兵》的资源循环为什么不是“简单发钻石”，而是把“挂机时长”当成最核心的底层货币？

## 总体判断

这款游戏的经济循环本质上是“时间即资源”：挂机时长决定了基础材料的产出上限，高级货币和抽卡道具负责加速或打破时长限制。玩家不是在单一货币里做选择，而是在“等时间自然产出”与“花钱买时间”之间做取舍。

## 怎么判断某个元素属于经济循环

- 它如果主要改变“拿到什么材料、花在哪、什么时候花”，就属于经济循环
- 它如果只改变战斗中谁打得更快，但不改变资源流向，就不属于经济循环
- 它如果能把挂机时长转化为英雄成长进度，就一定在经济循环里占有位置

## 资源分层怎么工作

### 1. 挂机时长负责基础产出

- 触发条件：玩家离线或在线挂机，按每分钟结算
- 具体结果：产出金币、英雄经验、低阶灵魂石等基础资源
- 作用：支撑日常的英雄升级与消耗，是游戏“护肝”标签的来源

### 2. 挂机收益上限控制推进节奏

- 触发条件：挂机时间达到存储上限12小时
- 具体结果：不再产生新的挂机收益，强制玩家上线收菜
- 为什么有效：用收益上限控制离线极限，同时制造每天至少2次的固定回流节点

### 3. 高稀缺资源负责路线解锁加速

- 触发条件：钻石、先知球等高级道具被使用
- 具体结果：玩家可以定向抽卡或快速获得关键阵营英雄
- 作用：控制路线成型速度，付费本质是买时间

### 4. 重复碎片负责长期沉淀

- 触发条件：抽到重复英雄或碎片被处理
- 具体结果：被转成升星材料或兑换代币
- 作用：减少重复掉落的负反馈，维持长线养成

## 经济循环的两个关键出口

- 一个出口是“基础成长材料”，直接喂给英雄成长底盘的等级系统
- 另一个出口是“定向构筑材料”，直接喂给抽卡与商店的先知祭坛兑换系统
- 如果一个资源不能明确流向这两个出口之一，它就更像装饰性奖励，不像核心经济

## 这个模块不负责什么

- 不负责具体关卡难度，那是关卡与放置节奏的职责
- 不负责单个英雄的战斗表现，那是执行层和成长底盘的职责
- 不负责羁绊结果本身，那是阵容构筑层的职责

## 可迁移结论

- If 放置RPG将基础经验与金币的产出唯一绑定于挂机时长（且12小时触顶），Then 玩家的付费动力就会转向购买“立即获取12小时收益”的特权，而不是购买数值特权
- If 离线收益在12小时后停止增长，Then 玩家每天至少会有两次固定的上线收菜回流节点，日活节拍变得可预测
