# 放置奇兵·关卡放置节奏拆解

## 这个专题要回答什么

《放置奇兵》的关卡节奏为什么能把"挂机"变成"阶段性卡点"？

## 总体判断

关卡节奏不是靠复杂的地图机制，而是靠"平滑推图-战力墙卡关-挂机积累-升星突破"的周期循环。玩家在推图中段感到轻松，在战力墙节点感到压力，然后退回挂机积累资源，直到底盘升星质变后突破卡点。节奏本质上是在控制"什么时候该用升星卡你"。

## 怎么判断某个东西属于关卡节奏

- 它如果主要控制压力何时升高、何时给喘息，就属于关卡节奏
- 它如果主要决定奖励在哪个节点掉落，属于节奏与奖励的交叉，但不等于奖励本体
- 它如果直接判断构筑是否有短板，那就更接近检定层而不是纯节奏

## 关卡节奏的关键结构

### 1. 前期平滑推图制造爽感

- 触发条件：开服初期或升星突破后，战力高于关卡需求
- 具体结果：可以连续推进多个章节，挂机收益效率快速提升
- 设计原因：用阶段性的爽感奖励前期的养成投入，并迅速拉开收益差距

### 2. 中后期战力墙制造养成需求

- 触发条件：关卡敌人战力或等级超过当前阵容上限
- 具体结果：推图停滞，挂机收益效率固定，必须通过同名卡升星质变才能继续
- 设计原因：放置游戏如果没有战力墙，养成系统就失去意义，玩家会迅速触及终局

### 3. 扫荡与快速战斗压缩垃圾时间

- 触发条件：已通关的关卡或使用快速战斗券
- 具体结果：瞬间结算一段时间内的挂机收益
- 设计原因：避免重复观看已稳赢的战斗，把时间转化为纯粹的数值结算

## 节奏和养成的关系

- 关卡节奏越快卡关，玩家对升星质变的需求就越迫切
- 如果战力墙拉得太快，玩家会觉得逼氪
- 如果战力墙拉得太慢，挂机收益的锚定作用会失效

## 这个模块不负责什么

- 不负责具体奖励内容，那是商店与充值奖励的职责
- 不负责英雄怎么升星，那是英雄养成底盘的职责
- 不负责阵容克制短板，那是阵营克制与挑战检定的职责

## 可迁移结论

- If 关卡节奏在平滑推进后设置必须由升星质变（而非等级提升）才能跨越的战力墙，Then 玩家在卡关期的行为会从日常收菜转为定向囤积同名碎片
- If 扫荡与快速战斗机制直接将离线时间转化为资源结算，Then 玩家在卡关期的游戏行为会完全剥离战斗观赏，退化为纯数值对撞验证
