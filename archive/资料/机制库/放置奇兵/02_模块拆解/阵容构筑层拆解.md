# 放置奇兵·阵容构筑层拆解

## 这个专题要回答什么

《放置奇兵》的阵容构筑为什么是“阵营路线声明”，而不是“谁强上谁”？

## 总体判断

阵容构筑系统不是简单的战力堆叠，而是把“凑齐6个高星英雄”变成“确定本队伍核心协同方向”的关键开关。玩家一旦决定走光明或暗影阵营路线，就等于确认了核心输出结构和后续碎片投入方向。

## 阵容构筑到底怎么工作

### 1. 同阵营上阵数量触发羁绊加成

- 触发条件：队伍中同阵营英雄达到3个或6个
- 具体结果：激活阵营光环，获得全属性加成（资料未覆盖具体数值，需后续补充）
- 为什么有效：把“上谁”从单卡比较变成组合决策，6人满编羁绊提供指数级收益

### 2. 核心输出与功能辅助的分工绑定

- 触发条件：玩家围绕1-2个核心输出（如西格蒙德、恶魔猎手）编队
- 具体结果：搭配提供控制、增伤、治疗的功能辅助英雄
- 识别边界：能独立终结关卡压力的是核心输出；不能独立终结，但能显著提高输出效率或生存的是辅助

### 3. 构筑会改变后续资源分配逻辑

- 触发条件：玩家确定了光明或暗影阵营路线
- 具体结果：抽卡资源、先知球等定向兑换更愿意投给该阵营，而非平均分配
- 为什么有效：构筑不只是结果，它会反向塑造资源投入决策

### 4. 构筑成型的标志是“排斥其他方向”

- 触发条件：玩家将某阵营英雄升至9星/10星
- 具体结果：其他阵营卡位和资源投入开始变得低效，沉没成本锁死路线
- 为什么有效：如果一个构筑不会让别的选择变差，它更像临时拼凑，不像真正的流派

## 这个层级不负责什么

- 不负责英雄的基础数值提升，那是英雄成长底盘的职责
- 不负责战斗中技能怎么放，那是英雄执行层的职责
- 不负责给你稳定掉落特定英雄，那是抽卡与商店奖励的职责

## 可迁移结论

- If 阵营羁绊的加成在6人满编时提供全属性+30%的指数级收益（远超3人+10%），Then 玩家的编队就会从“混搭最强单卡”转向“纯色阵营构筑”，因为单卡面板差距无法弥补3个羁绊乘区的损失
- If 阵营羁绊提供的是独立乘区（如全伤害+25%）而非线性加算（如攻击力+100），Then 它就不只是收集图鉴，而是决定构筑上限的路线层系统
