# 放置奇兵·商店与充值奖励拆解

## 这个专题要回答什么

《放置奇兵》的商店和充值为什么是缺口的即时填补器，而不是单纯的福利发放？

## 总体判断

这套奖励系统的作用不是"发福利"，而是把玩家的构筑缺口转成可被即时修补的节点。商店解决定向缺哪个英雄碎片，充值解决升星缺多少资源，日常奖励则解决今天想推进但时间不够的问题。

## 怎么判断某个东西属于奖励层

- 它如果在"缺什么就补什么"这个逻辑里工作，就属于奖励层
- 它如果主要决定资源进入速度，而不是直接决定关卡胜负，就属于奖励层
- 它如果可以被玩家按需用副货币购买或兑换，就更像奖励层而不是战斗层

## 奖励节点是怎么工作的

### 1. 碎片商店负责定向修正构筑

- 触发条件：玩家消耗特定货币（如竞技场币、公会币等）在商店兑换
- 具体结果：定向获取指定阵营或英雄的碎片
- 为什么有效：修正抽卡的随机性，保证指定阵营构筑有稳定的碎片来源，是路线成型的保底机制

### 2. 充值中心负责加速底盘质变

- 触发条件：玩家付费购买钻石、成长礼包或特权卡
- 具体结果：获得大量金币、抽卡券或直接获取高阶英雄
- 为什么有效：把升星卡关的时间成本转化为金钱成本，为急需突破的玩家提供捷径

### 3. 日常与活动奖励提供基础供给

- 触发条件：完成每日任务、登录或参与限时活动
- 具体结果：获得抽卡券、金币、经验粉尘等基础资源
- 为什么有效：维持0氪/微氪玩家的基础成长速度，保证生态底层活跃

## 奖励层和经济层的分工

- 奖励层决定"现在给你什么"
- 经济层决定"这些东西怎么被花掉、循环到哪里去"
- 只给奖励不建流向，会让资源显得散；只建流向不发奖励，会让玩家觉得系统空转

## 这个模块不负责什么

- 不负责关卡战力墙设置，那是关卡放置节奏的职责
- 不负责英雄战斗表现，那是英雄战斗执行的职责
- 不负责阵营羁绊规则，那是阵营职业构筑的职责

## 可迁移结论

- If 商店允许使用特定副货币（如竞技场币）定向兑换指定英雄碎片，Then 玩家对抽卡随机性的容忍度会上升，因为随机抽卡不再是指定路线成型的唯一途径
- If 付费礼包直接投放同名英雄碎片或自选阵营碎片，Then 付费行为就从购买随机抽卡机会转变为购买指定路线的成型保底，客单价意愿提升
