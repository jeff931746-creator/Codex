# 放置奇兵·英雄成长底盘拆解

## 这个专题要回答什么

《放置奇兵》的英雄成长为什么是“底盘”，而不是“路线”？

## 总体判断

英雄成长系统负责把阵容构筑看得到的强度变成可持续积累的底盘。它不直接告诉玩家该走哪条阵营路线，但它决定了这条路线能不能稳定兑现，以及兑现速度有多快。

## 成长底盘由哪些部分组成

### 1. 等级和升星是基础兑现

- 触发条件：投入金币、英雄经验及自身英雄碎片
- 具体结果：基础属性提升，技能等级上升，解锁更高阶形态
- 为什么有效：最容易理解的成长方式，适合承接放置游戏用户的预期

### 2. 重复卡牌消耗把抽卡冗余转成长期价值

- 触发条件：抽到重复的英雄卡牌或灵魂石
- 具体结果：转化为升星材料，9星升10星时必须消耗1个同名本体英雄
- 设计原因：把“抽到重复卡”的挫败转成“重复也有用”的长期沉淀

### 3. 装备与神器提供独立乘区

- 触发条件：英雄穿戴装备或神器
- 具体结果：获得额外属性加成或特殊效果
- 为什么有效：用装备系统拉开同英雄在不同阶段的差距

### 4. 升星阈值会改变实际决策

- 触发条件：英雄达到9星或10星
- 具体结果：解锁质变被动（如开局满能量），玩家会优先将资源集中给即将到达9星的英雄
- 设计原因：阈值型成长比线性成长更容易制造“养成选择题”

## 这个层级不负责什么

- 不负责单局里技能怎么放，那是英雄执行层的职责
- 不负责谁跟谁组队有羁绊，那是阵容构筑层的职责
- 不负责奖励从哪里来，那是放置资源循环和抽卡与商店奖励的职责

## 可迁移结论

- If 英雄升星必须消耗同名英雄本体（9星升10星需1个同名本体），Then 抽卡系统的冗余产出就会转化为长线养成刚需，降低“抽歪”负反馈
- If 英雄在9星和10星节点解锁质变被动（如开局满能量），Then 玩家会自然形成“倾家荡产把核心英雄推到9星”的集中式养成行为，而不是平均分配资源
