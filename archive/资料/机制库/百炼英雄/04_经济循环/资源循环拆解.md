# 百炼英雄·资源循环拆解

## 这个专题要回答什么

《百炼英雄》的资源循环为什么不是“简单发钻石”，而是把抽卡难度和英雄生命周期绑在一起？

## 总体判断

这款游戏的经济循环本质上是“稀有度驱动的分层获取”：低稀有度英雄负责前期平滑过渡，高稀有度英雄（橙卡如天使、卡卡西）负责后期上限，但获取难度极高。玩家不是在单一货币里做选择，而是在“确定性养成”和“赌博性抽取”之间做取舍。

## 怎么判断某个元素属于经济循环

- 它如果主要改变“拿到什么英雄、花在哪、什么时候花”，就属于经济循环
- 它如果只改变战斗中谁打得更快，但不改变资源流向，就不属于经济循环
- 它如果能把日常推图收益转成高稀有度英雄的获取机会，就一定在经济循环里占有位置

## 资源分层怎么工作

### 1. 推图资源负责日常养成兜底

- 触发条件：通关副本、完成日常任务
- 具体结果：获得经验、金币和绿/蓝卡英雄碎片
- 作用：支撑当前阵容的等级和升星，保证推图进度不卡死

### 2. 抽卡货币负责路线解锁

- 触发条件：使用游戏内货币或真实货币进行抽卡
- 具体结果：获取橙卡（如天使、卡卡西）或重复碎片
- 作用：控制路线成型速度，橙卡的高难度获取直接拉长了付费和肝度周期

### 3. 觉醒材料负责长期沉淀

- 触发条件：重复英雄碎片被转化为觉醒材料
- 具体结果：英雄获得觉醒质变（如王昭君觉醒20提升30%攻击）
- 作用：把抽卡歪掉/重复的负反馈转成定向养成长线

### 4. 分模式产出引导多维度养成

- 触发条件：参与试炼塔、矿坑、龙巢等不同模式
- 具体结果：获得试炼塔代币/矿坑矿石/龙巢龙鳞等专属养成资源
- 作用：让玩家不能只靠一套阵容挂机所有模式，必须分散资源培养多维度能力

## 经济循环的两个关键出口

- 一个出口是“英雄本体与碎片”，直接喂给英雄成长底盘，决定阵容上限
- 另一个出口是“抽卡机会”，决定玩家能否获得改变路线的核心件（如天使的无敌技能）
- 如果一个资源不能明确流向这两个出口之一，它就更像装饰性奖励，不像核心经济

## 这个模块不负责什么

- 不负责具体关卡难度，那是节奏层和检定层的职责
- 不负责单个英雄的战斗表现，那是执行层和成长底盘的职责
- 不负责阵容搭配逻辑，那是构筑层的职责

## 可迁移结论

- If 抽卡难度与英雄的发力期绑定（如卡卡西极难抽但45级质变），Then 玩家的长线留存动力会显著增加，但前期挫败感也需要用易获取英雄（双枪/猎人）来对冲
- If 觉醒系统消耗重复掉落的英雄碎片并提供质变觉醒，Then 玩家对抽卡歪掉的抵触会降低，冗余产出转化为长线养料
