# 百炼英雄·英雄执行层拆解

## 这个专题要回答什么

《百炼英雄》的英雄为什么是执行层，而不是路线层？

## 总体判断

英雄执行层负责把阵容的搭配意图自动兑现成战斗结果。它决定“这局能不能顶住/打死”，但不决定“这局到底走增益流还是续航流”，路线声明来自阵容构筑和养成阈值。玩家在战斗中不操作，所以执行层的技能机制必须足够直觉，才能让玩家在局外理解为什么要这么配队。

## 英雄执行层到底在做什么

### 1. 英雄上场后按定位自动执行战斗职责

- 触发条件：玩家将英雄编入队伍并进入战斗
- 具体结果：输出英雄（如李白、卡卡西）造成伤害，坦克（如圣骑士）承受伤害，奶妈（如天使）提供治疗，控制（如白无常）施加破甲与流血
- 设计原因：放置游戏执行层必须是零操作成本的，玩家看得懂结果才会去研究搭配

### 2. 100%减伤和无敌技能提供生存质变

- 触发条件：圣骑士释放大招提供100%减伤，或天使释放二技能提供无敌
- 具体结果：队伍在高难度Boss/试炼塔高层中免于团灭，硬扛原本数值无法承受的伤害
- 设计原因：线性数值堆叠无法解决生存压力，必须有100%减伤/无敌这类机制型技能做质变拐点

### 3. 全队增益技能改变输出结构

- 触发条件：王昭君释放全队攻击+攻速BUFF
- 具体结果：主C（如李白、狂战士）的输出频率和单次伤害同时被放大
- 设计原因：让辅助位的价值不在于自身伤害，而在于对执行层整体输出的乘区放大

### 4. 近战与远程的承伤差异塑造站位逻辑

- 触发条件：近战英雄（如狂战士、白无常）攻击时需靠近敌人
- 具体结果：近战更容易被AOE或集火秒杀，远程相对安全
- 设计原因：用物理位置逻辑强制制造阵容弱点，逼玩家在输出和生存间取舍

## 这个层级的边界在哪里

- 它不负责声明“这局要玩流血还是爆发”，那是阵容构筑层的职责
- 它不负责英雄基础数值的长期成长，那是英雄成长底盘的职责
- 它不负责产出抽卡资源，那是资源循环的职责

## 典型识别标准

如果一个元素满足下面任意两条，就更接近执行层：

- 上场后立刻自动参与伤害或治疗结算
- 其强度主要体现为“当前关卡能不能过/扛住”
- 技能效果可以直接用“造成伤害/提供100%减伤/回血”归纳

如果它主要满足下面这些条件，就不是执行层：

- 需要靠局外资源长期养成才能解锁关键效果
- 本身不直接参与战斗结算，而是改变其他英雄的结算方式
- 更像路线选择或资源转化

## 可迁移结论

- If 一款放置RPG的执行层存在“100%减伤/无敌”技能，且玩家无法靠操作规避伤害，Then 玩家的养成目标就会从“堆防御数值”转向“凑技能机制”，构筑深度质变
- If 近战受击范围和承伤频率远高于远程（如近战吃AOE，远程脱战），Then 近战C位的生存必须依赖阵容提供外部减伤/无敌（如圣骑士），自身无法独立兑现输出
