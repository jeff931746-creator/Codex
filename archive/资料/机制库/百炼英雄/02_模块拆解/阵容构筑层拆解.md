# 百炼英雄·阵容构筑层拆解

## 这个专题要回答什么

《百炼英雄》的阵容搭配为什么是路线声明，而不是简单的“把稀有英雄放一起”？

## 总体判断

阵容构筑层不是单纯的战力堆叠，而是把“凑齐5个位置”变成“确定本局打法方向”的关键开关。玩家一旦决定用李白+王昭君的增益爆发流，或用圣骑士+天使的生存续航流，就等于在确认当前阶段的攻坚方向，并主动暴露出对应的短板。

## 阵容构筑到底怎么工作

### 1. 核心输出决定阵容的攻坚上限

- 触发条件：玩家选择李白或卡卡西作为主C
- 具体结果：阵容的资源倾斜和辅助搭配全部围绕该核心展开
- 为什么有效：核心输出（如李白、卡卡西）先确定，辅助增益（如王昭君）与生存保障（如圣骑士、天使）才有存在意义，否则辅助增益无处兑现

### 2. 辅助增益与核心输出形成乘区绑定

- 触发条件：王昭君的攻击+攻速BUFF施加给李白或狂战士
- 具体结果：核心输出的DPS被成倍放大，远超单纯提升等级的收益
- 识别边界：如果一个英雄的技能描述是“提升全队属性”，且该属性能被主C的技能系数放大，它就是构筑放大件；如果只是自身造成伤害，它只是执行件

### 3. 生存位填补核心输出的承伤短板

- 触发条件：玩家将圣骑士或天使编入队伍
- 具体结果：近战C位（如狂战士、白无常）不再被轻易秒杀，队伍获得输出空间
- 为什么有效：没有生存位的队伍在高难度Boss战/试炼塔高层中活不到输出结算，生存件是路线兑现的前提

### 4. 破甲与控制提供攻坚效率

- 触发条件：白无常对Boss施加破甲和流血
- 具体结果：高防目标的承伤增加，队伍整体输出效率提升
- 为什么有效：让破甲/控制位（如白无常）不抢核心资源，但能实质性改变战斗时长

### 5. 构筑成型的标志是“排斥其他路线”

- 触发条件：玩家将资源集中投入李白+王昭君组合
- 具体结果：不再有余力培养另一套以狂战士为核心的近战流血体系
- 为什么有效：真正的构筑必须有代价，如果什么都想养，最后就是没有成型路线

## 这个层级不负责什么

- 不负责英雄具体数值成长，那是英雄成长底盘的职责
- 不负责关卡何时变难，那是节奏层的职责
- 不负责抽卡概率和资源获取，那是资源循环的职责

## 可迁移结论

- If 阵容构筑存在“乘区绑定”（如王昭君攻速增益与李白技能系数乘算），Then 替换辅助位的收益会高于替换同战力输出位，构筑调整优先级改变
- If 生存位能提供免于秒杀的机制拐点（如圣骑士100%减伤），Then 核心输出的养成可以极端偏向输出属性而无需兼顾生存，构筑分工深化
