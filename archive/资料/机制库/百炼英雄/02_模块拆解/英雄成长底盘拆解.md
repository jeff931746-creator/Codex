# 百炼英雄·英雄成长底盘拆解

## 这个专题要回答什么

《百炼英雄》的英雄成长为什么是“底盘”，而不是“路线”？

## 总体判断

英雄成长系统负责把单局里看得到的强度变成可持续积累的底盘。它不直接告诉玩家该走哪条流派，但它决定了这条流派能不能稳定兑现，以及兑现速度有多快。特别是觉醒系统和等级阈值，把线性养成变成了质变拐点。

## 成长底盘由哪些部分组成

### 1. 等级和升星是基础兑现

- 触发条件：投入经验值、金币和英雄碎片
- 具体结果：基础攻击、生命、技能强度上升
- 为什么有效：最容易理解的成长方式，承接放置游戏用户的线性预期

### 2. 觉醒系统把重复卡转成长期质变

- 触发条件：英雄达到35级/45级/55级/75级指定星级后，消耗碎片觉醒（如觉醒20、觉醒60）
- 具体结果：王昭君觉醒20提升30%攻击力，卡卡西觉醒后后期输出超越李白
- 设计原因：把“抽到重复卡”的挫败转成“质变前置条件”的长期沉淀

### 3. 45级阈值改变英雄生命周期

- 触发条件：卡卡西达到45级
- 具体结果：卡卡西从前期弱势直接变为后期最猛输出
- 设计原因：用等级阈值制造英雄发力点，让前期和后期的阵容选择发生分化，增加养成节奏感

### 4. 55级与75级决定资源投入优先级

- 触发条件：英雄升到55级或75级（白5星觉醒60）
- 具体结果：玩家必须选择优先将顶级英雄拉满，还是将工具人停在55级
- 为什么有效：阈值型成长比线性成长更容易制造“停手点”和“养成选择题”

## 这个层级不负责什么

- 不负责单局里具体怎么配队，那是阵容构筑层的职责
- 不负责技能效果本身（如减伤100%），那是执行层的职责
- 不负责奖励从哪里来，那是资源循环的职责

## 可迁移结论

- If 成长系统存在明显的等级发力点（如卡卡西45级质变），且关卡难度同步加压，Then 玩家会产生“熬过前期弱势期”的长线期待，前期英雄自然退环境
- If 觉醒系统把重复掉落转成特定乘区的质变（如王昭君觉醒20提升30%全队攻击），Then 养成就从“全属性提升”变成了“定向构筑强化”，底盘开始反向定义路线层的最优解
