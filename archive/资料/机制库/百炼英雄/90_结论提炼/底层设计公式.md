# 百炼英雄·底层设计公式

## 这一层的用途
这里的结论脱离本游戏语境仍应成立。它们揭示了“放置RPG+英雄收集”品类在底层结构上的因果关系——当游戏试图在自动战斗框架下制造构筑深度、并通过长线养成与检定节点维持留存时，必须遵守的机制传导逻辑。这些命题不依赖具体的英雄题材或技能名称，而是该设计范式成立的基石。

## 结论列表

**结论 1：机制拐定律**

If 放置战斗中存在无视数值的机制型生存技能（如无敌/全额减伤），且玩家无法靠操作规避伤害，Then 玩家的养成决策会从线性堆防转向凑机制组合，构筑深度质变。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立。若只有数值型减伤，玩家只会堆防御属性，不会产生凑技能的组合需求。

**结论 2：生命周期迭代律**

If 英雄的强度曲线被设计为在不同等级阈值发生质变，且关卡难度与该阈值同步加压，Then 玩家会在该节点产生阵容迭代需求，前期英雄自然退环境。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立。若英雄强度曲线线性且无质变，或关卡不与阈值同步，阵容迭代不会在特定节点集中爆发，前期英雄也不会被强制退环境。

**结论 3：偏科检定律**

If 检定节点专门针对阵容偏科设置高压门槛（如纯输出缺生存、缺续航），Then 玩家受挫后的行为会转向补齐短板而非纯堆数值。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立。若检定节点只看总战力，玩家受挫后只会继续堆同一属性。

**结论 4：冗余转化律**

If 局外成长底盘能无损吸收局内重复掉落的角色碎片并转化为质变觉醒，Then 冗余资源产出将转化为长线养成驱动力而非负反馈。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立。若重复掉落无法转化，冗余产出只会增加玩家对抽卡概率的负反馈。

**结论 5：乘区协同律**

If 辅助角色的增益效果与核心角色的输出系数处于不同乘区，Then 辅助角色的战略价值将独立于其自身战力，构筑深度显著提升。

*反面测试*：把 If 取反后，Then 是否仍成立？不成立。若增益与输出在同一线性乘区，辅助角色只会沦为低配输出，无法独立成为战略核心。

## 边界
这套公式在缺乏局外长线养成或战斗结果完全由操作决定的动作游戏中不适用；在纯单机无重复抽卡的经济系统中也不适用。当游戏的核心乐趣不再依赖于“自动战斗下的阵容协同与长线底盘沉淀”时，上述因果链便会断裂。
