# 百炼英雄·Boss与挑战拆解

## 这个专题要回答什么

《百炼英雄》的试炼塔、矿坑和Boss战为什么更像“偏科检定器”，而不是单纯的数值墙？

## 总体判断

这套检定系统不是只看战力高低，而是看构筑有没有短板。只会堆单点输出的阵容可能在推图时很舒服，但一到试炼塔、矿坑或高难度Boss，就会暴露缺生存（无减伤无敌）、缺续航（无持续回血）、缺爆发（无破甲增伤）的缺口。试炼塔、矿坑、龙巢等挑战模式专门针对特定的阵容缺陷。

## 怎么判断某个节点属于检定层

- 它如果会强迫玩家暴露短板，而不是单纯给更多奖励，就属于检定层
- 它如果会改变玩家对“要不要继续补同一条线”的判断，就属于检定层
- 它如果只是在主线中平滑加难，而不会导致构筑修正，就更像节奏层而不是检定层

## 检定是怎么发生的

### 1. 高难度Boss检定生存和爆发

- 触发条件：进入高难度Boss战
- 具体结果：没有圣骑士减伤或天使无敌的队伍会被秒杀，没有破甲（如白无常）的队伍打不动Boss
- 为什么有效：Boss把“当前阵容生存够不够、爆发足不足”这件事显性化

### 2. 试炼塔检定输出上限与增益效率

- 触发条件：进入试炼塔高层
- 具体结果：没有王昭君全队增益的队伍无法在限定时间内清层，狂战士等高频输出英雄在此模式表现突出
- 为什么有效：逼玩家区分“持续输出”和“短时爆发”，验证增益辅助的放大效率

### 3. 矿坑检定续航与恢复能力

- 触发条件：进入矿坑长时间作战
- 具体结果：没有天使普攻回血的队伍在持久战中逐渐崩盘
- 为什么有效：让后期不是纯数值比拼，而是阵容鲁棒性比拼

### 4. 检定的本质是逼玩家修正偏科

- 触发条件：某个流派在推图时效率很高，但在试炼塔、矿坑、龙巢里失效
- 具体结果：玩家必须补控制、补群伤、补生存或补续航
- 识别边界：只要一个节点不是在检验“生存/续航/爆发缺口”，它就不算检定层

## 检定层和节奏层的分界

- 节奏层回答“什么时候更难”
- 检定层回答“难的到底是哪一类问题”
- 同样是加压，只有能指出短板类型的节点才算检定

## 这个模块不负责什么

- 不负责长期资源怎么来，那是资源循环的职责
- 不负责关卡中间的平滑节奏，那是节奏层的职责
- 不负责英雄具体数值成长，那是成长底盘的职责

## 可迁移结论

- If 检定节点专门针对偏科行为（如矿坑检定续航缺天使回血），Then 玩家受挫后的行为会转向补齐阵容短板而非纯堆数值
- If 检定节点只看战力数值，Then 玩家就会继续堆同一种属性，系统不会逼出真正的构筑修正
