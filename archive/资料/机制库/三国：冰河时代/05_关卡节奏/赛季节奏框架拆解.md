# 三国：冰河时代·赛季节奏框架拆解

## 这个专题要回答什么

《三国：冰河时代》的赛季更迭为什么能把“无限数值膨胀”变成“阶段性重置与追新”，而不是单纯的滚服？

## 总体判断

赛季节奏框架是这款SLG的节拍器。它通过固定周期的赛季更替、跨服匹配池扩大和剧本轮换，把传统SLG的无限内卷切成有终点的阶段竞争。每个赛季都是一个完整的“起步-发育-争霸-结算”循环，老赛季的领土积累被清零，但武将底盘被保留，玩家带着老将打新图。

## 怎么判断某个东西属于赛季节奏框架

- 只要它主要控制“什么时候开新图、匹配池扩大到哪、重置什么保留什么”，就属于节奏框架层
- 如果它只改变武将数值，不属于节奏层
- 如果它只决定单场战斗怎么打，属于执行层

## 核心结构

### 1. 赛季演进提供阶段化竞争目标

- 触发条件：服务器时间推进到赛季节点（S1-S8及以后）
- 具体结果：开放新的7级主城目标（洛阳→长安→官渡→襄阳）和对应剧本
- 设计原因：如果没有阶段化目标，SLG会陷入无止尽的磨洋工，赛季目标给玩家一个明确的“终点线”

### 2. 跨服匹配打破区内势力固化

- 触发条件：赛季进阶（S2跨4服，S4跨24服等）
- 具体结果：原有区服的霸主联盟面临外部挑战，弱者有机会重新站队
- 设计原因：防止一盟独大导致服务器死亡，用外部变量重置政治格局

### 3. 剧本轮换改变战场规则

- 触发条件：S5开启多元剧本（如兖州之战-冬，2v2对抗；S6富甲天下-春，八方争夺）
- 具体结果：玩家不能只用一套打法，必须适应新剧本的规则（如2v2对抗要求更紧密的同盟，富甲天下要求经济博弈）
- 设计原因：用规则变化对抗玩法的枯燥感，重置老玩家的战略优势

### 4. 赛季结算清零领土，保留底盘

- 触发条件：赛季结束
- 具体结果：领土、地图资源点重置，武将等级、星级、觉醒状态保留
- 设计原因：既保证了重新竞争的公平性（清零外在领土），又保留了玩家的长期投入价值（保留内在底盘）

## 这个模块不负责什么

- 不负责武将觉醒的具体数值和技能表现，那是武将养成底盘的职责
- 不负责跑商的具体操作和资金转化，那是联盟协作与构筑的职责
- 不负责元宝怎么花，那是资源与经济循环的职责

## 可迁移结论

- If SLG的核心领土目标（如7级主城）随赛季重置并更换地图，Then 旧赛季的领土积累会被强制清零，竞争压力会被重新点燃，避免了死服固化
- If 赛季制只清零领土而不清零武将底盘（保留星级与觉醒），Then 老玩家的沉没成本被尊重，同时新玩家可以通过赛季重置获得相对公平的起跑线
