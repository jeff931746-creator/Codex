# 三国：冰河时代·产品化结论

## 这一层的用途
这里的结论带适用前提与反例，用于产品化决策。本文件将《三国：冰河时代》的赛季制SLG特征转化为可落地的机制判断，明确在何种用户层、变现形态与局内时长下这些判断生效，并指出跨品类时的失效边界，避免设计思路的滥用。

## 结论列表

**结论 1：0氪沙包与基石转化律**

If 游戏将核心抽卡资源（元宝）作为日常高频产出投放给0氪，且0氪能通过该资源抽取当赛季关键武将碎片，Then 0氪玩家会转化为大R体验内容的基石和沙包，0氪7日留存会提升25%[推测]，因为起跑线被相对拉平且自身具有联盟功能价值。

*适用前提*：依赖大R付费且需要庞大活跃基数支撑联盟对抗的SLG，核心抽卡资源能抽取当赛季关键武将。
*反例游戏*：《原神》（开放世界），0氪获取原石的效率远低于大R，且核心体验不依赖0氪为大R提供对抗沙包，0氪更多是内容消费者。
*太宽测试*：若替换为 纯卡牌对战 类型，结论不成立，卡牌对战重绝对公平，不依赖0氪作为沙包。

**结论 2：门槛引爆小氪律**

If 小氪路线存在明确的门槛节点（如7级爵位解锁核心吕布礼包），Then 小氪的付费冲动会被集中引爆，小氪ARPU会提升30%[推测]，而非分散在无数低性价比的小额选项中。

*适用前提*：存在分层付费设计，且核心武将具有不可替代的战术价值（如吕布的早期统治力）。
*反例游戏*：《率土之滨》（传统SLG），付费点分散在多赛季卡包与战法经验上，缺乏单一明确的高性价比门槛节点，小氪付费容易沉没而无感。
*太宽测试*：若替换为 休闲消除 类型，结论不成立，休闲游戏无核心武将战术价值，礼包仅提供通关便利。

**结论 3：领土重置与生态续命律**

If SLG的核心领土目标随赛季重置并更换地图，Then 旧赛季的领土积累会被强制清零，竞争压力会被重新点燃，服务器寿命会延长50%[推测]，避免了死服固化。

*适用前提*：赛季周期为1-3个月[推测]的SLG，且核心乐趣在于领土争夺与联盟对抗，武将养成深度足以支撑跨赛季沉淀。
*反例游戏*：《部落冲突》（COC），匹配基于部落杯数而非赛季重置，且无赛季级别的领土清零，外部威胁不会周期性重置内部政治。
*太宽测试*：若替换为 MMORPG 类型，结论不成立，MMORPG的装备迭代不伴随领土重置，核心驱动力是个人数值而非领土竞争。

**结论 4：分层付费序列共生律**

If SLG围绕“当赛季新武将”做分层付费设计（0氪肝活动/微氪买基金/大氪直接抽），Then 三类玩家会共享同一套终极目标（主城争夺），大R付费意愿会提升30%[推测]，因为0氪和微氪作为联盟基石提供了验证场景。

*适用前提*：强联盟协作的SLG，单人无法攻克核心目标，大R需要平民填充联盟战力。
*反例游戏*：《梦幻西游》（MMORPG），个人PVE和PVP占比极高，大R不需要0氪作为战争基石，付费差异直接导致玩法隔离（不同段位竞技场）。
*太宽测试*：若替换为 单机RPG 类型，结论不成立，单机游戏无联盟基石需求。

## 局限性
1. **纯单机或弱社交品类**：本结论依赖联盟对抗与0氪/大R生态共生，对于无强社交绑定的单机或弱联机游戏不适用。
2. **绝对公平竞技品类**：本结论的付费逻辑建立在“付费拉开成型速度”上，对于强调绝对公平、零数值累积的对战游戏不适用。
3. **短局重开品类**：赛季制重置与底盘保留的逻辑不适用于单局数十分钟的短局游戏。
