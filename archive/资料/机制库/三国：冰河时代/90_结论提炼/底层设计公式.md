# 三国：冰河时代·底层设计公式

## 这一层的用途
这里的结论脱离本游戏语境仍应成立。它们揭示了“赛季制SLG”品类在底层结构上的因果关系——当游戏试图将无限内卷的SLG切成阶段化竞争、并让0氪与大R共生时，必须遵守的机制传导逻辑。这些命题不依赖具体的三国题材或特定武将名，而是该设计范式成立的基石。

## 结论列表

**结论 1：底盘相对优势律**

If 赛季更迭引入上位替代武将（如S2弓兵周瑜替代S1步兵将），Then 玩家的养成重心会从全量收集转向当赛季定向追新，底盘沉淀从“永久绝对值”变为“赛季相对优势”。

*反面测试*：把 If 取反后（不引入上位替代新武将），Then 是否仍成立？不成立，不引入新武将则底盘为绝对值，老玩家永远碾压，养成重心停留在全量收集补图鉴。

**结论 2：多线占城结构检定律**

If SLG的终极占领目标（如洛阳多关卡城门）需要多梯队连战与多线铺路才能攻克，Then 这个目标就会成为联盟整体结构的检定器，而非单一战力的排行榜。

*反面测试*：把 If 取反后（终极目标只需单梯队单次战斗攻克），Then 是否仍成立？不成立，单线目标退化为高战撞钟，无法检定联盟多梯队结构与持续作战能力。

**结论 3：路线互斥声明律**

If 联盟科技树的选择存在互斥研发进度（如军事科技与经济科技抢夺捐献资源），且会显著改变联盟整体的资源获取与战斗属性，Then 联盟就不再只是社交容器，而是路线声明器。

*反面测试*：把 If 取反后（联盟科技只提供线性数值加成且无互斥），Then 是否仍成立？不成立，联盟退化为数值累加器，无路线互斥则无战略选择，加入任何联盟体验同质化。

**结论 4：经济战略绑定律**

If 跑商类玩法的资金产出直接挂钩战争军费，Then 经济行为就会升格为战略博弈，而非纯粹的休闲副玩法。

*反面测试*：把 If 取反后（跑商只产出外观或无关紧要的休闲道具），Then 是否仍成立？不成立，经济行为与战争脱钩则沦为边缘副玩法，无法对主循环产生战略影响。

**结论 5：外部威胁激活律**

If 赛季进阶伴随跨服匹配池的扩大（如4服→24服），Then 区内一盟独大的局面会被打破，外部威胁强制激活内部政治。

*反面测试*：把 If 取反后（不扩大匹配池，始终同区对抗），Then 是否仍成立？不成立，区内一旦固化无外部威胁，政治格局死亡，弱者无重新站队机会。

## 边界
这套公式在缺乏赛季重置机制或无强联盟对抗的SLG中不适用；在重绝对公平竞技、零长线养成沉淀的对战游戏中不适用。当游戏的核心乐趣不再依赖于“阶段性竞争重燃与0氪大R生态共生”时，上述因果链便会断裂。
