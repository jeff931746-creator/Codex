# 三国：冰河时代·武将养成底盘拆解

## 这个专题要回答什么

《三国：冰河时代》的武将养成为什么是跨赛季的底盘，而不是单赛季的消耗品？

## 总体判断

武将养成系统负责把玩家的时间和付费沉淀为可持续积累的底盘。它不直接决定单场战斗的胜负走向，但它决定了玩家在每个赛季能否稳定兑现联盟分配的战斗职责。通过引入觉醒和二代将，底盘从“一次性满级”变成了“持续追新”的阶梯。

## 怎么判断某个东西属于武将养成底盘

- 只要它的主要作用是“抬高武将的可用上限或长期属性（如升星提升基础攻防、觉醒解锁质变技能）”，它就属于底盘
- 如果它只改变当前赛季的地图目标或匹配规则，它属于节奏层
- 如果它只改变联盟的集体增益（如联军科技），它属于联盟协作层

## 核心结构

### 1. 乾坤寻将与三顾茅庐负责碎片获取

- 触发条件：玩家消耗元宝参与乾坤寻将或三顾茅庐活动
- 具体结果：获得指定武将的碎片或完整武将，例如S1的貂蝉、S2的郭嘉等核心将
- 设计原因：把抽卡行为和特定活动绑定，控制核心武将的产出节奏，避免玩家过早集齐全图鉴

### 2. 升星与觉醒把重复资源转成长期价值

- 触发条件：武将碎片积累达到升星门槛（如3星升4星需30碎片）或觉醒门槛（如S7觉醒郭嘉需专属觉醒碎片+本体满星）
- 具体结果：武将基础属性提升，觉醒武将解锁质变技能（如S7觉醒郭嘉的全体冰冻）或新形态
- 设计原因：把“抽到重复卡”的负反馈转成“重复也有用”的长期沉淀，拉长核心将的养成周期

### 3. 赛季更迭引入上位替代，制造养成缺口

- 触发条件：新赛季开启，解锁二代将或新觉醒将（如S2弓兵周瑜/刘备/荀彧，S4骑兵曹操/关羽/甘宁）
- 具体结果：旧赛季的主力阵容面临迭代压力，玩家必须追新将才能保持战力优势
- 设计原因：如果不引入新将，底盘满级后玩家失去追求；引入新将能重新激活养成需求

### 4. 兵种迭代强制重构阵容结构

- 触发条件：赛季演进带来特定兵种的崛起（如S2弓兵崛起，S3步兵深化，S4骑兵时代）
- 具体结果：玩家不能只堆一种兵种的武将，必须根据赛季环境补齐弓、步、骑的二代将
- 设计原因：用兵种克制和赛季环境强迫玩家拓宽养成面，而不是单线极限堆叠

## 这个模块不负责什么

- 不负责联军科技里对全盟攻防百分比的加成，那是联盟协作与构筑的职责
- 不负责7级主城多关卡怎么打，那是赛季战斗与检定的职责
- 不负责元宝从哪里来，那是资源与经济循环的职责

## 可迁移结论

- If 游戏在赛季更迭时引入上位替代武将（如S2弓兵周瑜替代S1步兵将），Then 玩家的养成重心会从全量收集转向当赛季定向追新，底盘沉淀从“永久绝对值”变为“赛季相对优势”
- If 养成系统存在兵种迭代环境（如S2弓兵崛起、S4骑兵时代），Then 玩家会被迫维持多兵种阵容的底线养成，而不是单一兵种的极限堆叠
