# 三国：冰河时代·赛季战斗与检定拆解

## 这个专题要回答什么

《三国：冰河时代》的7级主城攻坚战为什么是检定器，而不是单纯的战力墙？

## 总体判断

赛季战斗系统不只是看双方战力总和，更是检验联盟协作、兵种搭配和战略执行的综合检定。7级主城（洛阳/长安/官渡/襄阳）作为终极目标，强迫联盟暴露出是“单点高战碾压”还是“整体结构完整”。

## 怎么判断某个东西属于战斗与检定层

- 只要它会强迫玩家暴露阵容或联盟短板（如多梯队连战缺人、兵种被克制），就属于检定层
- 如果它只决定地图怎么铺、怎么走，属于节奏层
- 如果它只决定武将数值怎么算，属于底盘层

## 核心结构

### 1. 7级主城攻坚战检定联盟整体战力结构

- 触发条件：赛季进程推进到7级主城开放争夺的阶段
- 具体结果：7级主城（如洛阳）包含多道城门与内城，需多梯队连战与多线铺路，单支高战队伍无法同时防守多门或连续攻坚至内城
- 设计原因：把“谁更强”的判断从个体战力比拼转为集体结构检定

### 2. 跨服匹配把检定对象从区内转向区间

- 触发条件：赛季进阶（如S2跨四服，S4跨24服）
- 具体结果：玩家面对的不再是固定对手，而是更广池的跨服联盟
- 设计原因：防止区内势力固化导致检定失效，用外部压力重新激活竞争

### 3. 战场维度扩展检定战略适应力

- 触发条件：S8赛季引入水路进攻、港口建筑、天气系统
- 具体结果：陆地平推的固定解法失效，联盟必须分配兵力防守水路港口和适应天气Debuff
- 设计原因：如果战斗维度不变，老玩家会形成固定解法，新维度能打破套路

### 4. 攻城斗阵与兵机阁检定微操与配队

- 触发条件：S5引入攻城斗阵和兵机阁机制
- 具体结果：玩家在攻城时需要考虑斗阵的站位与技能释放，而非纯数值对撞
- 设计原因：在宏观SLG框架内嵌入局部策略点，提高战斗的操作上限

## 这个模块不负责什么

- 不负责武将怎么升星觉醒，那是武将养成底盘的职责
- 不负责决定匹配到哪些对手及赛季何时开服，那是赛季节奏框架的职责
- 不负责日常讨伐和种田产出，那是资源与经济循环的职责

## 可迁移结论

- If SLG的终极占领目标（如洛阳多关卡城门）需要多梯队连战与多线铺路才能攻克，Then 这个目标就会成为联盟整体结构的检定器，而非单一战力的排行榜
- If 战斗系统随赛季增加新维度（如S8水路/港口），Then 老玩家的陆地平推固定解法会被强制打破，策略深度得以重置
