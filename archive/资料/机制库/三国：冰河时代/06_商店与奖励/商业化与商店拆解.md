# 三国：冰河时代·商业化与商店拆解

## 这个专题要回答什么

《三国：冰河时代》的付费设计为什么能把0氪、小氪、大氪捆在同一套武将迭代序列里，而不是让三者各玩各的？

## 总体判断

这套商业化系统不是单纯的卖数值，而是围绕“武将迭代序列（貂蝉→郭嘉→太史慈→觉醒将等）”做分层榨取。0氪靠元宝追下限，小氪靠基金和礼包卡关键节点（如7级爵位吕布礼包），大氪靠高级招募直接拉满当赛季新将。三者共享同一个终极检定（7级主城），大氪需要0氪和微氪作为联盟基石，0氪需要大氪带飞拿奖励。

## 怎么判断某个东西属于商业化与商店

- 只要它主要决定“花现实货币或高级代币换取特定养成材料”，就属于商业化层
- 如果它只决定日常打怪掉什么，属于经济循环
- 如果它只改变联盟科技，属于联盟协作

## 核心结构

### 1. 元宝基金与爵位礼包卡小氪关键节点

- 触发条件：小氪玩家购买元宝基金，快速提升爵位至7级
- 具体结果：解锁购买吕布礼包和月卡等特权，获得性价比极高的核心武将和资源
- 设计原因：用明确的“7级爵位”门槛和“吕布礼包”奖励，给小氪一条清晰的付费攻略路线，避免漫无目的地小额付费

### 2. 高级招募供大R直接拉满当赛季新将

- 触发条件：大R玩家在赛季初使用高级招募抽取二代将或觉醒将
- 具体结果：迅速获得如S2郭嘉、S4曹操等当赛季统治级武将，建立战力优势
- 设计原因：大R的核心诉求是“早享受、早碾压”，高级招募满足其第一时间获取新将的需求

### 3. 乾坤寻将与三顾茅庐作为0氪与微氪的抽卡出口

- 触发条件：玩家消耗积攒的元宝参与活动
- 具体结果：获取指定核心武将（如貂蝉、郭嘉）的碎片
- 设计原因：把抽卡行为和特定活动绑定，让0氪的元宝消耗有明确的预期和保底，维持长期留存

### 4. 赛季庆典活动集中释放付费刺激

- 触发条件：新赛季开启或特定节日
- 具体结果：开放限定兑换、累充奖励和特殊礼包
- 设计原因：在玩家竞争欲最强的赛季初，用集中投放的付费刺激最大化营收

## 这个模块不负责什么

- 不负责武将觉醒后的具体技能表现，那是武将养成底盘的职责
- 不负责联盟跑商的资金流转，那是联盟协作与经济循环的职责
- 不负责7级主城的匹配规则，那是赛季节奏框架的职责

## 可迁移结论

- If SLG围绕“当赛季新武将”做分层付费设计（0氪肝乾坤寻将/微氪买7级爵位基金/大氪直接高级招募），Then 三类玩家会共享同一套养成语言和终极目标，付费差异体现在成型速度而非玩法隔离
- If 小氪路线存在明确的门槛节点（如7级爵位解锁核心礼包），Then 小氪的付费冲动会被集中引爆，而非分散在无数低性价比的小额选项中
