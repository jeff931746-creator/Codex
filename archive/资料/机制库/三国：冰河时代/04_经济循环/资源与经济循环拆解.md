# 三国：冰河时代·资源与经济循环拆解

## 这个专题要回答什么

《三国：冰河时代》的元宝经济为什么能维持0氪玩家的长期留存，而不是让0氪早早流失？

## 总体判断

这款游戏的经济循环核心在于“元宝的高频日常投放与关键消耗”。0氪玩家通过日常和讨伐稳定获取元宝，再用元宝参与乾坤寻将和三顾茅庐，获得了追下限的资格；大R则通过付费加速和高级招募拉开上限。两者共享同一套养成语言，0氪的存在为大R提供了竞争的沙包和联盟的基石。

## 怎么判断某个东西属于经济循环

- 只要它主要改变“拿到什么资源、花在哪、什么时候花”，就属于经济循环
- 如果它只改变战斗中的伤害结算，不属于经济循环
- 如果它能把日常时间转化为核心抽卡资源（如元宝），它就在经济循环里占有核心位置

## 核心结构

### 1. 日常与种田负责元宝的基础供给

- 触发条件：玩家完成日常活动、讨伐任务和种田收获
- 具体结果：稳定获取元宝和基础资源
- 设计原因：0氪的留存依赖于“只要上线就有核心资源拿”，元宝的日常投放是留存底线

### 2. 元宝的消耗优先级决定养成节奏

- 触发条件：玩家积累元宝后面临消耗选择
- 具体结果：0氪的优先级为“乾坤寻将+三顾茅庐 > 庆典活动 > 角色抽取 > 元宝商城 > 元宝加速”
- 设计原因：强制0氪做取舍，把元宝花在刀刃上（抽核心将），而非无脑加速，拉长养成周期

### 3. 万人跑商建立联盟级经济循环

- 触发条件：S6赛季跑商系统开启
- 具体结果：联盟成员通过跑商获取资金，资金转化为军费和联盟资源
- 设计原因：把个体经济行为汇聚成联盟经济循环，增加联盟成员间的经济依赖

### 4. 讨伐任务提供武将碎片与材料

- 触发条件：消耗体力或讨伐次数挑战NPC
- 具体结果：获取指定武将碎片、装备材料等
- 设计原因：在纯挂机种田外提供主动战斗获取资源的通道，满足不同活跃度玩家

## 这个模块不负责什么

- 不负责联军科技的加成效果，那是联盟协作的职责
- 不负责7级主城的战斗胜负，那是战斗与检定的职责
- 不负责赛季重置规则，那是节奏框架的职责

## 可迁移结论

- If 游戏将核心抽卡资源（元宝）作为日常高频产出投放给0氪，Then 0氪玩家会转化为大R体验内容的基石和沙包，而非纯粹的流失用户
- If 联盟级经济循环（跑商）的资金产出直接挂钩战争军费，Then 个体玩家的经济行为会被强制绑定到联盟战略中，脱盟成本极高
