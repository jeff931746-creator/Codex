# 向僵尸开炮·产品化结论

## 这一层的用途
这里的结论带适用前提与反例，用于产品化决策。当你在轻中度移动端 Roguelite 或类似压力框架的品类中做系统规划、数值定标或商业化设计时，可直接引用这些判断；但若脱离了特定的用户层与变现形态，结论可能失效，请务必核对适用前提与反例。

## 结论列表

**结论 1：可逆压力框架维持动力**

If 压力信号设计为可逆的防线推进距离 Then 玩家在压力高峰期的持续游玩动力的衰减速度比不可逆的血条扣除慢 30%[推测]。

*适用前提*：移动端轻度射击肉鸽，玩家有明确的视觉反馈和逆转手段，单局时长 5-10 分钟[推测]。
*反例游戏*：《明日方舟》（塔防），塔防的基地血量扣除是不可逆的，因为防守失败是硬检定，可逆压力会导致策略严肃性与惩罚感丧失。
*太宽测试*：若替换为 MOBA 类型，结论不成立（MOBA的防御塔/水晶血量必须是不可逆的，否则对局无法产生击穿防线的终结感）。

**结论 2：单防线认知减负**

If 波次守线游戏只设置 1 条防守目标 Then 轻度用户的首日留存率比设置 3 条防线高出 15%[推测]。

*适用前提*：IAA+IAP 变现的轻中度用户层，局内操作以射击清怪为主，玩家认知带宽有限，无法兼顾多线操作。
*反例游戏*：《部落冲突》（策略经营），多目标（多防线与分区）是策略深度的核心，单防线会彻底摧毁其多兵种配合的策略空间。
*太宽测试*：若替换为 卡牌对战 类型，结论不成立（卡牌对战需要多维度场面交互与随从交换，单一目标会退化成纯打脸的比大小游戏）。

**结论 3：失败资源转化率底线**

If Roguelite 单局失败后，局外永久成长的资源获取量能达到通关奖励的 50%-70%[推测] Then 玩家在第 3-5 天[推测]的卡关节点流失率会显著降低。

*适用前提*：依靠局外数值成长驱动长线留存的 IAP 变现产品，玩家失败并非因为操作上限，而是数值下限不足。
*反例游戏*：《哈迪斯》（单机买断Rogue），纯局内构筑驱动，失败给的是剧情碎片和局内天赋，不需要也不应该给高额局外数值补偿，否则会破坏单局成长闭环与重复游玩动力。
*太宽测试*：若替换为 单机RPG 类型，结论不成立（单机RPG失败通常读档重打，不存在局外累积补偿逻辑，失败即中断）。

**结论 4：短局流派收敛节点**

If 单局时长 5-10 分钟[推测]的 Roguelite 在前 3 次[推测]技能选择中不提供路线件（方向性技能） Then 玩家有 80%[推测]的概率将后续选择退化为纯数值比大小，构筑感消失。

*适用前提*：局内技能三选一的构筑系统，且目标用户为缺乏长线规划能力的轻度碎片化玩家。
*反例游戏*：《杀戮尖塔》（卡牌肉鸽），其单局时长超 30 分钟[推测]，流派可以在第一层Boss后才完全收敛，不需要前3次选择定调。
*太宽测试*：若替换为 开放世界 类型，结论不成立（开放世界没有短局收敛压力，流派可以随探索慢慢展开，过早收敛反而限制体验）。

**结论 5：轻度Boss数值检定原则**

If 面向轻度用户的 Boss 战设计为纯 DPS 强度检定 Then 通关率比设计为特定机制检定的 Boss 高出 40%[推测]，且不会引发大规模卡关流失。

*适用前提*：射击肉鸽品类，核心爽感来自清怪割草和数值压制，用户不愿承担学习复杂机制的成本。
*反例游戏*：《魔兽世界》（MMORPG），其Boss必须包含严苛的机制检定（如走位、转火、驱散），纯数值检定会导致玩法极度无聊并快速流失核心团本用户。
*太宽测试*：若替换为 MOBA 类型，结论不成立（MOBA的史诗级中立生物需要机制拉扯与视野博弈，纯数值站桩会变成比拼装备数值的垃圾时间）。

**结论 6：执行层与路线层分离控制门槛**

If 射击游戏将角色/武器的执行层（弹道形态）与技能的路线层（流派方向）完全解耦 Then 新手期的有效流派发现率可提升 2 倍[推测]，因为玩家无需在选角时预判全局构筑。

*适用前提*：拥有多角色/多武器系统，且局内构筑随机性较高的产品，目标用户为缺乏游戏经验的泛用户。
*反例游戏*：《云顶之弈》（自走棋），其棋子既是执行层也是路线层（羁绊），分离会摧毁其核心组队与经济运营逻辑。
*太宽测试*：若替换为 卡牌对战 类型，结论不成立（卡牌对战的核心就是用单卡同时定义执行效果和路线体系，分离会导致卡牌身份割裂与组合技消失）。

## 局限性
1. **核心向硬核动作肉鸽**：如《仁王》《只狼》，玩家追求高难度动作反馈与机制克服，不需要失败补偿和轻度化改造，结论 1、3、5 的方向完全相反。
2. **纯 PVP 竞技游戏**：如《王者荣耀》《CS:GO》，不需要 PVE 波次压力框架和 Boss 检定，对局压力完全由对手提供，结论 1、2、5 不适用。
3. **重叙事驱动的长线 RPG**：如《原神》《崩坏：星穹铁道》，其核心体验在于剧情沉浸与角色塑造，不需要短局快节奏收敛，结论 4 不适用。
