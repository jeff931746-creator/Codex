# 养成结构拆解（局外成长 + 变现）

> 模块：局外成长 + 变现结构  
> 数据状态：公测第 9 天，部分细节待验证

---

## 局外成长体系

### 全员共享养成（最重要的设计决策）

**触发条件**：玩家获得任意新角色。  
**结果**：新角色**直接以当前全局等级上场**，无需从 1 级重新培养。  
**原因**：打破了传统手游"出新角色→养成成本→选择性培养"的循环，降低了抽到新角色的时间成本。  
**边界**：是否有角色专属技能/被动需要单独解锁待验证；全局等级的成长上限待确认。

**If-Then**：
- If 全员共享等级，Then 抽到新角色的即时满足感更强（不用等养成）。
- If 无需为每个角色单独养成，Then 玩家愿意尝试更多角色组合，构筑多样性提升。
- 但 If 养成成本过低，Then 付费动机减弱——这是该设计的取舍代价（用 ARPU 换留存和口碑）。

---

### 离线挂机资源积累

**触发条件**：玩家离开游戏后（甚至不开游戏时）。  
**结果**：游戏自动积累部分资源。  
**功能**：确保轻度玩家不因缺少游戏时间而感到落后，维持其长期留存意愿。  
**边界**：挂机资源的种类（是否包括抽卡货币）和积累速率待验证。

---

### 肉鸽随机 Buff（局内成长）

**触发条件**：进入无限游戏厅模式，每隔一段时间出现 Buff 选择界面。  
**结果**：本局获得随机增益，不带出局外。  
**边界**：Buff 是否与局外角色技能联动（放大已有技能 vs 独立 Buff 池）待验证。

## 变现结构

### 抽卡体系

| 指标 | 乱涂彩世界 | 行业标准（参考） |
|---|---|---|
| 单次保底 | 50 抽 | 80-100 抽 |
| 歪卡机制 | 无（"指定不歪"） | 有（50% 歪率后再保底） |
| 新手福利 | 首周 400 抽 + 首月 1000 抽 | 首月约 50-100 抽 |
| 开服选角 | 五选一 SSR | 无或一选一 |
| 联动角色 | 洛天依（签到免费） | 联动角色通常付费或难获取 |

**设计意图**：明确以"低门槛获角色"作为竞争优势，面向不愿深度氪金的玩家群体。  
**代价**：首月放出大量角色后，后续如何维持付费动机是核心挑战。

---

### 付费层次

根据已有信息推断（部分待验证）：

| 层次 | 内容 | 面向玩家 |
|---|---|---|
| 免费层 | 首月 1000 抽、签到福利、日常任务货币 | 零氪/微氪 |
| 基础付费 | 月卡（待确认）、成长礼包 | 轻氪 |
| 进阶付费 | 限时礼包、新角色池专属道具 | 中氪 |
| 大额付费 | 限定角色皮肤、高价礼包 | 重氪 |

> 注：具体价格和月卡金额未找到公开信息，待补充。

---

### 变现结构评价

**优势**：
1. 低保底 + 不歪 = 降低了"运气焦虑"，玩家体验更舒适
2. 首月大量免费抽 = 快速建立角色池，短期内玩家满意度高
3. 联动角色免费 = 降低联动内容的付费门槛，提升参与度

**风险**：
1. 首月超大福利后，后续付费钩子（留住花钱的理由）尚未验证
2. "不氪也能玩"的定位在大 DAU 场景下可持续，但依赖高用户量摊薄成本
3. 抽卡货币获取难度偏高（评测指出），实际上是用"低保底"弥补"难获取"的体验

---

## If-Then 总结

- If 全员共享养成降低培养成本，Then 口碑和留存更好，但单用户 LTV 下降；适合用量取胜的腾讯流量打法。
- If 首月 1000 抽建立了很高的免费预期，Then 后续版本若减少免费抽，会产生明显的期望落差。
- If 挂机资源包含抽卡货币，Then 离线玩家的留存动机增强；If 不包含，Then 离线功能更多是"生活质量"而非留存钩子。
