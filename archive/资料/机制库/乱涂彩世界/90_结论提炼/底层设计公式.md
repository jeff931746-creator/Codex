# 底层设计公式 + 竞品分析

> 分析目的：借鉴可迁移的设计原则 + 竞品定位对比  
> 生成日期：2026-05-07

---

## 一、可迁移设计公式

### 公式 1：副游心理契约公式

**问题**：玩家为什么愿意在已有主游的情况下打开一个新游戏？  
**乱涂彩世界的回答**：极限压缩时间承诺，消除"你需要投入的焦虑"。

**公式**：
```
副游定位 = 固定短局时（90秒）+ 完整情绪曲线 + 离线自动进度 + 明确可停顿
```

**If-Then**：
- If 局时固定且短，Then 玩家开启成本极低（"就玩一局"）。
- If 情绪曲线完整（积累→爆发→结算），Then 短局不等于空洞，玩家有满足感。
- If 离线自动积累，Then 没时间玩的玩家仍有"登录价值"（不落后的安全感）。
- If 可随时停且不惩罚，Then 副游不与主游竞争时间，而是填补碎片时间。

**适用条件**：适合非独占市场定位的游戏，与已有大型 IP 竞争时的差异化打法。  
**失效条件**：如果局时固定但情绪曲线不完整（如只有积累，没有爆发高潮），效果失效。

---

### 公式 2：操作解耦 + 策略保留

**问题**：如何同时覆盖休闲玩家（不想思考）和策略玩家（想思考）？  
**乱涂彩世界的回答**：将操作层自动化，将策略层保留在局外构筑。

**公式**：
```
低门槛感知 = 自动瞄准（操作负担为零）
策略出口 = 色系搭配 + 职业组合（局外决策，有深度）
情绪主控权 = 走位躲避（玩家主观感受"我在操控"）
```

**If-Then**：
- If 核心操作自动化，Then 休闲玩家可以上手，降低流失率。
- If 策略层保留在局外，Then 有意愿的玩家有深度探索空间，不会感到"无脑"。
- If 保留玩家走位的自主控制，Then 玩家不会觉得"自己只是看戏"，保留了主体感。

**适用条件**：弹幕射击类游戏，想覆盖休闲玩家但不想完全去除策略空间时。  
**失效条件**：如果策略层影响不够显著（通关不需要优化搭配），玩家不会探索构筑，策略层变成装饰。

---

### 公式 3：低保底不歪 = 用抽卡体验换 LTV

**问题**：如何在竞争激烈的市场中快速建立口碑？  
**乱涂彩世界的回答**：用低于行业标准的抽卡保底（50 抽 vs 100 抽）+ 指定不歪，换玩家满意度和传播。

**公式**：
```
口碑杠杆 = 超出预期的保底低 + 首月大量免费抽
代价 = 单用户 LTV 下降
补偿 = 高 DAU × 低 ARPU 仍可盈利（腾讯流量打法）
```

**If-Then**：
- If 保底低且不歪，Then 玩家"运气焦虑"消失，抽卡体验变为正向，愿意口口相传。
- If 首月大量免费抽，Then 初期角色池建立速度快，留存更好。
- If 主要靠量而非高 ARPU，Then 此策略仅适合拥有高流量渠道的发行商（腾讯适用，独立游戏不适用）。

**失效条件**：中小团队无法承受低 ARPU；首月高福利后续若收紧，会产生期望落差。

---

### 公式 4：染色作为进度可视化手段

**问题**：如何让玩家在 90 秒内感受到明确的进度感？  
**乱涂彩世界的回答**：颜色积累（印记）→ 颜色反应（爆发）是视觉上最直接的进度感表达。

**公式**：
```
进度可视化 = 颜色印记叠加（颜色变化是视觉反馈）+ 乱涂等级进度条
爆发可视化 = 无双模式全屏颜色爆发（情绪高点的视觉对应）
```

**If-Then**：
- If 进度通过颜色变化体现，Then 玩家不需要看数字也能感知"我在变强"。
- If 爆发时全屏视觉效果强烈，Then 情绪高潮点的记忆强度高，愿意再来一局。

**适用条件**：任何需要在短局时间内塞入完整进度感的游戏。

## 二、竞品定位对比

### 同类竞品矩阵

| 游戏 | 局时 | 操作复杂度 | 构筑深度 | 核心差异化 | 定位 |
|---|---|---|---|---|---|
| 乱涂！彩世界 | 90 秒 | 极低（单指走位） | 低-中 | 副副副游定位 + 染色视觉 | 极轻度休闲副游 |
| 弹壳特攻队 | 3-5 分钟 | 中（双摇杆） | 中 | 武器多样性 + 暴力爽快 | 轻度主游 / 副游 |
| 吸血鬼爬行者 | 15-20 分钟 | 低（单摇杆） | 高 | Synergy 深度 + 局内构筑 | 中度主游 |
| 向僵尸开炮 | 5-10 分钟 | 低（走位为主） | 中-高 | 波次压力 + 武器多元化 | 中度副游 |

### 乱涂彩世界的核心差异化

1. **唯一的副副副游定位**：明确声明"不是主游，甚至不是副游"，是赛道差异化而非功能差异化。
2. **90 秒最短局时**：在同类游戏中局时最短，碎片化时间利用率最高。
3. **染色视觉系统**：颜色印记和反应是功能与美学统一的机制设计（同时是视觉表达和伤害系统）。
4. **全员共享养成**：降低了角色养成的时间摩擦，是对传统手游养成模式的逆向选择。

### 乱涂彩世界的主要劣势

1. **玩法深度不足**：选择副游定位意味着主动放弃深度，但深度不足意味着核心玩家 2-4 周后流失。
2. **长期重复感强**：90 秒固定节奏 × 同质化关卡 = 内容消耗速度快。
3. **留存钩子不明确**：5 分钟清完日常后，玩家继续游玩的理由是什么？目前主要靠角色收集和联动活动。
4. **社交层薄弱**：3v3 联机是否能真正形成社交网络尚未验证；副游玩家的社交意愿天然低。

## 三、对立项的启示

### 可借鉴的设计决策

1. **副游心理契约先行**：如果打算做副游定位，必须在设计之初就把"时间承诺极限压缩"写进核心循环，而不是上线后再调整节奏。

2. **视觉与机制统一**：染色/涂色既是视觉表达（画风 identity）也是核心机制（印记反应），这种统一降低了向玩家解释机制的成本——眼睛看到的就是在发生的。

3. **反传统养成作为差异化**：全员共享养成打破了"新角色必须重新养"的惯性认知，是一个值得在立项阶段主动考虑的反向选择。

4. **抽卡体验可以成为核心卖点**：不氪也能玩 + 低保底不歪，将"公平感"作为产品卖点，与其他游戏形成直接对比。

### 需要规避的问题

1. **副游内容深度是长期留存的隐患**：短局时 + 低门槛 = 短期满意度高，但如果肉鸽模式和构筑深度没有支撑，核心玩家 1 个月内流失。

2. **首月高福利是双刃剑**：建立了过高的期望基准，后续版本维持这一标准的成本很高。

3. **不要用"副游定位"逃避内容深度问题**：说"我是副游所以不需要深度"是用定位掩盖设计不足。真正的副游仍然需要有足够的深度留住那 20% 会变成核心玩家的用户。
