# 色系职业搭配拆解

> 模块：构筑层  
> 数据状态：公测第 9 天，部分关系待验证

## 职业分类

已确认的角色定位分为 4 个职业类型（游戏内称"神明"）：

| 职业 | 功能 |
|---|---|
| 炮击 | 远程范围输出 |
| 近战 | 高频/近距离输出（如天照：扇形攻击） |
| 射击 | 远程单体或直线输出 |
| 魔法 | 特殊效果（控制/buff/debuff） |

**角色角色定位分层**（从攻略来源归纳）：

| 定位 | 代表角色 | 功能说明 |
|---|---|---|
| 核心输出 | 天照 | 高频近战，扇形范围，被动激活后基础伤害+100% |
| 破防 | 菠萝、潘潘 | 降低敌人防御，为输出角色打开伤害窗口 |
| 加速/充能 | 托莉 | 提升全队攻速或技能充能效率 |
| 持续治疗 | 狄安娜 | 高频群体回血，适合长局或密集弹幕环境 |
| 辅助治疗+复活 | 丘比特 | 低频但有复活功能，保底容错 |
| 护盾+减益反制 | 伊邪那美 | 提供护盾并反弹/减弱敌方减益效果 |
| 聚怪/控制 | 月读 | 控制敌方走位，辅助全队集火 |
| 控场 | 丹塔莉 | 减速或限制敌方行动 |
| 生存 | 希尔米 | 提升全队生存能力 |

## 色系机制

### 已确认色系

| 色系 | 元素效果 | 搭配价值 |
|---|---|---|
| 红色 | 灼烧（持续伤害） | 与其他色系叠加触发颜色反应 |
| 黄色 | 感电（可能附带麻痹效果）[待验证] | 黄色角色多为功能性（破防/加速），感电可能影响目标行动 |
| 蓝色 | 寒冷（减速）[待验证] | 减速使敌人更容易被集火 |

> 其他色系（紫色、绿色等）的元素效果待补充。

### 颜色搭配规则

**核心约束**：队伍搭配需要覆盖至少 2 种色系，才能持续触发颜色反应爆发伤害。  
**隐性设计**：游戏不直接显示"你需要两种颜色"，但颜色反应是最大伤害来源，玩家通过攻略或试错发现这一规律。

## 推荐阵容分析

### 阵容 1：高爆发速攻（托莉 + 天照 + 伊邪那美）

**运作链路**：加速（托莉）→ 持续输出（天照）→ 护盾减伤 + 承伤反制（伊邪那美）  
**适合场景**：PVE 高难副本，需要快速清怪  
**构筑逻辑**：托莉的充能加速让天照的大招释放频率翻倍，伊邪那美提供生存底板

**颜色覆盖**：待验证（需要确认三者色系是否覆盖 2 种以上）

---

### 阵容 2：新手均衡过渡（月读 + 丘比特 + 狄安娜）

**运作链路**：聚怪（月读）→ 辅助治疗/复活（丘比特）→ 持续群体治疗（狄安娜）  
**适合场景**：初期资源积累阶段  
**构筑逻辑**：月读聚怪让天然分散的敌人集火变得容易；双奶组合极大降低被歼灭风险  
**代价**：输出密度低，清怪速度慢，适合容错优先而非速度优先

---

### 阵容 3：控制流生存（月读 + 丹塔莉 + 希尔米）

**运作链路**：控场（月读+丹塔莉）→ 生存（希尔米）  
**适合场景**：资源积累期或高难度卡关时  
**代价**：输出不足，Boss 战难以在有限时间内击杀

---

## 构筑系统评价

### 深度分析

**表面深度**：角色分工清晰，职业和色系双维度搭配，初看有一定策略空间。  
**实际深度**：由于操作层自动化（自动瞄准），构筑决策是唯一的策略出口，但受限于以下因素：
1. 关卡难度梯度低（副游定位），大多数内容不需要精确搭配也能通过
2. 构筑调整窗口有限（有评测指出自动进入下关后无法调整阵容）
3. 色系机制不够显性，新手不易自主发现颜色反应规律

**与同类游戏对比**：

| 游戏 | 构筑深度 | 构筑可见性 |
|---|---|---|
| 乱涂彩世界 | 低-中 | 低（颜色反应需要自行发现） |
| 弹壳特攻队 | 中 | 高（武器 DPS 数字明确） |
| 吸血鬼爬行者 | 高 | 中（synergy 需要尝试发现） |
| 小丑牌 | 极高 | 中（需要深度理解牌组逻辑） |

## If-Then 结论

- If 队伍缺少破防角色（菠萝/潘潘），Then 高防御敌人和 Boss 的伤害效率大幅下降。
- If 队伍只有单一色系，Then 颜色反应无法触发，整体伤害上限受限。
- If 搭配充能加速角色（托莉），Then 无双爆发期内技能频率翻倍，爆发窗口价值最大化。
- If 关卡难度提升，Then 治疗 + 输出的平衡权衡开始有意义；否则全输出阵容即可。
