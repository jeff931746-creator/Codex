# 战斗_染色射击与无双爆发

> 本文是 `01_系统结构/` 两份详细拆解的模块摘要，方便跨游戏对比检索

---

## 核心战斗层结构

```
自动瞄准射击（玩家仅走位）
  ↓
染色印记叠加（攻击附带色系属性）
  ↓
颜色反应触发（2+ 色系印记叠加 → 巨额伤害 + 降抗性）
  ↓
乱涂等级积累 Lv1→5（持续走位攻击驱动）
  ↓
Lv5 触发无双爆发（攻速/充能大幅提升，清屏）
```

---

## 模块边界

**战斗_染色射击与无双爆发** 覆盖：
- 基础射击自动化设计
- 印记叠加 / 颜色反应 / 着色条机制
- 乱涂等级积累规则
- 无双爆发触发条件和效果

**不覆盖**：
- 角色职业分工和搭配决策 → `构筑_色系职业搭配.md`
- 关卡节奏和时长设计 → `05_关卡节奏/`

---

## If-Then 核心结论（战斗层）

- If 自动瞄准，Then 操作门槛降为走位，适合极度休闲场景；策略差距转移到局外阵容构筑。
- If 颜色反应是主要爆发来源，Then 构筑决策的核心约束是"覆盖 ≥2 种色系"。
- If 乱涂等级满时触发无双，Then 90 秒局内有清晰的情绪高潮节点，短局体验不空洞。
- If 玩家停止移动，Then 乱涂值积累停止，鼓励主动走位而非站桩——操作中保留了最小技巧空间。
