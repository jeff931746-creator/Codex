# 关卡模式分工拆解

> 模块：关卡节奏  
> 数据状态：结构基本确认，部分模式细节待验证

---

## 已知游戏模式

| 模式 | 类型 | 主要机制 | 功能定位 |
|---|---|---|---|
| 主线推图 | PVE 单人 | 逐关清怪，线性推进 | 主要关卡进度，资源产出 |
| 乱涂挑战 | PVE 单人 | 快速升级共鸣等级（待验证具体规则） | 角色/共鸣等级提升 |
| 颜料保卫战 | PVE 防守 | 塔防玩法，保护目标不被敌人突破 | 玩法多样性，可能有专属资源 |

## 模式分工逻辑

### 核心功能轴

```
主要资源产出        → 主线推图 + 乱涂挑战
挑战/成就感出口     → 无限游戏厅（肉鸽随机）
玩法变化（防守）    → 颜料保卫战
社交/竞技出口       → 喷溅对决（3v3）
长期留存目标        → 环球冒险
```

### 副游模式设计的关键取舍

**正确的地方**：
- 多模式组合覆盖不同玩家需求（速刷党/肉鸽玩家/社交玩家/探索玩家）
- 每种模式时间成本独立，不强制全部完成
- 挂机/离线资源积累覆盖无时间玩家

**存在的问题**：
- 模式间的差异化不够显著——大多数模式仍是"90 秒走位射击"，只是场景目标不同
- 颜料保卫战的塔防玩法是最明显的玩法差异，但深度待验证
- 无限游戏厅（肉鸽随机 Buff）是唯一提供横向构筑变化的模式，但与局外养成是否有交集待确认

---

## 肉鸽模式（无限游戏厅）

**触发条件**：进入无限游戏厅模式  
**机制**：每隔一定间隔提供随机 Buff 选择（局内获取，本局有效）  
**与主线的差异**：提供了局内构筑决策空间，打破了主线推图的机械重复感  
**边界**：Buff 池深度和选择策略复杂度待验证；与局外养成强度的交互待确认

**If-Then**：If 肉鸽 Buff 池足够深，Then 无限游戏厅是高留存玩家的主要内容出口；If Buff 池浅或选择同质化，Then 重复感问题同样存在。

---

## 3v3 联机模式（喷溅对决）

**机制**：3 名玩家分组，通过爆杀 Boss 和地图染色进行伤害 PK  
**功能**：引入社交层，给玩家提供展示养成成果的舞台  
**设计意图**：副游通常缺乏社交钩子，3v3 模式是主要补充手段  
**边界**：是否有排名/积分系统待验证；是否有专属资源奖励待验证

---

## If-Then 总结

- If 副游的日常只需 5 分钟，Then 模式多样性是留住非核心玩家的关键——他们不会重复主线，需要新鲜的"第二关卡"。
- If 肉鸽模式构筑深度足够，Then 核心玩家的留存锚点存在；If 不够，Then 核心玩家 2 周后流失。
- If 3v3 没有专属奖励和排名压力，Then 大多数副游玩家不会主动参与——联机模式对副游受众的吸引力天然有限。
