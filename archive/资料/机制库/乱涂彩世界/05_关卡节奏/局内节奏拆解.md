# 局内节奏拆解（90 秒闭环）

> 模块：关卡节奏  
> 数据状态：基本确认

---

## 单局时间结构

**单局时长**：90 秒（固定）  
**日常耗时**：约 5 分钟（清完日常内容）  
**定位意图**："九十秒涂一局，五分钟清日常"——极限降低时间承诺

---

## 局内节奏分段

根据乱涂等级机制，一局自然分为三段：

```
[0-30s] 积累段
  - 乱涂等级 Lv1→3
  - 稳定走位，印记叠加
  - BUFF 逐渐积累，颜色反应频率开始提升

[30-70s] 爬坡段
  - 乱涂等级 Lv3→5
  - 颜色反应触发频率较高
  - Boss 的着色条开始快速积累
  - 玩家进入"等待爆发"心理状态

[70-90s] 爆发段（无双触发后）
  - 乱涂等级 Lv5，无双模式激活
  - 攻速/充能效率大幅提升
  - 清屏效果，情绪高潮
  - 局结算
```

> 注：时间分段为推算值，实际阶段长度受玩家走位效率影响。

---

## 节奏设计的结构逻辑

### 1. 固定时长的价值

90 秒固定时长是"副游心理契约"的核心：  
- 玩家开局前知道 90 秒后可以停，不存在"再玩一局"的开放式时间陷阱  
- 但情绪曲线（积累→爬坡→爆发）在固定时间内完整，不会因为时间短而感到浅薄  
- **If-Then**：If 局时固定且短，Then 玩家更容易启动一局（降低开始成本）；但 If 每局体验同质化，Then 长期重复感强（这是该游戏最主要的批评点）。

### 2. 乱涂等级作为节奏锚点

没有乱涂等级的话，90 秒是平铺的——玩家不知道自己在哪个阶段。乱涂等级的作用：
- 给玩家一个可视化的进度感（进度条）
- 将单调的自动射击拆分成"前中后期"有意义的体验段
- 提供情绪管理的手段：玩家在 Lv4 时已经开始期待 Lv5 爆发

### 3. 主要问题点

**同质化**：多数关卡套路相同（走位→积累→爆发），缺少节奏变化和意外事件。  
**自动切关**：通关后自动进入下一关，无法在关间调整阵容——评测指出这是体验不流畅的来源。  
**深度不足**：对于主动玩家，90 秒内的走位决策质量影响有限，体验与挂机差距不大。

---

## If-Then 总结

- If 副游定位需要最低时间承诺，Then 固定短局时是正确选择；但需要在局内塞入完整情绪曲线，否则体验空洞。
- If 节奏全靠等级积累驱动，Then 不同关卡会趋向同质化；需要特殊关卡/Boss 战破节奏。
- If 清日常只需 5 分钟，Then 玩家留存钩子必须来自局外（角色养成/收藏/社交），不能靠局内体验本身。
