# 无双模式触发机制

> 模块：执行层  
> 数据状态：基本确认

---

## 机制描述

### 乱涂等级系统

**触发条件**：玩家控制角色持续向任意方向移动并攻击，命中敌人后积累"乱涂值"（显示为进度条）。  
**等级划分**：Lv1 → Lv2 → Lv3 → Lv4 → Lv5（满级）。  
**中间收益**：每提升一级，角色获得一定程度的属性 BUFF（具体数值待验证）。  
**结果**：等级越高，稳定输出阶段的强度越强；Lv5 时触发无双爆发。  
**边界**：乱涂值是否会衰减（停止攻击后是否下降）待验证；90 秒局内是否能多次触发待验证。

---

### 无双爆发（乱涂爆发模式）

**触发条件**：乱涂等级达到 Lv5（进度条满）。  
**效果**：
- 全队攻速大幅提升
- 充能效率大幅提升（技能释放频率加快）
- 进入"清屏"状态，适合集中爆发清怪

**持续时间**：待验证（是否有固定时长或持续到局结束）。  
**重置规则**：爆发结束后是否重置为 Lv1 待验证。

---

## 结构功能分析

### 乱涂等级的设计意图

1. **正反馈循环**：玩家越积极攻击，等级越高，BUFF 越强，正向激励不停手。
2. **局内节奏分段**：90 秒内自然形成"积累阶段 → 爆发阶段"的节奏，解决了无明显高潮点的问题。
3. **视觉成就感**：Lv5 触发时的全屏效果和清屏感是主要情绪释放点。

### 与其他副游爆发机制的对比

| 游戏 | 爆发积累方式 | 爆发表现 |
|---|---|---|
| 乱涂彩世界 | 持续攻击积累乱涂值 | 全屏清怪 + 攻速/充能提升 |
| 弹壳特攻队 | 击杀积累超级武器 | 召唤大型攻击 |
| 向僵尸开炮 | 技能冷却自动回转 | 技能连发爆发段 |
| 吸血鬼爬行者 | 经验升级选技能 | 技能组合产生 synergy |

乱涂彩世界的差异点：爆发是玩家**主动积累行为的结果**，而非时间 / 击杀数自动触发，给玩家制造了"我操控了这次爆发"的掌控感。

---

## If-Then 结论

- If 玩家持续走位攻击不停，Then 乱涂值积累速度最快，Lv5 触发更频繁。
- If 队伍搭配了充能加速角色（如托莉），Then 无双爆发期内技能释放量翻倍，爆发窗口价值最大化。
- If 局内能触发多次无双，Then 90 秒内的伤害曲线呈现多个波峰，整体感受为"解压爽快"。
- If 乱涂值有衰减机制，Then 被动挂机玩法的输出效率会明显低于主动走位玩法（等待验证后更新）。
