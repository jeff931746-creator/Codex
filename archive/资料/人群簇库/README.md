## 添加新人群簇

按 [填库工作流](../../../reference/部门标准/立项/三位一体框架/05-填库工作流.md) 操作：

1. 主 agent 先 plan，说明拟新增的人群簇
2. 用户审批后，按 [人群簇模板](../../../reference/部门标准/立项/三位一体框架/模板/人群簇模板.md) 填写草稿
3. 通过自检和交叉引用检查
4. 分配 ID 并归档
5. 更新 [人群索引](人群索引.md)
