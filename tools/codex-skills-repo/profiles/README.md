# Profiles

Profiles map the shared governance assets in this repo to specific runtimes or IDEs.

Current intent:

- `cursor/`: Cursor-facing structure such as `.cursor/rules` and related mappings
- `claude/`: Claude Code or Claude desktop facing structure such as `.claude/commands`, `.claude/agents`, or other runtime-specific projections

Profiles should describe how shared `rules/`, `skills/`, and `references/` are exposed in each tool, rather than duplicating the content.
