# 买量组合评估 Skill

## 目的

对 **题材 × 画风** 组合进行系统性评估，生成完整的买量效果分析报告。

## 输入

- **题材名称**：如"暗黑童话"
- **画风名称**：如"第五人格画风"
- **可选**：参考游戏/案例

## 输出

在 `/Users/mt/Documents/Codex/research/资料/买量组合库/组合案例库/` 下生成：
- `{题材名}×{画风名}.md`：完整评估报告

## 工作流程

### 1. 准备阶段

- 读取 `/Users/mt/Documents/Codex/research/资料/买量组合库/评估模板/组合评估模板.md`
- 检查题材库和画风库中是否已有基础信息
  - 如有，读取作为背景
  - 如无，在评估过程中补充基础信息

### 2. 市场调研阶段

使用 WebSearch 收集以下信息：

**题材维度**：
- 该题材在游戏市场的热度和趋势
- 主流受众画像和情感诉求
- 跨媒体表现（影视/小说/动漫）

**画风维度**：
- 该画风的代表作品
- 视觉特征和情绪传达能力
- 制作门槛和团队要求

**组合维度**：
- 搜索类似组合的成功/失败案例
- 分析竞品的素材策略和市场表现
- 识别市场空白和创新空间

### 3. 评估阶段

按照组合评估模板的维度逐项分析：

#### 受众契合度 (25%)
- 题材的情感诉求 × 画风的情绪传达 = 是否产生共鸣
- 目标受众的审美偏好是否匹配
- 文化认知门槛是否合理

#### 素材表现力 (30%)
- **视觉冲击力**：首帧能否吸引停留
- **情感钩子**：3秒内能否传达核心卖点
- **制作效率**：单条素材的制作周期
- **迭代空间**：能否持续产出差异化素材

#### 竞品密度 (20%)
- 列出成功案例（产品名、市场表现、素材特点）
- 列出失败案例（失败原因、踩坑点）
- 判断红海/蓝海状态
- 识别差异化空间

#### 长线潜力 (15%)
- 素材迭代空间（节日/事件/剧情素材可行性）
- IP 延展性（续作/衍生品潜力）
- 社区粘性（UGC 和玩家文化）

#### 团队适配度 (10%)
- 20人团队能否驾驭该画风
- 美术/技术门槛评估
- 内容生产成本预估

### 4. 评分阶段

- 对每个维度给出 1-10 分评分
- 计算加权总分
- 给出推荐指数 (1-5星)

### 5. 输出阶段

- 按模板生成完整报告
- 写入 `组合案例库/{题材名}×{画风名}.md`
- 更新 `组合案例库/组合索引.md`

## 评分标准

### 受众契合度
- 9-10分：题材情感与画风情绪高度契合，目标受众明确且规模大
- 7-8分：契合度良好，受众清晰
- 5-6分：契合度一般，存在部分冲突
- 3-4分：契合度较差，受众模糊
- 1-2分：严重不匹配

### 素材表现力
- 9-10分：视觉冲击力强，情感钩子明确，制作效率高，迭代空间大
- 7-8分：表现力良好，具备买量优势
- 5-6分：表现力一般，需要优化
- 3-4分：表现力较弱，买量风险高
- 1-2分：不适合买量

### 竞品密度
- 9-10分：蓝海市场，成功案例少但验证可行
- 7-8分：竞争适中，有差异化空间
- 5-6分：红海市场，需要强差异化
- 3-4分：严重红海，难以突围
- 1-2分：市场饱和，不建议进入

### 长线潜力
- 9-10分：素材迭代空间大，IP 延展性强
- 7-8分：长线潜力良好
- 5-6分：长线潜力一般
- 3-4分：长线潜力较弱
- 1-2分：只适合短期买量

### 团队适配度
- 9-10分：20人团队完全可驾驭，成本可控
- 7-8分：适配度良好，有一定挑战
- 5-6分：需要补强团队或外包
- 3-4分：门槛较高，风险较大
- 1-2分：不适合小团队

## 注意事项

- 如果是全新组合（无市场案例），明确标注"待验证"状态
- 买量数据（CPI、ROI）如果无法获取，使用"预估"并说明依据
- 政策风险必须明确标注
- 评估结论必须基于市场数据，避免主观臆断
