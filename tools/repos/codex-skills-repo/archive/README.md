# Archive

Retired or superseded skill copies are stored here for traceability.

## Policy

- Keep runtime backups here instead of under the active runtime directory.
- Prefer moving whole skill folders instead of cherry-picking files so historical structure stays intact.
- If an archived copy differs from the active skill, preserve the whole archived folder and note the reason.
